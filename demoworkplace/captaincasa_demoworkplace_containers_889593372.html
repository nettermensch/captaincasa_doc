The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • Containers
  • Scroll Pane - Extended Functions
  • Scroll to top/bottom events

The SCROLLPANE component sends certain scroll events to the server-side:

1. when defining WITHSCROLLEVENTTOP to "true" then the scroll pane sends an event when the user scrolls from/to the top scroll position.

2. when defining WITHSCROLLEVENTBOTTOM to "true" then the scroll pane sends an event when the user scrolls from/to the bottom scroll position.

You may use these events to update e.g. some header or footer area - or to receive some event that the user scrolled to the end, e.g. when presenting legal texts (license terms...).

package workplace;

import java.io.Serializable;

import org.eclnt.editor.annotations.CCGenClass;
import org.eclnt.jsfserver.elements.events.BaseActionEventScroll;
import org.eclnt.workplace.IWorkpageDispatcher;
import org.eclnt.workplace.WorkpageDispatchedPageBean;

@CCGenClass (expressionBase="#{d.DemoScrollpaneScrollListener}")

public class DemoScrollpaneScrollListener
    extends PageBean 
    implements Serializable
{
    // ------------------------------------------------------------------------
    // members
    // ------------------------------------------------------------------------

    boolean m_renderedExtendedHeadline = true;

    // ------------------------------------------------------------------------
    // constructors & initialization
    // ------------------------------------------------------------------------

    public DemoScrollpaneScrollListener(IWorkpageDispatcher workpageDispatcher)
    {
        super(workpageDispatcher);        
    }

    public String getPageName() { return "/workplace/demoscrollpanescrolllistener.jsp"; }
    public String getRootExpressionUsedInPage() { return "#{d.DemoScrollpaneScrollListener}"; }

    // ------------------------------------------------------------------------
    // public usage
    // ------------------------------------------------------------------------

    public void onScrollAction(javax.faces.event.ActionEvent event) 
    {
        if (event instanceof BaseActionEventScroll)
        {
            BaseActionEventScroll e = (BaseActionEventScroll)event;
            if ("top".equals(e.getPosition()))
            {
                if (e.checkIfPositionWasLeft() == true)
                    m_renderedExtendedHeadline = false;
                else
                    m_renderedExtendedHeadline = true;
            }
        }
    }

    public boolean getRenderedExtendedHeadline() { return m_renderedExtendedHeadline; }
    public void setRenderedExtendedHeadline(boolean value) { this.m_renderedExtendedHeadline = value; }

    // ------------------------------------------------------------------------
    // private usage
    // ------------------------------------------------------------------------
}
