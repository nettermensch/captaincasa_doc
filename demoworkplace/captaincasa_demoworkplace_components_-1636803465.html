The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • Components
  • Straight Controls
  • Menus
  • Open popup menu from server side

By using attribute TRIGGERPOPUPMENU you can update the popup menu of the corresponding component from server side processing.

Please check the demo:

(.) the button executed the trigger which is defines with the field
(.) the popup menu opens as if the user clicked with the right mouse button onto the field
(.) the functions are executed just as normal when the user clicks a menu item

In the example the popup menu (id "PM") is defined on ROWBODYPANE level. This means: even though the popup menu is opened on top of the field, the processing is done on ROWBODYPANE level (and as always: on MENUITEM level).

This is the XML layout definition:

<?xml version="1.0" encoding="utf-8"?>
<t:layoutdefinition xmlns:t="http://www.captaincasa.com/controllibrary/t" >
<t:beanprocessing id="g_1" beanbinding="#{d.DemoTriggerPopupMenu}" />
<t:rowdemobodypane id="g_4" actionListener="#{d.DemoTriggerPopupMenu.onPaneAction}" objectbinding="#{d.DemoTriggerPopupMenu}" popupmenu="PM" rowdistance="#{ccstylevalue[&#39;@defaultrowdistance@&#39;]}" >
<t:row id="g_6" >
<t:button id="g_2" actionListener="#{d.DemoTriggerPopupMenu.onButtonAction}" text="Show popup menu" />
</t:row>
<t:rowline id="g_13" />
<t:row id="g_10" >
<t:field id="g_11" actionListener="#{d.DemoTriggerPopupMenu.onButtonAction}" labeltext="Hallo" text="Some text..." triggerpopupmenu="#{d.DemoTriggerPopupMenu.fieldPMTrigger}" width="200" />
</t:row>
</t:rowdemobodypane>
<t:popupmenu id="PM" >
<t:menuitem id="g_8" actionListener="#{d.DemoTriggerPopupMenu.onMenuItemCMD1}" command="CMD1" text="Command 1" />
</t:popupmenu>
</t:layoutdefinition>

This is the Java code:

package workplace;

import java.io.Serializable;

import org.eclnt.editor.annotations.CCGenClass;
import org.eclnt.jsfserver.defaultscreens.OKPopup;
import org.eclnt.jsfserver.elements.events.BaseActionEventPopupMenuItem;
import org.eclnt.jsfserver.elements.util.Trigger;
import org.eclnt.workplace.IWorkpageDispatcher;

@CCGenClass (expressionBase="#{d.DemoTriggerPopupMenu}")

public class DemoTriggerPopupMenu
    extends WorkpageDispatchedPageBean 
    implements Serializable
{
    // ------------------------------------------------------------------------
    // inner classes
    // ------------------------------------------------------------------------

    // ------------------------------------------------------------------------
    // members
    // ------------------------------------------------------------------------

    Trigger m_fieldPMTrigger = new Trigger();

    // ------------------------------------------------------------------------
    // constructors & initialization
    // ------------------------------------------------------------------------

    public DemoTriggerPopupMenu(IWorkpageDispatcher workpageDispatcher)
    {
        super(workpageDispatcher);        
    }

    public String getPageName() { return "/workplace/demotriggerpopupmenu.xml"; }
    public String getRootExpressionUsedInPage() { return "#{d.DemoTriggerPopupMenu}"; }

    // ------------------------------------------------------------------------
    // public usage
    // ------------------------------------------------------------------------

    public Trigger getFieldPMTrigger() { return m_fieldPMTrigger; }
    
    public void onButtonAction(javax.faces.event.ActionEvent event) 
    {
        m_fieldPMTrigger.trigger();
    }

    /*
     * Action listener within ROWBODYPANE.
     */
    public void onPaneAction(javax.faces.event.ActionEvent event) 
    {
        if (event instanceof BaseActionEventPopupMenuItem)
        {
            OKPopup.createInstance("","onPaneAction: Popup menu was invoked!");
        }
    }

    /*
     * Action listener within MENUITEM.
     */
    public void onMenuItemCMD1(javax.faces.event.ActionEvent event) 
    {
        OKPopup.createInstance("","onMenuItemCMD1: Popup menu was invoked!");
    }

    // ------------------------------------------------------------------------
    // private usage
    // ------------------------------------------------------------------------
}
