The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • Components
  • Reporting, PDF, SVG
  • Show PDF

The PDFRENDERER component is able to render a PDF bytestream directly into the user interface. There is no necessity for having some Adobe Reader installed in the frontend - the rendering is done through pure Java.

The PDF format must adhere to documents display-able in Adobe Acrobat Reader 5.0.

...of course: you can also use the BROWSER component and pass a URL ending with ".pdf". But: the PDFRENDERER component gives you a tighter level of intergration. As you see in the server side Java program the PDF is rendered directly into a property string which is references by the component ...that's it.

When using the component as part of your screen definition make sure that the PDFRenderer.jar library is referenced within your applet of webstart definition.

The PDFRENDER component supports clients printing as well. Printing is started by using the TRIGGERPRINT attribute. You can pre-select if the user should see a print dialog and/or you can pre-select the client side printer by defining its name. If on client side no printer definition can be found, then the print dialog is opened automatically.

This is the Java code:

package workplace;

import java.io.Serializable;

import javax.faces.event.ActionEvent;

import org.eclnt.editor.annotations.CCGenClass;
import org.eclnt.jsfserver.elements.events.BaseActionEventUpload;
import org.eclnt.jsfserver.elements.util.Trigger;
import org.eclnt.jsfserver.managedbean.IDispatcher;
import org.eclnt.jsfserver.util.WebResourceReader;
import org.eclnt.util.file.FileManager;
import org.eclnt.util.valuemgmt.ValueManager;

@CCGenClass (expressionBase="#{d.DemoPdfRenderer}")

public class DemoPdfRenderer
    extends PageBean
    implements Serializable
{
    // ------------------------------------------------------------------------
    // inner classes
    // ------------------------------------------------------------------------
    
    // ------------------------------------------------------------------------
    // members
    // ------------------------------------------------------------------------
    
    protected Trigger m_trigger = new Trigger();
    protected String m_pdf;
    
    // ------------------------------------------------------------------------
    // constructors
    // ------------------------------------------------------------------------
    
    public DemoPdfRenderer(IDispatcher dispatcher)
    {
        super(dispatcher);
        initRefComponent("pdfrenderer");
        byte[] initialPdf = WebResourceReader.readFileIntoByteArray("/pdf/pdftest1.pdf");
        m_pdf = ValueManager.encodeHexString(initialPdf);
    }
    
    // ------------------------------------------------------------------------
    // public usage
    // ------------------------------------------------------------------------
    
    public Trigger getTrigger() { return m_trigger; }
    public void setTrigger(Trigger value) { m_trigger = value; }

    public String getPdf() { return m_pdf; }
    public void setPdf(String value) { m_pdf = value; }

    public void onLoadPdfAction(ActionEvent event) 
    {
        if (event instanceof BaseActionEventUpload)
        {
            BaseActionEventUpload baeu = (BaseActionEventUpload)event;
            byte[] bytes = baeu.getHexBytes();
            m_pdf = ValueManager.encodeHexString(bytes);
        }
    }

    public void onTriggerPrint(ActionEvent event) 
    {
        m_trigger.trigger();
    }
    
    public void onClear(ActionEvent event)
    {
        m_pdf = null;
    }

    // ------------------------------------------------------------------------
    // private usage
    // ------------------------------------------------------------------------

}
