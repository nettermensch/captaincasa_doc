The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • Components
  • Straight Controls
  • Drag & Drop
  • Fine control drag&drop by regular expressions

Check the demo: all the involved drag/drop components use the drag/drop type "DEMO", i.e. but there is a fine-granular, regular expression based selection, where a dragged information can be dropped.

In the DROPRECEIVE definition of a component you can add a regular expression as 3rd argument, each one separated by a colon (":"):

type:zone:regex(...)

The TYPE is the basic guidance if a drag operation (i.e. a DRAGSEND value) is applicable for the component - or not. If the TYPE does match, then the component will be open to accept a drop of the information.

The ZONE is "just" an indicator about the highlighting of the component during drag-over, there is no logic behind.

The REGEX now is the 3rd parameter - containing a regular expression: if the TYPE matches then the value part of the DRAGSEND definition (type:value) is in addition checked against this regular expression - and the component will only accept the drop if the values matches the regular expression.

This is the Java code:

package workplace;

import java.io.Serializable;

import org.eclnt.editor.annotations.CCGenClass;
import org.eclnt.jsfserver.defaultscreens.Statusbar;
import org.eclnt.jsfserver.elements.events.BaseActionEventDrop;
import org.eclnt.workplace.IWorkpageDispatcher;

@CCGenClass (expressionBase="#{d.DemoDragWithregex}")

public class DemoDragWithregex
    extends PageBean 
    implements Serializable
{
    public DemoDragWithregex(IWorkpageDispatcher workpageDispatcher)
    {
        super(workpageDispatcher);        
    }

    public String getPageName() { return "/workplace/demodragwithregex.xml"; }
    public String getRootExpressionUsedInPage() { return "#{d.DemoDragWithregex}"; }

    public void onDropAction(javax.faces.event.ActionEvent event) 
    {
        if (event instanceof BaseActionEventDrop)
        {
            BaseActionEventDrop e = (BaseActionEventDrop)event;
            Statusbar.outputSuccess("Drop was processed: " + e.getDragInfo());
        }
    }
}
