The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • General
  • Arranging Content
  • Popup Dialogs
  • Pass Values into Popup

This example demonstrates how data is passed between the calling page and the popped up page.

The calling page creates the popped up page and calls "prepare" method, passing all the information that the popped up page requires to exist.

Especially take a look onto the management of closing the popup via the explicit "close" button: the button invokes an action listener within the page bean of the popup, the action listener calls back via an interface the page bean that opened the popup.

This is the Java code:

package workplace;

import java.io.Serializable;

import javax.faces.event.ActionEvent;

import org.eclnt.editor.annotations.CCGenClass;
import org.eclnt.jsfserver.defaultscreens.ModalPopup;
import org.eclnt.workplace.IWorkpageDispatcher;

@CCGenClass (expressionBase="#{d.DemoPassValueCaller}")

public class DemoPassValueCaller
    extends PageBean
    implements Serializable
{
    // ------------------------------------------------------------------------
    // inner classes
    // ------------------------------------------------------------------------
    
    // ------------------------------------------------------------------------
    // members
    // ------------------------------------------------------------------------
    
    protected String m_lastName;
    protected String m_firstName;
    
    // ------------------------------------------------------------------------
    // constructors
    // ------------------------------------------------------------------------
    
    public DemoPassValueCaller(IWorkpageDispatcher dispatcher)
    {
        super(dispatcher);
    }
    
    // ------------------------------------------------------------------------
    // public usage
    // ------------------------------------------------------------------------
    
    public String getLastName() { return m_lastName; }
    public void setLastName(String value) { m_lastName = value; }

    public String getFirstName() { return m_firstName; }
    public void setFirstName(String value) { m_firstName = value; }
    
    public void onOpenPopup(ActionEvent event) 
    {
        final DemoPassValuePopup bean = new DemoPassValuePopup(getOwningDispatcher());
        openModalPopup(bean,"Demo Popup",300,300,new ModalPopup.IModalPopupListener()
        {
            public void reactOnPopupClosedByUser()
            {
                closePopup(bean);
            }
        });
        bean.prepare(m_lastName,
                     m_firstName,
                     new DemoPassValuePopup.ICloseListener()
                     {
                         public void reactOnClose()
                         {
                             closePopup(bean);
                         }
                     });
    }
    
    // ------------------------------------------------------------------------
    // private usage
    // ------------------------------------------------------------------------

}
