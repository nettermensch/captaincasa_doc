The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • Components
  • Straight Controls
  • Images
  • Image (double) click

Many components (such as the IMAGE component) provide a property INVOKEEVENT. When setting to "click" or "doubleclick" then the component will react correspondingly and will send invoke-events to the server side processing.

On server side you can evaluate the exact click-position. Please pay attention: images and texts may be scaled on client side. When evaluating the click position, then always take the width and height information as reference, that is available via the event's API as well.

This is the Java code:

package workplace;

import java.io.Serializable;

import javax.faces.event.ActionEvent;

import org.eclnt.editor.annotations.CCGenClass;
import org.eclnt.jsfserver.elements.events.BaseActionEventInvoke;
import org.eclnt.jsfserver.util.ClientTestLog;
import org.eclnt.jsfserver.util.IClientLogAware;
import org.eclnt.workplace.IWorkpageDispatcher;

@CCGenClass (expressionBase="#{d.DemoImageInvoke}")

public class DemoImageInvoke
    extends PageBean
    implements Serializable
{
    String m_result;
    String m_result2;
    
    public DemoImageInvoke(IWorkpageDispatcher dispatcher)
    {
        super(dispatcher);
        initRefComponent("image");
    }

    public String getResult() { return m_result; }
    public String getResult2() { return m_result2; }
    
    public void onInvoke(ActionEvent event) 
    {
        if (event instanceof BaseActionEventInvoke)
        {
            ClientTestLog.addClientTestLogMessage("click executed");
            BaseActionEventInvoke baei = (BaseActionEventInvoke)event;
            m_result = "Click executed, coordinates: " + baei.getX() + ", " + 
                                                         baei.getY() + ", " + 
                                                         baei.getWidth() + ", " + 
                                                         baei.getHeight() + ", " +
                                                         baei.getClickCount();
        }
    }

    public void onInvoke2(ActionEvent event) 
    {
        if (event instanceof BaseActionEventInvoke)
        {
            ClientTestLog.addClientTestLogMessage("double click executed");
            BaseActionEventInvoke baei = (BaseActionEventInvoke)event;
            m_result2 = "Double click executed, coordinates: " + baei.getX() + ", " + 
                                                                 baei.getY() + ", " + 
                                                                 baei.getWidth() + ", " + 
                                                                 baei.getHeight() + ", " +
                                                                 baei.getClickCount();
        }
    }

}
