The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • General
  • Client Integration
  • Dynamic browser content

The TempFileManager is a server side utility to store files within some temporary directory that is associated with a session.

Due to a specific servlet the content of this temporary directory is access-able by URL. When saving a file in the TempFileManager then a URL is returned that can be passed back to the browser.

The content of the directory is removed once a session is destroyed.

Thie TempFileManager is a much easier to use variant to store dynamically created data on server side that is available by URL than e.g. using BufferedContentMgr. ...but: it is only useful to use in scenarios in which the URL that is provided is actually used!

This is the Java code:

package workplace;

import java.io.Serializable;
import org.eclnt.editor.annotations.CCGenClass;
import org.eclnt.jsfserver.defaultscreens.Statusbar;
import org.eclnt.jsfserver.util.HttpSessionAccess;
import org.eclnt.jsfserver.util.tempfile.TempFileManager;
import org.eclnt.util.valuemgmt.UniqueIdCreator;
import org.eclnt.workplace.IWorkpageDispatcher;
import org.eclnt.workplace.WorkpageDispatchedPageBean;

import javax.faces.event.ActionEvent;

@CCGenClass (expressionBase="#{d.DemoTempFileAccess}")

public class DemoTempFileAccess
    extends PageBean 
    implements Serializable
{
    // ------------------------------------------------------------------------
    // members
    // ------------------------------------------------------------------------

    String m_url = null;
    String m_html = "<html>\n<body>\n<h1>Some headline</h1>\n<p>\nSome content.\n</p>\n</body>\n</html>";
    
    // ------------------------------------------------------------------------
    // constructors & initialization
    // ------------------------------------------------------------------------

    public DemoTempFileAccess(IWorkpageDispatcher workpageDispatcher)
    {
        super(workpageDispatcher);        
    }

    public String getPageName() { return "/workplace/demotempfileaccess.jsp"; }
    public String getRootExpressionUsedInPage() { return "#{d.DemoTempFileAccess}"; }

    // ------------------------------------------------------------------------
    // public usage
    // ------------------------------------------------------------------------

    public String getUrl() { return m_url; }

    public String getHtml() { return m_html; }
    public void setHtml(String value) { this.m_html = value; }

    public void onCreateFileAction(javax.faces.event.ActionEvent event) 
    {
        try
        {
            byte[] bytes = m_html.getBytes("UTF-8");
            String path = "demo.html";
            m_url = TempFileManager.saveTempFile(path,bytes);
        }
        catch (Throwable t)
        {
            Statusbar.outputError(t.getMessage());
        }
    }

    // ------------------------------------------------------------------------
    // private usage
    // ------------------------------------------------------------------------
}
