The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • Containers
  • Pile
  • Pile component

The PILE component is used for stacking components vertically.

You could do the same with a PANE component, arranging each content within a ROW component - but the PILE does automatically arrange it this way.

Pleas note: by default each contained component gets assigned a width of 100% so that it fills the pile. You of course can assign an individual width to each contained component.

The pile is e.g. useful for traditional LABEL-FIELD combinations, in which the LABEL is not integrated into the FIELD component.
