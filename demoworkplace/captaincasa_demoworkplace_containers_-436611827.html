The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • Containers
  • Rolling Container Area

Scroll the content area to the bottom - and observe what happens in the title area:

The title of the top-most area stays on top, so that the user always sees the content of the area within its context.

A ROLLINGCONTAINERAREA contains several ROWCONTAINER components. Each of the ROWCONTAINER components has the following attributes:

. TEXT - a title
. ICONSLEFT - a list of icons on the left of the title
. ICONSRIGHT - a list of icons on the right of the title

The list of icons is following the scheme "<id>:<image>;<id>:<image>;...".

A ROLLINGCONTAINER has an action listener through which the event is passed, if user presses one of the icons.

This is the Java code:

package workplace;

import java.io.Serializable;
import org.eclnt.editor.annotations.CCGenClass;
import org.eclnt.jsfserver.defaultscreens.Statusbar;
import org.eclnt.jsfserver.elements.events.BaseActionEventRollingContainerIconInvoke;
import org.eclnt.workplace.IWorkpageDispatcher;
import org.eclnt.workplace.WorkpageDispatchedPageBean;

import javax.faces.event.ActionEvent;

@CCGenClass (expressionBase="#{d.DemoRollingContainerArea}")

public class DemoRollingContainerArea
    extends PageBean 
    implements Serializable
{
    public DemoRollingContainerArea(IWorkpageDispatcher workpageDispatcher)
    {
        super(workpageDispatcher);        
    }

    public String getPageName() { return "/workplace/demorollingcontainerarea.jsp"; }
    public String getRootExpressionUsedInPage() { return "#{d.DemoRollingContainerArea}"; }

    public void onContainer1Action(javax.faces.event.ActionEvent event) 
    {
        if (event instanceof BaseActionEventRollingContainerIconInvoke)
        {
            BaseActionEventRollingContainerIconInvoke e = (BaseActionEventRollingContainerIconInvoke)event;
            Statusbar.outputSuccessWithPopup("Icon was pressed in Container 1, id = " + e.getIconId() +".");
        }
    }

    public void onContainer2Action(javax.faces.event.ActionEvent event) 
    {
        if (event instanceof BaseActionEventRollingContainerIconInvoke)
        {
            BaseActionEventRollingContainerIconInvoke e = (BaseActionEventRollingContainerIconInvoke)event;
            Statusbar.outputSuccessWithPopup("Icon was pressed in Container 2, id = " + e.getIconId() +".");
        }
    }
}
