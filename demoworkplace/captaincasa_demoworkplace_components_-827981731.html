The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • Components
  • Straight Controls
  • Field Control
  • Formatted Field
  • Formatting by data type

By using the component FORMATTEDFIELD you can assign a data type that is checked as part of the client processing. The following data types are currently supported by passing the information via the attribute FORMAT:

int
long
float
double
date
time
datetime
bigdecimal

By using the attribute FORMATMASK you can pass certain additional type information in order to influence the rendering of values. In the case of using format date / time / datetime you can e.g. pass information if you want to output date values in short / medium / long format.

This is the Java code:

package workplace;

import java.math.BigDecimal;
import java.util.Date;

import org.eclnt.workplace.IWorkpageDispatcher;

public class DemoFormattedfield extends PageBean
{
    int m_intValue = 100;
    float m_floatValue = 99.99f;
    BigDecimal m_bidDecimalValue = new BigDecimal("99.99");
    Date m_dateValue = new Date();
    Date m_timeValue = new Date();
    
    public DemoFormattedfield(IWorkpageDispatcher dispatcher)
    {
        super(dispatcher);
        initRefComponent("formattedfield"); 
    }
    
    public int getIntValue()
    {
        return m_intValue;
    }
    public void setIntValue(int intValue)
    {
        m_intValue = intValue;
    }
    public float getFloatValue()
    {
        return m_floatValue;
    }
    public void setFloatValue(float floatValue)
    {
        m_floatValue = floatValue;
    }
    public BigDecimal getBidDecimalValue()
    {
        return m_bidDecimalValue;
    }
    public void setBidDecimalValue(BigDecimal bidDecimalValue)
    {
        m_bidDecimalValue = bidDecimalValue;
    }
    public Date getDateValue()
    {
        return m_dateValue;
    }
    public void setDateValue(Date dateValue)
    {
        m_dateValue = dateValue;
    }
    public Date getTimeValue()
    {
        return m_timeValue;
    }
    public void setTimeValue(Date timeValue)
    {
        m_timeValue = timeValue;
    }
    
}
