The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • Grids & Trees
  • Grids and Trees
  • Column Sequence and Widths
  • Only show full columns

Change the size horizontal size of this grid by e.g. changing the browser window size or by changing the size of the workpage container.

You will see: the grid will always only show full columns. If there is no sufficient screen space then some columns will be (optically) removed. This is due to defining attributr FIXGRID-ONLYSHOWFULLCOLUMNS to "true".

On GRIDCOL level you define the priority of columns using attribute GRIDCOL-ONLYSHOWFULLCOLUMNSPRIO. In the demo you see that the priorities are not reflecting the lef-to-right order of the columns: the lowest priority is assigned to the "zipCode" column, then the "street" column, etc.

Typical use case of this feature: if using a grid in mobile scenarios then you do not wnat the user to scroll horizontally - but better come up with a selection of appropriate columns.

This is the Java code:

package workplace;

import java.io.Serializable;

import org.eclnt.editor.annotations.CCGenClass;
import org.eclnt.jsfserver.elements.impl.FIXGRIDItem;
import org.eclnt.jsfserver.elements.impl.FIXGRIDListBinding;
import org.eclnt.workplace.IWorkpageDispatcher;

import workplace.logic.RandomLogic;

@CCGenClass (expressionBase="#{d.DemoGridFullColumns}")

public class DemoGridFullColumns
    extends WorkpageDispatchedPageBean 
    implements Serializable
{
    // ------------------------------------------------------------------------
    // inner classes
    // ------------------------------------------------------------------------

    public class GridItem extends FIXGRIDItem implements java.io.Serializable
    {
        String i_town;
        String i_zipCode;
        String i_street;
        String i_lastName;
        String i_firstName;
        public String getTown() { return i_town; }
        public void setTown(String value) { this.i_town = value; }
        public String getZipCode() { return i_zipCode; }
        public void setZipCode(String value) { this.i_zipCode = value; }
        public String getStreet() { return i_street; }
        public void setStreet(String value) { this.i_street = value; }
        public String getLastName() { return i_lastName; }
        public void setLastName(String value) { this.i_lastName = value; }
        public String getFirstName() { return i_firstName; }
        public void setFirstName(String value) { this.i_firstName = value; }
        public void onRowSelect()
        {
        }
        public void onRowExecute()
        {
        }
    }

    // ------------------------------------------------------------------------
    // members
    // ------------------------------------------------------------------------

    FIXGRIDListBinding<GridItem> m_grid = new FIXGRIDListBinding<GridItem>();

    // ------------------------------------------------------------------------
    // constructors & initialization
    // ------------------------------------------------------------------------

    public DemoGridFullColumns(IWorkpageDispatcher workpageDispatcher)
    {
        super(workpageDispatcher);        
        for (int i=0; i<100; i++)
        {
            GridItem gi = new GridItem();
            gi.setFirstName(RandomLogic.getFirstName());
            gi.setLastName(RandomLogic.getLastName());
            gi.setStreet("Street " + i);
            gi.setZipCode(""+(10000+i));
            gi.setTown("Town " + i);
            m_grid.getItems().add(gi);
        }
    }

    public String getPageName() { return "/workplace/demogridfullcolumns.xml"; }
    public String getRootExpressionUsedInPage() { return "#{d.DemoGridFullColumns}"; }

    // ------------------------------------------------------------------------
    // public usage
    // ------------------------------------------------------------------------

    public FIXGRIDListBinding<GridItem> getGrid() { return m_grid; }

    // ------------------------------------------------------------------------
    // private usage
    // ------------------------------------------------------------------------
}
