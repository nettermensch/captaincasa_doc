The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • Components
  • Straight Controls
  • Field Control
  • Field Background

The field controls (FIELD, COMBOFIELD, FORMATTEDFIELD) allow to define a BGPAINT attribute - either statically or dynamically. This is very useful for painting information into the field that guides the user.

In the example screen some variants are shown for rendering a field as mandatory field and as field containing an error.

Have a look onto the definition of the BGPAINT attributes: there are two special painting methods (in addition to the "normal" ones like "rectangle", "image", etc.) which are:

(1) error()
(2) mandatory()

"error()" marks the control as control holding an error value. "mandatory()" shows the user that he/she needs to input the field. The rendering behind "mandatory()" is only done, if the corresponding element does not contain a value. When the user inputs a value then the painting for "mandatory()" is no longer done.

This is the Java code:

package workplace;

import javax.faces.event.ActionEvent;

import org.eclnt.jsfserver.defaultscreens.ISetId;
import org.eclnt.jsfserver.defaultscreens.IdTextSelection;
import org.eclnt.workplace.IWorkpageDispatcher;


public class DemoFieldBgpaint extends PageBean
{
    static String BGPAINT_ERROR = "error()";
    
    String m_value;
    String m_valueBgpaint;
    
    public DemoFieldBgpaint(IWorkpageDispatcher dispatcher)
    {
        super(dispatcher);
        initRefComponent("field"); 
        m_valueBgpaint = BGPAINT_ERROR;
    }
    
    public void setValue(String value) { m_value = value; }
    public String getValue() { return m_value; }
    
    public void setValueBgpaint(String value) { m_valueBgpaint = value; }
    public String getValueBgpaint() { return m_valueBgpaint; }
    
    public void onCheckInput(ActionEvent ae)
    {
        if (m_value != null && m_value.equals("OK"))
            m_valueBgpaint = null;
        else
            m_valueBgpaint = BGPAINT_ERROR;
    }
    
    public void onComboFieldAction(ActionEvent ae)
    {
        IdTextSelection idts = IdTextSelection.createInstance();
        idts.addLine("Option1","Explanation for Option1");
        idts.addLine("Option2","Explanation for Option2");
        idts.addLine("Option3","Explanation for Option3");
        ISetId callBack = new ISetId() 
        {
            public void setId(String id)
            {
                m_value = id;
            }
        };
        idts.setCallBack(callBack);
    }
    
}
