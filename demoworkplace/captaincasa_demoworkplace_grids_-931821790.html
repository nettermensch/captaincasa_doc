The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • Grids & Trees
  • Grids and Trees
  • Load on Demand (2)

This example shows how to scroll through a list with potencially infinite length (it is limited to 250 in this example).

The total grid increases its size automatically always checking that there is some "down-scroll" possibility until all items are loaded. On client side the effect of course is, that the scroll bar gets smaller and smaller because the total number of grid items is increased.

This is the XML layout definition:

<?xml version="1.0" encoding="utf-8"?>
<t:layoutdefinition xmlns:t="http://www.CaptainCasa.com/controllibrary/t" xmlns:tpbc="http://www.CaptainCasa.com/controllibrary/tpbc" >
<t:rowdemobodypane id="g_1" objectbinding="#{d.demoGridloadondemand2}" >
<t:row id="g_2" >
<t:fixgrid id="g_3" avoidroundtrips="true" bordercolor="#{ccstylevalue.ccLightBorderColor}" borderheight="1" borderwidth="1" objectbinding="#{d.demoGridloadondemand2.grid}" sbvisibleamount="20" width="100%" >
<t:gridcol id="g_4" text="First Name" width="50%" >
<t:field id="g_5" text=".{firstName}" />
</t:gridcol>
<t:gridcol id="g_6" text="Last Name" width="50%" >
<t:field id="g_7" text=".{lastName}" />
</t:gridcol>
</t:fixgrid>
</t:row>
</t:rowdemobodypane>
<t:pageaddons id="g_pa"/>
</t:layoutdefinition>

This is the Java code:

package workplace;

import org.eclnt.jsfserver.elements.impl.FIXGRIDItem;
import org.eclnt.jsfserver.elements.impl.FIXGRIDListBinding;
import org.eclnt.workplace.IWorkpageDispatcher;

public class DemoGridloadondemand2 extends WorkpageDispatchedPageBean
{
    // ------------------------------------------------------------------------
    // inner classes
    // ------------------------------------------------------------------------
    
    public class MyRow extends FIXGRIDItem
    {
        String m_firstName;
        String m_lastName;
        public void setLastName(String value) { m_lastName = value; }
        public String getLastName() { return m_lastName; }
        public void setFirstName(String value) { m_firstName = value; }
        public String getFirstName() { return m_firstName; }
    }
    
    public class MyGrid extends FIXGRIDListBinding<MyRow>
    {
        @Override
        public void renderingBegins()
        {
            super.renderingBegins();
            // ensure that always sbvalue+2*sbvisibleamount items are loaded
            int loadToIndex = getSbvalue() + 2 * getSbvisibleamount();
            if (loadToIndex > 250)
                loadToIndex = 250;
            for (int index=getItems().size(); index <= loadToIndex; index++)
            {
                getItems().add(readItem(index));
            }
        }
    }
    
    // ------------------------------------------------------------------------
    // members
    // ------------------------------------------------------------------------
    
    MyGrid m_grid = new MyGrid();
    
    // ------------------------------------------------------------------------
    // constructors
    // ------------------------------------------------------------------------
    
    public DemoGridloadondemand2(IWorkpageDispatcher dispatcher)
    {
        super(dispatcher);
    }
    
    // ------------------------------------------------------------------------
    // public usage
    // ------------------------------------------------------------------------
    
    public FIXGRIDListBinding<MyRow> getGrid() { return m_grid; }
    
    // ------------------------------------------------------------------------
    // private usage
    // ------------------------------------------------------------------------
    
    private void initializeGridWithNullItems()
    {
    }
    
    private MyRow readItem(int index)
    {
        MyRow row = new MyRow();
        row.m_firstName = "First Name " + index;
        row.m_lastName = "Last Name " + index;
        return row;
    }

}
