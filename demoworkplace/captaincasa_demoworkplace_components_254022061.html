The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • Components
  • Animation & Graphics & Sound
  • Paint Area
  • Line drawing
  • Intersections

By using attribute PAINTAREA-LINEINTERSECTIONMODE you can control how lines are drawn if they cross one another.

By default lines are drawn without checking intersections.

By assigning value "gap", the crossing line is interrupted, so that the user can better recognize the direction of lines. By assigning "curve" small "bridges" will be drawn.

In some scenarios it is useful to not to consider individual lines within the intersection management. You can do so by defining PAINTAREALINEITEM-IGNOREINTERSECTIONS to "true".

Please pay attention: calculating the intersection points and drawing the intersection gaps takes performance on client side! Check the performance prior to using it in scenarios with e.g. hundreds of lines!
