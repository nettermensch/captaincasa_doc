The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • Containers
  • Foldable / Titled Pane
  • Vertical

The component FOLABLEVERTICALPANE is a titled area that can be collapsed by the user by either pressing the text link or the icon on the left.

This is the Java code:

package workplace;

import javax.faces.event.ActionEvent;

import org.eclnt.workplace.IWorkpageDispatcher;

public class DemoFoldableVertical extends PageBean
{
    boolean m_leftIsOpened = true;
    boolean m_rightIsOpened = false;

    public DemoFoldableVertical(IWorkpageDispatcher dispatcher)
    {
        super(dispatcher);
    }

    public boolean getLeftIsOpened() { return m_leftIsOpened; }
    public void setLeftIsOpened(boolean leftIsOpened) { m_leftIsOpened = leftIsOpened; }

    public boolean getRightIsOpened() { return m_rightIsOpened; }
    public void setRightIsOpened(boolean rightIsOpened) { m_rightIsOpened = rightIsOpened; }

    public void onFoldableLeftAction(ActionEvent event)
    {
        m_rightIsOpened = !m_leftIsOpened;
    }

    public void onFoldableRightAction(ActionEvent event)
    {
        m_leftIsOpened = !m_rightIsOpened;
    }

    public void onButtonHideLeftAction(ActionEvent event)
    {
        m_leftIsOpened = false;
        m_rightIsOpened = true;
    }

    public void onButtonHideRightAction(ActionEvent event)
    {
        m_leftIsOpened = true;
        m_rightIsOpened = false;
    }
}
