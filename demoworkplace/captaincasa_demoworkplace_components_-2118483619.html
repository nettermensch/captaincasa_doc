The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • Components
  • Straight Controls
  • Field Control
  • Formatted Field
  • Adjust values during input

While performing the input of figures, the first field will automatically arrange the separators according to the FORMAT and the FORMATMASK definition. In addtion the user is not able to define more decimal digits than defined by the FORMATMASK definition.

The second field will format the user's input after leaving the field. (Default behavior.)

Use attribute FORMATTEDFIELD-WITHUPDATEDURINGINPUT to switch on the automatic arrangement of separators.

Please pay attention: this attribute currently is only supported with number formats. Which area: FORMATTEDFIELD-FORMAT "bigdecimal", "double", "float", "biginteger", "long", "int", "short".
