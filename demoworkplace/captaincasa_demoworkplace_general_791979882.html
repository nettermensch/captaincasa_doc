The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • General
  • Workplace Management
  • Drop Info onto Node

This example shows that you can drop information from any area, e.g. the content area, into the workplace function tree. Drop the icons of this page onto the node "Drop Info onto Node" on the left - this is the node you started this page from. A corresponding message will show up and a new node will be added within the function node tree.

In order to process the drop event you neeed to override the "FunctionNode" class - you find the code within the program "workplace/WPFunctionTreeGeneral.java".
