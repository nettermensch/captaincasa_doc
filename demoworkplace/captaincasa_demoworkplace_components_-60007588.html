The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • Components
  • Touch Screen
  • Numeric field

The component TOUCHFIELDNUMERIC allows to input a number by touch keyboard.

This is the XML layout definition:

<?xml version="1.0" encoding="utf-8"?>
<t:layoutdefinition xmlns:t="http://www.CaptainCasa.com/controllibrary/t" xmlns:tpbc="http://www.CaptainCasa.com/controllibrary/tpbc" >
<t:rowdemobodypane id="g_1" objectbinding="#{d.DemoTouch}" >
<t:row id="g_2" >
<t:label id="g_3" text="Value via Touch Field" width="150" />
<t:touchfieldnumeric id="g_4" flush="true" height="200" value="#{d.DemoTouch.numericValue}" width="200" />
</t:row>
<t:rowdistance id="g_5" />
<t:row id="g_6" >
<t:label id="g_7" text="Value via Text Field" width="150" />
<t:field id="g_8" flush="true" rowalignmenty="top" text="#{d.DemoTouch.numericValue}" width="100%" />
<t:coldistance id="g_9" />
</t:row>
<t:rowdistance id="g_10" />
</t:rowdemobodypane>
<t:pageaddons id="g_pa"/>
</t:layoutdefinition>

This is the Java code:

package workplace;

import java.math.BigDecimal;

import org.eclnt.workplace.IWorkpageDispatcher;

public class DemoTouch extends PageBean
{
    BigDecimal m_numericValue = new BigDecimal("100");
    
    public DemoTouch(IWorkpageDispatcher dispatcher)
    {
        super(dispatcher);
        initRefComponent("touchfieldnumeric");
    }
    
    public void setNumericValue(BigDecimal value) { m_numericValue = value; }
    public BigDecimal getNumericValue() { return m_numericValue; }
}
