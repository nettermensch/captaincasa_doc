The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • General
  • Arranging Content
  • Popup Dialogs
  • Closing popup by moving mouse

Open the popup - you will see: the popup will close when moving the mouse.

Use this mode in order to define popup dialogs e.g. containing some message to the user, which should not interrupt the user too much - but which need some minimum level of user acknowledgement.

Please note: the internal listening to mouse-move-events starts after a short duration of time (1000 ms). This duration ensures that the user actually is able to register the message in scenarios in which he/she is working with the mouse. Without this the user would sometimes only see some short flickering of the dialog, without any change to check the content.

This is the Java code:

package workplace;

import java.io.Serializable;

import org.eclnt.editor.annotations.CCGenClass;
import org.eclnt.jsfserver.defaultscreens.ModelessPopup;
import org.eclnt.jsfserver.defaultscreens.OKPopup;
import org.eclnt.jsfserver.elements.util.DefaultModelessPopupListener;
import org.eclnt.workplace.IWorkpageDispatcher;
import org.eclnt.workplace.WorkpageDispatchedPageBean;

import javax.faces.event.ActionEvent;

@CCGenClass (expressionBase="#{d.DemoPopupCloseOnMouseActivity}")

public class DemoPopupCloseOnMouseActivity
    extends PageBean 
    implements Serializable
{
    public DemoPopupCloseOnMouseActivity(IWorkpageDispatcher workpageDispatcher)
    {
        super(workpageDispatcher);        
    }

    public String getPageName() { return "/workplace/demopopupcloseonmouseactivity.jsp"; }
    public String getRootExpressionUsedInPage() { return "#{d.DemoPopupCloseOnMouseActivity}"; }

    public void onOpenPopupAction(javax.faces.event.ActionEvent event) 
    {
        OKPopup okp = OKPopup.createInstance("Message Popup","This popup will be closed when moving the mouse some pixels. - Use this type of closing for \"ultra-lightweight\" dialogs.");
        okp.getModalPopup().setCloseonmouseactivity(true);
        okp.getModalPopup().setUndecorated(true);
    }

    public void onOpenOwnPopupAction(javax.faces.event.ActionEvent event) 
    {
        final DemoHelloWorld ui= new DemoHelloWorld(getOwningDispatcher());
        ModelessPopup p = openModelessPopup(ui,"Hello world!" ,800,480,new DefaultModelessPopupListener(this,ui));
        p.setCloseonmouseactivity(true);
        p.setCloseonclickoutside(true);
    }

    public void onOpenPopupExitAction(javax.faces.event.ActionEvent event) 
    {
        OKPopup okp = OKPopup.createInstance("Message Popup","This popup will be closed when moving the mouse out of the area of the popup dialog.");
        okp.getModalPopup().setCloseonmouseexit(true);
//        okp.getModalPopup().setUndecorated(true);
    }

    public void onOpenOwnPopupExitAction(javax.faces.event.ActionEvent event) 
    {
        final DemoHelloWorld ui= new DemoHelloWorld(getOwningDispatcher());
        ModelessPopup p = openModelessPopup(ui,"Hello world!" ,800,480,new DefaultModelessPopupListener(this,ui));
        p.setCloseonmouseexit(true);
        p.setCloseonclickoutside(true);
    }

}
