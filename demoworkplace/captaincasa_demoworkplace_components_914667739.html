The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • Components
  • Animation & Graphics & Sound
  • Sound

Use the SOUND component for adding sound features to your page. The SOUND component on the one hand references a sound file. On the other hand it provides a trigger which needs to be activated when the sound should be played.

There is an implicitly created SOUND component that is automatically available with each client. Use API "Client.instance().playSound(...)" to directly call it.

This is the Java code:

package workplace;

import java.io.Serializable;

import javax.faces.event.ActionEvent;

import org.eclnt.editor.annotations.CCGenClass;
import org.eclnt.jsfserver.defaultscreens.Client;
import org.eclnt.jsfserver.elements.util.Trigger;
import org.eclnt.jsfserver.managedbean.IDispatcher;

@CCGenClass (expressionBase="#{d.DemoSound}")

public class DemoSound 
    extends PageBean
    implements Serializable
{
    // ------------------------------------------------------------------------
    // members
    // ------------------------------------------------------------------------
    
    protected String m_sound = "/sounds/beep.wav";
    protected Trigger m_trigger = new Trigger();
    
    // ------------------------------------------------------------------------
    // constructors
    // ------------------------------------------------------------------------

    public DemoSound(IDispatcher dispatcher)
    {
        super(dispatcher);
        initRefComponent("sound");
    }
    
    // ------------------------------------------------------------------------
    // public usage
    // ------------------------------------------------------------------------
    
    public void onPlayBeep(javax.faces.event.ActionEvent event) 
    {
        Client.instance().playSound("/sounds/beep.wav");
    }
    
    public Trigger getTrigger() { return m_trigger; }
    public void setTrigger(Trigger value) { m_trigger = value; }

    public String getSound() { return m_sound; }
    public void setSound(String value) { m_sound = value; }
    
    public void onPlaySound(ActionEvent event) 
    {
        m_trigger.trigger();
    }
}
