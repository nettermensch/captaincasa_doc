The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • Containers
  • Adaptive Containers - Controls
  • Adaptive toolbar

Resize the screen horizontally - either the toolbar content is shown as one line of components - or the toolbar content is condensed into one icon and shown when the user presses this icon.

The adaptive toolbar is a container holding components - typically buttons, icons, search fields etc. If there is enough screen space then the content is rendered in one row, one after the other. If the screen space is not sufficient then only one icon is drawn. If the user presses the icon then a dialog shows up in which the controls are shown one below the other.

If opened as popup dialog, the componentes by default will show up with their original size. By setting attribute EXPANDCOMPONENTSINPOPUP to "true" the component's will be automatically rendered with a width of 100% so that they all have the same width.

The default styling of the toolbar (style variant "default") comes with some own rendering of padding and borders. For embedding the toolbar into other components (e.g. into the ROWHEADER) use style "ccembeddable" which comes without padding, border and background coloring.

This is the Java code:

package workplace;

import java.io.Serializable;

import javax.faces.event.ActionEvent;

import org.eclnt.editor.annotations.CCGenClass;
import org.eclnt.jsfserver.defaultscreens.Statusbar;
import org.eclnt.jsfserver.elements.events.BaseActionEventRearrange;
import org.eclnt.workplace.IWorkpageDispatcher;

@CCGenClass (expressionBase="#{d.DemoAdaptiveToolbar}")

public class DemoAdaptiveToolbar
    extends PageBean 
    implements Serializable
{
    // ------------------------------------------------------------------------
    // inner classes
    // ------------------------------------------------------------------------

    // ------------------------------------------------------------------------
    // members
    // ------------------------------------------------------------------------

    String m_alignment = "left";
    boolean m_expandComponentsInPopup = false;
    
    // ------------------------------------------------------------------------
    // constructors & initialization
    // ------------------------------------------------------------------------

    public DemoAdaptiveToolbar(IWorkpageDispatcher workpageDispatcher)
    {
        super(workpageDispatcher);        
    }

    public String getPageName() { return "/workplace/demoadaptivetoolbar.jsp"; }
    public String getRootExpressionUsedInPage() { return "#{d.DemoAdaptiveToolbar}"; }

    // ------------------------------------------------------------------------
    // public usage
    // ------------------------------------------------------------------------

    public String getAlignment() { return m_alignment; }
    public void setAlignment(String value) { this.m_alignment = value; }

    public void onAction1(ActionEvent event)
    {
        Statusbar.outputSuccess("Button1 was pressed.");
    }
    
    public void onAction2(ActionEvent event)
    {
        Statusbar.outputSuccess("Button2 was pressed.");
    }
    
    public void onAction3(ActionEvent event)
    {
        Statusbar.outputSuccess("Button3 was pressed.");
    }
    
    public void onRearrangeAction(javax.faces.event.ActionEvent event) 
    {
        if (event instanceof BaseActionEventRearrange)
        {
            BaseActionEventRearrange e = (BaseActionEventRearrange)event;
            Statusbar.outputSuccess("Content of toolbar was rearranged. Current mode is:" + e.getRearrangeMode());
        }
    }

    public boolean getExpandComponentsInPopup() { return m_expandComponentsInPopup; }
    public void setExpandComponentsInPopup(boolean value) { this.m_expandComponentsInPopup = value; }

    // ------------------------------------------------------------------------
    // private usage
    // ------------------------------------------------------------------------
}
