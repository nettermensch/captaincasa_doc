The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • Components
  • Animation & Graphics & Sound
  • Draw Area

The DRAWAREA is a component that allows to draw certain 2d-commands. You can draw:

. rectangles
. ovals
. lines
. text
. images
. raster grids
. polygones

Draw commands can be concatenated with ";". Pay attention: spaces are meaningful!

x,y,width,height values can be either defined as

. direct pixel values ("20")
. percentage values ("10")
. percentage value minus pixel value ("100%-10")

Internally the DRAWAREA component takes usage of the SVG drawing.

This is the Java code:

package workplace;

import java.io.Serializable;

import org.eclnt.editor.annotations.CCGenClass;
import org.eclnt.jsfserver.util.WebResourceReader;
import org.eclnt.util.valuemgmt.ValueManager;
import org.eclnt.workplace.IWorkpageDispatcher;

@CCGenClass (expressionBase="#{d.DemoDrawArea}")

public class DemoDrawArea
    extends PageBean
    implements Serializable
{
    String m_hexImageDrawCommand;

    public DemoDrawArea(IWorkpageDispatcher workpageDispatcher)
    {
        super(workpageDispatcher);
        // read image and provide as "heximage(...)"
        byte[] image = new WebResourceReader().readFileIntoByteArray("/images/pictures/ph0.jpg");
        m_hexImageDrawCommand = "heximage(" +
                                "10,10," +
                                ValueManager.encodeHexString(image) +
                                ",lefttop);" +
                                "heximage(" +
                                "250,10,30,30," +
                                ValueManager.encodeHexString(image) +
                                ",lefttop);";
    }

    @Override
    public String getPageName() { return "/workplace/demodrawarea.jsp"; }
    @Override
    public String getRootExpressionUsedInPage() { return "#{d.DemoDrawArea}"; }

    public String getHexImageDrawCommand() { return m_hexImageDrawCommand; }
    public void setHexImageDrawCommand(String value) { this.m_hexImageDrawCommand = value; }

}
