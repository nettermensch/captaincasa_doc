The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • General
  • Server Side Value Binding
  • Data Type Resolution in Maps

This example shows by using two maps, how the data type of the values to be put into the map is found:

Enter some values into the fields - and observe the protocol output.

For the map "data" the following rules apply:
(.) int1 is already filled into the map - as result the type will be kept (String).
(.) the same happens with int2 - but now the type is Integer
(.) with int3 the map was not filled at all, so the FORMATTEDFIELD's format definition is guiding the type resolution

For the map "datawi" the interface "IPropertyTypeResolver" is provided, exactly telling which data type to select.

This is the Java code:

package workplace;

import java.io.Serializable;
import java.util.HashMap;

import javax.faces.event.ActionEvent;

import org.eclnt.editor.annotations.CCGenClass;
import org.eclnt.jsfserver.util.ClientTestLog;
import org.eclnt.jsfserver.util.IClientLogAware;
import org.eclnt.jsfserver.util.IPropertyTypeResolver;
import org.eclnt.workplace.IWorkpageDispatcher;

@CCGenClass (expressionBase="#{d.DemoDynamicBindingTypes}")

public class DemoDynamicBindingTypes
    extends PageBean 
    implements Serializable, IClientLogAware
{
    public class MyMap extends HashMap<String,Object>
    {
        @Override
        public Object put(String key, Object value)
        {
            if (value != null)
                m_protocol = m_protocol + "\n" + key + "/" + value + "/" + value.getClass();
            else
                m_protocol = m_protocol + "\n" + key + "/null";
            return super.put(key, value);
        }
    }
    
    public class MyMapWithInfo extends MyMap
        implements IPropertyTypeResolver
    {
        public Class resolveType(String propertyName)
        {
            if ("int4".equals(propertyName)) return String.class;
            if ("int5".equals(propertyName)) return Integer.class;
            return String.class;
        }
    }
    
    
    String m_protocol = "";
    MyMap m_data = new MyMap();
    MyMapWithInfo m_datawi = new MyMapWithInfo();
    
    public DemoDynamicBindingTypes(IWorkpageDispatcher workpageDispatcher)
    {
        super(workpageDispatcher);        
        m_data.put("int1","100");
        m_data.put("int2",100);
        m_protocol = "Protocol:";
    }

    public MyMap getData() { return m_data; }
    public MyMapWithInfo getDatawi() { return m_datawi; }
    public String getProtocol() { return m_protocol; }
    
    public void onFlush(ActionEvent event) {}

    public boolean onCreateClientTestLogMessage()
    {
        ClientTestLog.addClientTestLogMessage(m_protocol);
        return false;
    }
    
}
