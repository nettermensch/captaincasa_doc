The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • Components
  • Straight Controls
  • Combobox and List
  • Dynamic definition of items

In this demo the list of COMBOBOX-items is defined on server-side.

In addition to the COMBOBOX-VALUE (which points to the property which is shown/edited) you need to implement a property of class "ValidValuesBinding" and bind it by the COMBOBOX-VALIDVALUESBINDING attribute.

As you can see in the demo by pressing the "Show 10 items" button, the server-side logic may change the number of items any time.

Please note: as any other input component a user selection is not immediately sent to the server-side but is queued within the client. You may switch attribute FLUSH to true if you want to immediately trigger a round-trip after user-selection.

This is the XML layout definition:

<?xml version="1.0" encoding="utf-8"?>
<t:layoutdefinition xmlns:t="http://www.CaptainCasa.com/controllibrary/t" >
<t:beanprocessing id="g_1" beanbinding="#{d.DemoComboboxDyn}" />
<t:rowdemobodypane id="g_4" objectbinding="#{d.DemoComboboxDyn}" rowdistance="10" >
<t:row id="g_6" >
<t:combobox id="g_7" labeltext="Department" validvaluesbinding="#{d.DemoComboboxDyn.departmentVVB}" value="#{d.DemoComboboxDyn.department}" width="200" />
</t:row>
<t:rowdistance id="g_11" height="50" />
<t:row id="g_3" />
<t:row id="g_5" >
<t:textpane id="g_13" text="Press the following button - the number of available combo box items will be changed from 4 values to 10 values." width="100%" />
</t:row>
<t:row id="g_9" >
<t:button id="g_2" actionListener="#{d.DemoComboboxDyn.onShow10Items}" text="Show 10 items" />
</t:row>
</t:rowdemobodypane>
</t:layoutdefinition>

This is the Java code:

package workplace;

import java.io.Serializable;

import javax.faces.event.ActionEvent;

import org.eclnt.editor.annotations.CCGenClass;
import org.eclnt.jsfserver.elements.util.ValidValuesBinding;
import org.eclnt.workplace.IWorkpageDispatcher;

@CCGenClass (expressionBase="#{d.DemoComboboxDyn}")

public class DemoComboboxDyn
    extends PageBean 
    implements Serializable
{
    // ------------------------------------------------------------------------
    // inner classes
    // ------------------------------------------------------------------------

    /* Listener to the user of the page bean. */
    public interface IListener extends Serializable
    {
    }

    // ------------------------------------------------------------------------
    // members
    // ------------------------------------------------------------------------

    ValidValuesBinding m_departmentVVB = new ValidValuesBinding();
    String m_department;

    // ------------------------------------------------------------------------
    // constructors & initialization
    // ------------------------------------------------------------------------

    public DemoComboboxDyn(IWorkpageDispatcher workpageDispatcher)
    {
        super(workpageDispatcher);
        fillComboboxValidValuesWith4Values();
    }

    public String getPageName() { return "/workplace/democomboboxdyn.xml"; }
    public String getRootExpressionUsedInPage() { return "#{d.DemoComboboxDyn}"; }

    // ------------------------------------------------------------------------
    // public usage
    // ------------------------------------------------------------------------

    public ValidValuesBinding getDepartmentVVB() { return m_departmentVVB; }

    public String getDepartment() { return m_department; }
    public void setDepartment(String value) { this.m_department = value; }
    
    public void onShow10Items(ActionEvent event)
    {
        fillComboboxValidValuesWith10Values();
    }

    // ------------------------------------------------------------------------
    // private usage
    // ------------------------------------------------------------------------
    
    private void fillComboboxValidValuesWith4Values()
    {
        m_departmentVVB.clear();
        m_departmentVVB.addValidValue("0001","Sales");
        m_departmentVVB.addValidValue("0002","Production");
        m_departmentVVB.addValidValue("0003","Human Resources");
        m_departmentVVB.addValidValue("0004","Development");
    }

    private void fillComboboxValidValuesWith10Values()
    {
        m_departmentVVB.clear();
        m_departmentVVB.addValidValue("0001","Sales");
        m_departmentVVB.addValidValue("0002","Production");
        m_departmentVVB.addValidValue("0003","Human Resources");
        m_departmentVVB.addValidValue("0004","Development");
        m_departmentVVB.addValidValue("0005","Department 5");
        m_departmentVVB.addValidValue("0006","Department 6");
        m_departmentVVB.addValidValue("0007","Department 7");
        m_departmentVVB.addValidValue("0008","Department 8");
        m_departmentVVB.addValidValue("0009","Department 9");
        m_departmentVVB.addValidValue("0010","Department 10");
    }

}
