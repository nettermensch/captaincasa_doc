The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • Containers
  • Foldable / Titled Pane
  • Light Style / Titled Pane

In addition to the default styling (attribute STYLEVARIANT "default") there is a second style variant "cclight" (attribute STYLEVARIANT "cclight") which comes without explicit coloring.

Please note: by setting attribute ENABLED to "false" you can switch off the dynamic behavior of the FOLDABLEPANE - and just use it as a titled area.

package workplace;

import java.io.Serializable;

import org.eclnt.editor.annotations.CCGenClass;
import org.eclnt.jsfserver.pagebean.PageBean;
import org.eclnt.workplace.IWorkpageDispatcher;

import javax.faces.event.ActionEvent;

@CCGenClass (expressionBase="#{d.DemoFoldableLight}")

public class DemoFoldableLight
    extends PageBean
    implements Serializable
{
    public DemoFoldableLight(IWorkpageDispatcher dispatcher)
    {
        super(dispatcher);
    }

    public String getPageName() { return "/workplace/demofoldablelight.jsp"; }
    public String getRootExpressionUsedInPage() { return "#{d.DemoFoldableLight}"; }
}
