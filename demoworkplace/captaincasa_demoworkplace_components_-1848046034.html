The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • Components
  • Straight Controls
  • Structure Components
  • Title Bar

This demo shows various configuration options that are provided by the ROWTITLEBAR component.

You may place additional components on the right of the text into the title bar.
By default the width of the title bar's label is set to 100%, so additional components are "pressed" to the very right of the title bar's internal row, in which the component are arranged. But you may override by using attribute TEXTWIDTH.
