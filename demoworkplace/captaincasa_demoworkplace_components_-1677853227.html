The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • Components
  • Straight Controls
  • Combobox and List
  • Width of Popup

Open the different combo boxes and observe the width of the popup in which the list of values is shown.



The popup that is opened by the COMBOBOX can be sized using the attribute COMBOPOPUPWIDTH.

By default the width of the popup equals the width of the COMBOBOX component - with a minimum width of 150. This ensures that event with small COMBOBOX instances the text of the popup is readable for the user.

By setting the attribute COMBOPOPUPWIDTH you can define the minimum width of the popup on your own.
