The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • Components
  • Straight Controls
  • Field Control
  • Regular Expressions
  • Regular Expressions

You can assign a regular expression to a FIELD using the REGEX attribute. As consequence the field's content is checked against the regular expression.

If the check determines an error, then the field is highlighted with red background colors.

There is an attribute REGEXMODE which defines the exact behavior:

0 (default): the user gets notified about invalid input of data - that's it. The value is accepted and passed to the logic.
1: the user is not allowed to input invalid data. Whenever invalid data is input and the user leaves the field, a popup dialog is shown notifying the suer that the input was reset to the last value.
2: same as 0, invalid values are accepted, the user in addition receives a popup dialog when leaving the field

By default the field only is drawn as invalid during input of data. By adding 10 to the existing REGEXMODE-values you define that invalid values are always shown.

There is the possibility to show the user a certain comment (attribute REGEXHINT) in case he/she is entering a value, that does not match the regular expression.

Regular expression can be - as with all other attributes - either assigned statically (as in the example) or can be assigned by expression binding to a server side property ("#{...}").

You find a nice description about the syntax of regular expressions at:
http://java.sun.com/docs/books/tutorial/essential/regex/intro.html

package workplace;

import javax.faces.event.ActionEvent;

import org.eclnt.editor.annotations.CCGenClass;
import org.eclnt.jsfserver.defaultscreens.Statusbar;
import org.eclnt.workplace.IWorkpageDispatcher;

@CCGenClass (expressionBase="#{d.demoFieldregex}")

public class DemoFieldregex extends PageBean
{
    public DemoFieldregex(IWorkpageDispatcher dispatcher)
    {
        super(dispatcher);
        initRefComponent("field"); 
    }

    protected String m_mailAddress;
    public String getMailAddress() { return m_mailAddress; }
    public void setMailAddress(String value) { m_mailAddress = value; }

    protected String m_phoneNumer = "abcde";
    public String getPhoneNumer() { return m_phoneNumer; }
    public void setPhoneNumer(String value) { m_phoneNumer = value; }

    protected String m_zipCode;
    public String getZipCode() { return m_zipCode; }
    public void setZipCode(String value) { m_zipCode = value; }

    public void onSynchData(ActionEvent event) 
    {
        Statusbar.outputSuccessWithPopup("Synchronization done.");
    }

}
