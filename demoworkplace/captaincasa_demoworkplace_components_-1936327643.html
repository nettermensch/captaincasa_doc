The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • Components
  • Animation & Graphics & Sound
  • Paint Area
  • Layered Selection

If selecting items of a PAINTAREA then by default the selection follows the z-hierarchy of the items. The z-hiearchy is either explicitly defined (the BOUNDS definition may have 5 parameters: "x;y;width;height;z") or it is implicitly defined by the drawing sequence.

This means: when one item covers another item, then the covered item cannot be reach by mouse clicks.

By setting attribute PAINTAREA-WITHLAYEREDSELECTION to "true" you can overcome this: you can continue to click onto a certain point and all the items that are on top of this point are selected one after the next.

This is the XML layout definition:

This is the Java code:

package workplace;

import java.io.Serializable;
import java.util.ArrayList;
import java.util.List;

import org.eclnt.editor.annotations.CCGenClass;
import org.eclnt.jsfserver.elements.componentnodes.PAINTAREAITEMNode;
import org.eclnt.jsfserver.elements.impl.DYNAMICCONTENTBinding;
import org.eclnt.jsfserver.elements.impl.ROWDYNAMICCONTENTBinding.ComponentNode;
import org.eclnt.workplace.IWorkpageDispatcher;

@CCGenClass (expressionBase="#{d.DemoPaintAreaLayeredSelection}")

public class DemoPaintAreaLayeredSelection
    extends PageBean 
    implements Serializable
{
    // ------------------------------------------------------------------------
    // members
    // ------------------------------------------------------------------------

    boolean m_withLayeredSelection = true;
    DYNAMICCONTENTBinding m_dynContent = new DYNAMICCONTENTBinding();

    // ------------------------------------------------------------------------
    // constructors & initialization
    // ------------------------------------------------------------------------

    public DemoPaintAreaLayeredSelection(IWorkpageDispatcher workpageDispatcher)
    {
        super(workpageDispatcher);      
        render();
    }

    public String getPageName() { return "/workplace/demopaintarealayeredselection.xml"; }
    public String getRootExpressionUsedInPage() { return "#{d.DemoPaintAreaLayeredSelection}"; }

    // ------------------------------------------------------------------------
    // public usage
    // ------------------------------------------------------------------------

    public boolean getWithLayeredSelection() { return m_withLayeredSelection; }
    public void setWithLayeredSelection(boolean value) { this.m_withLayeredSelection = value; }

    public DYNAMICCONTENTBinding getDynContent() { return m_dynContent; }

    // ------------------------------------------------------------------------
    // private usage
    // ------------------------------------------------------------------------
    
    private void render()
    {
        List<ComponentNode> l = new ArrayList<ComponentNode>();
        for (int i=0; i<10; i++)
        {
            PAINTAREAITEMNode pin = new PAINTAREAITEMNode()
                .setBounds((20+i*20)+";"+(20+i*20)+";200;200")
                .setBackground("#00000010")
                .setBorder("#C0C0C0")
                .setSelectionenabled(true)
                .setMovingenabled(true);
            l.add(pin);
        }
        m_dynContent.setContentNodes(l);
    }
}
