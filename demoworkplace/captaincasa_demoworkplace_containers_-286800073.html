The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • Containers
  • Adaptive Containers - Controls
  • Adaptive line

(FX only - this component is not supported in Swing yet.)

Resize the window horizontally - if the width is below a certain level then the controls in the lines are rearranged.

The component ROWADAPTIVELINE is a row that by default arranges its content into one line.

If the available horizontal width is below a certain level (defined by attribute BREADPIXELS) then the line is split into two lines. All the components left of the BREAKDINDEX (by default 1) are moved into the first line, the other ones are moved into the second line.
