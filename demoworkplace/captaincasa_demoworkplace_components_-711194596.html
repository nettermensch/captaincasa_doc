The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • Components
  • Straight Controls
  • Text Input/Output
  • Text input with AI support
  • Ad hoc text translation

The component TPBC:TEXTPANEWITHTRANSLATION is a "just normal text pane" with some built in translation feature.

When hovering with the mouse over the component area, then a little icon will pop up - by which the user can translate the text into various languages.

It connects to a translation service (here: Deepl) by using an abstracted interface (ITranslationService). The translation then is done - in case of Deepl - by calling a REST service.

Please note when using this component together with Deepl: Deepl is accessed through CaptainCasa's ccee-library, the API class is "DeeplFreeTranslation". It requires an API key that you need to obtain from Deepl and which you either have to directly set into the API-class or which you have to provide as property of config file "ccee_config.properties".

Last but not least: the translated text is buffered in the stream store, buffer key is the hash code of the text. Deepl is only called when the buffer is empty.

This is the Java code:

package workplace;

import java.io.Serializable;

import org.eclnt.ccaddons.pbc.translate.util.TranslationService;
import org.eclnt.ccaddons.pbc.translate.util.TranslationServiceImplDeeplFreeTranslation;
import org.eclnt.editor.annotations.CCGenClass;
import org.eclnt.util.file.ClassloaderReader;
import org.eclnt.workplace.IWorkpageDispatcher;

@CCGenClass (expressionBase="#{d.DemoTextPaneWithTranslation}")

public class DemoTextPaneWithTranslation
    extends PageBean 
    implements Serializable
{
    // ------------------------------------------------------------------------
    // inner classes
    // ------------------------------------------------------------------------

    // ------------------------------------------------------------------------
    // members
    // ------------------------------------------------------------------------

    String m_text;

    // ------------------------------------------------------------------------
    // constructors & initialization
    // ------------------------------------------------------------------------

    static
    {
        TranslationService.initialize(new TranslationServiceImplDeeplFreeTranslation());
    }
    
    public DemoTextPaneWithTranslation(IWorkpageDispatcher workpageDispatcher)
    {
        super(workpageDispatcher);        
        readText();
    }

    public String getPageName() { return "/workplace/demotextpanewithtranslation.xml"; }
    public String getRootExpressionUsedInPage() { return "#{d.DemoTextPaneWithTranslation}"; }

    // ------------------------------------------------------------------------
    // public usage
    // ------------------------------------------------------------------------

    public String getText() { return m_text; }

    // ------------------------------------------------------------------------
    // private usage
    // ------------------------------------------------------------------------
    
    private void readText()
    {
        try
        {
            byte[] textBytes = new ClassloaderReader(true).readFile("workplace/DemoTextPaneWithTranslation1.html",true);
            m_text = new String(textBytes,"cp1252");
        }
        catch (Throwable t)
        {
            throw new Error(t);
        }
    }

}
