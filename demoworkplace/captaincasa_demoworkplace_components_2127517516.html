The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • Components
  • Straight Controls
  • Client Clock
  • Client Clock

The CLOCK component is a pure output component - it presents the current time or date to the user.

By using attribute CLOCKFORMAT you can specify various output formats. Use the attribute TIMEZONE if you want to explicitly define the time zone to be used for displaying - if not defined then the client's default time zone will be used.

    }
