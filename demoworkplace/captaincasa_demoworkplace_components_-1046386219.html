The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • Components
  • Straight Controls
  • Drag & Drop
  • Drop Files (Java Client only)

Pick one ore more files from the file explorer and drop the information into the correpsonding area. You will see that the name(s) of the file(s) are transferred.

Apart from defining information types within DRAGSEND and DROPRECEIVE in any "free" way, there are some predefined types:

(.) "ccfilename" is the type definition that corresponds to exactly one file
(.) "ccfilenames" is the type definition that corresponds to one or more file names

In the example the DROPRECEIVE definition is applied on PANE level.

This is the Java code:

package workplace;

import java.io.Serializable;

import javax.faces.event.ActionEvent;

import org.eclnt.editor.annotations.CCGenClass;
import org.eclnt.jsfserver.elements.events.BaseActionEventDrop;
import org.eclnt.util.valuemgmt.ValueManager;
import org.eclnt.workplace.IWorkpageDispatcher;

@CCGenClass (expressionBase="#{d.DemoDragFiles}")

public class DemoDragFiles
    extends PageBean
    implements Serializable
{
    protected String m_droppedInfo;
    
    public DemoDragFiles(IWorkpageDispatcher dispatcher)
    {
        super(dispatcher);
    }

    public String getDroppedInfo() { return m_droppedInfo; }
    
    public void onFileDrop(ActionEvent event) 
    {
        StringBuffer info = new StringBuffer();
        if (event instanceof BaseActionEventDrop)
        {
            BaseActionEventDrop baed = (BaseActionEventDrop)event;
            String s = baed.getDragInfo();
            info.append(s);
            if (s.startsWith("ccfilename:"))
            {
                info.append("\n\nFilename:");
                info.append("\n--------------------------");
                info.append("\n"+s.substring("ccfilename:".length()));
            }
            else if (s.startsWith("ccfilenames:"))
            {
                info.append("\n\nFilename(s):");
                info.append("\n--------------------------");
                String csv = s.substring("ccfilenames:".length());
                String[] fns = ValueManager.decodeCSV(csv);
                for (String fn: fns)
                {
                    info.append("\n" + fn);
                }
            }
        }
        m_droppedInfo = info.toString();
    }


}
