The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • General
  • Client Integration
  • JavaScript Method Caller

The invisible component CLIENTJSMETHODCALLER calls a JavaScript method on client side.

The attribute JSCALL contains the JavaScript call of the method. Pleas note - you may only pass one JavaScript call - so you cannot pass a sequence of statements.

Dependent on the attribute WITHCALLBACK the result of the calls is transferred back to the server.

All parameters are interpreted as String parameters - both the parameters you pass into the JavaScript statement and the parameter that you return as result (if having set WITHCALLBACK to true).

In the RISC client you may add JavaScript libraries using system.xml configuration file ():

<system...>
...
<riscclientscript src="xxx.js" type="text/javascript"/>
...
</system>

In the JavaSwing client you may add JavaScript processing by simply updating the .html file in which the client-applet is embedded.

This is the XML layout definition:

This is the Java code:

package workplace;

import java.io.Serializable;

import javax.faces.event.ActionEvent;

import org.eclnt.editor.annotations.CCGenClass;
import org.eclnt.jsfserver.defaultscreens.Statusbar;
import org.eclnt.jsfserver.elements.events.BaseActionEventJsCallResult;
import org.eclnt.jsfserver.elements.util.Trigger;
import org.eclnt.jsfserver.pagebean.PageBean;
import org.eclnt.util.valuemgmt.ValueManager;
import org.eclnt.workplace.IWorkpageDispatcher;

@CCGenClass (expressionBase="#{d.DemoClientJSMethodCaller}")

public class DemoClientJSMethodCaller
    extends PageBean
    implements Serializable
{
    // ------------------------------------------------------------------------
    // inner classes
    // ------------------------------------------------------------------------
    
    // ------------------------------------------------------------------------
    // members
    // ------------------------------------------------------------------------
    
    Trigger m_trigger = new Trigger();
    String m_jsCall = "";
    String m_p1 = "Captain";
    String m_p2 = "Casa";
    
    
    // ------------------------------------------------------------------------
    // constructors & initialization
    // ------------------------------------------------------------------------

    public DemoClientJSMethodCaller(IWorkpageDispatcher dispatcher)
    {
        super(dispatcher);
    }
    
    public String getPageName() { return "/workplace/democlientjsmethodcaller.jsp"; }
    public String getRootExpressionUsedInPage() { return "#{d.DemoClientJSMethodCaller}"; }

    // ------------------------------------------------------------------------
    // public usage
    // ------------------------------------------------------------------------

    public String getJsCall() { return m_jsCall; }
    public void setJsCall(String value) { this.m_jsCall = value; }

    public Trigger getTrigger() { return m_trigger; }
    public void setTrigger(Trigger value) { this.m_trigger = value; }

    public void onTriggerAction(ActionEvent event)
    {
        // the concatenation via encodeMethod is important - because it
        // properly encodes "," characters within the parameters!
        m_jsCall = ValueManager.encodeMethod("testJSMethodCall",new String[] 
        {
            m_p1,
            m_p2
        });
        m_trigger.trigger();
    }

    public String getP2() { return m_p2; }
    public void setP2(String value) { this.m_p2 = value; }

    public String getP1() { return m_p1; }
    public void setP1(String value) { this.m_p1 = value; }

    public void onJsCallAction(ActionEvent event) 
    {
        if (event instanceof BaseActionEventJsCallResult)
        {
            BaseActionEventJsCallResult e = (BaseActionEventJsCallResult)event;
            if (e.getSuccess() == true)
                Statusbar.outputSuccess("Result of call: " + e.getSuccess() + ", " + e.getResult());
            else
                Statusbar.outputError("Result of call: " + e.getSuccess() + ", " + e.getResult());
        }
    }

    // ------------------------------------------------------------------------
    // private usage
    // ------------------------------------------------------------------------
}
