The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • Components
  • Straight Controls
  • Statusbar and Messages
  • Notification messages

Notifications are messages that are output by the operating system - independent from the browser. The browser itself may be invisible - e.g. minimized or your dialog might be positioned on a hidden brwoser tab.

Notifications require some permission that needs to be granted by the user. This permission only can be obtained in the context of a user interaction.

PLEASE NOTE: there are a couple of security restrictions that might be active in concrete scenarios that depend on the environment (browser type) and/or the browser configuration. It is important to check your usage scenarios prior to using notifications!

There are diverse attributes that are part of the NOTIFICATION component. By setting attribute ONLYSENDIFHIDDEN to "true" you ensure that notifications will only be sent if the current dialog is not visible to the user - e.g. if it is minimized.

This is the Java code:

package workplace;

import java.io.Serializable;

import org.eclnt.editor.annotations.CCGenClass;
import org.eclnt.jsfserver.elements.util.Trigger;
import org.eclnt.workplace.IWorkpageDispatcher;

@CCGenClass (expressionBase="#{d.DemoNotification}")

public class DemoNotification
    extends PageBean 
    implements Serializable
{
    // ------------------------------------------------------------------------
    // inner classes
    // ------------------------------------------------------------------------

    // ------------------------------------------------------------------------
    // members
    // ------------------------------------------------------------------------

    boolean m_onlySendIfHidden = false;;
    boolean m_requireInteraction = false;
    Trigger m_delayTrigger = new Trigger();
    Trigger m_notificationTrigger = new Trigger();
    String m_bodyText = "This notification was sent by the NOTIFICATION component. If it appears, then the permission for receiving notifications was granted by the user.";
    String m_titleText = "Hello User!";
    
    // ------------------------------------------------------------------------
    // constructors & initialization
    // ------------------------------------------------------------------------

    public DemoNotification(IWorkpageDispatcher workpageDispatcher)
    {
        super(workpageDispatcher);        
    }

    public String getPageName() { return "/workplace/demonotification.xml"; }
    public String getRootExpressionUsedInPage() { return "#{d.DemoNotification}"; }

    // ------------------------------------------------------------------------
    // public usage
    // ------------------------------------------------------------------------

    public String getBodyText() { return m_bodyText; }
    public void setBodyText(String value) { this.m_bodyText = value; }

    public String getTitleText() { return m_titleText; }
    public void setTitleText(String value) { this.m_titleText = value; }

    public boolean getOnlySendIfHidden() { return m_onlySendIfHidden; }
    public void setOnlySendIfHidden(boolean value) { this.m_onlySendIfHidden = value; }

    public boolean getRequireInteraction() { return m_requireInteraction; }
    public void setRequireInteraction(boolean value) { this.m_requireInteraction = value; }

    public Trigger getDelayTrigger() { return m_delayTrigger; }

    public Trigger getNotificationTrigger() { return m_notificationTrigger; }

    public void onCreateNotificationAction(javax.faces.event.ActionEvent event) 
    {
        m_notificationTrigger.trigger();
    }

    public void onCreateNotificationActionDelayed(javax.faces.event.ActionEvent event) 
    {
        m_delayTrigger.trigger();
    }

    // ------------------------------------------------------------------------
    // private usage
    // ------------------------------------------------------------------------
}
