The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • Containers
  • Container sizing
  • Row distance in panes

A pane is a container component, in which rows are placed. By default the rows follow one another, i.e. there is no vertical distance between the rows. You can specify a general distance that is placed between the rows by using the attribute ROWDISTANCE.

Please note: there is also a component ROWDISTANCE that can be placed into a container. If using the component ROWDISTANCE then also for this component an additional distance is calculated, depending from the value of the pane container's ROWDISTANCE attribute.
