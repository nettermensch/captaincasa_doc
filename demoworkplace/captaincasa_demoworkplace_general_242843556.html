The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • General
  • Focus Management
  • Focus sequence

You can explicitly define the tab/focus-sequence within a page using the attribute FOCUSSEQUENCE, that is available on all relevant components.

The attribute value, that you need to pass, is built out of two definitions: <area>/<index>

Within one page there may be one or several area definitions. In the example page there is the "A" area, the "B" area, the "C" area. Within each area you define the indices for these components for which you want to explicitly define the tab/focus-order.

The focus sequence definitions are defined on page level: if you use area "A" on one page, and you use area "A" on another page as well - then both "A" areas are no interlinked, but are managed as totally separated areas.

Btw.: the index definitions may contain gaps, i.e. you may define FOCUSSEQUENCE values "A/0", "A/2", "A/5" without having to fill in the missing indices.

This is the Java code:

package workplace;

import java.io.Serializable;

import org.eclnt.editor.annotations.CCGenClass;
import org.eclnt.jsfserver.elements.impl.FIXGRIDItem;
import org.eclnt.jsfserver.elements.impl.FIXGRIDListBinding;
import org.eclnt.workplace.IWorkpageDispatcher;
import org.eclnt.workplace.WorkpageDispatchedBean;

@CCGenClass (expressionBase="#{d.DemoFocusSequence}")

public class DemoFocusSequence
    extends PageBean 
    implements Serializable
{
    public class GridItem extends FIXGRIDItem implements java.io.Serializable
    {
        String m_town;
        public String getTown() { return m_town; }
        public void setTown(String value) { this.m_town = value; }

        String m_street;
        public String getStreet() { return m_street; }
        public void setStreet(String value) { this.m_street = value; }

        String m_lastName;
        public String getLastName() { return m_lastName; }
        public void setLastName(String value) { this.m_lastName = value; }

        String m_firstName;
        public String getFirstName() { return m_firstName; }
        public void setFirstName(String value) { this.m_firstName = value; }

    }

    FIXGRIDListBinding<GridItem> m_grid = new FIXGRIDListBinding<GridItem>();

    public DemoFocusSequence(IWorkpageDispatcher workpageDispatcher)
    {
        super(workpageDispatcher);
        for (int i=0; i<20; i++)
        {
            GridItem gi = new GridItem();
            m_grid.getItems().add(gi);
        }
    }

    public FIXGRIDListBinding<GridItem> getGrid() { return m_grid; }
}
