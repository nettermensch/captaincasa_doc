The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • Components
  • Straight Controls
  • Text Input/Output
  • Text transform to upper/lower case

Most of the components rendering some text content provide an attribute TEXTTRANSFORM which either can be set to "uppercase" or "lowercase". The text that is output is updated inside the client accordingly.

This is extremely useful when e.g. defining text in resource files for multi-language usage: you do not want to define literals in upper case within the resource files, but only want the output in the control to be upper case.

Please also note:

1. for FIELD-components there is an equivalent UPPERCASE / LOWERCASE property. Here the user input is transferred correspondingly.

2. LABELTEXT values (the little labels shown on top left of a e.g. field) can be transformed using the LABELTEXTTRANSFORM attribute

This is the Java code:

package workplace;

import java.io.Serializable;

import org.eclnt.editor.annotations.CCGenClass;
import org.eclnt.jsfserver.elements.impl.FIXGRIDItem;
import org.eclnt.jsfserver.elements.impl.FIXGRIDListBinding;
import org.eclnt.workplace.IWorkpageDispatcher;
import org.eclnt.workplace.WorkpageDispatchedPageBean;

import workplace.logic.RandomLogic;

@CCGenClass (expressionBase="#{d.DemoTextTransform}")

public class DemoTextTransform
    extends PageBean 
    implements Serializable
{
    // ------------------------------------------------------------------------
    // inner classes
    // ------------------------------------------------------------------------

    public class GridItem extends FIXGRIDItem implements java.io.Serializable
    {
        String i_lastName;
        public String getLastName() { return i_lastName; }
        public void setLastName(String value) { this.i_lastName = value; }

        public void onRowSelect()
        {
        }
        public void onRowExecute()
        {
        }
    }

    // ------------------------------------------------------------------------
    // members
    // ------------------------------------------------------------------------

    FIXGRIDListBinding<GridItem> m_grid = new FIXGRIDListBinding<GridItem>();

    // ------------------------------------------------------------------------
    // constructors & initialization
    // ------------------------------------------------------------------------

    public DemoTextTransform(IWorkpageDispatcher workpageDispatcher)
    {
        super(workpageDispatcher);    
        for (int i=0; i<10; i++)
        {
            GridItem gi = new GridItem();
            gi.setLastName(RandomLogic.getLastName());
            m_grid.getItems().add(gi);
        }
    }

    public String getPageName() { return "/workplace/demotexttransform.xml"; }
    public String getRootExpressionUsedInPage() { return "#{d.DemoTextTransform}"; }

    // ------------------------------------------------------------------------
    // public usage
    // ------------------------------------------------------------------------

    public FIXGRIDListBinding<GridItem> getGrid() { return m_grid; }

    // ------------------------------------------------------------------------
    // private usage
    // ------------------------------------------------------------------------
}
