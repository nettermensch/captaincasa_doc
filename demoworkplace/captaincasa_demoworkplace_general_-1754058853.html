The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • General
  • Workplace Management
  • Inter Workpage Events
  • Receiving Page

This is the "receiver" part of the demo. It contains a grid, which is empty at the beginning. Open the "sender" part of the demo and press the corresponding button in the sender screen. Every time you press the button in the "sender" screen then a new item will be generated within the "receiver" screen.

What's all this about? - This example demonstrates how two workpages that run isolated from one another can and should talk to one another: through workpage eventing. The one workpage throws an event, the other one receives it and reacts correspondingly.

This demo is a quite technical one: typical examples of the real world are: in the one workpage you see a list of objects, in a second you see the detail of one object. When saving the detail the list needs to be updated. Instead of hard-wiring the pages you couple the pages through eventing.

Please, do not get confused: the workpage events have nothing to do with thw JSF events!

This is the Java code:

package workplace;

import java.io.Serializable;

import org.eclnt.editor.annotations.CCGenClass;
import org.eclnt.jsfserver.elements.impl.FIXGRIDItem;
import org.eclnt.jsfserver.elements.impl.FIXGRIDListBinding;
import org.eclnt.jsfserver.managedbean.IDispatcher;
import org.eclnt.workplace.IWorkpageProcessingEventListener;
import org.eclnt.workplace.WorkpageProcessingEvent;

@CCGenClass (expressionBase="#{d.DemoWpReceive}")

public class DemoWpReceive
    extends PageBean
    implements Serializable
{
    // ------------------------------------------------------------------------
    // inner classes
    // ------------------------------------------------------------------------
    
    public class MyWorkpageProcessingEventListener
        implements IWorkpageProcessingEventListener
    {
        public void processEvent(WorkpageProcessingEvent event)
        {
            if (event instanceof DemoWpAddItemEvent)
                addNewItem();
        }
    }
    
    public class MyRow extends FIXGRIDItem implements java.io.Serializable
    {
        protected String m_lastName;
        public String getLastName() { return m_lastName; }
        public void setLastName(String value) { m_lastName = value; }
        
        protected String m_firstName;
        public String getFirstName() { return m_firstName; }
        public void setFirstName(String value) { m_firstName = value; }
    }
    
    // ------------------------------------------------------------------------
    // members
    // ------------------------------------------------------------------------
    
    protected FIXGRIDListBinding<MyRow> m_rows = new FIXGRIDListBinding<MyRow>();
    
    // ------------------------------------------------------------------------
    // constructors
    // ------------------------------------------------------------------------
    
    public DemoWpReceive(IDispatcher dispatcher)
    {
        super(dispatcher);
        getWorkpage().addWorkpageProcessingEventListener(new MyWorkpageProcessingEventListener());
    }
    
    // ------------------------------------------------------------------------
    // public usage
    // ------------------------------------------------------------------------
    
    public FIXGRIDListBinding<MyRow> getRows() { return m_rows; }
    
    // ------------------------------------------------------------------------
    // private usage
    // ------------------------------------------------------------------------
    
    private void addNewItem()
    {
        MyRow row = new MyRow();
        row.setFirstName("New first name");
        row.setLastName("New last name");
        m_rows.getItems().add(row);
    }
    
}
