The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • Containers
  • Split Pane - Extended Functions
  • Quick Navigation

By setting attribute ONETOUCHEXPANDABLE to true two icons will be added to the divider between the split pane parts.

The user can quickly collapse/re-open the corresponding part by clicking onto the corresponding icon.

This is the XML layout definition:
