The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • Components
  • Straight Controls
  • Structure Components
  • Outlook Bar - Separator line

The management of lines between the parts of the OUTLOOKBAR is simple by using the attribute SEPARATORCOLOR.

The client side will automatically update the borders of the contained items and of the contained content, so that always the bottom border is set for all items - despite the last one.

Please note: the border management will override any border the you assign by style. We recommend to style the OUTLOOKBARITEM components without any border as consequence.
