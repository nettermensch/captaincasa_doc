The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • General
  • Workplace Management
  • Set title at runtime

Define a new title and press the button - the titel that you input will be the new workpage title. Both the title bar and the workpage selector (on the bottom of the screen) will be updated accordingly.

In addition to changing the title of the workpage you can change the id as well - using the IWorkpage.setId() method.

This is the Java code:

package workplace;

import java.io.Serializable;

import javax.faces.event.ActionEvent;

import org.eclnt.editor.annotations.CCGenClass;
import org.eclnt.jsfserver.managedbean.IDispatcher;
import org.eclnt.workplace.Workpage;

@CCGenClass (expressionBase="#{d.DemoWorkpageSetTitle}")

public class DemoWorkpageSetTitle 
    extends PageBean
    implements Serializable
{
    public DemoWorkpageSetTitle(IDispatcher dispatcher)
    {
        super(dispatcher);
        if (getWorkpage() != null)
        {
            // take over current title
            m_title = getWorkpage().getTitle();
            m_selectorTitle = getWorkpage().getSelectorTitle();
            getWorkpage().setIconURL("/eclntjsfserver/images/iconssvg/calendar_16x16.svg");
        }
    }

    protected String m_title = "Update Workpage Title";
    public String getTitle() { return m_title; }
    public void setTitle(String value) { m_title = value; }
    
    protected String m_selectorTitle = "Update Workpage Title";
    public void setSelectorTitle(String value) { m_selectorTitle = value; }
    public String getSelectorTitle() { return m_selectorTitle; }
    
    public void onUpdateTitle(ActionEvent event) 
    {
        if (getWorkpage() != null)
        {
            getWorkpage().setTitle(m_title);
            getWorkpage().setSelectorTitle(m_selectorTitle);
        }
    }

    public void onUpdateImage(ActionEvent event) 
    {
        if (getWorkpage() != null)
        {
            getWorkpage().setIconURL("/images/iconssvg/demoshow_16x16.svg");
        }
    }

}
