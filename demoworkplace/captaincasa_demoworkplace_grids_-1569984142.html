The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • Grids & Trees
  • Grids and Trees
  • Flexible row heights
  • Dynamic height sizing with images

This demo shows how the IMAGE component behaves when being used in a grid with dynamic height sizing (FIXGRID-DYNAMICHEIGHTSIZING set to "true").

Change the column width of the image-column and observe how the image size adapts - and as result the row height adapts as well.

In general the IMAGE component always tries to fit into its given space - by default keeping the image ratio. The height of the image cell is limited by the pixel height of the image (or the IMAGEWIDTH/HEIGHT that you explicitly assign).

<t:row id="g_6" >

This is the Java code:

package workplace;

import java.io.Serializable;

import org.eclnt.editor.annotations.CCGenClass;
import org.eclnt.jsfserver.elements.impl.FIXGRIDItem;
import org.eclnt.jsfserver.elements.impl.FIXGRIDListBinding;
import org.eclnt.workplace.IWorkpageDispatcher;

@CCGenClass (expressionBase="#{d.DemoGridDynHeightSizingImage}")

public class DemoGridDynHeightSizingImage
    extends WorkpageDispatchedPageBean 
    implements Serializable
{
    // ------------------------------------------------------------------------
    // inner classes
    // ------------------------------------------------------------------------

    public class GridItem extends FIXGRIDItem implements java.io.Serializable
    {
        String i_image;
        public String getImage() { return i_image; }
        public void setImage(String value) { this.i_image = value; }
        public void onRowSelect()
        {
        }
        public void onRowExecute()
        {
        }
    }


    // ------------------------------------------------------------------------
    // members
    // ------------------------------------------------------------------------

    public static String[] s_images = new String[]
    {
        "/images.iconssvg.cancel.#606060.16x16.ccsvg",
        "/images.iconssvg.cancel.#606060.32x32.ccsvg",
        "/images.iconssvg.cancel.#606060.48x48.ccsvg",
        "/images.iconssvg.cancel.#606060.64x64.ccsvg",
        "/images/pictures/avatar0.jpg",
        "/images/pictures/ph0.jpg",
        "/images/pictures/photo1.gif",
    };
    FIXGRIDListBinding<GridItem> m_grid = new FIXGRIDListBinding<GridItem>();

    // ------------------------------------------------------------------------
    // constructors & initialization
    // ------------------------------------------------------------------------

    public DemoGridDynHeightSizingImage(IWorkpageDispatcher workpageDispatcher)
    {
        super(workpageDispatcher);
        for (int i=0; i<100; i++)
        {
            GridItem gi = new GridItem();
            int imageIndex =i % s_images.length;
            gi.setImage(s_images[imageIndex]);
            m_grid.getItems().add(gi);
        }
    }

    public String getPageName() { return "/workplace/demogriddynheightsizingimage.xml"; }
    public String getRootExpressionUsedInPage() { return "#{d.DemoGridDynHeightSizingImage}"; }

    // ------------------------------------------------------------------------
    // public usage
    // ------------------------------------------------------------------------

    public FIXGRIDListBinding<GridItem> getGrid() { return m_grid; }

    // ------------------------------------------------------------------------
    // private usage
    // ------------------------------------------------------------------------
}
