The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • Containers
  • Scale Pane
  • Keep center during scaling

If setting SCALEPANE-WITHSCROLLPOSITIONUPDATE to "true" then the SCALEPANE will search for the SCROLLPANE that it is embedded in - and will update the scroll position so that the center of the current view is kept table.

Using this attribute requires the SCALEPANE to be placed inside a SCROLLPANE!

This is the Java code:

package workplace;

import java.io.Serializable;

import org.eclnt.editor.annotations.CCGenClass;
import org.eclnt.workplace.IWorkpageDispatcher;

@CCGenClass (expressionBase="#{d.DemoScalePosition}")

public class DemoScalePosition
    extends PageBean 
    implements Serializable
{
    // ------------------------------------------------------------------------
    // members
    // ------------------------------------------------------------------------

    int m_sliderScale = 100;
    float m_scale = 1f;
    
    // ------------------------------------------------------------------------
    // constructors & initialization
    // ------------------------------------------------------------------------

    public DemoScalePosition(IWorkpageDispatcher workpageDispatcher)
    {
        super(workpageDispatcher);        
    }

    public String getPageName() { return "/workplace/demoscaleposition.jsp"; }
    public String getRootExpressionUsedInPage() { return "#{d.DemoScalePosition}"; }

    // ------------------------------------------------------------------------
    // public usage
    // ------------------------------------------------------------------------

    public void onScalePaneAction(javax.faces.event.ActionEvent event) 
    {
        m_sliderScale = Math.round(m_scale * 100);
        m_sliderScale = (m_sliderScale * 10) / 10;
    }

    public void onSliderAction(javax.faces.event.ActionEvent event) 
    {
        m_scale = (float)m_sliderScale / 100f;
    }

    public int getSliderScale() { return m_sliderScale; }
    public void setSliderScale(int value) { this.m_sliderScale = value; }

    public float getScale() { return m_scale; }
    public void setScale(float value) { this.m_scale = value; }
}
