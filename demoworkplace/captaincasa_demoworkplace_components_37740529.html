The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • Components
  • Straight Controls
  • Invokers
  • Button - Image Sizing

When defining an image with a BUTTON / ICON / ICONSEPTEXT component then you can explicitly define an image sizing by using the attributes IMAGEHEIGHT and IMAGEWIDTH.

This is the XML layout definition:
