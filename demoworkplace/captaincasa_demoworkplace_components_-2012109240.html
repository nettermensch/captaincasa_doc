The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • Components
  • Straight Controls
  • Field Control
  • Date Input
  • Local Date/Time

As counter part of any date component you may use

(1.) a java.util.Date value
(2.) a Long/long value (same as java.util.Date, but representing the milliseconds time

or

(3.) a LocalDate or LocalTime value

LocalDate and LocalTime were introduced with Java 1.8 and are a representation of a date/time without reference to a time zone.

Define time zone "LOCAL" if using LocalDate/LocalTime!

This is the Java code:

package workplace;

import java.io.Serializable;
import java.time.LocalDate;
import java.time.LocalDateTime;
import java.time.LocalTime;

import org.eclnt.editor.annotations.CCGenClass;
import org.eclnt.workplace.IWorkpageDispatcher;

@CCGenClass (expressionBase="#{d.DemoLocalDate}")

public class DemoLocalDate
    extends PageBean 
    implements Serializable
{
    // ------------------------------------------------------------------------
    // inner classes
    // ------------------------------------------------------------------------

    // ------------------------------------------------------------------------
    // members
    // ------------------------------------------------------------------------

    LocalDate m_datum = LocalDate.now();
    LocalTime m_time = LocalTime.now();
    LocalDateTime m_dateTime = LocalDateTime.now();
    String m_protocol;

    // ------------------------------------------------------------------------
    // constructors & initialization
    // ------------------------------------------------------------------------

    public DemoLocalDate(IWorkpageDispatcher workpageDispatcher)
    {
        super(workpageDispatcher);        
    }

    public String getPageName() { return "/workplace/demolocaldate.jsp"; }
    public String getRootExpressionUsedInPage() { return "#{d.DemoLocalDate}"; }

    // ------------------------------------------------------------------------
    // public usage
    // ------------------------------------------------------------------------

    public void setTime(LocalTime time) { m_time = time; }
    public LocalTime getTime() { return m_time; }

    public LocalDate getDatum() { return m_datum; }
    public void setDatum(LocalDate datum) { m_datum = datum; }

    public LocalDateTime getDateTime() { return m_dateTime; }
    public void setDateTime(LocalDateTime value) { m_dateTime = value; }
    
    public String getProtocol() { return m_protocol; }
    public void setProtocol(String value) { this.m_protocol = value; }

    public void onShowContentAction(javax.faces.event.ActionEvent event) 
    {
        StringBuffer sb = new StringBuffer();
        sb.append("LocalDate: " + m_datum.toString());
        sb.append("\nLocalTime: " + m_time.toString());
        sb.append("\nLocalDateTime: " + m_dateTime.toString());
        m_protocol = sb.toString();
    }

    // ------------------------------------------------------------------------
    // private usage
    // ------------------------------------------------------------------------
}
