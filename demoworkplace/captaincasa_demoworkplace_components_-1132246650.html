The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • Components
  • Straight Controls
  • Images
  • SVG images

SVG images can be used just as normal images.

The default size of the SVG image is derived out of its SVG element:
"<svg ... width="..." height=".." ...>"

If no height is available then the size is set to 16x16.

Of course you can explicitly set the width and height. Unlike with e.g. PNG or JPG images, SVG images always keept their ratio.

package workplace;

import org.eclnt.editor.annotations.CCGenClass;

@CCGenClass (expressionBase="#{d.DemoSVGImages}")

}
