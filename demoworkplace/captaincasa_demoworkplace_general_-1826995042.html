The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • General
  • Client Integration
  • Client Monitoring Info (Java Client only)

The CLIENTMONITOR component sends basic monitoring information from the client to the server.

You may include this information inside your server side monitoring in order to compare server side performance figures with client side performance.

The information that is sent from the client side is always related to the previous roundtrip.

This is the Java code:

package workplace;

import java.io.Serializable;
import java.util.Map;

import org.eclnt.editor.annotations.CCGenClass;
import org.eclnt.util.valuemgmt.ValueManager;
import org.eclnt.workplace.IWorkpageDispatcher;
import org.eclnt.workplace.WorkpageDispatchedPageBean;

import javax.faces.event.ActionEvent;

@CCGenClass (expressionBase="#{d.DemoClientMonitor}")

public class DemoClientMonitor
    extends PageBean 
    implements Serializable
{
    public void onRoundtripAction(ActionEvent event) 
    {}

    String m_monitorValue;
    public String getMonitorValue() { return m_monitorValue; }
    public void setMonitorValue(String value) 
    { 
        m_monitorValue = value;
        Map<String,String> map = ValueManager.decodeComplexValue(value);
        m_protocol += "\n\nLast request: ";
        m_protocol += " roundtrip " + map.get("roundtrip") + " ms";
        m_protocol += "\nUI update " + map.get("uiupdate") + " ms";
        m_protocol += "\nFull information: " + value;
        // use ValueManager.decodeComplexValue(value) for transferring the 
        // string into a map
    }
    
    String m_protocol = "Protocol:";
    public String getProtocol() { return m_protocol; }
    

    // ------------------------------------------------------------------------
    // inner classes
    // ------------------------------------------------------------------------


    /* Listener to the user of the page bean. */
    public interface IListener
    {
    }

    // ------------------------------------------------------------------------
    // members
    // ------------------------------------------------------------------------

    private IListener m_listener;

    // ------------------------------------------------------------------------
    // constructors & initialization
    // ------------------------------------------------------------------------

    public DemoClientMonitor(IWorkpageDispatcher workpageDispatcher)
    {
        super(workpageDispatcher);        
    }

    public String getPageName() { return "/workplace/democlientmonitor.jsp"; }
    public String getRootExpressionUsedInPage() { return "#{d.DemoClientMonitor}"; }

    // ------------------------------------------------------------------------
    // public usage
    // ------------------------------------------------------------------------

    /* Initialization of the bean. Add any parameter that is required within your scenario. */
    public void prepare(IListener listener)
    {
        m_listener = listener;
    }

    // ------------------------------------------------------------------------
    // private usage
    // ------------------------------------------------------------------------
}
