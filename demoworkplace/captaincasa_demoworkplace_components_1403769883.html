The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • Components
  • Animation & Graphics & Sound
  • Video
  • Videos with subtitles ("tracks")

Below the VIDEO component you may place one or several TRACK elements - e.g. for adding sub-titles to the video.

This is the XML layout definition:

<?xml version="1.0" encoding="utf-8"?>
<t:layoutdefinition xmlns:t="http://www.CaptainCasa.com/controllibrary/t" xmlns:tpbc="http://www.CaptainCasa.com/controllibrary/tpbc" >
<t:beanprocessing id="g_1" beanbinding="#{d.DemoVideoWithTrack}" />
<t:rowdemobodypane id="g_2" objectbinding="#{d.DemoVideoWithTrack}">
<t:row id="g_3" >
<t:video id="g_4" height="100%" videosrc="/videos/sample.webm" width="100%" >
<t:track id="g_5" default="true" kind="subtitles" label="Subtitles for video" src="/videos/sample.vtt" srclang="en" />
</t:video>
</t:row>
</t:rowdemobodypane>
<t:pageaddons id="g_pa"/>
</t:layoutdefinition>

This is the Java code:

package workplace;

import java.io.Serializable;

import org.eclnt.editor.annotations.CCGenClass;
import org.eclnt.workplace.IWorkpageDispatcher;

@CCGenClass (expressionBase="#{d.DemoVideoWithTrack}")

public class DemoVideoWithTrack
    extends PageBean 
    implements Serializable
{
    // ------------------------------------------------------------------------
    // constructors & initialization
    // ------------------------------------------------------------------------

    public DemoVideoWithTrack(IWorkpageDispatcher workpageDispatcher)
    {
        super(workpageDispatcher);        
    }

    public String getPageName() { return "/workplace/demovideowithtrack.jsp"; }
    public String getRootExpressionUsedInPage() { return "#{d.DemoVideoWithTrack}"; }

}
