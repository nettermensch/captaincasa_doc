The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • General
  • Client Integration
  • Button for directly copying content to clipboard

The component BUTTONCOPYTOCLIPBOARD represents a normal button that copies a text to the client side clipboard. The text is the one that is referenced by the component's attribute CLIPBOARDCONTENT.

Please note: there is a more generic component CLIENTCLIPBOARDSET that is not bound to a specific component rendering. The component BUTTONCOPYTOCLIPBOARD is a "short cut" for the typical case that a text that is available as expression anyway (e.g. due to being shown as LABEL) also can be copied into the clipboard.

package workplace;

import java.io.Serializable;
import java.time.LocalDateTime;

import org.eclnt.editor.annotations.CCGenClass;
import org.eclnt.workplace.IWorkpageDispatcher;

@CCGenClass (expressionBase="#{d.DemoButtonCopyToClipboard}")

public class DemoButtonCopyToClipboard
    extends PageBean 
    implements Serializable
{
    // ------------------------------------------------------------------------
    // inner classes
    // ------------------------------------------------------------------------

    /* Listener to the user of the page bean. */
    public interface IListener extends Serializable
    {
    }

    // ------------------------------------------------------------------------
    // members
    // ------------------------------------------------------------------------

    private IListener m_listener;

    // ------------------------------------------------------------------------
    // constructors & initialization
    // ------------------------------------------------------------------------

    public DemoButtonCopyToClipboard(IWorkpageDispatcher workpageDispatcher)
    {
        super(workpageDispatcher);        
    }

    public String getPageName() { return "/workplace/demobuttoncopytoclipboard.xml"; }
    public String getRootExpressionUsedInPage() { return "#{d.DemoButtonCopyToClipboard}"; }

    // ------------------------------------------------------------------------
    // public usage
    // ------------------------------------------------------------------------

    /* Initialization of the bean. Add any parameter that is required within your scenario. */
    public void prepare(IListener listener)
    {
        m_listener = listener;
    }

    public String getServerText() { return "This text comes from the server side. " + LocalDateTime.now().toString(); }

    // ------------------------------------------------------------------------
    // private usage
    // ------------------------------------------------------------------------
}
