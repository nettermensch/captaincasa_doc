The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • Grids & Trees
  • Grids and Trees
  • Special Issues
  • Re-selection of rows

This is something quite special about row selection: in the top grid a selection event is passed to the server side processing every time the user clicks onto a value - have a look onto the timestamp output within the statusbar message.

In the bottom grid, row selections are only sent to the server, when the user selects a line first time. Staying on the same line and re-selecting it will not cause any server side processing.

Please note: double clicks (onExecute() processing on server side) are always transferred.

This is the Java code:

package workplace;

import java.io.Serializable;
import java.util.Date;

import org.eclnt.editor.annotations.CCGenClass;
import org.eclnt.jsfserver.defaultscreens.Statusbar;
import org.eclnt.jsfserver.elements.impl.FIXGRIDItem;
import org.eclnt.jsfserver.elements.impl.FIXGRIDListBinding;
import org.eclnt.jsfserver.managedbean.IDispatcher;

@CCGenClass (expressionBase="#{d.DemoGridReselect}")

public class DemoGridReselect
    extends PageBean
    implements Serializable
{
    public class MyRow extends FIXGRIDItem implements java.io.Serializable
    {
        public MyRow(String value)
        {
            m_value = value;
        }
        protected String m_value;
        public String getValue() { return m_value; }
        public void setValue(String value) { m_value = value; }
        @Override
        public void onRowSelect()
        {
            Statusbar.outputSuccess("SELECTION of row: " + m_value + " / Timestamp: " + (new Date()).getTime());
        }
        
    }
    
    public DemoGridReselect(IDispatcher dispatcher)
    {
        super(dispatcher);
        for (int i=0; i<25; i++)
        {
            m_rows1.getItems().add(new MyRow("Value " + i));
            m_rows2.getItems().add(new MyRow("Value " + i));
        }
    }
    
    protected FIXGRIDListBinding<MyRow> m_rows1 = new FIXGRIDListBinding<MyRow>();
    protected FIXGRIDListBinding<MyRow> m_rows2 = new FIXGRIDListBinding<MyRow>();
    
    public FIXGRIDListBinding<MyRow> getRows1() { return m_rows1; }
    public FIXGRIDListBinding<MyRow> getRows2() { return m_rows2; }

}
