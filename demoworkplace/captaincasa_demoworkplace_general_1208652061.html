The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • General
  • Workplace Management
  • Statusbar Output

By default you use the static Statusbar methods for displaying a message within the central status bar.

For workplace content scenarios there is an optional, a bit more sophisticated framework. Background: workpage content may be opened within a modeless dialog box - the central statusbar may be hidden as result.

The frameworks consists of:

(.) When a page is openes as modeless popup then an additional header and an additional statusbar is drawn. The statusbar that is now part of the modeless popup is the one were messagesa are going to - i.e. the central status bar is not used, as long as the page it opened within the popup.

(.) In the coding on the server side you do not use the static methods, but you use the write*-methods to write your output statements.

This is the Java code:

package workplace;

import java.io.Serializable;

import javax.faces.event.ActionEvent;

import org.eclnt.editor.annotations.CCGenClass;
import org.eclnt.jsfserver.managedbean.IDispatcher;

@CCGenClass (expressionBase="#{d.DemoWpStatusbar}")

public class DemoWpStatusbar 
    extends PageBean // implicitly: extends WorkpageDispatchedBean
    implements Serializable
{

    // ------------------------------------------------------------------------
    // inner classes
    // ------------------------------------------------------------------------
    
    // ------------------------------------------------------------------------
    // members
    // ------------------------------------------------------------------------
    
    // ------------------------------------------------------------------------
    // constructors
    // ------------------------------------------------------------------------
    
    public DemoWpStatusbar(IDispatcher dispatcher)
    {
        super(dispatcher);
    }
    
    // ------------------------------------------------------------------------
    // public usage
    // ------------------------------------------------------------------------
    
    public void onOutputAlert(ActionEvent ae)
    {
        getOwningDispatcher().getStatusbar().writeAlert("This message is output as ALERT message.","Alert Message");
    }
    
    public void onOutputError(ActionEvent ae)
    {
        getOwningDispatcher().getStatusbar().writeError("This message is output as ERROR message.");
    }
    
    public void onOutputSuccess(ActionEvent ae)
    {
        getOwningDispatcher().getStatusbar().writeSuccess("This message is output as SUCCESS message.");
    }
    
    public void onOutputMessage(ActionEvent ae)
    {
        getOwningDispatcher().getStatusbar().writeMessage("This message is output as normal message 1.");
        getOwningDispatcher().getStatusbar().writeMessage("This message is output as normal message 2.");
        getOwningDispatcher().getStatusbar().writeMessage("This message is output as normal message 3.");
        getOwningDispatcher().getStatusbar().writeMessage("This message is output as normal message 4.");
        getOwningDispatcher().getStatusbar().writeMessage("This message is output as normal message 5.");
        getOwningDispatcher().getStatusbar().writeMessage("This message is output as normal message.");
    }
    
    public void onOutputWarning(ActionEvent ae)
    {
        getOwningDispatcher().getStatusbar().writeWarning("This message is output as WARNING message.");
    }
    
    // ------------------------------------------------------------------------
    // private usage
    // ------------------------------------------------------------------------
    
}
