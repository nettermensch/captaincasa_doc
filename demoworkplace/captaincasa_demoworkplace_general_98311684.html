The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • General
  • Workplace Management
  • Favorites List

It is very simple to create favorite areas - in which the user can drag&drop functions w.g. coming from the workplace's function tree management.

Just create an instance of class "WorkplaceFavorites" and bind it to the ROWWORKPAGEFAVORITES component.

There is one special favorite area, that is part of the workplace management already: the history. Every time something in the workplace is started using WorkpageStartInfo-instances, a corresponding history item is added.

The history is persisted in the local stream store.

This is the Java code:

package workplace;

import java.io.Serializable;

import org.eclnt.editor.annotations.CCGenClass;
import org.eclnt.jsfserver.managedbean.DefaultDispatchedBean;
import org.eclnt.jsfserver.managedbean.IDispatcher;
import org.eclnt.workplace.WorkpageStartInfo;
import org.eclnt.workplace.WorkplaceFavorites;

@CCGenClass (expressionBase="#{d.DemoWPFavorites}")

public class DemoWPFavorites 
    extends PageBean
    implements Serializable
{
    WorkplaceFavorites m_favs;
    
    public DemoWPFavorites(IDispatcher dispatcher)
    {
        super(dispatcher);
        m_favs = new WorkplaceFavorites(dispatcher,"demo");
        if (m_favs.getFaovriteInfos().size() == 0)
        {
            WorkpageStartInfo fi = new WorkpageStartInfo();
            fi.setImage("/images/icons/person.png");
            fi.setText("Hallo");
            fi.setJspPage("/workplace/demohelloworld.jsp");
            m_favs.addFavorite(fi);
        }
    }
    
    public WorkplaceFavorites getFavs() { return m_favs; }
    
    public WorkplaceFavorites getHistory() { return getWorkpageContainer().getHistory(); }
}
