The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • General
  • Tool Tips / User Hints
  • User hint - more dynamic...

This example shows a usage variant of the USERHINT attribute.

1. The USERHINT attributes are dynamically set.
2. When pressing the Check-button then all user hints are shown. Once the user steps through the fields they disappear.

There is a corresponding USERHINTTRIGGER attribute. If triggered then the corresponding user hint is shown. In the example both FIELDs refer to the same trigger property.

This is the Java code:

package workplace;

import java.io.Serializable;

import javax.faces.event.ActionEvent;

import org.eclnt.editor.annotations.CCGenClass;
import org.eclnt.jsfserver.elements.util.Trigger;
import org.eclnt.workplace.IWorkpageDispatcher;

@CCGenClass (expressionBase="#{d.DemoUserHintTrigger}")

public class DemoUserHintTrigger
    extends PageBean 
    implements Serializable
{
    public void onLastNameAction(ActionEvent event) { check(); }
    public void onFirstNameAction(ActionEvent event) { check(); }

    Trigger m_userHintTrigger = new Trigger();

    String m_firstNameUserHint;
    String m_lastNameUserHint;
    String m_lastNameBgpaint;
    String m_firstNameBgpaint;

    protected String m_firstName = "Test";
    protected String m_lastName = "Test";
        
    public DemoUserHintTrigger(IWorkpageDispatcher dispatcher)
    {
        super(dispatcher);
        onCheckAction(null);
    }
    
    public Trigger getUserHintTrigger() { return m_userHintTrigger; }
    
    public String getFirstName() { return m_firstName; }
    public void setFirstName(String value) { m_firstName = value; }
    
    public String getLastName() { return m_lastName; }
    public void setLastName(String value) { m_lastName = value; }

    public String getFirstNameUserHint() { return m_firstNameUserHint; }
    public String getLastNameUserHint() { return m_lastNameUserHint; }
    public String getLastNameBgpaint() { return m_lastNameBgpaint; }
    public String getFirstNameBgpaint() { return m_firstNameBgpaint; }

    public void onCheckAction(ActionEvent event) 
    {
        boolean success = check();
        if (success == false)
        {
            m_userHintTrigger.trigger();
        }
    }

    private boolean check()
    {
        boolean result = true;
        m_lastNameBgpaint = null;
        m_lastNameUserHint = null;
        m_firstNameBgpaint = null;
        m_firstNameUserHint = null;
        if (m_firstName == null || m_firstName.length()<10)
        {
            m_firstNameUserHint = "Please input at least 10 characters.";
            m_firstNameBgpaint = "error()";
            result = false;
        }
        if (m_lastName == null || m_lastName.length()<20)
        {
            m_lastNameUserHint = "Please input at least 20 characters.";
            m_lastNameBgpaint = "error()";
            result = false;
        }
        return result;
    }
}
