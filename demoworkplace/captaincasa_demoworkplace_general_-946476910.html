The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • General
  • Client Integration
  • Pooled browser

The POOLEDBROWSER from a first perspective looks like a normal BROWSER component: there is an area in which a browser instance (iframe) is started.

...but: this browser instance is not managed as child of the corresponding area! It is managed on a top level within the client side component tree.

Reason: when switching from one workplace content page to the next then the internal browser instance is not removed from the component tree and then re-created when being shown again, but it stays stable at the top level so that the navigation state does not get lost.

This is the XML layout definition:

<?xml version="1.0" encoding="utf-8"?>
<t:layoutdefinition xmlns:t="http://www.CaptainCasa.com/controllibrary/t" xmlns:tpbc="http://www.CaptainCasa.com/controllibrary/tpbc" >
<t:rowdemobodypane id="g_1" objectbinding="#{d.DemoPooledBrowser}" padding="0" >
<t:row id="g_2" >
<t:field id="g_3" enabled="false" labeltext="URL that was passed by bean" text="#{d.DemoPooledBrowser.url2Show}" width="100%" />
</t:row>
<t:rowdistance id="g_4" height="10" />
<t:rowline id="g_5" />
<t:rowdistance id="g_6" height="10" />
<t:row id="g_7" >
<t:pooledbrowser id="g_8" height="100%" poolid="#{d.DemoPooledBrowser.poolId}" url="#{d.DemoPooledBrowser.url2Show}" width="100%" />
</t:row>
</t:rowdemobodypane>
<t:pageaddons id="g_pa"/>
</t:layoutdefinition>

This is the Java code:

package workplace;

import java.io.Serializable;

import org.eclnt.editor.annotations.CCGenClass;
import org.eclnt.jsfserver.elements.util.POOLEDBROWSERUtil;
import org.eclnt.util.valuemgmt.UniqueIdCreator;
import org.eclnt.workplace.IWorkpageDispatcher;
import org.eclnt.workplace.WorkpageDefaultLifecycleListener;

@CCGenClass (expressionBase="#{d.DemoPooledBrowser}")

public class DemoPooledBrowser
    extends PageBean 
    implements Serializable
{
    // ------------------------------------------------------------------------
    // members
    // ------------------------------------------------------------------------

    // ensure that the URL is "new" for the pooled browser; the pooled browser
    // only updates its conent if there is a change in the URL that is sent
    final static String URL = "/html/firstName.html"; 
    String m_url2Show = URL+"?now="+UniqueIdCreator.createUUID();
    String m_poolId = "HARRY";

    
    // ------------------------------------------------------------------------
    // constructors & initialization
    // ------------------------------------------------------------------------

    public DemoPooledBrowser(IWorkpageDispatcher workpageDispatcher)
    {
        super(workpageDispatcher);        
        getWorkpage().addLifecycleListener(new WorkpageDefaultLifecycleListener() 
        {
            @Override
            public void reactOnDestroyed()
            {
                super.reactOnDestroyed();
                // it's important to explicitly remove the pool instance on client side
                // when the browser instance is not required any more
                POOLEDBROWSERUtil.removePooledBrowserInClient(m_poolId);
            }
        });
    }

    public String getPageName() { return "/workplace/demopooledbrowser.jsp"; }
    public String getRootExpressionUsedInPage() { return "#{d.DemoPooledBrowser}"; }

    // ------------------------------------------------------------------------
    // public usage
    // ------------------------------------------------------------------------

    public String getUrl2Show() { return m_url2Show; }
    public void setUrl2Show(String value) { this.m_url2Show = value; }
    public String getPoolId() { return m_poolId; }

    // ------------------------------------------------------------------------
    // private usage
    // ------------------------------------------------------------------------
}
