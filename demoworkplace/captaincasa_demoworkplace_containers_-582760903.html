The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • Containers
  • Scroll Pane - Extended Functions
  • Scroll to end of text

In this example a SCROLLANCHOR is set into the SCROLLPANE - directly below the TEXTPANE component, showing the protocol.

Every time a new message is added to the protocol, the SCROLLANCHOR is triggered.

This is the XML layout definition:

<?xml version="1.0" encoding="utf-8"?>
<t:layoutdefinition xmlns:t="http://www.CaptainCasa.com/controllibrary/t" xmlns:tpbc="http://www.CaptainCasa.com/controllibrary/tpbc" >
<t:rowdemobodypane id="g_1" objectbinding="#{d.DemoMessageProtocol}" rowdistance="#{ccstylevalue['@defaultrowdistance@']}" >
<t:row id="g_2" >
<t:button id="g_3" actionListener="#{d.DemoMessageProtocol.onAddMessage}" doubleclickenabled="true" text="Add a message" />
</t:row>
<t:row id="g_4" >
<t:scrollpane id="g_5" border="#808080" height="100%" width="100%" >
<t:row id="g_6" >
<t:textpane id="g_7" text="#{d.DemoMessageProtocol.message}" width="100%" />
</t:row>
<t:row id="g_8" >
<t:scrollanchor id="g_9" scrolltrigger="#{d.DemoMessageProtocol.scrollTrigger}" />
</t:row>
</t:scrollpane>
</t:row>
</t:rowdemobodypane>
<t:pageaddons id="g_pa"/>
</t:layoutdefinition>

This is the Java code:

package workplace;

import java.io.Serializable;

import javax.faces.event.ActionEvent;

import org.eclnt.editor.annotations.CCGenClass;
import org.eclnt.jsfserver.elements.util.Trigger;
import org.eclnt.workplace.IWorkpageDispatcher;
import org.eclnt.workplace.WorkpageDispatchedBean;

@CCGenClass (expressionBase="#{d.DemoMessageProtocol}")

public class DemoMessageProtocol
    extends PageBean 
    implements Serializable
{
    Trigger m_scrollTrigger = new Trigger();
    String m_message = 
        "Message Protocol\n\n" +
        "\n" +
        "Fill the protocol by clicking the button.\n" +
        "The text of the protocol will always scroll to the\n" +
        "end, so that the most up to date information is available.\n\n\n";
    int m_counter = 1;
    
    public DemoMessageProtocol(IWorkpageDispatcher workpageDispatcher)
    {
        super(workpageDispatcher);        
    }

    public Trigger getScrollTrigger() { return m_scrollTrigger; }

    public String getMessage() { return m_message; }
    public void setMessage(String value) { this.m_message = value; }

    public void onAddMessage(ActionEvent event) 
    {
        m_message += "Button pressed " + m_counter + "\n";
        m_counter++;
        m_scrollTrigger.trigger();
    }
    
}
