The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • General
  • Client Integration
  • Automated File Transfer (Java Client only)

This page includes a CLIENTFILETRANSFER component.

Create the directory C:\temp\transfer and the directory c:\temp\transferbackup within your client file system.

Then copy files into the c:\temp\transfer directory.

The files will be picked, uploaded and removed afterwards.


The component observes a certain directory and checks the directory for new files regularly. New files are transferred to the server side. After successful transfer the files are removed from the client file system. - It is possible to define an additional backup directory, in which all transferred files are copied before being removed.

This is the XML layout definition:

<?xml version="1.0" encoding="utf-8"?>
<t:layoutdefinition xmlns:t="http://www.CaptainCasa.com/controllibrary/t" xmlns:tpbc="http://www.CaptainCasa.com/controllibrary/tpbc" >
<t:beanprocessing id="g_1" >
<t:clientfiletransfer id="g_2" actionListener="#{d.DemoClientFileTransfer.onClientFileTransferAction}" asynchronousuploadurl="#{d.DemoClientFileTransfer.uploadURL}" createdirectories="false" darkupload="false" scanduration="5000" separatebackupdirectories="true" transferbackupdirectory="c:\temp\transferbackup" transferdirectory="c:\temp\transfer" />
</t:beanprocessing>
<t:rowdemobodypane id="g_3" objectbinding="#{d.DemoClientFileTransfer}" rowdistance="#{ccstylevalue['@defaultrowdistance@']}" >
<t:row id="g_4" >

</t:row>
</t:rowdemobodypane>
<t:pageaddons id="g_pa"/>
</t:layoutdefinition>

This is the Java code:

package workplace;

import java.io.Serializable;
import org.eclnt.editor.annotations.CCGenClass;
import org.eclnt.jsfserver.bufferedcontent.DefaultUploadContent;
import org.eclnt.jsfserver.bufferedcontent.UploadContentMgr;
import org.eclnt.jsfserver.defaultscreens.Statusbar;
import org.eclnt.workplace.IWorkpageDispatcher;
import org.eclnt.workplace.WorkpageDefaultLifecycleListener;
import org.eclnt.workplace.WorkpageDispatchedBean;

import javax.faces.event.ActionEvent;

@CCGenClass (expressionBase="#{d.DemoClientFileTransfer}")

public class DemoClientFileTransfer
    extends PageBean 
    implements Serializable
{
    class UploadContent extends DefaultUploadContent
    {
        public void beginPassing()
        {
            m_protocol += "File Upload:\n"; 
        }
        public void passClientFile(String fileName, byte[] content)
        {
            m_protocol += fileName + ": " + content.length + "\n"; 
        }
        public void endPassing()
        {
            m_protocol += "\n"; 
        }
    }
    
    String m_protocol = "Copy any files into your client directory c:\\temp\\transfer - they will be automatically transferred to the server.\n\n";
    UploadContent m_uploadContent;
    
    public DemoClientFileTransfer(IWorkpageDispatcher workpageDispatcher)
    {
        super(workpageDispatcher);        
        m_uploadContent = new UploadContent();
        UploadContentMgr.add(m_uploadContent);
        getWorkpage().addLifecycleListener(new WorkpageDefaultLifecycleListener()
        {
            @Override
            public void reactOnDestroyed()
            {
                UploadContentMgr.remove(m_uploadContent);
            }
        });
    }
    
    public String getUploadURL() { return m_uploadContent.getURL(); }

    public String getProtocol() { return m_protocol; }
    public void setProtocol(String value) { this.m_protocol = value; }

    public void onClientFileTransferAction(ActionEvent event) 
    {
        Statusbar.outputSuccess("Client file transfer was finished. " + System.currentTimeMillis());
    }

}
