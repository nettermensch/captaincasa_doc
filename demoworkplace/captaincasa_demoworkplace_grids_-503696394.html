The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • Grids & Trees
  • Grids and Trees
  • Tree
  • Explicit Status Image

By default there are three icons that represent the status of each tree node (closed, opened, end-node).

You can override these images either centrally by using client parameters "imagetreenode*". These central definitions are used by default for all tree nodes.

Or you can individually define the status images for the individual TREENODE component.

The individual definition can be done in two ways:

1. You may define the three images using the FIXSTATUSIMAGE* attributes. In this case the definition you do is used for all tree nodes of the tree.
2. You may define a specific image using attribute STATUSIMAGE, which points to a server side property, providing the name of the image. In this case you can individually assign status image for each individual node instance.

Please note: a tree node provides two images: the status image is the one indicating if the tree node is opened or closed or if the tree node is an end node.
Independent from this status image there is the optional normal image, which is assigned by calling the setImage()-method in the server side tree node representation. This image is arranged just in front of the text of the node.

This is the Java code:

package workplace;

import java.io.Serializable;
import java.util.Random;

import org.eclnt.editor.annotations.CCGenClass;
import org.eclnt.jsfserver.elements.impl.FIXGRIDTreeBinding;
import org.eclnt.jsfserver.elements.impl.FIXGRIDTreeItem;
import org.eclnt.workplace.IWorkpageDispatcher;
import org.eclnt.workplace.WorkpageDispatchedBean;

@CCGenClass (expressionBase="#{d.DemoTreeStatusImage}")

public class DemoTreeStatusImage
    extends PageBean
    implements Serializable
{
    static Random RANDOM = new Random();
    
    public class TreeNode extends FIXGRIDTreeItem implements java.io.Serializable
    {
        // i_statusImage used for dynamic example only!
        String i_statusImage;
        public String getStatusImage() { return i_statusImage; }
        public TreeNode(FIXGRIDTreeItem parentNode)
        {
            super(parentNode);
            int r = RANDOM.nextInt() % 3;
            if (r == 0) i_statusImage = "/images/icons/accept.png";
            else if (r == 1) i_statusImage = "/images/icons/disk.png";
            else i_statusImage = "/images/icons/car.png";
        }
    }

    protected FIXGRIDTreeBinding<TreeNode> m_tree = new FIXGRIDTreeBinding<TreeNode>();
    public DemoTreeStatusImage(IWorkpageDispatcher workpageDispatcher)
    {
        super(workpageDispatcher);    
        for (int i=0; i<5; i++)
        {
            TreeNode top = new TreeNode(m_tree.getRootNode());
            top.setText("Top " + i);
            top.setStatus(TreeNode.STATUS_OPENED);
            for (int j=0; j<5; j++)
            {
                TreeNode sub = new TreeNode(top);
                sub.setText("Sub " + i + "/" + j);
                sub.setStatus(TreeNode.STATUS_ENDNODE);
            }
        }
    }

    public FIXGRIDTreeBinding<TreeNode> getTree() { return m_tree; }

}
