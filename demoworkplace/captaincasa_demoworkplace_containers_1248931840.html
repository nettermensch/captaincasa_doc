The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • Containers
  • Runtime update of size
  • Dynamic Pane Sizing

A component provides a set of attributes, e.g. a pane component provides a height attribute.

An attribute can either be defined in a fix way (height="100") or it can be bound to a property of a managed bean. This example demonstrates how the height of a pane is controlled by a server side property.

This is the Java code:

package workplace;

import javax.faces.event.ActionEvent;

import org.eclnt.workplace.IWorkpageDispatcher;

public class DemoDynamicPaneSizing extends PageBean
{
    // ------------------------------------------------------------------------
    // inner classes
    // ------------------------------------------------------------------------
    
    // ------------------------------------------------------------------------
    // members
    // ------------------------------------------------------------------------
    
    int m_height = 100;
    
    // ------------------------------------------------------------------------
    // constructors
    // ------------------------------------------------------------------------
    
    public DemoDynamicPaneSizing(IWorkpageDispatcher dispatcher)
    {
        super(dispatcher);
    }

    // ------------------------------------------------------------------------
    // public usage
    // ------------------------------------------------------------------------
        
    public int getHeight() { return m_height; }    
    
    public void onIncreaseHeight(ActionEvent ae)
    {
        m_height += 50;
    }
    
    public void onDecreaseHeight(ActionEvent ae)
    {
        m_height -= 50;
    }
    
    public void onHideShow(ActionEvent ae)
    {
        if (m_height != 0)
            m_height = 0;
        else
            m_height = 100;
    }
    
    // ------------------------------------------------------------------------
    // private usage
    // ------------------------------------------------------------------------
    

}
