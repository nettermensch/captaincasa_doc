The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • Components
  • Straight Controls
  • Invokers
  • Button image definition

BUTTON and ICON provide some attributes to explicitly define the image that may be shown. These attributes are:

(.) IMAGEHEIGHT/WIDTH for sizing the image (please always use both!)
(.) IMAGEBACKGROUND for coloring the background of the image area
(.) IMAGEBORDERWIDTH, IMAGEBORDERCOLOR for defining the border
(.) IMAGEROUNDING for defining the radius that is applied to background and border

As result you are able to customize the image part, e.g. to have a rounded background - and to not influence e.g. the background of the whole button - including the button text.
