The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • Components
  • Straight Controls
  • Images
  • Image as background

This demo shows how to arrange a picture-image as background.

(.) Use OVERLAYAREA and OVERLAYREAITEM to arrange two z-layers
(.) paint the IMAGE on the background layer
(.) use IMAGE-KEEPRATIONFILLMODE to make the image cover the whole space with keeping its ratio
