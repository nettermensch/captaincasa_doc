The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • Components
  • Straight Controls
  • Field Control
  • Formatted Field
  • High precision number input

By default there is a FORMATMASK attribute which is used both for input and for displaying the value.

But you can separate by defining a value in attribute FORMATMASKDURINGINPUT. In this case the format mask is switched automatically when the component is focused - and switches back automatically when the focus is lost.

The definition of FORMATMASKDURINGINPUT only makes sense for FORMAT "float", "double", "bigdecimal". And it only makes sense a format mask in FORMATMASKDURINGINPUT which is more detailed than the one defined in FORMATMASK.

This is the Java code:

package workplace;

import java.io.Serializable;
import java.math.BigDecimal;

import org.eclnt.editor.annotations.CCGenClass;
import org.eclnt.jsfserver.elements.BaseActionEvent;
import org.eclnt.workplace.IWorkpageDispatcher;
import org.eclnt.workplace.WorkpageDispatchedPageBean;

@CCGenClass (expressionBase="#{d.DemoFloatFormatUpdate}")

public class DemoFloatFormatUpdate
    extends PageBean 
    implements Serializable
{
    // ------------------------------------------------------------------------
    // members
    // ------------------------------------------------------------------------

    BigDecimal m_value = new BigDecimal("100.999999");

    // ------------------------------------------------------------------------
    // constructors & initialization
    // ------------------------------------------------------------------------

    public DemoFloatFormatUpdate(IWorkpageDispatcher workpageDispatcher)
    {
        super(workpageDispatcher);        
    }

    public String getPageName() { return "/workplace/demofloatformatupdate.xml"; }
    public String getRootExpressionUsedInPage() { return "#{d.DemoFloatFormatUpdate}"; }

    // ------------------------------------------------------------------------
    // public usage
    // ------------------------------------------------------------------------

    public BigDecimal getValue() { return m_value; }
    public void setValue(BigDecimal value) { this.m_value = value; }
}
