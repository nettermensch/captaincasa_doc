The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • General
  • Arranging Content
  • Page Beans
  • Page Beans opening up content areas

This is an example for a page bean that internally opens up some area in which the using dialog can place own content:

The scale management is taken over by the page bean - and the content of the scale area is directly defined in the page.

For this purpose the inner component uses one or several PARENTEXIT components, each one defining an EXITID value. - The outside page bean can now place its content inside by using either the CONTENTPANE or by using the CONTENT component.

Use CONTENTPANE if it's some own area which you want to place. Use CONTENT if there are several items: here the items of the component are directly placed into the corresponding exit-area.

This is the Java code:

package workplace;

import java.io.Serializable;

import org.eclnt.editor.annotations.CCGenClass;
import org.eclnt.workplace.IWorkpageDispatcher;
import org.eclnt.workplace.WorkpageDispatchedPageBean;

@CCGenClass (expressionBase="#{d.DemoPBWithExitUser}")

public class DemoPBWithExitUser
    extends WorkpageDispatchedPageBean 
    implements Serializable
{
    // ------------------------------------------------------------------------
    // members
    // ------------------------------------------------------------------------

    DemoPBWithExit m_content = new DemoPBWithExit();
    String m_lastName = "Casa";
    String m_firstName = "Captain";

    // ------------------------------------------------------------------------
    // constructors & initialization
    // ------------------------------------------------------------------------

    public DemoPBWithExitUser(IWorkpageDispatcher workpageDispatcher)
    {
        super(workpageDispatcher);        
    }

    public String getPageName() { return "/workplace/demopbwithexituser.jsp"; }
    public String getRootExpressionUsedInPage() { return "#{d.DemoPBWithExitUser}"; }

    // ------------------------------------------------------------------------
    // public usage
    // ------------------------------------------------------------------------

    public void onAction(javax.faces.event.ActionEvent event) 
    {
        m_lastName += "?";
        m_firstName += "?";
    }

    public String getLastName() { return m_lastName; }
    public void setLastName(String value) { this.m_lastName = value; }

    public String getFirstName() { return m_firstName; }
    public void setFirstName(String value) { this.m_firstName = value; }

    public DemoPBWithExit getContent() { return m_content; }

    // ------------------------------------------------------------------------
    // private usage
    // ------------------------------------------------------------------------
}
