The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • Containers
  • Layering and absolute postioning
  • Layered Pane

The layered pane arranges PANE components in a vertical order. The first PANE below the component is drawn in the back, the next on top of the previous one.

All the PANEs cover the whole available area that is given by LAYEREDPANE. If no size is defined with LAYEREDPANE then the biggest contained PANE defines the size of the whole LAYEREDPANE.

PAY ATTENTION: the top pane (which is the last PANE in the sequence below LAYEREDPANE) is catching all the mouse events - so it does not make sense to arrange any input components within a background pane.

Typical usage: you display some information "in the back" and want to have some active components on top of it.

This is the Java code:

package workplace;

import java.io.Serializable;

import javax.faces.event.ActionEvent;

import org.eclnt.editor.annotations.CCGenClass;
import org.eclnt.workplace.IWorkpageDispatcher;

@CCGenClass (expressionBase="#{d.DemoLayeredPane}")

public class DemoLayeredPane 
    extends PageBean
    implements Serializable
{
    static Planet[] PLANET = new Planet[] 
    {
        new Planet("Mercury","This is a comment about Mercury. This is a comment about Mercury.","/images/pictures/photo0.gif"),
        new Planet("Venus","This is a comment about Venus. This is a comment about Venus.","/images/pictures/photo1.gif"),
        new Planet("Earth","This is a comment about the Earth. This is a comment about the Earth. This is a comment about the Earth.","/images/pictures/photo2.gif"),
    };
    
    public static class Planet implements java.io.Serializable
    {
        public Planet(String name, String comment, String image)
        {
            super();
            m_name = name;
            m_comment = comment;
            m_image = image;
        }

        protected String m_image;
        public String getImage() { return m_image; }
        public void setImage(String value) { m_image = value; }

        protected String m_comment;
        public String getComment() { return m_comment; }
        public void setComment(String value) { m_comment = value; }

        protected String m_name;
        public String getName() { return m_name; }
        public void setName(String value) { m_name = value; }
    }

    int m_selectedIndex = 0;
    
    public DemoLayeredPane(IWorkpageDispatcher dispatcher)
    {
        super(dispatcher);
    }

    public Planet getP() { return PLANET[m_selectedIndex]; }
    
    public void onNext(ActionEvent ae)
    {
        m_selectedIndex++;
        if (m_selectedIndex >= PLANET.length)
            m_selectedIndex = 0;
    }
}
