The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • Components
  • Reporting, PDF, SVG
  • Show SVG

The SVGVIEWER component renders SVG documents.
You can zoom in / out using the mouse wheel. When double clicking into the viewer then the graphics is right-sized as if it was newly loaded.

The document is passed via the attribute SVGVIEWER-SVG.

The user can use the mouse wheel for scrolling into the graphics and can move the graphics by clicking into it and moving the mouse.

Please pay attention: only plain SVG (without e.g. scripting) can be displayed.

This is the XML layout definition:

<?xml version="1.0" encoding="utf-8"?>

This is the Java code:

package workplace;

import java.io.Serializable;
import org.eclnt.editor.annotations.CCGenClass;
import org.eclnt.util.file.ClassloaderReader;
import org.eclnt.workplace.IWorkpageDispatcher;
import org.eclnt.workplace.WorkpageDispatchedBean;

import javax.faces.event.ActionEvent;

@CCGenClass (expressionBase="#{d.DemoSvg}")

public class DemoSvg
    extends WorkpageDispatchedPageBean 
    implements Serializable
{
    String m_svg;
    
    public DemoSvg(IWorkpageDispatcher workpageDispatcher)
    {
        super(workpageDispatcher);
        initRefComponent("svgviewer");
        loadSVG("workplace/demo1.svg");
    }
    
    public void setSvg(String value) { m_svg = value; }
    public String getSvg() { return m_svg; }
    
    public void onLoadDemo1(ActionEvent event) { loadSVG("workplace/demo1.svg"); }
    public void onClear(ActionEvent event) { m_svg = null; }

    private void loadSVG(String resource)
    {
        m_svg = (new ClassloaderReader(getClass().getClassLoader())).readUTF8File(resource,true);
    }
}
