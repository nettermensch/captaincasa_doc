The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • Components
  • Straight Controls
  • Images
  • Area selection (image cropping)

Drag and drop the red areas and see what happens...

The AREASELECTOR component is able to draw some selection that the user executes via dragging and dropping. The seleciton information is sent to the server side, so that you can perform any kind of activity there.

Typical scenario: you want to crop an image out of some existing image.

Important:

(.) The counter part of the component is some AREASELECTORBinding object which is references in the attribute OBJECTBINDING
(.) The AREASELECTORBinding provides the corresponding get-ter method to acces the selection data
(.) In principal you can select some area of any contained inner component. In the example an IMAGE component is positioned as content of the AREASELECTOR component.

The inner values of AREASELECTORBinding are transferred as just normal component content - and by default are queued on client side when being changed (like e.g. FIELD input, which is also queued). You may set the FLUSH attribute to "true" so that the values are causing some server side roundtrip directly after some update by the user.

This is the Java code:

package workplace;

import java.io.Serializable;
import java.util.Map;

import org.eclnt.ccee.image.ImageCropper;
import org.eclnt.editor.annotations.CCGenClass;
import org.eclnt.jsfserver.defaultscreens.Statusbar;
import org.eclnt.jsfserver.elements.impl.AREASELECTORBinding;
import org.eclnt.jsfserver.util.WebResourceReader;
import org.eclnt.util.valuemgmt.ValueManager;
import org.eclnt.workplace.IWorkpageDispatcher;

@CCGenClass (expressionBase="#{d.DemoAreaSelector}")

public class DemoAreaSelector
    extends PageBean
    implements Serializable
{
    // ------------------------------------------------------------------------
    // inner classes
    // ------------------------------------------------------------------------

    // ------------------------------------------------------------------------
    // members
    // ------------------------------------------------------------------------

    AREASELECTORBinding m_areaInfo = new AREASELECTORBinding();
    String m_croppedImageHex;
    boolean m_withMover = true;
    boolean m_withEdgeSizer = true;
    boolean m_withSideSizer = true;
    float m_areaRatio = 0;
    String m_selectedArea;

    // ------------------------------------------------------------------------
    // constructors & initialization
    // ------------------------------------------------------------------------

    public DemoAreaSelector(IWorkpageDispatcher workpageDispatcher)
    {
        super(workpageDispatcher);
    }

    @Override
    public String getPageName() { return "/workplace/demoareaselector.jsp"; }
    @Override
    public String getRootExpressionUsedInPage() { return "#{d.DemoAreaSelector}"; }

    // ------------------------------------------------------------------------
    // public usage
    // ------------------------------------------------------------------------

    public AREASELECTORBinding getAreaInfo() { return m_areaInfo; }

    public String getAreaInfoString()
    {
        return m_areaInfo.toString();
    }

    public String getCroppedImageHex() { return m_croppedImageHex; }

    public float getAreaRatio() { return m_areaRatio; }
    public void setAreaRatio(float value) { this.m_areaRatio = value; }
    
    public String getSelectedArea() { return m_selectedArea; }
    public void setSelectedArea(String value) { this.m_selectedArea = value; }

    public boolean getWithMover() { return m_withMover; }
    public void setWithMover(boolean value) { this.m_withMover = value; }

    public boolean getWithEdgeSizer() { return m_withEdgeSizer; }
    public void setWithEdgeSizer(boolean value) { this.m_withEdgeSizer = value; }

    public boolean getWithSideSizer() { return m_withSideSizer; }
    public void setWithSideSizer(boolean value) { this.m_withSideSizer = value; }

    public void onApplySelectedAreaAction(javax.faces.event.ActionEvent event) 
    {
        if (m_selectedArea == null)
        {
            Statusbar.outputError("Please define a selected area");
            return;
        }
        Map<String,String> m = ValueManager.decodeComplexValue(m_selectedArea);
        int x1 = ValueManager.decodeInt(m.get("x1"),-1);
        int y1 = ValueManager.decodeInt(m.get("y1"),-1);
        int x2 = ValueManager.decodeInt(m.get("x2"),-1);
        int y2 = ValueManager.decodeInt(m.get("y2"),-1);
        m_areaInfo.setPixelPosX1(x1);
        m_areaInfo.setPixelPosY1(y1);
        m_areaInfo.setPixelPosX2(x2);
        m_areaInfo.setPixelPosY2(y2);
    }

    public void onCreateImageAction(javax.faces.event.ActionEvent event)
    {
        byte[] originalImage = new WebResourceReader().readFileIntoByteArray("/images/pictures/carpicture.jpg");
        byte[] newImage = originalImage;
        if (m_areaInfo.getPixelPosX1() >= 0)
        {
            // the class ImageCropper is part of the eclnt_ccee.jar library
            // eclnt_ccee is a collection of server side utilities which is
            // available in the "resources" area of your CaptainCasa
            // installation
            newImage = ImageCropper.cropImage(newImage,
                                              m_areaInfo.getPixelPosX1(),m_areaInfo.getPixelPosY1(),
                                              m_areaInfo.getPixelPosX2(),m_areaInfo.getPixelPosY2());
            m_croppedImageHex = ValueManager.encodeHexString(newImage);
        }
        else
        {
            Statusbar.outputError("No area selection was done so far.");
        }
    }

    // ------------------------------------------------------------------------
    // private usage
    // ------------------------------------------------------------------------
}
