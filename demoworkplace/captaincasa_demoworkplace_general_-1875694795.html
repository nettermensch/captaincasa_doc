The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • General
  • RISC Client Integration
  • Camera / take photo

The CAMERA component shows what is currently is recorded by the web camera. By pressing the "snap"-button, the current image is sent to the server side in its original camera resolution.

The CAMERA component requires some explicit activation by the user for security reasons. It is NOT available on Internet Explorer 10/11.

This is the Java code:

package workplace;

import java.io.Serializable;

import org.eclnt.editor.annotations.CCGenClass;
import org.eclnt.jsfserver.elements.events.BaseActionEventCameraSnap;
import org.eclnt.jsfserver.elements.util.Trigger;
import org.eclnt.jsfserver.util.tempfile.TempFileManager;
import org.eclnt.workplace.IWorkpageDispatcher;

@CCGenClass (expressionBase="#{d.DemoCamera}")

public class DemoCamera
    extends PageBean 
    implements Serializable
{
    boolean m_withSnapIcon = true;
    public boolean getWithSnapIcon() { return m_withSnapIcon; }
    public void setWithSnapIcon(boolean value) { this.m_withSnapIcon = value; }

    public void onTriggerSnapAction(javax.faces.event.ActionEvent event) 
    {
        m_trigger.trigger();
    }

    Trigger m_trigger = new Trigger();
    public Trigger getTrigger() { return m_trigger; }

    String m_capturedImageURL = null;
    
    public DemoCamera(IWorkpageDispatcher workpageDispatcher)
    {
        super(workpageDispatcher);        
    }

    public String getPageName() { return "/workplace/democamera.jsp"; }
    public String getRootExpressionUsedInPage() { return "#{d.DemoCamera}"; }

    public String getCapturedImageURL() { return m_capturedImageURL; }

    public void onCameraAction(javax.faces.event.ActionEvent event) 
    {
        if (event instanceof BaseActionEventCameraSnap)
        {
            BaseActionEventCameraSnap e = (BaseActionEventCameraSnap)event;
            byte[] bytes = e.getPNGImageBytes();
            m_capturedImageURL = TempFileManager.saveTempFile("capturedImage.png",bytes);
        }
    }
}
