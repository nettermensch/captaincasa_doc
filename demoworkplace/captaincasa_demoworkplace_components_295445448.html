The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • Components
  • Straight Controls
  • Field Control
  • Spinner

The SPINNER component allows to input an integer value within a defined range. The user can scroll up and down the value either by using the keyboard or by using the corresponding icons.

By using the attribute EDITABLE you can decide if in addition keyboard input is possible.

This is the Java code:

package workplace;

import java.io.Serializable;
import org.eclnt.editor.annotations.CCGenClass;
import org.eclnt.workplace.IWorkpageDispatcher;

@CCGenClass (expressionBase="#{d.DemoSpinner}")

public class DemoSpinner
    extends PageBean
    implements Serializable
{
    public DemoSpinner(IWorkpageDispatcher dispatcher)
    {
        super(dispatcher);
        initRefComponent("spinner"); 
    }
    
    protected int m_spinnerValue1 = 5;
    protected int m_spinnerValue2 = 20;

    public int getSpinnerValue1()
    {
        return m_spinnerValue1;
    }
    public void setSpinnerValue1(int spinnerValue1)
    {
        m_spinnerValue1 = spinnerValue1;
    }
    public int getSpinnerValue2()
    {
        return m_spinnerValue2;
    }
    public void setSpinnerValue2(int spinnerValue2)
    {
        m_spinnerValue2 = spinnerValue2;
    }

}
