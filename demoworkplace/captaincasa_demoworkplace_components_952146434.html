The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • Components
  • Straight Controls
  • Invokers
  • Vertical Links

The LINKVERTICAL component is a text that can be clicked by the user. It provides certain roll-over effect that are similar to what users know from browser-links.

This is the XML layout definition:

<?xml version="1.0" encoding="utf-8"?>
<t:layoutdefinition xmlns:t="http://www.CaptainCasa.com/controllibrary/t" xmlns:tpbc="http://www.CaptainCasa.com/controllibrary/tpbc" >
<t:rowdemobodypane id="g_1" objectbinding="#{d.DemoLinkVertical}" >
<t:row id="g_2" >
<t:linkvertical id="g_3" actionListener="#{d.DemoLinkVertical.onLinkPressed}" rowalignmenty="bottom" text="Vertical Link" />
<t:linkvertical id="g_4" actionListener="#{d.DemoLinkVertical.onLinkPressed}" font="size:16;weight:bold" rowalignmenty="bottom" text="Vertical Link" />
<t:linkvertical id="g_5" actionListener="#{d.DemoLinkVertical.onLinkPressed}" font="size:26;weight:bold" foreground="#0000FF" linkinvokedcolor="#FFFF00" text="Vertical Link" />
</t:row>
</t:rowdemobodypane>
<t:pageaddons id="g_pa"/>
</t:layoutdefinition>

This is the Java code:

package workplace;

import java.io.Serializable;

import javax.faces.event.ActionEvent;

import org.eclnt.editor.annotations.CCGenClass;
import org.eclnt.workplace.IWorkpageDispatcher;

@CCGenClass (expressionBase="#{d.DemoLinkVertical}")

public class DemoLinkVertical
    extends PageBean
    implements Serializable
{
    public DemoLinkVertical(IWorkpageDispatcher dispatcher)
    {
        super(dispatcher);
        initRefComponent("linkvertical"); 
    }

    public void onLinkPressed(ActionEvent event) 
    {
        getOwningDispatcher().getStatusbar().outputSuccess("Link was pressed!");
    }
}
