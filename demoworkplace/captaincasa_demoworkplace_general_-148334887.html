The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • General
  • Long Operations
  • Progress info

The new server side API "BlockerInfo.sendProgressToClient(...)" allows to send progress messages to the client directly during application processing.

Internally the client opens up a websocket connection to the server that is responsible for transferring progress messages back to the client - without using the "big communication round trip".

This is the Java code:

package workplace;

import java.io.Serializable;

import org.eclnt.editor.annotations.CCGenClass;
import org.eclnt.jsfserver.defaultscreens.BlockerInfo;
import org.eclnt.jsfserver.defaultscreens.Statusbar;
import org.eclnt.workplace.IWorkpageDispatcher;

@CCGenClass (expressionBase="#{d.DemoBlockerInfo}")

public class DemoBlockerInfo
    extends PageBean 
    implements Serializable
{
    // ------------------------------------------------------------------------
    // constructors & initialization
    // ------------------------------------------------------------------------

    public DemoBlockerInfo(IWorkpageDispatcher workpageDispatcher)
    {
        super(workpageDispatcher);        
    }

    public String getPageName() { return "/workplace/demoblockerinfo.jsp"; }
    public String getRootExpressionUsedInPage() { return "#{d.DemoBlockerInfo}"; }

    // ------------------------------------------------------------------------
    // public usage
    // ------------------------------------------------------------------------

    public void onLongAction(javax.faces.event.ActionEvent event) 
    {
        try
        {
            for (int i=0; i<100; i++)
            {
                Thread.sleep(100);
                BlockerInfo.sendProgressToClient("Processing step " + (i+1),i+1);
            }
        }
        catch (Throwable t)
        {
            Statusbar.outputError(t.toString());
        }
    }

    public void onLongAngryAction(javax.faces.event.ActionEvent event) 
    {
        try
        {
            Thread.sleep(10000);
        }
        catch (Throwable t)
        {
            Statusbar.outputError(t.toString());
        }
    }

    
    // ------------------------------------------------------------------------
    // private usage
    // ------------------------------------------------------------------------
}
