The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • General
  • Workplace Management
  • Redirect client to URL/page

There are two components that redirect the client to a complete different page.

CLIENTREDIRECTURL navigates to a new URL - the client will be closed as consequence.

CLIENREDIRECTPAGE navigates to a different dialog of your project. You may decide if this navigation should happen within the existing session or if a new session is to be created.

Please pay attention: the purpose of redirecting is NOT to implement normal, logic-driven page navigation. The normal page navigation should be properly done by structuring page beans using the ROWPAGEBEANINCLUD component! (!!!)

The navigation via redirecting to a URL or page is used for exceptional cases only:
(.) you want to navigate to an error page
(.) you want to close the current client and navigate to some normal URL

This is the Java code:

package workplace;

import java.io.Serializable;

import org.eclnt.editor.annotations.CCGenClass;
import org.eclnt.jsfserver.elements.util.Trigger;
import org.eclnt.workplace.IWorkpageDispatcher;

@CCGenClass (expressionBase="#{d.DemoRedirect}")

public class DemoRedirect
    extends WorkpageDispatchedPageBean 
    implements Serializable
{
    // ------------------------------------------------------------------------
    // members
    // ------------------------------------------------------------------------

    boolean m_keepSession = false;
    String m_page = "/workplace/demohelloworld.jsp"; 
    String m_url = "http://www.captaincasa.com";
    Trigger m_pageTrigger = new Trigger();
    Trigger m_urlTrigger = new Trigger();

    // ------------------------------------------------------------------------
    // constructors & initialization
    // ------------------------------------------------------------------------

    public DemoRedirect(IWorkpageDispatcher workpageDispatcher)
    {
        super(workpageDispatcher);        
    }

    public String getPageName() { return "/workplace/demoredirect.jsp"; }
    public String getRootExpressionUsedInPage() { return "#{d.DemoRedirect}"; }

    // ------------------------------------------------------------------------
    // public usage
    // ------------------------------------------------------------------------

    public boolean getKeepSession() { return m_keepSession; }
    public void setKeepSession(boolean value) { this.m_keepSession = value; }

    public String getPage() { return m_page; }
    public void setPage(String value) { this.m_page = value; }

    public String getUrl() { return m_url; }
    public void setUrl(String value) { this.m_url = value; }

    public Trigger getPageTrigger() { return m_pageTrigger; }
    public Trigger getUrlTrigger() { return m_urlTrigger; }

    public void onRedirectPageAction(javax.faces.event.ActionEvent event) 
    {
        DemoHelloWorld dhw = (DemoHelloWorld)getOwningDispatcher().getTopOwner().getDispatchedBean(DemoHelloWorld.class);
        dhw.setName("Captain");
        m_pageTrigger.trigger();
    }

    public void onRedirectURLAction(javax.faces.event.ActionEvent event) 
    {
        m_urlTrigger.trigger();
    }

    // ------------------------------------------------------------------------
    // private usage
    // ------------------------------------------------------------------------
}
