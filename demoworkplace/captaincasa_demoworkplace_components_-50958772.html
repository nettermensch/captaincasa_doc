The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • Components
  • Straight Controls
  • Field Control
  • Input of "strange" characters
  • Key variant selection

The key variant popup is activated by adding KEYVARIANTSSUPPORT to "true".

The definition of the key variants is kept in a server-side XML file, by default the file "/eclntjsfserver/keyvariants/ccdefault.xml" is used - but you can define an own definition and reference this one using attribute KEYVARIANTSLAYOUT.

The hot key (default: alt-k) can be defined using attribute CLIENTCONFIG-KEYVARIANTSHOTKEY.

This is the XML layout definition:

<?xml version="1.0" encoding="utf-8"?>

<t:beanprocessing id="g_1" beanbinding="#{d.DemoFieldKeyVariants}" />
<t:rowdemobodypane id="g_2" objectbinding="#{d.DemoFieldKeyVariants}" >
<t:row id="g_3" >
<t:textpane id="g_4" text="Press alt-K (&#34;K&#34; for key) after the input of a character in the following text field. A dialog will show up in which you can select character variants." width="100%" />
</t:row>
<t:rowdistance id="g_5" height="30" />
<t:row id="g_6" >
<t:field id="g_7" keyvariantssupport="true" labeltext="Field with key-variant-support, press &#34;alt-K&#34;" text="#{d.DemoFieldKeyVariants.text}" width="100%" />
</t:row>
</t:rowdemobodypane>
<t:pageaddons id="g_pa"/>
</t:layoutdefinition>

This is the Java code:

package workplace;

import java.io.Serializable;

import org.eclnt.editor.annotations.CCGenClass;
import org.eclnt.workplace.IWorkpageDispatcher;

@CCGenClass (expressionBase="#{d.DemoFieldKeyVariants}")

public class DemoFieldKeyVariants
    extends PageBean 
    implements Serializable
{
    // ------------------------------------------------------------------------
    // members
    // ------------------------------------------------------------------------

    String m_text = "\u0054\u0068\u0069\u0073\u0020\u0074\u1EC5\u0078\u0074"
            + "\u0020\u0063\u006F\u006E\u0074\u00E6\u0069\u006E\u0073"
            + "\u0304\u0020\u0073\u0074\u0072\u0061\u006E\u01E5\u1EC7"
            + "\u0020\u0188\u0068\u0061\u0072\u0061\u0063\u021B\u0065\u0072\u0073\u0021";

    // ------------------------------------------------------------------------
    // constructors & initialization
    // ------------------------------------------------------------------------

    public DemoFieldKeyVariants(IWorkpageDispatcher workpageDispatcher)
    {
        super(workpageDispatcher);        
    }

    public String getPageName() { return "/workplace/demofieldkeyvariants.jsp"; }
    public String getRootExpressionUsedInPage() { return "#{d.DemoFieldKeyVariants}"; }

    // ------------------------------------------------------------------------
    // public usage
    // ------------------------------------------------------------------------

    public String getText() { return m_text; }
    public void setText(String value) { this.m_text = value; }
}
