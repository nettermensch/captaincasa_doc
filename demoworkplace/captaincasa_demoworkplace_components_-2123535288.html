The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • Components
  • Straight Controls
  • Combobox and List
  • Direct Select Mode

The attribute DIRECTSELECTMODE controls the behavior of the COMBOBOX during keyboard navigation.

Both COMBOBOX instances of the screen edit the same value ("department"). Both COMBOBOX flush their value immediately.

When opening the COMBOBOX-list in the first COMBOBOX and then navigating through the list of items by up/down keys, the value of the COMBOBOX is not updated. The user needs to explicitly press the return-key.

In the second COMBOBOX the value of the COMBOBOX is updated immediately when navigating up and down.

The advantage of the direct selection mode being switched on is:

1. The behavior is the traditional behavior of combo boxes.

The disadvantages are:

1. when flushing values directly to the server (FLUSH defined as "true") then there is a network roundtrip initiated with each keyboard navigation.

2. the default behavior of the default COMBOFIELD selection dialogs is different. With the COMBOFIELD component you typically open up a dialog and the value of the dialog is transferred explicitly. Because COMBOBOX and COMBOFIELD look the same the user may also expect the same selection behavior.

package workplace;

import java.io.Serializable;

import org.eclnt.editor.annotations.CCGenClass;
import org.eclnt.jsfserver.elements.util.ValidValuesBinding;
import org.eclnt.workplace.IWorkpageDispatcher;
import org.eclnt.workplace.WorkpageDispatchedPageBean;

@CCGenClass (expressionBase="#{d.DemoComboBoxDirectSelectionMode}")

public class DemoComboBoxDirectSelectionMode
    extends PageBean 
    implements Serializable
{
    ValidValuesBinding m_departmentVVS = new ValidValuesBinding();
    public ValidValuesBinding getDepartmentVVS() { return m_departmentVVS; }

    String m_department = null;
    public String getDepartment() { return m_department; }
    public void setDepartment(String value) { this.m_department = value; }

    // ------------------------------------------------------------------------
    // inner classes
    // ------------------------------------------------------------------------

    // ------------------------------------------------------------------------
    // members
    // ------------------------------------------------------------------------

    // ------------------------------------------------------------------------
    // constructors & initialization
    // ------------------------------------------------------------------------

    public DemoComboBoxDirectSelectionMode(IWorkpageDispatcher workpageDispatcher)
    {
        super(workpageDispatcher);        
        m_departmentVVS.addValidValue("0001","Sales");
        m_departmentVVS.addValidValue("0002","Production");
        m_departmentVVS.addValidValue("0003","Human Resources");
        m_departmentVVS.addValidValue("0004","Financials");
    }

    public String getPageName() { return "/workplace/democomboboxdirectselectionmode.jsp"; }
    public String getRootExpressionUsedInPage() { return "#{d.DemoComboBoxDirectSelectionMode}"; }
