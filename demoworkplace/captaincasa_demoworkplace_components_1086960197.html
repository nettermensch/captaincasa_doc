The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • Components
  • Animation & Graphics & Sound
  • Paint Area
  • Magnify Factor

The PAINTAREACOMPONENT provides an attribute MAGNIFYFACTOR.

When the user moves the mouse over a PAINTAREAITEM then this item is magnified - by the factor defined via MAGNIFYFACTOR. The value of MAGNIFYFACTOR is an integer percentage number (e.g. 25 for "25%").

package workplace;

import java.io.Serializable;

public class DemoPaintAreaMagnify
    extends PageBean
    implements Serializable
{
    public DemoPaintAreaMagnify(IWorkpageDispatcher dispatcher)

        super(dispatcher);
        initRefComponent("paintarea");
    }

    public void onImage4(ActionEvent event) { m_image = "/images/pictures/ph4.jpg"; }

    public void onImage2(ActionEvent event) { m_image = "/images/pictures/ph2.jpg"; }

    public void onImage1(ActionEvent event) { m_image = "/images/pictures/ph1.jpg"; }

    public void onImage0(ActionEvent event) { m_image = "/images/pictures/ph0.jpg"; }

    protected String m_image;
    public String getImage() { return m_image; }

}
