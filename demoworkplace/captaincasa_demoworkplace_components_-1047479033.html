The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • Components
  • Animation & Graphics & Sound
  • Paint Area
  • Line drawing
  • Editing of interim points

The PAINTARAEALINEITEM allows the user to edit its interim points.

To do so you need to:

1. switch the attribute WITHEDITINGOFINTERIMPOINTS to "true"
2. provide a property to be the counterpart of the INTERIMPOINTS attribute

The interim points are now both taken and written back into the property. By setting FLUSH to "true" the value is immediately transferred to the server side.
