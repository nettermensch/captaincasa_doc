The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • Components
  • Straight Controls
  • Field Control
  • Date Input
  • Date formatting (2)

When defining the FORMATMASK for date and time values (FORMAT date/ time/ datetime) then you...

...either can select the pre-configured format masks "short", "medium" and "long" (well, actually "long" is the same as "medium"...). These pre-configured format masks ensure that the sequence of year/ month/ date and the delimiters between follow the defaults of the client's country definition.

...or you can configure the format on your own by defining a pattern in which special characters indicate the placing of the date/time-values. In the pattern the following characters can be used:

yyyy - year (4 digits)
yy - year (2 digits)
M or MM - month
d or dd - day of month
E or EE or EEE or EEEE - abbreviated name of week day
H or HH - the hour of the day (0-23)
h or hh - the hour of the day (0-12)
a or aa - am/pm indicator

For all special characters you may either choose the one-character or t he twp-characters representation ("d" or "dd") - the result will always be the same, values will be shown as two digits always (e.g. "01", not "1").

You may see some analogies when comparing the format to the Java-SimpleDateFormat - but keep in mind: the conversion is taking place on client side (i.e. in the JavaScript part), so not all features of the Java SimpleDateFormat are available!

package workplace;

import java.io.Serializable;
import java.time.LocalDateTime;

import org.eclnt.editor.annotations.CCGenClass;
import org.eclnt.workplace.IWorkpageDispatcher;

@CCGenClass (expressionBase="#{d.DemoSimpleDateFormat}")

public class DemoSimpleDateFormat
    extends PageBean 
    implements Serializable
{
    // ------------------------------------------------------------------------
    // members
    // ------------------------------------------------------------------------

    LocalDateTime m_someTime = LocalDateTime.now();

    // ------------------------------------------------------------------------
    // constructors & initialization
    // ------------------------------------------------------------------------

    public DemoSimpleDateFormat(IWorkpageDispatcher workpageDispatcher)
    {
        super(workpageDispatcher);        
    }

    public String getPageName() { return "/workplace/demosimpledateformat.xml"; }
    public String getRootExpressionUsedInPage() { return "#{d.DemoSimpleDateFormat}"; }

    // ------------------------------------------------------------------------
    // public usage
    // ------------------------------------------------------------------------

    public LocalDateTime getSomeTime() { return m_someTime; }
    public void setSomeTime(LocalDateTime value) { this.m_someTime = value; }

    // ------------------------------------------------------------------------
    // private usage
    // ------------------------------------------------------------------------
}
