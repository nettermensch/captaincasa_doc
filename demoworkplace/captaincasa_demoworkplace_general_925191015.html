The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • General
  • Client Integration
  • Copy content of client clipboard

The component BUTTONCOPYCLIPBOARD copies the client side clipboard content to the server side. After pressing the button a corresponding action-listener will be called on server side, the content of the client clipboard is transferred within the event.

The content is defined by a "type" (e.g. "text/plain", "text/html", "image/png", ...).

The content is represented by a String. If the content is byte data (e.g. image) then the String is a hexadecimal representation of the byte data. You can access the byte data by the method BaseActionEventCopyClipboard.getClipboardContentAsDecodedByteArray().

Pay attention on security issues: the browser requests the right to access the client side clipboard when calling the component. The component is only use-able in https scenarios!

This is the Java code:

package workplace;

import java.io.Serializable;

import org.eclnt.editor.annotations.CCGenClass;
import org.eclnt.jsfserver.elements.events.BaseActionEventCopyClipboard;
import org.eclnt.workplace.IWorkpageDispatcher;

@CCGenClass (expressionBase="#{d.DemoButtonCopyClipboard}")

public class DemoButtonCopyClipboard
    extends PageBean 
    implements Serializable
{
    // ------------------------------------------------------------------------
    // members
    // ------------------------------------------------------------------------

    String m_content;
    String m_type;

    // ------------------------------------------------------------------------
    // constructors & initialization
    // ------------------------------------------------------------------------

    public DemoButtonCopyClipboard(IWorkpageDispatcher workpageDispatcher)
    {
        super(workpageDispatcher);        
    }

    public String getPageName() { return "/workplace/demobuttoncopylcipboard.xml"; }
    public String getRootExpressionUsedInPage() { return "#{d.DemoButtonCopyClipboard}"; }

    // ------------------------------------------------------------------------
    // public usage
    // ------------------------------------------------------------------------

    public void onClipboardAction(javax.faces.event.ActionEvent event) 
    {
        if (event instanceof BaseActionEventCopyClipboard)
        {
            BaseActionEventCopyClipboard e = (BaseActionEventCopyClipboard)event;
            m_type = e.getClipboardContentType();
            m_content = e.getClipboardContentAsString();
        }
    }

    public String getContent() { return m_content; }
    public void setContent(String value) { this.m_content = value; }

    public String getType() { return m_type; }
    public void setType(String value) { this.m_type = value; }

    public boolean getAvailableImage() 
    { 
        return m_type != null && m_type.startsWith("image/"); 
    }

    // ------------------------------------------------------------------------
    // private usage
    // ------------------------------------------------------------------------
}
