The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • Grids & Trees
  • Grids and Trees
  • Column Sequence and Widths
  • Column sequence / width

A grid contains columns that can be arranged by the user.

The user can re-size columns by picking the shadowed area on the right and dragging/dropping the columns size.

The user can re-arrange columns by dragging and dropping the whole title area of the column onto another one.

Please note: the information about sizes and sequences that might be changed by the user is available on server side as well. You may for example store a user's column sizing and column sequence in some kind of user profile.

Every time a request is sent from the client to the server size, sequence updates are made available within the server side FIXGRIDBinding object. - On the other hand you can set these values from your logic.

Sizes and sequence definitions are semicolon separated strings, each value representing the sequence / size of the corresponding column.

This is the Java code:

package workplace;

import javax.faces.event.ActionEvent;

import org.eclnt.jsfserver.elements.impl.FIXGRIDItem;
import org.eclnt.jsfserver.elements.impl.FIXGRIDListBinding;
import org.eclnt.workplace.IWorkpageDispatcher;

public class DemoGridCols extends PageBean
{
    public class MyRow extends FIXGRIDItem
    {
        String m_firstName;
        String m_lastName;
        String m_street;
        String m_town;
        public void setFirstName(String value) { m_firstName = value; }
        public String getFirstName() { return m_firstName; }
        public void setLastName(String value) { m_lastName = value; }
        public String getLastName() { return m_lastName; }
        public void setStreet(String value) { m_street = value; }
        public String getStreet() { return m_street; }
        public void setTown(String value) { m_town = value; }
        public String getTown() { return m_town; }
    }
    
    FIXGRIDListBinding<MyRow> m_rows = new FIXGRIDListBinding<MyRow>();
    String m_newSequence;
    String m_newWidths;
    
    public FIXGRIDListBinding<MyRow> getRows() { return m_rows; }
    
    public void setNewWidths(String value) { m_newWidths = value; }
    public String getNewWidths() { return m_newWidths; }
    public void setNewSequence(String value) { m_newSequence = value; }
    public String getNewSequence() { return m_newSequence; }
    
    public DemoGridCols(IWorkpageDispatcher dispatcher)
    {
        super(dispatcher);
        for (int i=0; i<20; i++)
        {
            MyRow mr = new MyRow();
            mr.setFirstName("FN/"+i);
            mr.setLastName("LN/"+i);
            mr.setStreet("S/"+i);
            mr.setTown("T/"+i);
            m_rows.getItems().add(mr);
        }
   }
    
    public void onAddRow(ActionEvent ae)
    {
        MyRow mr = new MyRow();
        m_rows.getItems().add(mr);
    }
    
    public void onResetAction(ActionEvent ae)
    {
        m_rows.resetPersistentData();
    }
    
    public void onSynchToServer(ActionEvent ae)
    {
    }
    
    public void onSynchToClient(ActionEvent ae)
    {
        if (m_newWidths != null && m_newWidths.length() != 0)
            m_rows.setModcolumnwidths(m_newWidths);
        else
            m_rows.setModcolumnwidths(null);
        if (m_newSequence != null && m_newSequence.length() != 0)
            m_rows.setColumnsequence(m_newSequence);
        else
            m_rows.setColumnsequence(null);
    }
    
}
