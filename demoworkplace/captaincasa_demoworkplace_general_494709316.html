The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • General
  • Style and Language Management
  • Define runtime styles

Style definitions can be outsourced into the Stream store so that they are kept outside the webcontent.

This is the XML layout definition:

<?xml version="1.0" encoding="utf-8"?>
<t:layoutdefinition xmlns:t="http://www.CaptainCasa.com/controllibrary/t" xmlns:tpbc="http://www.CaptainCasa.com/controllibrary/tpbc" >
<t:rowdemobodypane id="g_4" objectbinding="#{d.DemoStstStyleFiles}" >
<t:row id="g_3" >
<t:pane id="g_8" background="#00000008" height="100%" padding="10" width="100%" >
<t:row id="g_1" >
<t:pagebeancomponent id="g_6" pagebeanbinding="#{d.DemoStstStyleFiles.ststUI}" shownullcontent="true" />
</t:row>
</t:pane>
</t:row>
</t:rowdemobodypane>
<t:pageaddons id="g_pa"/>
</t:layoutdefinition>

This is the Java code:

package workplace;

import java.io.Serializable;

import org.eclnt.ccaddons.pbc.CCRuntimeStyleEditor;
import org.eclnt.editor.annotations.CCGenClass;
import org.eclnt.workplace.IWorkpageDispatcher;

@CCGenClass (expressionBase="#{d.DemoStstStyleFiles}")

public class DemoStstStyleFiles
    extends PageBean
    implements Serializable
{
    // ------------------------------------------------------------------------
    // inner classes
    // ------------------------------------------------------------------------

    // ------------------------------------------------------------------------
    // members
    // ------------------------------------------------------------------------

    CCRuntimeStyleEditor m_ststUI = new CCRuntimeStyleEditor();

    // ------------------------------------------------------------------------
    // constructors & initialization
    // ------------------------------------------------------------------------

    public DemoStstStyleFiles(IWorkpageDispatcher workpageDispatcher)
    {
        super(workpageDispatcher);
        m_ststUI.prepare("/workplace/workplaceRisc.jsp",null);
    }

    @Override
    public String getPageName() { return "/workplace/demostststylefiles.jsp"; }
    @Override
    public String getRootExpressionUsedInPage() { return "#{d.DemoStstStyleFiles}"; }

    // ------------------------------------------------------------------------
    // public usage
    // ------------------------------------------------------------------------

    public CCRuntimeStyleEditor getStstUI() { return m_ststUI; }

    // ------------------------------------------------------------------------
    // private usage
    // ------------------------------------------------------------------------
}
