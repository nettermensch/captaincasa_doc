The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • Components
  • Straight Controls
  • Menus
  • Button Popup - Mini popups in mouse over

This demo shows the usage of a BUTTONPOPUP component so that popup dialogs are shown on mouse over - opening from the left, right, top, bottom.

The features used in this demo are:

1. Usage of OVERLAYARE component in order to position the 4 side areas on top of the "base area"

2. Usage of BUTTONPOPUP with the following attributes
a. attribute ACTIVATEONMOUSEOVER set to true - so that not only a click invokes the popup but also moving the mouse on top
b. attribute MENUPOSITION set to some "overlay*" value so that the popup opens on top of the BUTTONMENU component
c. attribute CLOSEONMOUSEEXIT set to true so that popup closes when moving the mouse outside of its area
d. usage of BORDER AND ROUNDING so that the component optically is nicely arranged
