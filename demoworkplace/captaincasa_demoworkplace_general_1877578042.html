The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • General
  • Tool Tips / User Hints
  • Tooltip - popup sizing

If using HTML based tooltips, then some own popup is opened by the CaptainCasa client framework.

The sizing of the tooltip popup depends on the text content of the HTML tooltip. In the style of style class "riscuserhint_textpane" there is a definition of a width/height-ration (value "_popupAspectRatio"). The sizing tries to find a size fitting to this ratio, starting with a minimum width of 300 pixels.

You may also directly define a size in the text by starting your text with "<html data-tooltipwidth='xxx'>". Please pay attention: the text has to exactly start with this phrase - no variations ara allowed!
