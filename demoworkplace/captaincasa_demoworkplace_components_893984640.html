The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • Components
  • Straight Controls
  • Text Input/Output
  • Simple text editor
  • Caret Info - Drag Drop

This example shows drag & drop operations that are executed on top of a TEXTEDITOR.

The labels on the left holds the texts, e.g. "info:.{customer}" as DRAGSEND information.
The TEXTEDITOR is defined with DROPRECEIVE "info" in order to accept the drop.

Together wtih the text, the caret position is passed. The caret directly points into the HTML text that is kept on server side.

IMPORTANT: you need to set AVOIDSANITIZING with TEXTPANE to "true". Otherwise the text will be updated by some client side sanitizer (e.g. the contenteditable="false" will not survive the sanitizing process).
If not doing so, then the text on client side will be different to the text on server side, resulting in wrong caret positions.
Of course: you need to be pay attention to sanitizing the text yourself on server-side!

This is the Java code:

package workplace;

import java.io.Serializable;

import org.eclnt.editor.annotations.CCGenClass;
import org.eclnt.jsfserver.elements.events.BaseActionEventDrop;
import org.eclnt.workplace.IWorkpageDispatcher;

@CCGenClass (expressionBase="#{d.DemoSimpleTextEditorDragDrop}")

public class DemoSimpleTextEditorDragDrop
    extends WorkpageDispatchedPageBean 
    implements Serializable
{
    // ------------------------------------------------------------------------
    // inner classes
    // ------------------------------------------------------------------------

    // ------------------------------------------------------------------------
    // members
    // ------------------------------------------------------------------------

    String m_text = "Lorem ipsum dolor sit amet, consetetur sadipscing elitr, sed diam " 
        + "nonumy eirmod tempor invidunt ut labore et dolore magna aliquyam erat, "
        + "sed diam voluptua. At vero eos et accusam et justo duo dolores et ea rebum. " 
        + "Stet clita kasd gubergren, no sea takimata sanctus est Lorem ipsum dolor sit amet. "
        + "Lorem ipsum dolor sit amet, consetetur sadipscing elitr, sed diam nonumy "
        + "eirmod tempor invidunt ut labore et dolore magna aliquyam erat, sed diam voluptua. "
        + "At vero eos et accusam et justo duo dolores et ea rebum. Stet clita kasd gubergren, "
        + "no sea takimata sanctus est Lorem ipsum dolor sit amet.";

    // ------------------------------------------------------------------------
    // constructors & initialization
    // ------------------------------------------------------------------------

    public DemoSimpleTextEditorDragDrop(IWorkpageDispatcher workpageDispatcher)
    {
        super(workpageDispatcher);        
    }

    public String getPageName() { return "/workplace/demosimpletexteditordragdrop.xml"; }
    public String getRootExpressionUsedInPage() { return "#{d.DemoSimpleTextEditorDragDrop}"; }

    // ------------------------------------------------------------------------
    // public usage
    // ------------------------------------------------------------------------

    public String getText() { return m_text; }
    public void setText(String value) { this.m_text = value; }

    public void onTextAction(javax.faces.event.ActionEvent event) 
    {
        if (event instanceof BaseActionEventDrop)
        {
            BaseActionEventDrop e = (BaseActionEventDrop)event;
            String dragInfo = e.getDragInfo();
            if (dragInfo.startsWith("info:"))
            {
                String text = dragInfo.substring(5);
                // pay attention: no unnecessary blank spaces! otherwise browser client might hold
                // a different representation of the same content.
                // Example: 
                // WRONG:   <span    contenteditable = \"false\" style=\"text-weight:   bold;   color:blue\">
                // RIGHT:   <span contenteditable=\"false\" style=\"text-weight:bold;color:blue\">
                String insertText = "<span contenteditable=\"false\" style=\"text-weight:bold;color:blue\">"+text+"</span>";
                if (e.getCaretPosition() >= 0)
                {
                    m_text = m_text.substring(0,e.getCaretPosition()) + insertText + m_text.substring(e.getCaretPosition());
                }
            }
        }
    }

    // ------------------------------------------------------------------------
    // private usage
    // ------------------------------------------------------------------------
}
