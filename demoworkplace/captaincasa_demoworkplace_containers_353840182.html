The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • Containers
  • Transfer of actual Size
  • Category size

This SIZETRANSFER component can be used to not directly transfer pixel sizes, but to transfer size categories.

To do so you need to define the attributes HORIZONTALCATEGORIES and/or VERTICALCATEGORIES.

Each attribute is a string in which you define the name of each category and the corresponding pixel size.

Example: if you define HORIZONTALCATEGORIES to be "narrow:400;normal:800;wide:1200" then the width that is transferred is "narrow" if the screen is up to 400 pixels wide, it is "normal" if the screen is up to 800 pixels wide and it is "wide" for the rest. The highest category is always used if no other, higher category is defined.

The advantage of using the categories is: the client only transfers change of its size to the server, if the changes are relevant on category level. If the corresponding PANE's size changes, but still in the range of its current categories, then no server update is initiated.

Please note: you still need to define FLUSH to be true, if changes should be communicated to the server directly! Otherwise changes will wait for the next synch (e.g. due to a button being pressed).

<t:sizetransfer id="g_4" actionListener="#{d.DemoSizeTransferCategory.onSizeTransferAction}" flush="true" horizontalcategories="narrow:400;normal:800;wide:1200" sizevalue="#{d.DemoSizeTransferCategory.size}" verticalcategories="all:10000" />

<t:pageaddons id="g_pa"/>
</t:layoutdefinition>

This is the Java code:

package workplace;

import java.io.Serializable;

import javax.faces.event.ActionEvent;

import org.eclnt.editor.annotations.CCGenClass;
import org.eclnt.jsfserver.defaultscreens.Statusbar;
import org.eclnt.workplace.IWorkpageDispatcher;
import org.eclnt.workplace.WorkpageDispatchedPageBean;

@CCGenClass (expressionBase="#{d.DemoSizeTransferCategory}")

public class DemoSizeTransferCategory
    extends PageBean 
    implements Serializable
{
    boolean m_flush = false;
    protected String m_size;
    
    public DemoSizeTransferCategory(IWorkpageDispatcher workpageDispatcher)
    {
        super(workpageDispatcher);        
        initRefComponent("sizetransfer"); 
    }

    public String getSize() { return m_size; }
    public void setSize(String value) { m_size = value; }

    public boolean getFlush() { return m_flush; }
    public void setFlush(boolean value) { this.m_flush = value; }

    public void onButtonAction(ActionEvent event) 
    {
        Statusbar.outputSuccess("BUTTON: The size of the gray client area is: " + m_size);
    }
    
    public void onSizeTransferAction(ActionEvent event)
    {
        Statusbar.outputSuccess("SIZETRANSFER: The size of the gray client area is: " + m_size);
    }
}
