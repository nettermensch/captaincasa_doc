The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • Containers
  • Panes

This page is a summary of basic pane components that are provided.

Pane components are containers that themselves can hold other components. Pane components can be nested in any way.

This is the Java code:

package workplace;

import javax.faces.event.ActionEvent;

import org.eclnt.workplace.IWorkpageDispatcher;

public class DemoPanes extends PageBean
{
    int m_flashCounter = 0;

    public DemoPanes(IWorkpageDispatcher dispatcher)
    {
        super(dispatcher);
    }

    public void onFlash(ActionEvent ae)
    {
        m_flashCounter++;
    }
    
    public int getFlashCounter()
    {
        return m_flashCounter;
    }
    public void setFlashCounter(int flashCounter)
    {
        m_flashCounter = flashCounter;
    }
    
}
