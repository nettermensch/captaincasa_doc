The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • Components
  • Straight Controls
  • Field Control
  • Password
  • Display length

The length of the real password is only shown during input and until the password value was sent to the serer side.

In all other situations the number of characters shown in the password field is set to a fix value (by default: 20) - totally independent from the actual size of the password. Reason: by this, guessing the password is much harder.

The number of characters can be explicitly set using PASSWORD-ANONYMIZEDLENGTH. Please note: values below 4 are not accepted.

This is the Java code:

package workplace;

import java.io.Serializable;

import org.eclnt.editor.annotations.CCGenClass;
import org.eclnt.workplace.IWorkpageDispatcher;

@CCGenClass (expressionBase="#{d.DemoPasswordLength}")

public class DemoPasswordLength
    extends PageBean 
    implements Serializable
{
    // ------------------------------------------------------------------------
    // inner classes
    // ------------------------------------------------------------------------

    // ------------------------------------------------------------------------
    // members
    // ------------------------------------------------------------------------

    String m_password = "CaptainCasa";
    
    // ------------------------------------------------------------------------
    // constructors & initialization
    // ------------------------------------------------------------------------

    public DemoPasswordLength(IWorkpageDispatcher workpageDispatcher)
    {
        super(workpageDispatcher);        
    }

    public String getPageName() { return "/workplace/demopasswordlength.xml"; }
    public String getRootExpressionUsedInPage() { return "#{d.DemoPasswordLength}"; }

    // ------------------------------------------------------------------------
    // public usage
    // ------------------------------------------------------------------------

    public String getPassword() { return m_password; }
    public void setPassword(String value) { this.m_password = value; }

    // ------------------------------------------------------------------------
    // private usage
    // ------------------------------------------------------------------------
}
