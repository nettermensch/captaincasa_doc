The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • Components
  • Straight Controls
  • Structure Components
  • Tabbed Line

The tabbed line is a horizontal arrangement of tabs. Each tab can have a text (mandatory) and an image (optional).

Unlike the TABBEDPANE the TABLINE component does not provide an own pane/ container component for each tab - but can be used for any purpose as consequence.

In the example the tabbed line is associated with a pane that is located directly below the tabbed line. When clicking onto a tab the corresponding page is shown within the pane.

This is the Java code:

package workplace;

import javax.faces.event.ActionEvent;

import org.eclnt.editor.annotations.CCGenClass;
import org.eclnt.jsfserver.defaultscreens.Statusbar;
import org.eclnt.workplace.IWorkpageDispatcher;

@CCGenClass (expressionBase="#{d.demoTabbedline}")

public class DemoTabbedline extends PageBean
{
    protected int m_selectedIndex;
    public int getSelectedIndex() { return m_selectedIndex; }
    public void setSelectedIndex(int value) { m_selectedIndex = value; }

    String m_background = "#C0FFC0";
    
    public DemoTabbedline(IWorkpageDispatcher dispatcher)
    {
        super(dispatcher);
        initRefComponent("tabbedline");
    }
    
    public String getBackground() { return m_background; }
    
    public void onLeft(ActionEvent ae)
    {
        Statusbar.outputMessage("Left tab was invoked.");
    }
    
    public void onRight(ActionEvent ae)
    {
        Statusbar.outputMessage("Right tab was invoked.");
    }
    
    public void onCenter(ActionEvent ae)
    {
        Statusbar.outputMessage("Center tab was invoked.");
    }
    
    public void onGreen(ActionEvent ae)
    {
        m_background = "#C0FFC0";
    }
    
    public void onRed(ActionEvent ae)
    {
        m_background = "#FFC0C0";
    }
    
}
