The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • Grids & Trees
  • Grids and Trees
  • Sorting
  • Triple state sorting

By default, clicking into the column header of a grid will first sort the content in an ascending direction, then in a descending direction and then again into ascending direction.

But: the grid also supports a "triple state" sorting. Here the sequence is: ascending, descending and then back to the original sequence.

There are two issues you have to follow:

1. set FIXGRID-TRIPLESORT to "true"
2. on Java side you need to tell what the original sequence of the grid items is. This is done by keeping an "originalIndex" in each grid item.

The "originalIndex" can be set by calling the method "applyOriginalIndexToCurrentContent" - in this case the items of the grid are indexed with their current order.

Please make sure: in case of updating the grid (e.g. adding or removing items), you need to re-apply the index again. Otherwise new items will be treated with a default index of "-1".

This is the Java code:

package workplace;

import java.io.Serializable;

import org.eclnt.editor.annotations.CCGenClass;
import org.eclnt.jsfserver.elements.impl.FIXGRIDItem;
import org.eclnt.jsfserver.elements.impl.FIXGRIDListBinding;
import org.eclnt.workplace.IWorkpageDispatcher;

@CCGenClass (expressionBase="#{d.DemoGridTripleSort}")

public class DemoGridTripleSort
    extends WorkpageDispatchedPageBean 
    implements Serializable
{
    // ------------------------------------------------------------------------
    // inner classes
    // ------------------------------------------------------------------------

    public class GridItem extends FIXGRIDItem implements java.io.Serializable
    {
        String i_value1;
        String i_value2;
        String i_value3;
        public String getValue3() { return i_value3; }
        public void setValue3(String value) { this.i_value3 = value; }
        public String getValue2() { return i_value2; }
        public void setValue2(String value) { this.i_value2 = value; }
        public String getValue1() { return i_value1; }
        public void setValue1(String value) { this.i_value1 = value; }
        public void onRowSelect()
        {
        }
        public void onRowExecute()
        {
        }
    }

    // ------------------------------------------------------------------------
    // members
    // ------------------------------------------------------------------------

    boolean m_tripleState = true;
    FIXGRIDListBinding<GridItem> m_grid = new FIXGRIDListBinding<GridItem>();

    // ------------------------------------------------------------------------
    // constructors & initialization
    // ------------------------------------------------------------------------

    public DemoGridTripleSort(IWorkpageDispatcher workpageDispatcher)
    {
        super(workpageDispatcher);
        fillGrid();
    }

    public String getPageName() { return "/workplace/demogridtriplesort.jsp"; }
    public String getRootExpressionUsedInPage() { return "#{d.DemoGridTripleSort}"; }

    // ------------------------------------------------------------------------
    // public usage
    // ------------------------------------------------------------------------

    public FIXGRIDListBinding<GridItem> getGrid() { return m_grid; }

    public boolean getTripleState() { return m_tripleState; }
    public void setTripleState(boolean value) { this.m_tripleState = value; }

    // ------------------------------------------------------------------------
    // private usage
    // ------------------------------------------------------------------------
    
    private void fillGrid()
    {
        for (int i=0; i<100; i++)
        {
            GridItem gi = new GridItem();
            gi.setValue1("Value " + makeTwoDigitString((i+7)%8));
            gi.setValue2("Value " + makeTwoDigitString((i+3)%5));
            gi.setValue3("Value " + makeTwoDigitString((i+22)%50));
            m_grid.getItems().add(gi);
        }
        m_grid.applyOriginalIndexToCurrentContent();
    }
    
    private String makeTwoDigitString(int i)
    {
        String s = ""+i;
        if (s.length() == 1)
            return "0"+s;
        else
            return s;
    }
}
