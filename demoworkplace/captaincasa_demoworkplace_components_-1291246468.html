The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • Components
  • Animation & Graphics & Sound
  • Content "flying in"
  • Basic demo

The FLYINCONTENTAREA component animates an update of some screen content: the updated content is "flying in" from a dedicated position.

In the demo there are 4 components with the ids "AAA", "BBB", "CCC", "DDD". By pressing the button...

1. the content of the inner area is updated (by a counter)
2. the FLYINCONTENTAREA-TRIGGER is invoked so that the animation starts

The FLYINCONTENTAREA provides an attribute STARTPOINT. There are two possibilities to defined the start point:

1. by referencing the center of a component by using the component's id ("component(...id...)"). This is the way the start point is defined in this demo.
2. by directly referencing an absolute coordinage ("point(x,y)")

This is the Java code:

package workplace;

import java.io.Serializable;

import org.eclnt.editor.annotations.CCGenClass;
import org.eclnt.jsfserver.elements.util.Trigger;
import org.eclnt.workplace.IWorkpageDispatcher;

@CCGenClass (expressionBase="#{d.DemoFlyInContentArea2}")

public class DemoFlyInContentArea2
    extends PageBean 
    implements Serializable
{
    // ------------------------------------------------------------------------
    // inner classes
    // ------------------------------------------------------------------------

    /* Listener to the user of the page bean. */
    public interface IListener extends Serializable
    {
    }

    // ------------------------------------------------------------------------
    // members
    // ------------------------------------------------------------------------

    Trigger m_trigger = new Trigger();
    String m_fromId = "AAA";
    int m_counter = 1;

    // ------------------------------------------------------------------------
    // constructors & initialization
    // ------------------------------------------------------------------------

    public DemoFlyInContentArea2(IWorkpageDispatcher workpageDispatcher)
    {
        super(workpageDispatcher);        
    }

    public String getPageName() { return "/workplace/demoflyincontentarea2.xml"; }
    public String getRootExpressionUsedInPage() { return "#{d.DemoFlyInContentArea2}"; }

    // ------------------------------------------------------------------------
    // public usage
    // ------------------------------------------------------------------------

    public Trigger getTrigger() { return m_trigger; }

    public String getFromId() { return m_fromId; }
    public void setFromId(String value) { this.m_fromId = value; }

    public String getText() { return m_counter+""; }

    public void onFlyInAction(javax.faces.event.ActionEvent event) 
    {
        m_counter++;
        m_trigger.trigger();
    }

    public String getFylInStartPoint() 
    { 
        return "component("+m_fromId+")"; 
    }
    
    // ------------------------------------------------------------------------
    // private usage
    // ------------------------------------------------------------------------
}
