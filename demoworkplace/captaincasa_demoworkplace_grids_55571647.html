The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • Grids & Trees
  • Grids and Trees
  • Special Issues
  • Synchronized horizontal scrolling

There are situations in which you want to synchronize the horizontal scrolling of the grid content across multiple grids.

To do so:

1. Define an explicit ID for the grid that is the one to scroll.
2. From the other grid reference this ID by using the attribute SCROLLREFERENCEHORIZONTAL.
3. Make sure that the width of both grids is the same (i.e. the width of the contained columns).

In the example both grids are listening to one another, so that the user can both scroll in the top and in the bottom grid.

This is the XML layout definition:

<?xml version="1.0" encoding="utf-8"?>
<t:layoutdefinition xmlns:t="http://www.CaptainCasa.com/controllibrary/t" xmlns:tpbc="http://www.CaptainCasa.com/controllibrary/tpbc" >
<t:rowdemobodypane id="g_1" objectbinding="#{d.DemoGridScrollingSynch}">
<t:row id="g_2" >
<t:fixgrid id="MASTERGRID1" objectbinding="#{d.DemoGridScrollingSynch.grid1}" sbvisibleamount="5" scrollreferencehorizontal="MASTERGRID2" width="100%" >
<t:gridcol id="g_3" text="Name" width="2500" >
<t:label id="g_4" text=".{name}" />
</t:gridcol>
</t:fixgrid>
</t:row>
<t:row id="g_5" >
<t:fixgrid id="MASTERGRID2" objectbinding="#{d.DemoGridScrollingSynch.grid2}" sbvisibleamount="5" scrollreferencehorizontal="MASTERGRID1" width="100%" >
<t:gridcol id="g_6" columnresizingenabled="false" text="Street" width="1250" >
<t:label id="g_7" text=".{street}" />
</t:gridcol>
<t:gridcol id="g_8" columnresizingenabled="false" text="Street" width="1250" >
<t:label id="g_9" text=".{city}" />
</t:gridcol>
</t:fixgrid>
</t:row>
</t:rowdemobodypane>
<t:pageaddons id="g_pa"/>
</t:layoutdefinition>

This is the Java code:

package workplace;

import java.io.Serializable;

import org.eclnt.editor.annotations.CCGenClass;
import org.eclnt.jsfserver.elements.impl.FIXGRIDItem;
import org.eclnt.jsfserver.elements.impl.FIXGRIDListBinding;
import org.eclnt.workplace.IWorkpageDispatcher;
import org.eclnt.workplace.WorkpageDispatchedPageBean;

@CCGenClass (expressionBase="#{d.DemoGridScrollingSynch}")

public class DemoGridScrollingSynch
    extends PageBean 
    implements Serializable
{
    // ------------------------------------------------------------------------
    // inner classes
    // ------------------------------------------------------------------------

    public class GridItem1 extends FIXGRIDItem implements java.io.Serializable
    {
        String i_name;
        public String getName() { return i_name; }
    }

    public class GridItem2 extends FIXGRIDItem implements java.io.Serializable
    {
        String i_street;
        String i_city;
        public String getStreet() { return i_street; }
        public String getCity() { return i_city; }
    }

    // ------------------------------------------------------------------------
    // members
    // ------------------------------------------------------------------------

    FIXGRIDListBinding<GridItem1> m_grid1 = new FIXGRIDListBinding<GridItem1>();
    FIXGRIDListBinding<GridItem2> m_grid2 = new FIXGRIDListBinding<GridItem2>();

    // ------------------------------------------------------------------------
    // constructors & initialization
    // ------------------------------------------------------------------------

    public DemoGridScrollingSynch(IWorkpageDispatcher workpageDispatcher)
    {
        super(workpageDispatcher);
        for (int i=0; i<20; i++)
        {
            GridItem1 gi = new GridItem1();
            gi.i_name = "Name " + i;
            m_grid1.getItems().add(gi);
        }
        for (int i=0; i<50; i++)
        {
            GridItem2 gi = new GridItem2();
            gi.i_street = "Street " + i;
            gi.i_city = "City " + i;
            m_grid2.getItems().add(gi);
        }
    }

    public String getPageName() { return "/workplace/demogridscrollingsynch.jsp"; }
    public String getRootExpressionUsedInPage() { return "#{d.DemoGridScrollingSynch}"; }

    // ------------------------------------------------------------------------
    // public usage
    // ------------------------------------------------------------------------

    public FIXGRIDListBinding<GridItem2> getGrid2() { return m_grid2; }
    public FIXGRIDListBinding<GridItem1> getGrid1() { return m_grid1; }

    // ------------------------------------------------------------------------
    // private usage
    // ------------------------------------------------------------------------
}
