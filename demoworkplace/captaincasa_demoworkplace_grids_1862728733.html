The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • Grids & Trees
  • Grids and Trees
  • Tree
  • Only show node's subcomponent

When setting flag ONLYSHOWSUBCOMPONENT to true, then the TREENODE does not show its default text (and optional image) node, but it only shows its contained sub component.

In the example the sub component is a FIELD component, that is bound to the text of the tree node. As result you can directly edit the tree node's text.

The sub component is sized, so that it automatically (independent from its WIDTH/HEIGHT definition) receives the full available width/height. You may add a PANE as sub component internally splitting up into one or several rows that then contain explicitly sized components.

This is the XML layout definition:

<?xml version="1.0" encoding="utf-8"?>
<t:layoutdefinition xmlns:t="http://www.CaptainCasa.com/controllibrary/t" xmlns:tpbc="http://www.CaptainCasa.com/controllibrary/tpbc" >
<t:rowdemobodypane id="g_1" objectbinding="#{d.DemoTreeOnlyShowSubComponent}" >
<t:row id="g_2" >
<t:fixgrid id="g_3" background="#FFFFFF" height="100%" objectbinding="#{d.DemoTreeOnlyShowSubComponent.tree}" sbvisibleamount="25" width="100%" >
<t:gridcol id="g_4" text="Tree Node" width="100%" >
<t:treenode id="g_5" onlyshowsubcomponent="true" >
<t:field id="g_6" background="#00000000" border="left:0;right:0;top:0;bottom:0" text=".{text}" width="100%" />
</t:treenode>
</t:gridcol>
</t:fixgrid>
</t:row>
</t:rowdemobodypane>
<t:pageaddons id="g_pa"/>
</t:layoutdefinition>

This is the Java code:

package workplace;

import java.io.Serializable;

import org.eclnt.editor.annotations.CCGenClass;
import org.eclnt.jsfserver.elements.impl.FIXGRIDTreeBinding;
import org.eclnt.jsfserver.elements.impl.FIXGRIDTreeItem;
import org.eclnt.workplace.IWorkpageDispatcher;
import org.eclnt.workplace.WorkpageDispatchedPageBean;

@CCGenClass (expressionBase="#{d.DemoTreeOnlyShowSubComponent}")

public class DemoTreeOnlyShowSubComponent
    extends WorkpageDispatchedPageBean 
    implements Serializable
{
    FIXGRIDTreeBinding<TreeNode> m_tree = new FIXGRIDTreeBinding<TreeNode>();
    public FIXGRIDTreeBinding<TreeNode> getTree() { return m_tree; }

    public class TreeNode extends FIXGRIDTreeItem implements java.io.Serializable
    {

        public TreeNode(FIXGRIDTreeItem parentNode)
        {
            super(parentNode);
        }
        
    }

    // ------------------------------------------------------------------------
    // inner classes
    // ------------------------------------------------------------------------

    /* Listener to the user of the page bean. */
    public interface IListener
    {
    }

    // ------------------------------------------------------------------------
    // members
    // ------------------------------------------------------------------------

    private IListener m_listener;

    // ------------------------------------------------------------------------
    // constructors & initialization
    // ------------------------------------------------------------------------

    public DemoTreeOnlyShowSubComponent(IWorkpageDispatcher workpageDispatcher)
    {
        super(workpageDispatcher);   
        for (int i=0; i<10; i++)
        {
            TreeNode top = new TreeNode(m_tree.getRootNode());
            top.setText("Top " + i);
            top.setStatus(TreeNode.STATUS_CLOSED);
            for (int j=0; j<10; j++)
            {
                TreeNode sub = new TreeNode(top);
                sub.setText("Sub " + i + "/" + j);
                sub.setStatus(TreeNode.STATUS_ENDNODE);
            }
        }
    }

    public String getPageName() { return "/workplace/demotreeonlyshowsubcomponent.jsp"; }
    public String getRootExpressionUsedInPage() { return "#{d.DemoTreeOnlyShowSubComponent}"; }

    // ------------------------------------------------------------------------
    // public usage
    // ------------------------------------------------------------------------

    /* Initialization of the bean. Add any parameter that is required within your scenario. */
    public void prepare(IListener listener)
    {
        m_listener = listener;
    }

    // ------------------------------------------------------------------------
    // private usage
    // ------------------------------------------------------------------------
}
