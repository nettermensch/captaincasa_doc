The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • Components
  • Animation & Graphics & Sound
  • Media Player (Java Client only)

With the MEDIAPLAYER component you can play media files such as MP4 movies.

Please check the exact media formats that are suppored within the Java documentation (https://docs.oracle.com/javafx/2/media/overview.htm).

The component sends a notification to the server side if some important status change happens inside the component. Currently the component sends a message "ENDED" if the user has seen the fully video. You can as consequence make sure that a user must see a certain video to the end (e.g. if containing security instructions).

This is the Java code:

package workplace;

import java.io.Serializable;

import javax.faces.event.ActionEvent;

import org.eclnt.editor.annotations.CCGenClass;
import org.eclnt.jsfserver.defaultscreens.Statusbar;
import org.eclnt.jsfserver.elements.events.BaseActionEventMediaPlayer;
import org.eclnt.jsfserver.elements.util.Trigger;
import org.eclnt.workplace.IWorkpageDispatcher;

@CCGenClass (expressionBase="#{d.DemoMediaPlayer}")

public class DemoMediaPlayer
    extends PageBean 
    implements Serializable
{
    // ------------------------------------------------------------------------
    // members
    // ------------------------------------------------------------------------

    String m_mediaURL;

    Trigger m_triggerStop = new Trigger();
    Trigger m_triggerPause = new Trigger();
    Trigger m_triggerPlay = new Trigger();
    
    // ------------------------------------------------------------------------
    // constructors
    // ------------------------------------------------------------------------

    public DemoMediaPlayer(IWorkpageDispatcher workpageDispatcher)
    {
        super(workpageDispatcher);        
    }

    public String getPageName() { return "/workplace/demomediaplayer.jsp"; }
    public String getRootExpressionUsedInPage() { return "#{d.DemoMediaPlayer}"; }

    // ------------------------------------------------------------------------
    // public usage
    // ------------------------------------------------------------------------

    public String getMediaURL() { return m_mediaURL; }
    public void setMediaURL(String value) { this.m_mediaURL = value; }
    
    public void onMediaPlayerAction(ActionEvent event) 
    {
        if (event instanceof BaseActionEventMediaPlayer)
        {
            BaseActionEventMediaPlayer e = (BaseActionEventMediaPlayer)event;
            Statusbar.outputSuccess("Received message from Media Player: " + e.getMessage());
        }
    }

    public Trigger getTriggerStop() { return m_triggerStop; }
    public Trigger getTriggerPause() { return m_triggerPause; }
    public Trigger getTriggerPlay() { return m_triggerPlay; }

    public void onTriggerStopAction(ActionEvent event) { m_triggerStop.trigger(); }
    public void onTriggerPauseAction(ActionEvent event) { m_triggerPause.trigger(); }
    public void onTriggerPlayAction(ActionEvent event) { m_triggerPlay.trigger(); }

    // ------------------------------------------------------------------------
    // private usage
    // ------------------------------------------------------------------------
}
