The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • General
  • Tool Tips / User Hints
  • Direct text definition

The attribute TOOLTIP is available for (nearly) all components, that are optical components.

You either define a straight text - or you define simple HTML text. HTML text needs to be defined as "<html>....</html>".

This is the XML layout definition:

<?xml version="1.0" encoding="utf-8"?>

<t:row id="g_2" >
