The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • Containers
  • Adaptive Containers - Controls
  • Adaptive matrix area
  • Example with form content

The component ROWSPANMATRIXAREA is a very useful component for structuring adaptive forms. It arranges the content items within a matrix, so that width and height of the corresponding item are consistently positioned both vertically and horizontally.

The width is divided into columns - the size an number of columns is driven by the attribute MATRIXWIDTH.

Each content item (in the example: the FOLDABLEPANE instances) is placed into this column definition so that the result is a matrix of components, each one occupying the number of columns it requires by its content.

A content item (in the example: the FOLDABLEPANE holding the grid) can explicitly request the spanning of columns by setting its attribute COLSPAN.

This is the Java code:

package workplace;

import java.io.Serializable;

import org.eclnt.editor.annotations.CCGenClass;
import org.eclnt.workplace.IWorkpageDispatcher;

@CCGenClass (expressionBase="#{d.DemoAdaptiveSpans}")

public class DemoAdaptiveSpans
    extends PageBean 
    implements Serializable
{
    // ------------------------------------------------------------------------
    // inner classes
    // ------------------------------------------------------------------------

    // ------------------------------------------------------------------------
    // members
    // ------------------------------------------------------------------------

    int m_scaleInt = 60;
    
    // ------------------------------------------------------------------------
    // constructors & initialization
    // ------------------------------------------------------------------------

    public DemoAdaptiveSpans(IWorkpageDispatcher workpageDispatcher)
    {
        super(workpageDispatcher);        
    }

    public String getPageName() { return "/workplace/demoadaptivespans.xml"; }
    public String getRootExpressionUsedInPage() { return "#{d.DemoAdaptiveSpans}"; }

    // ------------------------------------------------------------------------
    // public usage
    // ------------------------------------------------------------------------

    public int getScaleInt() { return m_scaleInt; }
    public void setScaleInt(int value) { m_scaleInt = value; }
    
    public double getScaleDouble() 
    {
        if (m_scaleInt == 100) 
            return 1d;
        else
            return (double)m_scaleInt/100d;
    }

    public void onScaleIntAction(javax.faces.event.ActionEvent event) 
    {
    }

    // ------------------------------------------------------------------------
    // private usage
    // ------------------------------------------------------------------------
}
