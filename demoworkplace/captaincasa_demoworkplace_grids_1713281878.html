The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • Grids & Trees
  • Grids and Trees
  • Special Issues
  • Grid cell border

By default the border of a component that is used as grid cell is cleared out by the grid management. Reason: a grid does not look nice, if all cell components (e.g. buttons, fields) come with their own border rendering - especially if there is some additional grid cell border.

You can override this behavior by using attribute GRIDCOL-KEEPCELLBORDER. If settin the attribute to "true" then the border of the cell component will explicitly be rendered - there is no implicit clearing.

In the grid of this example you see the first column being rendered as default column: the contained FIELD component has some border, but this is cleared by the grid processing. The second column is defined to keep the border - and as consequence it is rendered accordingly.

This is the XML layout definition:

<?xml version="1.0" encoding="utf-8"?>
<t:layoutdefinition xmlns:t="http://www.CaptainCasa.com/controllibrary/t" xmlns:tpbc="http://www.CaptainCasa.com/controllibrary/tpbc" >
<t:rowdemobodypane id="g_1" objectbinding="#{d.DemoGridCellBorder}" >
<t:row id="g_2" >
<t:fixgrid id="g_3" height="100%" objectbinding="#{d.DemoGridCellBorder.grid}" sbvisibleamount="35" width="100%" >
<t:gridcol id="g_4" keepcellborder="false" text="First Name" width="50%;100" >
<t:field id="g_5" border="#FF0000" text=".{firstName}" />
</t:gridcol>
<t:gridcol id="g_6" keepcellborder="true" text="First Name" width="50%;100" >
<t:field id="g_7" border="#FF0000" text=".{firstName}" />
</t:gridcol>
</t:fixgrid>
</t:row>
</t:rowdemobodypane>
<t:pageaddons id="g_pa"/>
</t:layoutdefinition>

This is the Java code:

package workplace;

import java.io.Serializable;

import org.eclnt.editor.annotations.CCGenClass;
import org.eclnt.jsfserver.elements.impl.FIXGRIDItem;
import org.eclnt.jsfserver.elements.impl.FIXGRIDListBinding;
import org.eclnt.jsfserver.pagebean.PageBean;
import org.eclnt.workplace.IWorkpageDispatcher;

@CCGenClass (expressionBase="#{d.DemoGridCellBorder}")

public class DemoGridCellBorder
    extends PageBean 
    implements Serializable
{
    FIXGRIDListBinding<GridItem> m_grid = new FIXGRIDListBinding<GridItem>();
    public FIXGRIDListBinding<GridItem> getGrid() { return m_grid; }

    public class GridItem extends FIXGRIDItem implements java.io.Serializable
    {
        String i_lastName;
        public String getLastName() { return i_lastName; }
        public void setLastName(String value) { this.i_lastName = value; }

        String i_firstName;
        public String getFirstName() { return i_firstName; }
        public void setFirstName(String value) { this.i_firstName = value; }

        public void onRowSelect()
        {
        }
        public void onRowExecute()
        {
        }
    }

    // ------------------------------------------------------------------------
    // inner classes
    // ------------------------------------------------------------------------
    
    /* Listener to the user of the page bean. */
    public interface IListener
    {
    }
    
    // ------------------------------------------------------------------------
    // members
    // ------------------------------------------------------------------------
    
    private IListener m_listener;
    
    // ------------------------------------------------------------------------
    // constructors & initialization
    // ------------------------------------------------------------------------

    public DemoGridCellBorder(IWorkpageDispatcher dispatcher)
    {
        super(dispatcher);
        for (int i=0; i<100; i++)
        {
            GridItem gi = new GridItem();
            gi.setFirstName("First " + i);
            gi.setLastName("Last " +i);
            m_grid.getItems().add(gi);
        }
    }

    public String getPageName() { return "/workplace/demogridcellborder.jsp"; }
    public String getRootExpressionUsedInPage() { return "#{d.DemoGridCellBorder}"; }

    // ------------------------------------------------------------------------
    // public usage
    // ------------------------------------------------------------------------

    /* Initialization of the bean. Add any parameter that is required within your scenario. */
    public void prepare(IListener listener)
    {
        m_listener = listener;
    }

    // ------------------------------------------------------------------------
    // private usage
    // ------------------------------------------------------------------------
}
