The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • General
  • RISC Client Integration
  • Playing videos

The VIDEO component is a representation of the HTML "video" element - and plays videos.

Please pay attention: check the documentation of the HTML "video" element in order to find the exact video formats that are supported on various platforms!

This is the Java code:

package workplace;

import java.io.Serializable;

import org.eclnt.editor.annotations.CCGenClass;
import org.eclnt.workplace.IWorkpageDispatcher;

@CCGenClass (expressionBase="#{d.DemoVideo}")

public class DemoVideo
    extends PageBean 
    implements Serializable
{
    // ------------------------------------------------------------------------
    // inner classes
    // ------------------------------------------------------------------------

    // ------------------------------------------------------------------------
    // members
    // ------------------------------------------------------------------------

    String m_videoSrc ="/videos/sample.webm";

    // ------------------------------------------------------------------------
    // constructors & initialization
    // ------------------------------------------------------------------------

    public DemoVideo(IWorkpageDispatcher workpageDispatcher)
    {
        super(workpageDispatcher);        
    }

    public String getPageName() { return "/workplace/demovideo.jsp"; }
    public String getRootExpressionUsedInPage() { return "#{d.DemoVideo}"; }

    // ------------------------------------------------------------------------
    // public usage
    // ------------------------------------------------------------------------

    public String getVideoSrc() { return m_videoSrc; }
    public void setVideoSrc(String value) { this.m_videoSrc = value; }


    // ------------------------------------------------------------------------
    // private usage
    // ------------------------------------------------------------------------
}
