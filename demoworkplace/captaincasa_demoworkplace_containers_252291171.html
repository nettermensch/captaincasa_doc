The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • Containers
  • Adaptive Containers - Controls
  • Size transfer - "100% flexibility"

Use the SIZETRANSFER component in order to add any adaptive functions to your layout processing that you want to implement on your own.

The SIZETRANSFER component transfers the actual current width/height of a certain PAINE to the server side. The transfer is done either by pixel value (default) or by size categories.

In the example you see:

1. There are size categories both for transferring the width and the height. The height is not relevant for the adaptive functions in this example so there is only one general "ALL"-height transferred.

2. Changes of the size are transferred without any further activity required (FLUSH set to true)

3. Changed of the size are transferred without any latency (FLUSHTIMER set to 0)

On server side the sizing of the icons is "calculated" based on the width category that is sent from the client side.

This is the Java code:

package workplace;

import java.io.Serializable;
import org.eclnt.editor.annotations.CCGenClass;
import org.eclnt.jsfserver.managedbean.IDispatcher;
import org.eclnt.jsfserver.pagebean.PageBean;
import org.eclnt.util.valuemgmt.ValueManager;
import org.eclnt.workplace.IWorkpageDispatcher;

import javax.faces.event.ActionEvent;

@CCGenClass (expressionBase="#{d.DemoAdaptiveSizeTransfer1}")

public class DemoAdaptiveSizeTransfer1
    extends PageBean 
    implements Serializable
{
    public void onSizeAction(javax.faces.event.ActionEvent event) 
    {
        String[] widthHeight = ValueManager.decodeCSV(m_size);
        String width = widthHeight[0];
        if ("LARGE".equals(width)) m_iconWidthHeight = 64;
        else if ("MEDIUM".equals(width)) m_iconWidthHeight = 48;
        else if ("SMALL".equals(width)) m_iconWidthHeight = 32;
        else if ("VERYSMALL".equals(width)) m_iconWidthHeight = 16;
    }

    int m_iconWidthHeight = 64;
    public int getIconWidthHeight() { return m_iconWidthHeight; }
 
    String m_size = "LARGE";
    public String getSize() { return m_size; }
    public void setSize(String value) { this.m_size = value; }

    // ------------------------------------------------------------------------
    // inner classes
    // ------------------------------------------------------------------------
    
    /* Listener to the user of the page bean. */
    public interface IListener extends Serializable
    {
    }
    
    // ------------------------------------------------------------------------
    // members
    // ------------------------------------------------------------------------
    
    private IListener m_listener;
    
    // ------------------------------------------------------------------------
    // constructors & initialization
    // ------------------------------------------------------------------------

    public DemoAdaptiveSizeTransfer1(IDispatcher dispatcher)
    {
        super(dispatcher);
    }

    public String getPageName() { return "/workplace/demoadaptivesizetransfer1.jsp"; }
    public DemoAdaptiveSizeTransfer1(IWorkpageDispatcher dispatcher)
    {
        super(dispatcher);
        // TODO Auto-generated constructor stub
    }
    public String getRootExpressionUsedInPage() { return "#{d.DemoAdaptiveSizeTransfer1}"; }

    // ------------------------------------------------------------------------
    // public usage
    // ------------------------------------------------------------------------

    /* Initialization of the bean. Add any parameter that is required within your scenario. */
    public void prepare(IListener listener)
    {
        m_listener = listener;
    }

    // ------------------------------------------------------------------------
    // private usage
    // ------------------------------------------------------------------------
}
