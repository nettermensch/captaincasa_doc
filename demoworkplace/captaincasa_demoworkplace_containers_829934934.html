The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • Containers
  • Split Pane - Extended Functions
  • Hide split areas (+ divider)

Sometimes you want to completely hide the one of the split areas of a split pane.

To do so there are two attributes:

(.) HIDEFIRSTSPLIT
(.) HIDESECONDSPLIT

By passing boolean values you can control the visibility of both areas.

What's the difference to setting the DIVIDERLOCATION to "0"?
If hiding the areas, then also the divider between the areas is hidden so that the user is not able anymore to open the corresponding area by moving the divider.

This is the Java code:

package workplace;

import java.io.Serializable;

import org.eclnt.editor.annotations.CCGenClass;
import org.eclnt.workplace.IWorkpageDispatcher;

@CCGenClass (expressionBase="#{d.DemoSplitPaneHide}")

public class DemoSplitPaneHide
    extends PageBean 
    implements Serializable
{
    // ------------------------------------------------------------------------
    // members
    // ------------------------------------------------------------------------

    boolean m_rightHidden = false;
    boolean m_leftHidden = false;

    // ------------------------------------------------------------------------
    // constructors & initialization
    // ------------------------------------------------------------------------

    public DemoSplitPaneHide(IWorkpageDispatcher workpageDispatcher)
    {
        super(workpageDispatcher);        
    }

    public String getPageName() { return "/workplace/demosplitpanehide.jsp"; }
    public String getRootExpressionUsedInPage() { return "#{d.DemoSplitPaneHide}"; }

    // ------------------------------------------------------------------------
    // public usage
    // ------------------------------------------------------------------------

    public boolean getRightHidden() { return m_rightHidden; }
    public void setRightHidden(boolean value) { this.m_rightHidden = value; }

    public boolean getLeftHidden() { return m_leftHidden; }
    public void setLeftHidden(boolean value) { this.m_leftHidden = value; }

}
