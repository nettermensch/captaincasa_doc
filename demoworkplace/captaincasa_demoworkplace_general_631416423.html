The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • General
  • Client Integration
  • Start Mail Client

The MAILTO component is an interface to the client side mail system. Internally the data is passed by building a "mailto:..."-URL and sending this via the client side Desktop interface.

This is the XML layout definition:

This is the Java code:

package workplace;

import java.io.Serializable;

import javax.faces.event.ActionEvent;

import org.eclnt.editor.annotations.CCGenClass;
import org.eclnt.jsfserver.elements.util.Trigger;
import org.eclnt.workplace.IWorkpageDispatcher;

@CCGenClass (expressionBase="#{d.DemoMailTo}")

public class DemoMailTo
    extends PageBean
    implements Serializable
{
    public DemoMailTo(IWorkpageDispatcher dispatcher)
    {
        super(dispatcher);
    }

    protected Trigger m_sendTrigger = new Trigger();
    public Trigger getSendTrigger() { return m_sendTrigger; }

    protected String m_body = "Some body text.";
    public String getBody() { return m_body; }
    public void setBody(String value) { m_body = value; }

    protected String m_subject = "Some subject text";
    public String getSubject() { return m_subject; }
    public void setSubject(String value) { m_subject = value; }

    protected String m_receiver = "info@captaincasa.com";
    public String getReceiver() { return m_receiver; }
    public void setReceiver(String value) { m_receiver = value; }

    public void onOpenMail(ActionEvent event) 
    {
        m_sendTrigger.trigger();
    }

}
