The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • Components
  • Straight Controls
  • Statusbar and Messages
  • Auto Clearing

By calling method Statusbar.autoClearStatusbar(duration), the status bar message will be cleared after the duration that is passed as argument.

This is the XML layout definition:

<?xml version="1.0" encoding="utf-8"?>

<t:rowdemobodypane id="g_1" objectbinding="#{d.DemoStatusbarAutoClear}" rowdistance="#{ccstylevalue['@defaultrowdistance@']}" >

This is the Java code:

package workplace;

import java.io.Serializable;
import org.eclnt.editor.annotations.CCGenClass;
import org.eclnt.jsfserver.defaultscreens.Statusbar;
import org.eclnt.workplace.IWorkpageDispatcher;
import org.eclnt.workplace.WorkpageDispatchedBean;

import javax.faces.event.ActionEvent;

@CCGenClass (expressionBase="#{d.DemoStatusbarAutoClear}")

public class DemoStatusbarAutoClear
    extends PageBean 
    implements Serializable
{
    String m_message = "This is the message to be displayed.";
    public String getMessage() { return m_message; }
    public void setMessage(String value) { this.m_message = value; }

    public void onAlert(ActionEvent event) 
    {
        Statusbar.outputAlert(m_message);
        Statusbar.autoClearStatusbar(2000);
    }

    public void onError(ActionEvent event) 
    {
        Statusbar.outputError(m_message);
        Statusbar.autoClearStatusbar(2000);
    }

    // ------------------------------------------------------------------------
    // constructors & initialization
    // ------------------------------------------------------------------------

    public DemoStatusbarAutoClear(IWorkpageDispatcher workpageDispatcher)
    {
        super(workpageDispatcher);        
    }

    // ------------------------------------------------------------------------
}
