The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • Components
  • Mobile Periphery
  • Date and time input

The controls TOUCHDATE and TOUCHTIME are page bean component controls that are part of the eclnt_pbc package.

They are used to input a date or a time in a touch screen environment. The little green arrow icons are used for quicker navigation.

The server side binding can be a LocalDate, a Date, a Long or a String (containing a long value).

This is the Java code:

package workplace;

import java.io.Serializable;
import java.time.LocalDate;
import java.time.LocalTime;

import org.eclnt.editor.annotations.CCGenClass;
import org.eclnt.jsfserver.defaultscreens.Statusbar;
import org.eclnt.workplace.IWorkpageDispatcher;

@CCGenClass (expressionBase="#{d.DemoTouchDateTimeControl}")

public class DemoTouchDateTimeControl
    extends WorkpageDispatchedPageBean 
    implements Serializable
{
    // ------------------------------------------------------------------------
    // inner classes
    // ------------------------------------------------------------------------

    // ------------------------------------------------------------------------
    // members
    // ------------------------------------------------------------------------

    LocalDate m_myDate = LocalDate.now();
    LocalTime m_myTime = LocalTime.now();

    // ------------------------------------------------------------------------
    // constructors & initialization
    // ------------------------------------------------------------------------

    public DemoTouchDateTimeControl(IWorkpageDispatcher workpageDispatcher)
    {
        super(workpageDispatcher);        
    }

    public String getPageName() { return "/workplace/demotouchdatetimecontrol.xml"; }
    public String getRootExpressionUsedInPage() { return "#{d.DemoTouchDateTimeControl}"; }

    // ------------------------------------------------------------------------
    // public usage
    // ------------------------------------------------------------------------

    public LocalDate getMyDate() { return m_myDate; }
    public void setMyDate(LocalDate value) { this.m_myDate = value; }

    public LocalTime getMyTime() { return m_myTime; }
    public void setMyTime(LocalTime value) { this.m_myTime = value; }

    public void onTouchTimeAction(javax.faces.event.ActionEvent event) 
    {
        Statusbar.outputMessage("Time was changed by touch input: " + m_myTime.toString());
    }

    public void onTouchDateAction(javax.faces.event.ActionEvent event) 
    {
        Statusbar.outputMessage("Date was changed by touch input: " + m_myDate.toString());
    }

    // ------------------------------------------------------------------------
    // private usage
    // ------------------------------------------------------------------------
}
