The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • Containers
  • Adaptive Containers - Controls
  • Flexible columns

The items of the ROWFLEXCOLUMNCONTAINER are arranged in one or more columns - dependent from the current height of the component.

The control is the vertical counter part to the ROWFLEXLINECONTAINER, in which the items are arranged in a dynamic number of rows.

Please note: the control is available in the FX-Client only. In the Swing-client the control is the same as the control ROWFLEXLINECONTAINER.
