The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • Containers
  • Layering and absolute postioning
  • Blur / frozen glass styling

This examples shows how to nicely use the "backdrop-filter" styling of HTML/CSS. It is included in the PANE-stylevariant "ccblur" and is also available as style variant for ROWBODYPANE and as style sequence "riscdialog_popup_blur" for popups.

Of course you can add own style classes and style variants to take use of this feature as well.
