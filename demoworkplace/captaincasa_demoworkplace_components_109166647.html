The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • Components
  • Straight Controls
  • Text Input/Output
  • HTML Editor
  • Quill Editor

Quill is a JavaScript based text editor which is free and open source based. Please check https://quilljs.com/

To use the Quill-editor you need to define the attribute SIMPLEHTMLEDITOR-BRIDGEPAGE with value "eclnt/risc/ext_quill/quillbridge.html".

This is the XML layout definition:
