The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • General
  • Server Side Value Binding
  • Property Change Notification - Version 1

Input some values into the fields and press OK. You will see a log telling about which values were updated on server side.

There are two ways to get notified when values are passed from the user interface processing into managed beans:

(1) Beans that are accessed via expressions (example: "#{d.address.firstName}") may optionally implement the interface "IPropertyResolverAware". In this case the bean is notified immediately after the property resolver having upodated a poperty value.

(2) There is a session wide ValueBinding listener, in which the valuebinding (the expression referenced by components) and the value that is set is passed as well. Typically you register this listener one time at the start up of your application - not in a constructor of a "sub-bean" - this is an example only!

You may use both interfaces in order to get triggered when the user interface request processing updates a property of your bean.

This is the Java code:

package workplace;

import java.io.Serializable;

import javax.el.MethodExpression;
import javax.el.ValueExpression;
import javax.faces.el.MethodBinding;
import javax.faces.event.ActionEvent;

import org.eclnt.editor.annotations.CCGenClass;
import org.eclnt.jsfserver.elements.phases.PhaseManager;
import org.eclnt.jsfserver.util.HttpSessionAccess;
import org.eclnt.jsfserver.util.IMethodBindingListener;
import org.eclnt.jsfserver.util.IPropertyResolverAware;
import org.eclnt.jsfserver.util.IValueBindingListener;
import org.eclnt.workplace.IWorkpageDispatcher;

@CCGenClass (expressionBase="#{d.DemoPropertyResolverAware}")

public class DemoPropertyResolverAware 
    extends PageBean
    implements Serializable, IPropertyResolverAware
{
    
    public class MyValueBindingListener implements IValueBindingListener
    {
        public void reactoOnValueExpressionSet(String expressionString,
                                               Object value)
        {
            m_log += "IValueBindingListener, Property change: " + expressionString + " / " + value + "\n";
        }
    }
    
    public class MyMethodBindingListener implements IMethodBindingListener
    {
        public void reactOnActionListenerInvokeAfter(String expressionString, ActionEvent event)
        {
            m_log += "IMethodBindingListener, Method call, before: " + expressionString + "\n";
        }

        public boolean reactOnActionListenerInvokeBefore(String expressionString, ActionEvent event)
        {
            m_log += "IMethodBindingListener, Method call, after: " +expressionString + "\n";
            return true;
        }
        
    }
    
    public class ResetLog implements Runnable
    {
        public void run()
        {
            m_log = "";
        }
    }
    
    public DemoPropertyResolverAware(IWorkpageDispatcher dispatcher)
    {
        super(dispatcher);
        // create value binding listener
        // pay attention the value binding listener is active for the whole application
        // now - not only for this managed bean instance.
        HttpSessionAccess.setValueBindingListener(new MyValueBindingListener());
        // same with method binding listener
        HttpSessionAccess.setMethodBindingListener(new MyMethodBindingListener());
    }
    
    protected String m_log = "";
    public String getLog() { return m_log; }
    public void setLog(String value) { m_log = value; }

    protected String m_city;
    public String getCity() { return m_city; }
    public void setCity(String value) { m_city = value; }

    protected String m_street;
    public String getStreet() { return m_street; }
    public void setStreet(String value) { m_street = value; }

    protected String m_lastName;
    public String getLastName() { return m_lastName; }
    public void setLastName(String value) { m_lastName = value; }

    protected String m_firstName;
    public String getFirstName() { return m_firstName; }
    public void setFirstName(String value) { m_firstName = value; }

    public void onOK(ActionEvent event) 
    {
        m_log += "onOK was called\n";
        PhaseManager.runAfterRenderResponsePhase(new ResetLog());
    }
    
    public void reactOnSetValue(int index, Object value)
    {
        m_log += "IPropertyResolverAware, Property change: " + index + " / " + value + "\n";
    }
    
    public void reactOnSetValue(Object property, Object value)
    {
        m_log += "IPropertyResolverAware, Property change: " + property + " / " + value + "\n";
    }
    
}
