The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • Grids & Trees
  • Grids and Trees
  • Flexible row heights
  • Dynamic height-sizing of headline

There are two functions used in this grid.

1. The GRDICOL instances are set to multi line text - by setting attribute MULTILINE to "true".

2. The FIXGRID's attribute DYNAMICHEADLINEROWHEIGHTSIZING is set to "true". As consequence the size of the headline is calculated by its content.

The items of the grid are not affected by the dynamic height-calculation of the headline row. They still are sized according to the ROWHEIGHT that is defined in the FIXGRID (with a default being defined in the style definition).

This is the Java code:

package workplace;

import java.io.Serializable;

import org.eclnt.editor.annotations.CCGenClass;
import org.eclnt.jsfserver.elements.impl.FIXGRIDItem;
import org.eclnt.jsfserver.elements.impl.FIXGRIDListBinding;
import org.eclnt.workplace.IWorkpageDispatcher;

import workplace.logic.Person;
import workplace.logic.RandomLogic;

@CCGenClass (expressionBase="#{d.DemoGridDynHeadlineSizing}")

public class DemoGridDynHeadlineSizing
    extends PageBean 
    implements Serializable
{
    // ------------------------------------------------------------------------
    // inner classes
    // ------------------------------------------------------------------------

    public class GridItem extends FIXGRIDItem implements java.io.Serializable
    {
        Person i_person = new Person();
        public GridItem(Person person)
        { 
            i_person = person; 
        }
        public Person getPerson() { return i_person; }
        public void onRowSelect()
        {
        }
        public void onRowExecute()
        {
        }
    }

    // ------------------------------------------------------------------------
    // members
    // ------------------------------------------------------------------------

    FIXGRIDListBinding<GridItem> m_grid = new FIXGRIDListBinding<GridItem>();

    // ------------------------------------------------------------------------
    // constructors & initialization
    // ------------------------------------------------------------------------

    public DemoGridDynHeadlineSizing(IWorkpageDispatcher workpageDispatcher)
    {
        super(workpageDispatcher);        
        for (int i=0; i<50; i++)
            m_grid.getItems().add(new GridItem(RandomLogic.createPerson()));
    }

    public String getPageName() { return "/workplace/demogriddynheadlinesizing.xml"; }
    public String getRootExpressionUsedInPage() { return "#{d.DemoGridDynHeadlineSizing}"; }

    // ------------------------------------------------------------------------
    // public usage
    // ------------------------------------------------------------------------

    public FIXGRIDListBinding<GridItem> getGrid() { return m_grid; }

    // ------------------------------------------------------------------------
    // private usage
    // ------------------------------------------------------------------------
}
