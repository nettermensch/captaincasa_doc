The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • Components
  • Straight Controls
  • Text Input/Output
  • Text Area

The TEXTAREA is a multi-line text input component. It represents a plain text editor - there are no sophisticated formatting functions.

This is the XML layout definition:

<?xml version="1.0" encoding="utf-8"?>
<t:layoutdefinition xmlns:t="http://www.CaptainCasa.com/controllibrary/t" xmlns:tpbc="http://www.CaptainCasa.com/controllibrary/tpbc" >
<t:rowdemobodypane id="g_1" objectbinding="#{d.DemoTextArea2}" rowdistance="#{ccstylevalue['@defaultrowdistance@']}" >
<t:row id="g_2" >
<t:textarea id="g_3" flush="true" height="50%" labeltext="Description text" text="#{d.DemoTextArea2.text}" width="100%" tabtonextcomponent="true" />
</t:row>
<t:row id="g_4" >
<t:button id="g_5" actionListener="#{d.DemoTextArea2.onCount}" text="Count characters of Text" />
</t:row>
</t:rowdemobodypane>
<t:pageaddons id="g_pa"/>
</t:layoutdefinition>

This is the Java code:

package workplace;

import java.io.Serializable;

import javax.faces.event.ActionEvent;

import org.eclnt.editor.annotations.CCGenClass;
import org.eclnt.jsfserver.defaultscreens.Statusbar;
import org.eclnt.workplace.IWorkpageDispatcher;

@CCGenClass (expressionBase="#{d.DemoTextArae2}")

public class DemoTextArea2
    extends PageBean
    implements Serializable
{
    public DemoTextArea2(IWorkpageDispatcher dispatcher)
    {
        super(dispatcher);
        initRefComponent("textarea"); 
    }

    String m_text = "";
    public void setText(String value) { m_text = value; }
    public String getText() { return m_text; }
    
    public void onCount(ActionEvent event)
    {
        Statusbar.outputSuccess("Length of text = " + m_text.length());
    }
}
