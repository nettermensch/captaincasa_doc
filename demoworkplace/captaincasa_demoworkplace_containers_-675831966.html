The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • Containers
  • Grid layout pane
  • Cross-synchronization of columns - adaptive version

The same "harmonizing" of column sizes that is available in the GRIDLAYOUTPANE component is available as will within the ADAPTIVEGRIDLAYOUTPANE component.

In the example there are 3 instances of ADAPTIVEGRIDLAYOUTPANE - updating the column structure in one of them will automatically update the columns structure of the related ones.

This is the Java code:

package workplace;

import java.io.Serializable;

import org.eclnt.editor.annotations.CCGenClass;
import org.eclnt.workplace.IWorkpageDispatcher;

@CCGenClass (expressionBase="#{d.DemoGridLayoutSharedSizing}")

public class DemoGridLayoutSharedSizing2
    extends PageBean 
    implements Serializable
{
    // ------------------------------------------------------------------------
    // members
    // ------------------------------------------------------------------------

    final String M5 = "MMMMM";
    final String M20 = "MMMMMMMMMMMMMMMMMMMM";
    
    String m_labelLast = M5;
    boolean m_renderSecondPart = true;

    // ------------------------------------------------------------------------
    // constructors & initialization
    // ------------------------------------------------------------------------

    public DemoGridLayoutSharedSizing2(IWorkpageDispatcher workpageDispatcher)
    {
        super(workpageDispatcher);        
    }

    public String getPageName() { return "/workplace/demogridlayoutsharedsizing.xml"; }
    public String getRootExpressionUsedInPage() { return "#{d.DemoGridLayoutSharedSizing}"; }

    // ------------------------------------------------------------------------
    // public usage
    // ------------------------------------------------------------------------

    public String getLabelLast() { return m_labelLast; }
    public void setLabelLast(String value) { this.m_labelLast = value; }

    public boolean getRenderSecondPart() { return m_renderSecondPart; }
    public void setRenderSecondPart(boolean value) { this.m_renderSecondPart = value; }

    public void onChangeLabelAction(javax.faces.event.ActionEvent event) 
    {
        if (m_labelLast == M5)
            m_labelLast = M20;
        else
            m_labelLast = M5;
    }
}
