The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • Components
  • Straight Controls
  • Menus
  • Button Popup - Explicit closing/opening

The dialog popup which is managed by the BUTTONMENU component an be directly controlled by using triggers.

BUTTONPOPUP-CLOSETRIGGER will close the dialog (if openend).
BUTTONPOPUP-OPENTRIGGER will opene the dialog (if not yet opened).

Despite the fact that this is some simple way of opening/ closing small dialogs we do recommend to use real popups (ModalPopup, ModelessPopup) in general. Only there you have full control about the life cycle of the popup dialog!

This is the Java code:

package workplace;

import java.io.Serializable;

import org.eclnt.editor.annotations.CCGenClass;
import org.eclnt.jsfserver.elements.util.Trigger;
import org.eclnt.workplace.IWorkpageDispatcher;

@CCGenClass (expressionBase="#{d.DemoButtonPopupOpenClose}")

public class DemoButtonPopupOpenClose
    extends PageBean 
    implements Serializable
{
    // ------------------------------------------------------------------------
    // members
    // ------------------------------------------------------------------------

    Trigger m_openTrigger = new Trigger();
    Trigger m_closeTrigger = new Trigger();

    Trigger m_openTriggerMenu = new Trigger();
    Trigger m_closeTriggerMenu = new Trigger();
    
    // ------------------------------------------------------------------------
    // constructors & initialization
    // ------------------------------------------------------------------------

    public DemoButtonPopupOpenClose(IWorkpageDispatcher workpageDispatcher)
    {
        super(workpageDispatcher);        
    }

    public String getPageName() { return "/workplace/demobuttonpopupopenclose.xml"; }
    public String getRootExpressionUsedInPage() { return "#{d.DemoButtonPopupOpenClose}"; }

    // ------------------------------------------------------------------------
    // public usage
    // ------------------------------------------------------------------------

    public Trigger getOpenTrigger() { return m_openTrigger; }
    public Trigger getCloseTrigger() { return m_closeTrigger; }
    public Trigger getOpenTriggerMenu() { return m_openTriggerMenu; } 
    public Trigger getCloseTriggerMenu() { return m_closeTriggerMenu; }

    public void onCloseAction(javax.faces.event.ActionEvent event) 
    {
        m_closeTrigger.trigger();
    }

    public void onOpenAction(javax.faces.event.ActionEvent event) 
    {
        m_openTrigger.trigger();
    }

    public void onCloseMenuAction(javax.faces.event.ActionEvent event) 
    {
        m_closeTriggerMenu.trigger();
    }

    public void onOpenMenuAction(javax.faces.event.ActionEvent event) 
    {
        m_openTriggerMenu.trigger();
    }

    // ------------------------------------------------------------------------
    // private usage
    // ------------------------------------------------------------------------
}
