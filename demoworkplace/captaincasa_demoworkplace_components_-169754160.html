The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • Components
  • Straight Controls
  • Text Input/Output
  • Info Text Pane

There are three pre-configured style variants for the LABEL, which add some border and background to the label text in order to raise more attraction to the user.

The names are "ccinfored", "ccinfoyellow" and "ccinfogreen".

This is the Java code:

package workplace;

import java.io.Serializable;

import org.eclnt.editor.annotations.CCGenClass;
import org.eclnt.workplace.IWorkpageDispatcher;

@CCGenClass (expressionBase="#{d.DemoInfoLabel}")

public class DemoInfoLabel
    extends PageBean 
    implements Serializable
{
    public DemoInfoLabel(IWorkpageDispatcher workpageDispatcher)
    {
        super(workpageDispatcher);        
    }

    public String getPageName() { return "/workplace/demoinfolabel.jsp"; }
    public String getRootExpressionUsedInPage() { return "#{d.DemoInfoLabel}"; }
}
