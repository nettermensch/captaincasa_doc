The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • General
  • Client Integration
  • Embed Flash Player (Java Client only)

This example demonstrates how easy it is to integrate the Adobe Flash Player via ActiveX.

You just need to specifiy the PROGID (which is "ShockwaveFlash.ShockwaveFlash") and need to set the "Moview"-property of the ActiveX component!

This is the Java code:

package workplace;

import javax.faces.event.ActionEvent;

import org.eclnt.workplace.IWorkpageDispatcher;

public class DemoActivexFlashPlayer extends PageBean
{
    // ------------------------------------------------------------------------
    // inner classes
    // ------------------------------------------------------------------------
    
    // ------------------------------------------------------------------------
    // members
    // ------------------------------------------------------------------------

    String m_activexCommand = "set(Movie,http://www.captaincasa.com/tours/eclnt_tour_statement.swf)";
    
    // ------------------------------------------------------------------------
    // constructors
    // ------------------------------------------------------------------------
    
    public DemoActivexFlashPlayer(IWorkpageDispatcher dispatcher)
    {
        super(dispatcher);
    }

    // ------------------------------------------------------------------------
    // public usage
    // ------------------------------------------------------------------------

    public String getActivexCommand() { return m_activexCommand; }
    
    public void onSlides1(ActionEvent event)
    {
        m_activexCommand = "set(Movie,http://www.captaincasa.com/tours/eclnt_tour_statement.swf)";
    }
    
    public void onSlides2(ActionEvent event)
    {
        m_activexCommand = "set(Movie,http://www.captaincasa.com/tours/eclnt_tour_statement_technology.swf)";
    }
    
    public void onSlides3(ActionEvent event)
    {
        m_activexCommand = "set(Movie,http://www.captaincasa.com/tours/eclnt_tour_captaincasa.swf)";
    }
    
    // ------------------------------------------------------------------------
    // private usage
    // ------------------------------------------------------------------------
    
}
