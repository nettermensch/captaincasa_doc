The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • Components
  • Animation & Graphics & Sound
  • Animation used in mobile scenarios

Adding animation is simple:

1. select a PANE that should be animated
2. add a CAPTUREAINMATOR component into the PANE
3. assign a trigger and some animation type - that's it

Whenever the trigger is triggered the animation will be executed. In case of animations of type "hidecontent*", a snapshow of the current dialog is internally created and preserved. This snapshot is then overlayed on top of the updated dialog.

In the example the content of the PANE is changed by using different areas, one of them being rendered by using the RENDERED attribute. Of course you can build up different content areas in any other way, too. Example: you may switch between different content pages by using ROWPAGEBEANINCLUDE.

This is the Java code:

package workplace;

import java.io.Serializable;

import org.eclnt.editor.annotations.CCGenClass;
import org.eclnt.jsfserver.elements.util.Trigger;
import org.eclnt.workplace.IWorkpageDispatcher;
import org.eclnt.workplace.WorkpageDispatchedPageBean;

@CCGenClass (expressionBase="#{d.DemoCaptureAnimator3}")

public class DemoCaptureAnimator3
    extends WorkpageDispatchedPageBean 
    implements Serializable
{
    int m_areaCounter = 0;
    
    public void onNextAction(javax.faces.event.ActionEvent event) 
    {
        m_areaCounter++;
        if (m_areaCounter > 3) m_areaCounter = 0;
        m_animationType = "hidecontenttoleft";
        m_trigger.trigger();
    }

    public void onPreviousAction(javax.faces.event.ActionEvent event) 
    {
        m_areaCounter--;
        if (m_areaCounter < 0) m_areaCounter = 3;
        m_animationType = "hidecontenttoright";
        m_trigger.trigger();
    }

    String m_animationType;
    public String getAnimationType() { return m_animationType; }
    public void setAnimationType(String value) { this.m_animationType = value; }

    Trigger m_trigger = new Trigger();
    public Trigger getTrigger() { return m_trigger; }

    public boolean getRenderedArea3() { return m_areaCounter == 3; }

    public boolean getRenderedArea2() { return m_areaCounter == 2; }

    public boolean getRenderedArea1() { return m_areaCounter == 1; }

    public boolean getRenderedArea0() { return m_areaCounter == 0; }

    String m_title;
    public String getTitle() { return m_title; }
    public void setTitle(String value) { this.m_title = value; }

    // ------------------------------------------------------------------------
    // inner classes
    // ------------------------------------------------------------------------

    /* Listener to the user of the page bean. */
    public interface IListener
    {
    }

    // ------------------------------------------------------------------------
    // members
    // ------------------------------------------------------------------------

    private IListener m_listener;

    // ------------------------------------------------------------------------
    // constructors & initialization
    // ------------------------------------------------------------------------

    public DemoCaptureAnimator3(IWorkpageDispatcher workpageDispatcher)
    {
        super(workpageDispatcher);        
    }

    public String getPageName() { return "/workplace/democaptureanimator3.jsp"; }
    public String getRootExpressionUsedInPage() { return "#{d.DemoCaptureAnimator3}"; }

    // ------------------------------------------------------------------------
    // public usage
    // ------------------------------------------------------------------------

    /* Initialization of the bean. Add any parameter that is required within your scenario. */
    public void prepare(IListener listener)
    {
        m_listener = listener;
    }

    // ------------------------------------------------------------------------
    // private usage
    // ------------------------------------------------------------------------
}
