The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • General
  • Client Clipboard
  • Setting/Getting clipboard content

The CLIPBOARDSET component connects to the client side clipboard and simplifies the way of passing information into the clipboard.

In some browsers (IE 10/11) server-side content can be directly copied into the clipboard - in most browsers (Edge, Firefox, Chrome, ...) this is done via a helper dialog showing up. In the dialog the user can easily copy the text by pressing the ctrl-c key - which also closes the dialog.

There is a default CLIPBOARDSET component, that is automatically available on any page. This can be accessed by the Clipboard-class by using a static API.

This is the XML layout definition:

<?xml version="1.0" encoding="utf-8"?>
<t:layoutdefinition xmlns:t="http://www.CaptainCasa.com/controllibrary/t" xmlns:tpbc="http://www.CaptainCasa.com/controllibrary/tpbc" >
<t:beanprocessing id="g_1" >
<t:clipboardset id="g_2" trigger="#{d.DemoClipboard.triggerSET}" value="#{d.DemoClipboard.text}" />
</t:beanprocessing>
<t:rowdemobodypane id="g_3" objectbinding="#{d.DemoClipboard}" rowdistance="#{ccstylevalue['@defaultrowdistance@']}" >
<t:row id="g_4" >
<t:textarea id="g_5" height="100" text="#{d.DemoClipboard.text}" width="100%" />
</t:row>
<t:row id="g_6" >
<t:button id="g_7" actionListener="#{d.DemoClipboard.onCopyTextToClipboard}" text="Copy Text to Clipboard" />
</t:row>
</t:rowdemobodypane>
<t:pageaddons id="g_pa"/>
</t:layoutdefinition>

This is the Java code:

package workplace;

import java.io.Serializable;

import javax.faces.event.ActionEvent;

import org.eclnt.editor.annotations.CCGenClass;
import org.eclnt.jsfserver.defaultscreens.Clipboard;
import org.eclnt.jsfserver.defaultscreens.Statusbar;
import org.eclnt.jsfserver.elements.events.BaseActionEventClipboardGet;
import org.eclnt.jsfserver.elements.util.Trigger;
import org.eclnt.jsfserver.managedbean.IDispatcher;
import org.eclnt.jsfserver.util.WebResourceReader;

@CCGenClass (expressionBase="#{d.DemoClipboard}")

public class DemoClipboard
    extends PageBean
    implements Serializable
{
    protected String m_text = "Some\ttext";
    protected Trigger m_triggerSET = new Trigger();
    
    public DemoClipboard(IDispatcher dispatcher)
    {
        super(dispatcher);
    }
    
    public Trigger getTriggerSET() { return m_triggerSET; }
    public void setTriggerSET(Trigger value) { m_triggerSET = value; }

    public String getText() { return m_text; }
    public void setText(String value) { m_text = value; }

    public void onCopyTextToClipboard(ActionEvent event) 
    {
        m_triggerSET.trigger();
    }

}
