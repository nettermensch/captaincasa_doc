The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • Components
  • Animation & Graphics & Sound
  • Camera / record videos

The CAMERARECORDER component allows to record content of the user's camera.

Pay attention: it is only usable in https-scenarios for security reasons! (For development reasons it is also supported for localhost-scenarios.)

The video recording produces a ".webm" file containing both the video and the audio stream. The file is sent to the server by the same upload mechanism that is used for transferring big files in the area of managing client file uploads: the big file is divided into parts, so that network upload limits ("max post size") are avoided.

After successful upload an action "AsynchronousUploadFinished" is triggered on server side.

The photo function ("snap") is the same as with the normal CAMERA component.

We have to add: the component was born out of business requirements! So the primary reason was not to add a "vlog component"... The concrete use case is: in some industries (e.g. insurances) the user can sign a contract by recording his "yes!" as video.

There are many other cases as well: recording a video of a technical problem to send it to maintenance. Record some behavior in medical applications. Etc. Etc.!

This is the Java code:

package workplace;

import java.io.Serializable;

import org.eclnt.editor.annotations.CCGenClass;
import org.eclnt.jsfserver.bufferedcontent.DefaultUploadContent;
import org.eclnt.jsfserver.bufferedcontent.UploadContentMgr;
import org.eclnt.jsfserver.defaultscreens.Statusbar;
import org.eclnt.jsfserver.elements.events.BaseActionEventCameraSnap;
import org.eclnt.jsfserver.elements.events.BaseActionEventUploadAsynchronousFinished;
import org.eclnt.jsfserver.util.HttpSessionAccess;
import org.eclnt.jsfserver.util.tempfile.TempFileManager;
import org.eclnt.util.file.FileManager;
import org.eclnt.util.valuemgmt.UniqueIdCreator;
import org.eclnt.util.valuemgmt.ValueManager;
import org.eclnt.workplace.IWorkpageDispatcher;

@CCGenClass (expressionBase="#{d.DemoCameraVideoRecording}")
public class DemoCameraVideoRecording
    extends PageBean 
    implements Serializable
{
    // ------------------------------------------------------------------------
    // inner classes
    // ------------------------------------------------------------------------

    /*
     * Pay attention: all methods are processed in some own upload thread which is
     * NOT the main request/response thread!
     */
    public class MyUploadContent extends DefaultUploadContent
    {
        @Override
        public void beginPassing()
        {
        }
        @Override
        public void passClientFile(String clientFileName, byte[] content)
        {
            // the file is written to a temporary file - it is NOT kept in memory in order
            // to prevent memory overload
            m_currentClientFileName = clientFileName;
            m_currentFileName = s_directory + UniqueIdCreator.createUUID() + "_" + clientFileName;
            FileManager.ensureDirectoryForFileExists(m_currentFileName);
            FileManager.writeFile(m_currentFileName,content,true);
        }
        @Override
        public void endPassing()
        {
        }
    }
    
    // ------------------------------------------------------------------------
    // members
    // ------------------------------------------------------------------------
    
    public static String s_directory;
    String m_currentClientFileName;
    String m_currentFileName;
    MyUploadContent m_uploadContent;
    String m_imageLink;
    String m_videoLink;
    boolean m_videoLinkEnabled = false;
    
    // ------------------------------------------------------------------------
    // constructors & initialization
    // ------------------------------------------------------------------------

    static
    {
        s_directory = HttpSessionAccess.getServletTempDirectory() + "/videorecordings/";
        s_directory = ValueManager.encodeIntoValidFileName(s_directory,true);
    }
    
    public DemoCameraVideoRecording(IWorkpageDispatcher workpageDispatcher)
    {
        super(workpageDispatcher);        
        m_uploadContent = new MyUploadContent();
        UploadContentMgr.add(m_uploadContent);
    }

    public String getPageName() { return "/workplace/democameravideorecording.xml"; }
    public String getRootExpressionUsedInPage() { return "#{d.DemoCameraVideoRecording}"; }

    // ------------------------------------------------------------------------
    // public usage
    // ------------------------------------------------------------------------

    public String getUploadURL() { return m_uploadContent.getURL(); }

    public void onCameraRecorderAction(javax.faces.event.ActionEvent event) 
    {
        if (event instanceof BaseActionEventUploadAsynchronousFinished)
        {
            Statusbar.outputSuccess("Upload of video is finished. File stored as: " + this.m_currentFileName);
            byte[] videoBytes = FileManager.readFile(m_currentFileName,true);
            // @data protection: the TempFileManager creates one directory per content, the name of the 
            // directory is created using secure random id algorithms. Only the current user session
            // knows about this directory - it is not visible to other sessions. The content is removed
            // automatically when the dialog session ends.
            m_videoLink = TempFileManager.saveTempFile(this.m_currentClientFileName,videoBytes);
            // @data protection: explicitly delete video after it is moved to file manager
            FileManager.deleteFile(m_currentFileName);
            m_videoLinkEnabled = true;
        }
        else if (event instanceof BaseActionEventCameraSnap)
        {
            BaseActionEventCameraSnap e =(BaseActionEventCameraSnap)event;
            byte[] image = e.getPNGImageBytes();
            m_imageLink = TempFileManager.saveTempFile("uploadedImage.png",image);
            Statusbar.outputSuccess("Upload of image is finished.");
        }
    }

    public String getImageLink() { return m_imageLink; }
    public String getVideoLink() { return m_videoLink; }
    public boolean getVideoLinkEnabled() { return m_videoLinkEnabled; }

    // ------------------------------------------------------------------------
    // private usage
    // ------------------------------------------------------------------------
}
