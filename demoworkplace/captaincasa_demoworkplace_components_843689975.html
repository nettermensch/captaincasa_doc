The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • Components
  • Straight Controls
  • Field Control
  • Spell checking

When clicking/tabbing into the field/text area then the spell checking of the browser will be activated.

Use attribute WITHSPELLCHECK to activate the spell checking.

This is the XML layout definition:

<?xml version="1.0" encoding="utf-8"?>
<t:layoutdefinition xmlns:t="http://www.CaptainCasa.com/controllibrary/t" xmlns:tpbc="http://www.CaptainCasa.com/controllibrary/tpbc" >
<t:rowdemobodypane id="g_1" objectbinding="#{d.DemoFieldSpellCheck}" rowdistance="#{ccstylevalue['@defaultrowdistance@']}" >
<t:row id="g_2" >
<t:field id="g_3" text="#{d.DemoFieldSpellCheck.text1}" width="100%" withspellcheck="true" />
</t:row>
<t:row id="g_4" >
<t:textarea id="g_5" height="100" text="#{d.DemoFieldSpellCheck.text2}" width="100%" withspellcheck="true" />
</t:row>
</t:rowdemobodypane>
<t:pageaddons id="g_pa"/>
</t:layoutdefinition>

This is the Java code:

package workplace;

import java.io.Serializable;
import org.eclnt.editor.annotations.CCGenClass;
import org.eclnt.workplace.IWorkpageDispatcher;
import org.eclnt.workplace.WorkpageDispatchedPageBean;

import javax.faces.event.ActionEvent;

@CCGenClass (expressionBase="#{d.DemoFieldSpellCheck}")

public class DemoFieldSpellCheck
    extends PageBean 
    implements Serializable
{
    // ------------------------------------------------------------------------
    // members
    // ------------------------------------------------------------------------

    String m_text1 = "Tis text has manny erors.";
    String m_text2 = "Tis text has evn more erors. Tis text has evn more erors. Tis text has evn more erors. Tis text has evn more erors. ";

    // ------------------------------------------------------------------------
    // constructors & initialization
    // ------------------------------------------------------------------------

    public DemoFieldSpellCheck(IWorkpageDispatcher workpageDispatcher)
    {
        super(workpageDispatcher);        
    }

    public String getPageName() { return "/workplace/demofieldspellcheck.jsp"; }
    public String getRootExpressionUsedInPage() { return "#{d.DemoFieldSpellCheck}"; }

    // ------------------------------------------------------------------------
    // public usage
    // ------------------------------------------------------------------------

    public String getText2() { return m_text2; }
    public void setText2(String value) { this.m_text2 = value; }

    public String getText1() { return m_text1; }
    public void setText1(String value) { this.m_text1 = value; }
}
