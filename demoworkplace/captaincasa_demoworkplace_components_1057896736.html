The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • Components
  • Straight Controls
  • Structure Components
  • Tabbed Line - Image Definition

This demo shows the usage of the attributes...

(.) IMAGE
(.) IMAGEWIDTH/HEIGHT
(.) IMAGEBACKGROUND
(.) IMAGEROUNDING
(.) IMAGEBORDERWIDTH/COLOR

...within the TABBEDLINETAB and TABBEDPANE component.
