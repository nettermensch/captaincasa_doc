The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • Components
  • Straight Controls
  • Statusbar and Messages
  • Message toasts with call back on close

In this demo a server side listener is called when the user explicitly closes a message toast by clicking it.

To do so:

1. define your own MESSAGETOAST component inside your layout
2. define your own MESSAGETOASTBinding in your Java code and bind it within the MESSAGETOAST component
3. assign an own actionListener method

Per message toast that you open up you now may define a "callbackId". This is the one that is part of the event "BaseActionEventCloseByUser" that is triggered on server side when a user closes a message toast instance.

The actionListener is only called...

1. when the corresponding message toast was defined with a callbackId
2. when the message toast was closed by the user (and not automatically by timer)

This is the XML layout definition:

<?xml version="1.0" encoding="utf-8"?>
<t:layoutdefinition xmlns:t="http://www.captaincasa.com/controllibrary/t" >
<t:beanprocessing id="g_1" beanbinding="#{d.DemoMessageToastWithCallback}" >
<t:messagetoast id="g_8" actionListener="#{d.DemoMessageToastWithCallback.onMtAction}" messagedata="#{d.DemoMessageToastWithCallback.mtBinding}" />
</t:beanprocessing>
<t:rowdemobodypane id="g_4" objectbinding="#{d.DemoMessageToastWithCallback}" >
<t:row id="g_6" >
<t:button id="g_2" actionListener="#{d.DemoMessageToastWithCallback.onCreateAction}" text="Create message toast" />
</t:row>
</t:rowdemobodypane>
</t:layoutdefinition>

This is the Java code:

package workplace;

import java.io.Serializable;

import javax.faces.event.ActionEvent;

import org.eclnt.editor.annotations.CCGenClass;
import org.eclnt.jsfserver.defaultscreens.MessageToast;
import org.eclnt.jsfserver.defaultscreens.Statusbar;
import org.eclnt.jsfserver.elements.events.BaseActionEventCloseByUser;
import org.eclnt.jsfserver.elements.impl.MESSAGETOASTBinding;
import org.eclnt.jsfserver.elements.impl.MESSAGETOASTBinding.MessageDataItem;
import org.eclnt.workplace.IWorkpageDispatcher;

@CCGenClass (expressionBase="#{d.DemoMessageToastWithCallback}")

public class DemoMessageToastWithCallback
    extends WorkpageDispatchedPageBean 
    implements Serializable
{
    // ------------------------------------------------------------------------
    // inner classes
    // ------------------------------------------------------------------------

    // ------------------------------------------------------------------------
    // members
    // ------------------------------------------------------------------------
    
    MESSAGETOASTBinding m_mtBinding = new MESSAGETOASTBinding();
    int m_counter = 1;

    // ------------------------------------------------------------------------
    // constructors & initialization
    // ------------------------------------------------------------------------

    public DemoMessageToastWithCallback(IWorkpageDispatcher workpageDispatcher)
    {
        super(workpageDispatcher);        
    }

    public String getPageName() { return "/workplace/demomessagetoastwithcallback.xml"; }
    public String getRootExpressionUsedInPage() { return "#{d.DemoMessageToastWithCallback}"; }

    // ------------------------------------------------------------------------
    // public usage
    // ------------------------------------------------------------------------

    
    
    public void onCreateAction(javax.faces.event.ActionEvent event) 
    {
        MessageDataItem mdi = m_mtBinding.addMessageToast(MessageToast.TYPE_INFO,null,"Message " + m_counter,"This is message number " + m_counter + ".");
        mdi.setCallbackId("MSG"+m_counter);
        m_counter++;
    }

    public MESSAGETOASTBinding getMtBinding() { return m_mtBinding; }
    
    public void onMtAction(ActionEvent event)
    {
        if (event instanceof BaseActionEventCloseByUser)
        {
            BaseActionEventCloseByUser e = (BaseActionEventCloseByUser)event;
            Statusbar.outputSuccessWithPopup("Event from message toast: " + e.getCallbackId());
        }
    }

    // ------------------------------------------------------------------------
    // private usage
    // ------------------------------------------------------------------------
}
