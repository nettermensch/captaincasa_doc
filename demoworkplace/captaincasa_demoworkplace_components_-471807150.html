The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • Components
  • Animation & Graphics & Sound
  • Paint Area
  • Connected items

You may connect PAINTAREAITEM instances so that if one is moved the other ones are moved as well.

For this reason you need to assign an explicit ITEMID to each PATINAREAITEM instance. And you need defined by using attribute CONNECTEDITEMIDS which instances are connected. CONNECTEDITEMIDS contains a semicolon-separated list of ITEMID values.

In the example item 1 is connected with item 2 - and item 2 is connected with item 3. Result: when moving item 1 then item 2 is moved - and as result item 3 is moved as well.

This is the Java code:

package workplace;

import java.io.Serializable;
import org.eclnt.editor.annotations.CCGenClass;
import org.eclnt.workplace.IWorkpageDispatcher;
import org.eclnt.workplace.WorkpageDispatchedPageBean;

import javax.faces.event.ActionEvent;

@CCGenClass (expressionBase="#{d.DemoPaintareaConnectedItems}")

public class DemoPaintareaConnectedItems
    extends PageBean 
    implements Serializable
{
    // ------------------------------------------------------------------------
    // members
    // ------------------------------------------------------------------------

    String m_bounds1 = "100;100;200;100";
    String m_bounds2= "70;200;50;50";
    String m_bounds3= "120;200;10;10";

    // ------------------------------------------------------------------------
    // constructors & initialization
    // ------------------------------------------------------------------------

    public DemoPaintareaConnectedItems(IWorkpageDispatcher workpageDispatcher)

        super(workpageDispatcher);        
    }

    public String getPageName() { return "/workplace/demopaintareaconnecteditems.jsp"; }

    public String getBounds2() { return m_bounds2; }

    public String getBounds3() { return m_bounds3; }
    public void setBounds3(String value) { this.m_bounds3 = value; }

    public void onItem1Action(javax.faces.event.ActionEvent event) 
    {
    }
}
