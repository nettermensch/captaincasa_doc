The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • Grids & Trees
  • Grids and Trees
  • Special Types of Binding
  • List of Beans

There is a server side class FIXGRIDBeanListBinding that is a sub-class of FIXGRIDListBinding. This class directly renders a list of beans. It provides add/remove methods that directly manipulate the list, that is passed into the processing.

In the columns the bean properties are reached by expression ".{bean.PROPERTY}".

This is the XML layout definition:

This is the Java code:

package workplace;

import java.lang.reflect.Method;
import java.lang.reflect.Type;
import java.util.ArrayList;
import java.util.List;

import javax.faces.event.ActionEvent;

import org.eclnt.jsfserver.elements.util.FIXGRIDBeanListBinding;
import org.eclnt.workplace.IWorkpageDispatcher;

public class DemoGridBeanBinding extends PageBean
{
    // ------------------------------------------------------------------------
    // members
    // ------------------------------------------------------------------------
    
    FIXGRIDBeanListBinding<Employee> m_grid = new FIXGRIDBeanListBinding<Employee>(Employee.class);
    
    // ------------------------------------------------------------------------
    // constructors
    // ------------------------------------------------------------------------
    
    public DemoGridBeanBinding(IWorkpageDispatcher dispatcher)
    {
        super(dispatcher);
        List<Employee> employees = readListOfEmployees();
        m_grid.showList(employees);
    }

    // ------------------------------------------------------------------------
    // public usage
    // ------------------------------------------------------------------------

    public FIXGRIDBeanListBinding<Employee> getGrid() { return m_grid; }
    
    
    
    
    
    // ------------------------------------------------------------------------
    // the followig code would normally come "from outside"...
    // ------------------------------------------------------------------------
    
    /**
     * Bean class that is used inside the grid. Normally is a class
     * "coming from outside".
     */
    public static class Employee
    {
        String m_firstName;
        String m_lastName;
        String m_street;
        String m_city;
        public void setFirstName(String value) { m_firstName = value; }
        public String getFirstName() { return m_firstName; }
        public void setLastName(String value) { m_lastName = value; }
        public String getLastName() { return m_lastName; }
        public void setStreet(String value) { m_street = value; }
        public String getStreet() { return m_street; }
        public void setCity(String value) { m_city = value; }
        public String getCity() { return m_city; }
    }
    
    /**
     * Dummy method for reading a list of beans
     */
    private List<Employee> readListOfEmployees()
    {
        List<Employee> result = new ArrayList<Employee>();
        for (int i=0; i<100; i++)
        {
            Employee e = new Employee();
            e.setFirstName("FN " + i);
            e.setLastName("LN " + i);
            e.setStreet("ST " + i);
            e.setCity("CI " + i);
            result.add(e);
        }
        return result;
    }
    
}
