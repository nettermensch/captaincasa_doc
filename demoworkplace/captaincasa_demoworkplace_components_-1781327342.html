The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • Components
  • Straight Controls
  • Structure Components
  • Ribbon Menu Areas

The areas on top of the page are not a specific component but just an arrangement of existing ones. Have a look into the XML for more details.

(This demo just shows a certain arrangement of components - there is no function implemented behind the icons/fields.)
