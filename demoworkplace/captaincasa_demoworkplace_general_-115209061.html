The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • General
  • Client Integration
  • Start Browser

The component JSHOWURL allows to open up a URL within a native browser, that shows up as separate window.

By default you should switch JSHOWURL-USEDESKTOP to "true", so that on client side the Java Desktop API is used for opening the browser. There are other modes as well, that are only applicable with the Enterprise client being used in an applet environment. Please read the Developer's Guide for more information.

The component JSHOWURL is an invisible client component. On the one hand you pass the value of the URL that you want to open, on the other hand you start a trigger which makes the component start the browser.

This is the Java code:

package workplace;

import java.io.Serializable;

import javax.faces.event.ActionEvent;

import org.eclnt.editor.annotations.CCGenClass;
import org.eclnt.jsfserver.elements.util.Trigger;
import org.eclnt.jsfserver.elements.util.ValidValuesBinding;
import org.eclnt.workplace.IWorkpageDispatcher;

@CCGenClass (expressionBase="#{d.DemoJshowurl}")

public class DemoJshowurl
    extends PageBean
    implements Serializable
{
    String m_windowFeatures = null;
    protected Trigger m_browserTrigger = new Trigger();
    protected String m_url = "http://www.captaincasa.com";
    ValidValuesBinding m_targetVVS = new ValidValuesBinding();
    String m_target;
    
    
    public DemoJshowurl(IWorkpageDispatcher dispatcher)
    {
        super(dispatcher);
        m_targetVVS.addValidValue(null,"normal");
        m_targetVVS.addValidValue("test","test");
        m_targetVVS.addValidValue("popup","popup");
    }
    
    public Trigger getBrowserTrigger() { return m_browserTrigger; }
    public String getUrl() { return m_url; }
    public void setUrl(String value) { m_url = value; }
 
    public void onOpenBrowser(ActionEvent event) 
    {
        m_browserTrigger.trigger();
    }

    public ValidValuesBinding getTargetVVS() { return m_targetVVS; }

    public String getTarget() { return m_target; }
    public void setTarget(String value) { this.m_target = value; }

    public String getWindowFeatures() { return m_windowFeatures; }
    public void setWindowFeatures(String value) { this.m_windowFeatures = value; }
    
}
