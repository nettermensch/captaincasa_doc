The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • Components
  • Scheduling
  • Schedule Component

The SCHEDULE component represents one schedule line. You can place SCHEDULEITEM components inside, each item representing a bar within the SCHEDULE line.

The sizing logic is: in the schedule line there is a SCHEDULEMAX property. This is an integer value which represents the "most right" coordinate in the schedule (the most left one is represented by "0"). Each SCHEDULEITEM component defines a SCHEDULELEFT and a SCHEDULERIGHT attribute - defining its position in the SCHEDULE. If there are overlapping items then the height of the items is automatically adjusted, so that they are shown as parallel items.

Each SCHEDULEITEM has a text which is displayed - and there are attributes to define the colouring etc.

Both SCHEDULE and SCHEDULEITEM provide the drag & drop interface - please check the schedule grid demo to see how they can be used.

This is the Java code:

package workplace;

import java.io.Serializable;

import javax.faces.event.ActionEvent;

import org.eclnt.editor.annotations.CCGenClass;
import org.eclnt.jsfserver.defaultscreens.ModalPopup;
import org.eclnt.jsfserver.defaultscreens.Statusbar;
import org.eclnt.workplace.IWorkpageDispatcher;

@CCGenClass (expressionBase="#{d.DemoSchedule}")

public class DemoSchedule
    extends PageBean
    implements Serializable
{
    ModalPopup m_popup;
    
    public DemoSchedule(IWorkpageDispatcher dispatcher)
    {
        super(dispatcher);
        initRefComponent("schedule");
    }

    public void onStartScheduleGrid(ActionEvent event) 
    {
        m_popup = ModalPopup.createInstance();
        m_popup.open("/workplace/demoschedulegrid.jsp","Scheduling Grid",800,600,new ModalPopup.IModalPopupListener()
        {
            public void reactOnPopupClosedByUser()
            {
                m_popup.close();
            }
        });
    }
    
    public void onSelection1(ActionEvent ae)
    {
        Statusbar.outputSuccess("Selected first schedule. " + ae.getClass().getName());
    }
    
    public void onSelection2(ActionEvent ae)
    {
        Statusbar.outputSuccess("Selected second schedule " + ae.getClass().getName());
    }
    
    public void onSelection3(ActionEvent ae)
    {
        Statusbar.outputSuccess("Selected third schedule. " + ae.getClass().getName());
    }
}
