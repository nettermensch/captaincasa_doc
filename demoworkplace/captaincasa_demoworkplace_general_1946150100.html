The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • General
  • Asynchronous Processing
  • Long Operations - Nicer Variant

The page bean component CCCAsynchronousJobProtocol provides nice features to output information from some long lasting operation that is operating in a parallel thread.

During the processing of the long lasing operation the user interface is blocked by a modal popup, so that the user cannot interrupt.

This is the XML layout definition:

<?xml version="1.0" encoding="utf-8"?>
<t:layoutdefinition xmlns:t="http://www.CaptainCasa.com/controllibrary/t" xmlns:tpbc="http://www.CaptainCasa.com/controllibrary/tpbc" >
<t:beanprocessing id="g_1" beanbinding="#{d.DemoPBCCCAsynchronousJobProtocol}" />
<t:rowdemobodypane id="g_2" objectbinding="#{d.DemoPBCCCAsynchronousJobProtocol}" >
<t:row id="g_3" >
<t:button id="g_4" actionListener="#{d.DemoPBCCCAsynchronousJobProtocol.onStartAction}" text="Start long operation" width="200+" />
</t:row>
</t:rowdemobodypane>
<t:pageaddons id="g_pa"/>
</t:layoutdefinition>

This is the Java code:

package workplace;

import java.io.Serializable;

import org.eclnt.ccaddons.pbc.CCAsynchronousJobProtocol;
import org.eclnt.ccaddons.pbc.CCAsynchronousJobProtocol.OpenPopupResult;
import org.eclnt.ccaddons.pbc.CCAsynchronousJobProtocol.ProtocolItem;
import org.eclnt.editor.annotations.CCGenClass;
import org.eclnt.jsfserver.defaultscreens.Statusbar;
import org.eclnt.workplace.IWorkpageDispatcher;

@CCGenClass (expressionBase="#{d.DemoPBCCCAsynchronousJobProtocol}")

public class DemoPBCCCAsynchronousJobProtocol
    extends PageBean 
    implements Serializable
{
    // ------------------------------------------------------------------------
    // members
    // ------------------------------------------------------------------------

    // ------------------------------------------------------------------------
    // constructors & initialization
    // ------------------------------------------------------------------------

    public DemoPBCCCAsynchronousJobProtocol(IWorkpageDispatcher workpageDispatcher)
    {
        super(workpageDispatcher);        
    }

    public String getPageName() { return "/workplace/demopbcccasynchronousjobprotocol.jsp"; }
    public String getRootExpressionUsedInPage() { return "#{d.DemoPBCCCAsynchronousJobProtocol}"; }

    // ------------------------------------------------------------------------
    // public usage
    // ------------------------------------------------------------------------

    public void onStartAction(javax.faces.event.ActionEvent event) 
    {
        // open the modal popup
        final OpenPopupResult opr = CCAsynchronousJobProtocol.openAsBlockingPopup("Test long running operation",this);
        final CCAsynchronousJobProtocol jb = opr.getProtocolBean();
        // define the long running operation which is executed in a parallel thread
        Runnable lr = new Runnable()
        {
            @Override
            public void run()
            {
                for (int i=0; i<100; i++)
                {
                    try { Thread.sleep(100); } catch (Throwable t) {}
                    jb.setProgressValue(i+1);
                    if (i%5 != 2)
                        jb.addProtocolItemFromAsynchronousProcessing(new ProtocolItem("TEST","Test text " + i,"#C0FFC0","#000000"));
                    else
                        jb.addProtocolItemFromAsynchronousProcessing(new ProtocolItem("TEST","Test text " + i,"#800000","#FFFFFF"));
                }
            }
        };
        // define the operation that is executed in the UI thread after the long operation
        // has ended
        Runnable fr = new Runnable()
        {
            @Override
            public void run()
            {
                Statusbar.outputAlert("Finished!");
            }
        };
        // start
        CCAsynchronousJobProtocol.processLongOperation(jb,lr,fr);
    }

    // ------------------------------------------------------------------------
    // private usage
    // ------------------------------------------------------------------------
}
