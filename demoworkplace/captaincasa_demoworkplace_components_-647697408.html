The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • Components
  • Animation & Graphics & Sound
  • 3D Animations
  • Rotating content - simple example

The ROTATECONTENTAREA is a container that can rotate its content - so that the content of the container is still visible, but only requires some reduced screen space.

All controls within the container are fully use-able - also when being rotated.

This is the Java code:

package workplace;

import java.io.Serializable;

import org.eclnt.editor.annotations.CCGenClass;
import org.eclnt.workplace.IWorkpageDispatcher;

@CCGenClass (expressionBase="#{d.DemoRotateContentArea2}")

public class DemoRotateContentArea2
    extends PageBean 
    implements Serializable
{
    public void onToggleAction(javax.faces.event.ActionEvent event) 
    {
        m_rotated = !m_rotated;
    }

    boolean m_rotated = true;
    public boolean getRotated() { return m_rotated; }
    public void setRotated(boolean value) { this.m_rotated = value; }

    // ------------------------------------------------------------------------
    // inner classes
    // ------------------------------------------------------------------------

    /* Listener to the user of the page bean. */
    public interface IListener extends Serializable
    {
    }

    // ------------------------------------------------------------------------
    // members
    // ------------------------------------------------------------------------

    private IListener m_listener;

    // ------------------------------------------------------------------------
    // constructors & initialization
    // ------------------------------------------------------------------------

    public DemoRotateContentArea2(IWorkpageDispatcher workpageDispatcher)
    {
        super(workpageDispatcher);        
    }

    public String getPageName() { return "/workplace/demorotatecontentarea2.xml"; }
    public String getRootExpressionUsedInPage() { return "#{d.DemoRotateContentArea2}"; }

    // ------------------------------------------------------------------------
    // public usage
    // ------------------------------------------------------------------------

    /* Initialization of the bean. Add any parameter that is required within your scenario. */
    public void prepare(IListener listener)
    {
        m_listener = listener;
    }

    // ------------------------------------------------------------------------
    // private usage
    // ------------------------------------------------------------------------
}
