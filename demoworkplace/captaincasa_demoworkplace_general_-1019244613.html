The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • General
  • General Issues
  • Attribute RENDERED

The attribute RENDERED is available on most of the components. If set to "false", typically using a binding to a managed bean, then the corresponding component will not be sent to the client at all - and as consequence will not be rendered onto the user's screen.

This is the Java code:

package workplace;

import javax.faces.event.ActionEvent;

import org.eclnt.jsfserver.defaultscreens.Statusbar;
import org.eclnt.workplace.IWorkpageDispatcher;

public class DemoRendered extends PageBean
{
    boolean m_rendered = true;
    boolean m_rowRendered = true;
    boolean m_leftTabRendered = true;
    boolean m_rightTabRendered = true;
    boolean m_paneRendered = false;
    
    public DemoRendered(IWorkpageDispatcher dispatcher)
    {
        super(dispatcher);
    }
    
    public boolean getRendered() { return m_rendered; }
    public void setRendered(boolean value) { m_rendered = value; }
    
    public boolean getRowRendered() { return m_rowRendered; }
    public void setRowRendered(boolean value) { m_rowRendered = value; }

    public boolean getLeftTabRendered() { return m_leftTabRendered; }
    public void setLeftTabRendered(boolean leftTabRendered) { m_leftTabRendered = leftTabRendered; }
    
    public boolean getRightTabRendered() { return m_rightTabRendered; }
    public void setRightTabRendered(boolean rightTabRendered) { m_rightTabRendered = rightTabRendered; }
    
    public boolean getPaneRendered() { return m_paneRendered; }
    public void setPaneRendered(boolean paneRendered) { m_paneRendered = paneRendered; }
    
    public void onChangeRendered(ActionEvent ae)
    {
        m_rendered = !m_rendered;
        Statusbar.outputMessage("Changed m_rendered to: " + m_rendered);
    }
    
    public void onChangeRowRendered(ActionEvent ae)
    {
        m_rowRendered = !m_rowRendered;
        Statusbar.outputMessage("Changed m_rowRendered to: " + m_rendered);
    }
    
}
