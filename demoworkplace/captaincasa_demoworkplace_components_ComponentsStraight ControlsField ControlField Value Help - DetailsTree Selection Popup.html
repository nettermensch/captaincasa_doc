The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • Components
  • Straight Controls
  • Field Control
  • Field Value Help - Details
  • Tree Selection Popup

This demo shows how to bring together...

...COMBOFIELD-component
...PageBean Management
...Tree Management

...in order to support scenarios in which you want a tree to be opened for selecting items for a certain field.

(The code and the XML of the popup that is called is contained as comment inside the "Server side Java Code"-tab.)

This is the XML layout definition:

<?xml version="1.0" encoding="utf-8"?>
<t:layoutdefinition xmlns:t="http://www.CaptainCasa.com/controllibrary/t" xmlns:tpbc="http://www.CaptainCasa.com/controllibrary/tpbc" >
<t:rowdemobodypane id="g_1" objectbinding="#{d.DemoComboFieldTree}" >
<t:row id="g_2" >
<t:combofield id="g_3" actionListener="#{d.DemoComboFieldTree.onDepartmentAction}" labeltext="Department" text="#{d.DemoComboFieldTree.department}" width="200" />
</t:row>
</t:rowdemobodypane>
<t:pageaddons id="g_pa"/>
</t:layoutdefinition>

This is the Java code:

package workplace;

import java.io.Serializable;

import javax.faces.event.ActionEvent;

import org.eclnt.editor.annotations.CCGenClass;
import org.eclnt.jsfserver.defaultscreens.ModelessPopup;
import org.eclnt.jsfserver.elements.events.BaseActionEventValueHelp;
import org.eclnt.workplace.IWorkpageDispatcher;
import org.eclnt.workplace.WorkpageDispatchedPageBean;

@CCGenClass (expressionBase="#{d.DemoComboFieldTree}")

public class DemoComboFieldTree
    extends PageBean // ==> WorkpageDispatchedPageBean 
    implements Serializable
{
    // ------------------------------------------------------------------------
    // members
    // ------------------------------------------------------------------------
    
    String m_department;
    
    // ------------------------------------------------------------------------
    // constructors & initialization
    // ------------------------------------------------------------------------

    public DemoComboFieldTree(IWorkpageDispatcher workpageDispatcher)
    {
        super(workpageDispatcher);        
        initRefComponent("combofield"); 
    }
    
    public String getPageName() { return "/workplace/democombofieldtree.jsp"; }
    public String getRootExpressionUsedInPage() { return "#{d.DemoComboFieldTree}"; }
    
    // ------------------------------------------------------------------------
    // public usage
    // ------------------------------------------------------------------------
    
    public String getDepartment() { return m_department; }
    public void setDepartment(String value) { this.m_department = value; }

    public void onDepartmentAction(ActionEvent event) 
    {
        if (event instanceof BaseActionEventValueHelp)
        {
            final DemoComboFieldTreePopup dcfp = new DemoComboFieldTreePopup(getOwningDispatcher());
            dcfp.prepare(m_department,new DemoComboFieldTreePopup.IListener()
            {
                public void reactOnSelection(String selectedValue)
                {
                    m_department = selectedValue;
                    closePopup(dcfp);  
                }
            });
            ModelessPopup mp = openModelessPopup(dcfp,"",0,0,new ModelessPopup.IModelessPopupListener()
            {
                public void reactOnPopupClosedByUser()
                {
                    closePopup(dcfp);
                }
            });
            mp.setUndecorated(true);
            mp.setCloseonclickoutside(true);
        }
    }

}
