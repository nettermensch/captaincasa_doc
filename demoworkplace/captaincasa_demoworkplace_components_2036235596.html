The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • Components
  • Straight Controls
  • Field Control
  • Input of "strange" characters
  • Virtual keyboard on demand

You can explicitly open a touch dialog by defining a keyboard short cut. Use attribute TOUCHOPENSHOTKEY to do so.

Typical use case: you want to provide a selection of keys that are not reach-able via the user's keyboard.

The layout of the keyboard can be customized. Please check the demos in the area of "Touch Screen" to see details about the configuration.

This is the XML layout definition:

<?xml version="1.0" encoding="utf-8"?>
<t:layoutdefinition xmlns:t="http://www.CaptainCasa.com/controllibrary/t" xmlns:tpbc="http://www.CaptainCasa.com/controllibrary/tpbc" >
<t:beanprocessing id="g_1" beanbinding="#{d.DemoTouchOpenHotkey}" />
<t:rowdemobodypane id="g_2" objectbinding="#{d.DemoTouchOpenHotkey}" rowdistance="#{ccstylevalue['@defaultrowdistance@']}" >
<t:row id="g_3" >
<t:field id="g_4" labeltext="A text - press ctrl-space for opening touch support" text="#{d.DemoTouchOpenHotkey.text1}" touchlayout="strangekeys1" touchopenhotkey="ctrl-32" width="100%;100" />
</t:row>
<t:row id="g_5" >
<t:formattedfield id="g_6" format="int" labeltext="An integer number - press ctrl-space for opening touch support" touchopenhotkey="ctrl-32" value="#{d.DemoTouchOpenHotkey.number1}" width="100%;100" />
</t:row>
</t:rowdemobodypane>
<t:pageaddons id="g_pa"/>
</t:layoutdefinition>

This is the Java code:

package workplace;

import java.io.Serializable;

import org.eclnt.editor.annotations.CCGenClass;
import org.eclnt.workplace.IWorkpageDispatcher;

@CCGenClass (expressionBase="#{d.DemoTouchOpenHotkey}")

public class DemoTouchOpenHotkey
    extends WorkpageDispatchedPageBean 
    implements Serializable
{
    // ------------------------------------------------------------------------
    // members
    // ------------------------------------------------------------------------

    int m_number1;
    String m_text1 = "Some text";
    
    // ------------------------------------------------------------------------
    // constructors & initialization
    // ------------------------------------------------------------------------

    public DemoTouchOpenHotkey(IWorkpageDispatcher workpageDispatcher)
    {
        super(workpageDispatcher);        
    }

    public String getPageName() { return "/workplace/demotouchopenhotkey.jsp"; }
    public String getRootExpressionUsedInPage() { return "#{d.DemoTouchOpenHotkey}"; }

    // ------------------------------------------------------------------------
    // public usage
    // ------------------------------------------------------------------------

    public int getNumber1() { return m_number1; }
    public void setNumber1(int value) { this.m_number1 = value; }

    public String getText1() { return m_text1; }
    public void setText1(String value) { this.m_text1 = value; }

}
