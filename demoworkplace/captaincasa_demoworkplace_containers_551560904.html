The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • Containers
  • Tab Pane
  • Tab Pane with close-Icon

On TABBEDPANETAB level you can define attribute WITHCLOSEICON to be "true". In this case an additional close-icon is shown within the corresponding tab.

On server side an action listener (again on TABBEDPANETAB level) is called, the event that is passed is an instance of class BaseActionEventTabClose.

In the example the closing of the tabs is done using a RENDERED attribute that is defined with each TABBEDPANE instance.

This is the Java code:

package workplace;

import java.io.Serializable;

import javax.faces.event.ActionEvent;

import org.eclnt.editor.annotations.CCGenClass;
import org.eclnt.workplace.IWorkpageDispatcher;

@CCGenClass (expressionBase="#{d.DemoTabbedPaneWithClose}")

public class DemoTabbedPaneWithClose
    extends PageBean
    implements Serializable
{
    protected boolean m_rightRendered = true;
    protected boolean m_leftRendered = true;

    public DemoTabbedPaneWithClose(IWorkpageDispatcher dispatcher)
    {
        super(dispatcher);
    }
    
    public boolean getRightRendered() { return m_rightRendered; }
    public boolean getLeftRendered() { return m_leftRendered; }

    public void onRightClose(ActionEvent event) 
    {
        m_rightRendered = false;
    }

    public void onLeftClose(ActionEvent event) 
    {
        m_leftRendered = false;
    }
    
    public void onShowAllTabs(ActionEvent event)
    {
        m_leftRendered = true;
        m_rightRendered = true;
    }

}
