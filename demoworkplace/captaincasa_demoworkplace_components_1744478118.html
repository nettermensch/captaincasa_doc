The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • Components
  • Straight Controls
  • Menus
  • Client side copy&paste (Java Client only)

There are special menu items that you can add to either the MENUBAR-menus or to the POPUPMENU-menus:

The so called SYSTEMMENUITEM-components are executing a client command - i.e. they do not communitcate to the server side as normal MENUITEM-components.

Currently there are three commands:
(.) copy
(.) cut
(.) paste
The three commands are operating with the client side clipboard and are linked to the text-input-components (FIELD, FORMATTEDFIELD, ...) within your page.

You need to define the command within the attribute CLIENTSYSTEMCOMMAND.

This is the Java code:

package workplace;

import java.io.Serializable;

import javax.faces.event.ActionEvent;

import org.eclnt.editor.annotations.CCGenClass;
import org.eclnt.jsfserver.defaultscreens.Statusbar;
import org.eclnt.jsfserver.util.ClientTestLog;
import org.eclnt.jsfserver.util.IClientLogAware;
import org.eclnt.util.log.CLog;
import org.eclnt.workplace.IWorkpageDispatcher;

@CCGenClass (expressionBase="#{d.DemoMenuClientSystemCommand}")

public class DemoMenuClientSystemCommand 
    extends WorkpageDispatchedPageBean
    implements Serializable, IClientLogAware
{
    protected String m_comment = "Comment on CaptainCasa.";
    protected String m_lastName = "Casa";
    protected String m_firstName = "Captain";
    
    public DemoMenuClientSystemCommand(IWorkpageDispatcher dispatcher)
    {
        super(dispatcher);
        initRefComponent("systemmenuitem");
    }
    
    public String getComment() { return m_comment; }
    public void setComment(String value) { m_comment = value; }

    public String getLastName() { return m_lastName; }
    public void setLastName(String value) { m_lastName = value; }

    public String getFirstName() { return m_firstName; }
    public void setFirstName(String value) { m_firstName = value; }

    public void onSynchronize(ActionEvent event) 
    {
        CLog.L.log(CLog.LL_INF,"***** First Name = " + m_firstName);
        CLog.L.log(CLog.LL_INF,"***** Last Name = " + m_lastName);
        CLog.L.log(CLog.LL_INF,"***** Comment = " + m_lastName);
        Statusbar.outputSuccess(m_firstName + " / " + m_lastName + " / " + m_comment);
    }
    
    public boolean onCreateClientTestLogMessage()
    {
        ClientTestLog.addClientTestLogMessage(m_firstName + "/" + m_lastName + "/" + m_comment);
        return false;
    }
    
    
}
