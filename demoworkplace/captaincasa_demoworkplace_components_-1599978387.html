The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • Components
  • Touch Screen
  • Signature

Sign within the yellow area and then press OK. (If you do not have a touch screen monitor, you have to use your mouse for signing...)

The TOUCHSIGNATURE components allows to directly "draw" lines with the finger or a pen into its sensitive area. The lines are transferred to server side as pixel values - a semicolon separated sequence of coordinates.

The intention of the component is to be used as control for capturing human signatures.

You want to convert the singature-"vector" into an image? Then use class SignatureImageCreator, which directly consukmes the input fomr the TOUCHSIGNATURE component and converts into an image.

This is the Java code:

package workplace;

import java.io.Serializable;
import org.eclnt.editor.annotations.CCGenClass;
import org.eclnt.workplace.IWorkpageDispatcher;
import org.eclnt.workplace.WorkpageDispatchedBean;

import javax.faces.event.ActionEvent;

@CCGenClass (expressionBase="#{d.DemoTouchSignature}")

public class DemoTouchSignature
    extends PageBean
    implements Serializable
{
    String m_pixelValues;
    public String getPixelValues() { return m_pixelValues; }
    public void setPixelValues(String value) { this.m_pixelValues = value; }

    public DemoTouchSignature(IWorkpageDispatcher workpageDispatcher)
    {
        super(workpageDispatcher);        
        initRefComponent("touchsignature");
    }

    public void onOK(ActionEvent event) {}

}
