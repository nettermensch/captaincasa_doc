The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • Components
  • Straight Controls
  • Field Control
  • Regular Expressions
  • Special words

Sometimes there is the necessity to build regular expressions, in which the locale/client-specific separators for decimal numbers are contained.

You can do this by using the reserved words "@@decimalSeparator@@" and "@@groupingSeparator@@". The words are replaced by their corresponding client-side representation at runtime.

This is the Java code:

package workplace;

import java.io.Serializable;
import java.math.BigDecimal;

import org.eclnt.editor.annotations.CCGenClass;
import org.eclnt.workplace.IWorkpageDispatcher;
import org.eclnt.workplace.WorkpageDispatchedPageBean;

@CCGenClass (expressionBase="#{d.DemoRegexSpecial}")

public class DemoRegexSpecial
    extends PageBean 
    implements Serializable
{
    public DemoRegexSpecial(IWorkpageDispatcher workpageDispatcher)
    {
        super(workpageDispatcher);        
    }

    public String getPageName() { return "/workplace/demoregexspecial.jsp"; }
    public String getRootExpressionUsedInPage() { return "#{d.DemoRegexSpecial}"; }

}
