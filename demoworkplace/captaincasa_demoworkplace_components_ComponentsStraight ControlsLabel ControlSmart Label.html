The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • Components
  • Straight Controls
  • Label Control
  • Smart Label

The SMARTLABEL component is a ...smart label!

You can pass text definitions in which characters are formatted individually to a certain extent. (The normal LABEL component is only able to render all text in one and the same font.)

The format of passing is either an XML format or a comma/seimcolon separated format:

The XML tags are:
<b></b>
<u></u>
<sub></sub>
<sup></sup>
<highlighted></highlighted>

The CSV definition is done in the following way:
<text>[;beginOffset;endOffset;format]

The valid formats are:
10 => bold
20 => underlined
30 => subscript
40 => superscript
50 => highlighted


Please note: it is not possible to nest format definitions! E.g. <b><u>text</u></b> is NOT allowed. If you need to do such definitions you should use the more general TEXTPANE component.
