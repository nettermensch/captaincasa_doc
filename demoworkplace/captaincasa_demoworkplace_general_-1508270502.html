The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • General
  • Arranging Content
  • Popup Dialogs
  • Undecorated Popups

Popups can be opened in various modes.

This example shows a nice mode of usage: the popup is opened as modeless popup, with additional parameters:
(.) it is opened "undecorated" - i.e. there is no title, no explicit border
(.) it is opened with "closeonclickoutside" - i.e. it automatically closes once the user clicks with the mouse outside the popup.

As consequence you can build nice add-on-dialogs in which you arrange optional functions, that you do not want to add to your main screen.

This is the XML layout definition:

<?xml version="1.0" encoding="utf-8"?>
<t:layoutdefinition xmlns:t="http://www.CaptainCasa.com/controllibrary/t" xmlns:tpbc="http://www.CaptainCasa.com/controllibrary/tpbc" >
<t:rowdemobodypane id="g_5" objectbinding="#{d.DemoPopupUndecorated}" >
<t:rowheader id="g_9" >
<t:button id="g_10" actionListener="#{d.DemoPopupUndecorated.onOpenSearch}" contentareafilled="false" horizontaltextposition="left" image="/images/icons/magnifier.png" text="Search..." />
</t:rowheader>
<t:rowdistance id="g_1" height="10" />
<t:row id="g_6" >
<t:label id="g_7" text="Press the &#34;Search...&#34; button!" />
</t:row>
</t:rowdemobodypane>
<t:pageaddons id="g_pa"/>
</t:layoutdefinition>

This is the Java code:

package workplace;

import java.io.Serializable;

import javax.faces.event.ActionEvent;

import org.eclnt.editor.annotations.CCGenClass;
import org.eclnt.jsfserver.defaultscreens.ModelessPopup;
import org.eclnt.jsfserver.elements.util.DefaultModelessPopupListener;
import org.eclnt.workplace.IWorkpageDispatcher;

@CCGenClass (expressionBase="#{d.DemoPopupUndecorated}")

public class DemoPopupUndecorated
    extends PageBean
    implements Serializable
{
    
    public DemoPopupUndecorated(IWorkpageDispatcher dispatcher)
    {
        super(dispatcher);
    }

    public void onOpenSearch(ActionEvent event) 
    {
        final DemoPopupUndecoratedSearch bean = new DemoPopupUndecoratedSearch(getOwningDispatcher());
        ModelessPopup popup = openModelessPopup(bean,"Search",250,300,new ModelessPopup.IModelessPopupListener()
        {
            public void reactOnPopupClosedByUser()
            {
                closePopup(bean);
            }
        });
        popup.setUndecorated(true);
        popup.setSizeableIfUndecorated(true);
        popup.setCloseonclickoutside(true);
    }
}
