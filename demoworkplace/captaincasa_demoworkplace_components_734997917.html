The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • Components
  • Animation & Graphics & Sound
  • Paint Area
  • Editing of interim points - fix start/end

When editing line points there are situation in which the first and/or the last interim point of the line should be fixed - and must not be moved by the user.

By setting the attributes FIXINTERIMPOINTFIRST and FIXINTERIMPOINTLAST to "true" you can do so.

package workplace;

public class DemoPaintAreaLineFixPoints

{

    }
