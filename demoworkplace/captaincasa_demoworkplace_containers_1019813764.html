The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • Containers
  • Split Pane - Extended Functions
  • Keep minimum / Cut content

Use attribute SPLITPANE-CUTCONTENT to define if the content minimum size should be respected when moving the slider - or not.
