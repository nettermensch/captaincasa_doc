The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • Components
  • Straight Controls
  • Menus
  • Flexible Menu Item Container

You can insert the component MENUITEMCONTAINER into menus. Inside the MENUITEMCONTAINER you can arrange - in principal - any content.

But: you should only use this component for simple content - e.g. if you want to integrate an up/download link into a menu. Definitely do not use this component for complex form scenario! The menu may close any time (e.g. if losing the focus)!

And: it is only possible to define one menu level - i.e. it's not possible to open a second menu within the menu.

Please note: MENUITEMCONTAINER items cannot be disabled, you need to explicitly disable their content components.
