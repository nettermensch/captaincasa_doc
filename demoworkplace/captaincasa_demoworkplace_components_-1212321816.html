The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • Components
  • Straight Controls
  • Combobox and List
  • Double Line Item

Open the combo box item popup: you will see that the items are rendered as two lines - the first representing the text that is passed, the second reperesenting a so called value-comment.

In order two use the value-comment you need to:

(.) set COMBOBOX-WITHVALUECOMMENT to "true"
(.) passe the value comments as part of the item definition - in addition to the value and text that you normally pass

This is the XML layout definition:

<?xml version="1.0" encoding="utf-8"?>

This is the Java code:

package workplace;

import java.io.Serializable;

import org.eclnt.editor.annotations.CCGenClass;
import org.eclnt.jsfserver.elements.util.ValidValuesBinding;
import org.eclnt.workplace.IWorkpageDispatcher;

@CCGenClass (expressionBase="#{d.DemoComboWithComment}")

public class DemoComboWithComment 
    extends PageBean
    implements Serializable
{
    protected String m_company;
    protected String m_department;
    protected ValidValuesBinding m_departmentVVS = new ValidValuesBinding();
    
    public DemoComboWithComment(IWorkpageDispatcher dispatcher)
    {
        super(dispatcher);
        initRefComponent("combobox"); 
        m_departmentVVS.addValidValue("S","Sales","Sales of our outstanding products","/images/pictures/photo0.gif");
        m_departmentVVS.addValidValue("D","Development","Fabulous development department","/images/pictures/photo1.gif");
        m_departmentVVS.addValidValue("Q","Quality Assurance","Yes, seems that we really need these guys","/images/pictures/photo2.gif");
        m_departmentVVS.addValidValue("HR","Human Resources",null,"/eclntjsfserver/images/delete.png");
        m_department = "S"; // preset to "Sales"
    }

    public ValidValuesBinding getDepartmentVVS() { return m_departmentVVS; }

    public String getDepartment() { return m_department; }
    public void setDepartment(String value) { m_department = value; }

    public String getCompany() { return m_company; }
    public void setCompany(String value) { m_company = value; }

}
