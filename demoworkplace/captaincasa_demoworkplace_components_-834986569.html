The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • Components
  • Straight Controls
  • Field Control
  • Pass text selection info

Both FIELD and TEXTAREA component are able to send the current text selection information to the server side.

You may use this information in order to e.g. insert/replace text at a certain position - or to start functions based on the currently selected text.

The information is passed as semicolon-separated string of three digits:
(.) the current caret position
(.) the current start of selection
(.) the current end of selection
You may use ValuManager.decodeStraightIntCSV(...) to transfer the string into int-values.
