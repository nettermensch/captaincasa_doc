The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • Containers
  • Tab Pane
  • Hotkey Navigation

You can assign HOTKEY definitions to inidividual tabs.

As you see in the example, there are equal definitions used for different tabs (e.g. ctrl-1 is used both in the top-TABBEDPANE and in the bottom-TABBEDPANE).

The hotkey management first tries to resolve the definition, in which the current focus is located. Otherwise the first definition will be used.
