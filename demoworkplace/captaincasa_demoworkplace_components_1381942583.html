The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • Components
  • Straight Controls
  • Images
  • data:image-URIs

You may use data:image-URIs as well. For proper multi-browser support and for proper sizing, we recommend to only use base64-encoded URIs.

In the example the images are read from the deployed web application using the WebResourceReader-API.

This is the XML layout definition:

<?xml version="1.0" encoding="utf-8"?>
<t:layoutdefinition xmlns:t="http://www.CaptainCasa.com/controllibrary/t" xmlns:tpbc="http://www.CaptainCasa.com/controllibrary/tpbc" >
<t:rowdemobodypane id="g_1" rowdistance="#{ccstylevalue['@defaultrowdistance@']}" objectbinding="#{d.DemoImageDataURL}">
<t:row id="g_2" >
<t:label id="g_3" text="PNG Image" width="100" />
<t:image id="g_4" image="#{d.DemoImageDataURL.imagePNG}" />
</t:row>
<t:row id="g_5" >
<t:label id="g_6" text="JPG Image" width="100" />
<t:image id="g_7" image="#{d.DemoImageDataURL.imageJPG}" />
</t:row>
<t:row id="g_8" >
<t:label id="g_9" text="SVG Image" width="100" />
<t:image id="g_10" image="#{d.DemoImageDataURL.imageSVG}" />
</t:row>
</t:rowdemobodypane>
<t:pageaddons id="g_pa"/>
</t:layoutdefinition>

This is the Java code:

package workplace;

import java.io.Serializable;
import org.eclnt.editor.annotations.CCGenClass;
import org.eclnt.jsfserver.util.WebResourceReader;
import org.eclnt.util.valuemgmt.Base64;
import org.eclnt.workplace.IWorkpageDispatcher;
import org.eclnt.workplace.WorkpageDispatchedPageBean;

import javax.faces.event.ActionEvent;

@CCGenClass (expressionBase="#{d.DemoImageDataURL}")

public class DemoImageDataURL
    extends PageBean 
    implements Serializable
{
    // ------------------------------------------------------------------------
    // inner classes
    // ------------------------------------------------------------------------

    // ------------------------------------------------------------------------
    // members
    // ------------------------------------------------------------------------

    String m_imageJPG;
    String m_imageSVG;
    String m_imagePNG;

    // ------------------------------------------------------------------------
    // constructors & initialization
    // ------------------------------------------------------------------------

    public DemoImageDataURL(IWorkpageDispatcher workpageDispatcher)
    {
        super(workpageDispatcher);        
        byte[] jpg = WebResourceReader.readFileIntoByteArray("images/pictures/ph0.jpg");
        byte[] png = WebResourceReader.readFileIntoByteArray("images/pictures/adressbuch.png");
        byte[] svg = WebResourceReader.readFileIntoByteArray("images/iconssvg/add_16x16.svg");
        m_imageJPG = "data:image/jpg;base64," + Base64.encode(jpg);
        m_imagePNG = "data:image/png;base64," + Base64.encode(png);
        m_imageSVG = "data:image/svg+xml;base64," + Base64.encode(svg);
    }

    public String getPageName() { return "/workplace/demoimagedataurl.jsp"; }
    public String getRootExpressionUsedInPage() { return "#{d.DemoImageDataURL}"; }

    // ------------------------------------------------------------------------
    // public usage
    // ------------------------------------------------------------------------

    public String getImageJPG() { return m_imageJPG; }
    public String getImageSVG() { return m_imageSVG; }
    public String getImagePNG() { return m_imagePNG; }

    // ------------------------------------------------------------------------
    // private usage
    // ------------------------------------------------------------------------
}
