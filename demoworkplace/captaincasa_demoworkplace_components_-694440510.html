The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • Components
  • Straight Controls
  • Text Input/Output
  • Code / XML Editor
  • Passing ACE-property-parameters

By using attribute CODEEDITORPARAMS you can pass any specific properties to the internally used ACE editor

The format of the value is: name:value;name:value;name:value

The name is pointing to a set-ter method of or a setOption method of the ACE-editor object.

(.) "xxx:value" ==> editor.setXxx(value)
(.) "session/xxx:value ==> editor.getSession().setXxx(value)
(.) "renderer/xxx:value ==> editor.renderer.setXxx(value)
(.) "renderer.option/xxx:value ==> editor.renderer.setOption(xxx,value)
(.) "editor.option/xxx:value ==> editor.setOption(xxx,value)

Please check the ACE editor documentation for the concrete values you might pass. - Some of the values can be set one time, and can not be set back to their previous value. Please check the ACE documentation.

If you want to configure the ACE editor in a more sophisticated please also consider to implement an own bridge page, which then can be set by using the BRIDGEPAGE attribute. The default is: "/eclnt/risc/ext_ace/ace_brige.html".

This is the Java code:

package workplace;

import java.io.Serializable;
import java.util.HashMap;
import java.util.Map;

import javax.faces.event.ActionEvent;

import org.eclnt.editor.annotations.CCGenClass;
import org.eclnt.jsfserver.elements.BaseActionEvent;
import org.eclnt.util.file.ClassloaderReader;
import org.eclnt.workplace.IWorkpageDispatcher;

@CCGenClass (expressionBase="#{d.DemoCodeEditorParams}")

public class DemoCodeEditorParams
    extends WorkpageDispatchedPageBean 
    implements Serializable
{
    // ------------------------------------------------------------------------
    // members
    // ------------------------------------------------------------------------

    String m_codeType = "xml";
    Map<String,String> m_texts = new HashMap<String, String>(); // codeType, text
    String m_editorParameterString = "theme:ace/theme/monokai";
    String m_newEditorParamString  = m_editorParameterString;
    
    // ------------------------------------------------------------------------
    // constructors & initialization
    // ------------------------------------------------------------------------

    public DemoCodeEditorParams(IWorkpageDispatcher workpageDispatcher)
    {
        super(workpageDispatcher);   
        readText();
    }

    public String getPageName() { return "/workplace/democodeeditorparams.jsp"; }
    public String getRootExpressionUsedInPage() { return "#{d.DemoCodeEditorParams}"; }

    // ------------------------------------------------------------------------
    // public usage
    // ------------------------------------------------------------------------
    
    public String getText() { return m_texts.get(m_codeType); } 
    public void setText(String text) { m_texts.put(m_codeType,text); }
    
    public String getCodeType() 
    { 
        if ("xmas".equals(m_codeType)) return "java"; // ;-)
        return m_codeType; 
    }
    public void setCodeType(String codeType) { m_codeType = codeType; }

    public void onLoadText(ActionEvent event)
    {
        m_codeType = ((BaseActionEvent)event).getSourceReference();
        readText();
    }

    public String getEditorParameterString() { return m_editorParameterString; }
    public void setEditorParameterString(String value) { this.m_editorParameterString = value; }

    public String getNewEditorParamString() { return m_newEditorParamString; }
    public void setNewEditorParamString(String value) { this.m_newEditorParamString = value; }

    public void onApplyAction(javax.faces.event.ActionEvent event) 
    {
        m_editorParameterString = m_newEditorParamString;
    }

    // ------------------------------------------------------------------------
    // private usage
    // ------------------------------------------------------------------------
    
    private void readText()
    {
        if (m_texts.containsKey(m_codeType) == false)
        {
            String text = new ClassloaderReader(true).readUTF8File("workplace/DemoCodeEditor_"+m_codeType+".txt",true);
            m_texts.put(m_codeType,text);
        }
    }
}
