The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • General
  • General Issues
  • Unicode

All literals and values that are managed within CaptainCasa Enterprise Client are managed as Unicode characters - both inside the Java programming (which is default in Java) and within the communication, that is based on UTF-8.

Please note that the whole client can be started in righ-to-left mode as well.
