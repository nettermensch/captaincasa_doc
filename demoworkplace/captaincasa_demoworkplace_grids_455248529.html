The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • Grids & Trees
  • Grids and Trees
  • Radio Button in Grid (I)

Radio buttons can be used in a grid as well, of course. When using please pay attention to:

(1) A radio button group never spans several lines. Remember: all the radio buttons of one group maintain exactly one and the same property instance. In case of a grid, each row represents a different object instance, and as consequence a different property instance, on server side.

(2) The name of the group must be different from row to row. You achieve this by binding the name to a server side value that is part of the row data.

This is the Java code:

package workplace;

import org.eclnt.jsfserver.elements.impl.FIXGRIDItem;
import org.eclnt.jsfserver.elements.impl.FIXGRIDListBinding;
import org.eclnt.workplace.IWorkpageDispatcher;

public class DemoGridwithrb extends PageBean
{
    // ------------------------------------------------------------------------
    // inner classes
    // ------------------------------------------------------------------------

    public class MyRow extends FIXGRIDItem
    {
        String m_prio;
        String m_text;
        String m_group;
        public void setPrio(String value) { m_prio = value; }
        public String getPrio() { return m_prio; }
        public void setText(String value) { m_text = value; }
        public String getText() { return m_text; }
        public void setGroup(String value) {  m_group = value; }        
        public String getGroup() {  return m_group; }        
    }
    
    // ------------------------------------------------------------------------
    // members
    // ------------------------------------------------------------------------
    
    FIXGRIDListBinding<MyRow> m_rows = new FIXGRIDListBinding<MyRow>();
    
    // ------------------------------------------------------------------------
    // constructors
    // ------------------------------------------------------------------------
    
    public DemoGridwithrb(IWorkpageDispatcher dispatcher)
    {
        super(dispatcher);
        for (int i=0; i<20; i++)
        {
            MyRow mr = new MyRow();
            mr.setPrio("A");
            mr.setText("Text " + i);
            mr.setGroup("GROUP"+i);
            m_rows.getItems().add(mr);
        }
    }
    
    // ------------------------------------------------------------------------
    // public usage
    // ------------------------------------------------------------------------
    
    public FIXGRIDListBinding<MyRow> getRows() { return m_rows; }
    
    // ------------------------------------------------------------------------
    // private usage
    // ------------------------------------------------------------------------
    

}
