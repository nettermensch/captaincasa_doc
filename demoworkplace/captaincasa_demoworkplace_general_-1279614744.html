The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • General
  • Arranging Content
  • Popup Dialogs
  • Auto sizing of popups
  • Example

When passing a width and height of "0" into a popup then the popup is automatically sized according to its content.

Pay attention: tpyically pages internally contain a ROWBODYPANE-component. This component internally provides a scroll area for its content. From sizing perspective a scroll area is quite "destructive": the purpose of a scroll area is to allow the user to scroll if its inner conntent is exceeding the scroll area's dimension. Result: if a scroll are is asked from outside what its minimum size is, it will return "something very small" even if its contained content is big - the inner content's size is hidden.

When using the possibility to automatically size a popup then the JSP page that is contained in the popup should not contain any scroll are as consequence.

If changing the content of a popup at runtime (as shown if clicking onto the link "Show/hide additional lines") you may retrigger the automated resizing of the popup by calling the method Modal/ModelessPopup.setWidthHeightToDesiredSize().

This is the Java code:

package workplace;

import java.io.Serializable;

import javax.faces.event.ActionEvent;

import org.eclnt.editor.annotations.CCGenClass;
import org.eclnt.jsfserver.defaultscreens.ModalPopup;
import org.eclnt.jsfserver.util.ClientTestLog;
import org.eclnt.jsfserver.util.IClientLogAware;
import org.eclnt.workplace.IWorkpageDispatcher;

import workplace.DemoPopupPackedPopup.IListener;

@CCGenClass (expressionBase="#{d.DemoPopupPacked}")

public class DemoPopupPacked 
    extends PageBean
    implements Serializable, IClientLogAware
{
    ModalPopup m_popup;
    
    public DemoPopupPacked(IWorkpageDispatcher dispatcher)
    {
        super(dispatcher);
    }

    public void onOpenDefaultPosition(ActionEvent event) 
    {
        openPopup();
    }
    
    public void onOpenCentered(ActionEvent event) 
    {
        openPopup();
        m_popup.setLeftTopReferenceCentered();
    }
    
    public void onOpenRight(ActionEvent event) 
    {
        openPopup();
        m_popup.setLeftTopReferenceRight();
    }
    
    public void openPopup() 
    {
        // screen navigation
        final DemoPopupPackedPopup bean = new DemoPopupPackedPopup();
        m_popup = openModalPopup(bean,"Packed Popup",0,0,new ModalPopup.IModalPopupListener()
        {
            public void reactOnPopupClosedByUser()
            {
                closePopup(bean);
            }
        });
        // data navigation
        bean.prepare(new IListener()
        {
            public void reactOnClose()
            {
                closePopup(bean);
                m_popup = null;
            }
            public void reactOnToBeResized()
            {
                m_popup.setWidthHeightToDesiredSize();
            }
        });
    }
    
    public boolean onCreateClientTestLogMessage()
    {
        if (m_popup == null)
          ClientTestLog.addClientTestLogMessage("Popup is closed");  
        else
            ClientTestLog.addClientTestLogMessage("Popup is opened");  
        return false;
    }

}
