The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • General
  • Workplace Management
  • Close by Timer

This is the XML layout definition:

<?xml version="1.0" encoding="utf-8"?>
<t:layoutdefinition xmlns:t="http://www.CaptainCasa.com/controllibrary/t" xmlns:tpbc="http://www.CaptainCasa.com/controllibrary/tpbc" >
<t:beanprocessing id="g_1" >
<t:timer id="g_2" actionListener="#{d.DemoWpCloseTimer.onClose}" duration="3000" />
</t:beanprocessing>
<t:rowbodypane id="g_3" >
<t:row id="g_4" >
<t:label id="g_5" text="This workpage will be closed after 3 seconds." />
</t:row>
</t:rowbodypane>
<t:pageaddons id="g_pa"/>
</t:layoutdefinition>

This is the Java code:

package workplace;

import java.io.Serializable;

import javax.faces.event.ActionEvent;

import org.eclnt.editor.annotations.CCGenClass;
import org.eclnt.jsfserver.defaultscreens.Statusbar;
import org.eclnt.workplace.IWorkpageDispatcher;

@CCGenClass (expressionBase="#{d.DemoWpCloseTimer}")

public class DemoWpCloseTimer
    extends PageBean
    implements Serializable
{
    public DemoWpCloseTimer(IWorkpageDispatcher dispatcher)
    {
        super(dispatcher);
    }

    public void onClose(ActionEvent event) 
    {
        try
        {
            getWorkpageContainer().closeWorkpage(getWorkpage());
        }
        catch (Throwable t)
        {
            Statusbar.outputAlert("Error when closing: " + t.toString());
        }
    }

}
