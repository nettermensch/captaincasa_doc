The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • General
  • Client Integration
  • Mini Browser

The JBROWSER component is the Java-base HTML browsert that comes with the Java runtime. Well, to be honest, you should not compare this browser with any type of native browser (Internet Explorer, Firexfox, etc.) - it only renderes basics HTML. But typically it's good enough to render static HTML, e.g. help texts within the online help.

There is a "real browser" component as well: the BROWSER. This component integrated the native browser of the environment, and as consequence is able to render any HTML. But: it requires the availability of the so called swt.jar on client side, which integrated the real browser into the Java processing. The size of the swt.jar is 2 MB approximately.

So in many cases it makes sense to use JBROWSER for uncritical, static HTML text.

This is the XML layout definition:

<?xml version="1.0" encoding="utf-8"?>
<t:layoutdefinition xmlns:t="http://www.CaptainCasa.com/controllibrary/t" xmlns:tpbc="http://www.CaptainCasa.com/controllibrary/tpbc" >
<t:rowdemobodypane id="g_1" objectbinding="#{d.DemoJbrowser}" rowdistance="#{ccstylevalue['@defaultrowdistance@']}" >
<t:row id="g_2" >
<t:label id="g_3" text="URL" width="80" />
<t:field id="g_4" text="#{d.DemoJbrowser.url}" width="100%" />
<t:coldistance id="g_5" width="5" />
<t:button id="g_6" actionListener="#{d.DemoJbrowser.onShowURL}" text="Load" />
</t:row>
<t:row id="g_7" >
<t:jbrowser id="g_8" height="100%" url="#{d.DemoJbrowser.shownURL}" width="100%" />
</t:row>
</t:rowdemobodypane>
<t:pageaddons id="g_pa"/>
</t:layoutdefinition>

This is the Java code:

package workplace;

import java.io.Serializable;

import javax.faces.event.ActionEvent;

import org.eclnt.editor.annotations.CCGenClass;
import org.eclnt.jsfserver.defaultscreens.Statusbar;
import org.eclnt.jsfserver.elements.events.BaseActionEventLink;
import org.eclnt.workplace.IWorkpageDispatcher;

@CCGenClass (expressionBase="#{d.DemoJbrowser}")

public class DemoJbrowser
    extends PageBean
    implements Serializable
{
    public DemoJbrowser(IWorkpageDispatcher dispatcher)
    {
        super(dispatcher);
    }

    protected String m_shownURL;
    public String getShownURL() { return m_shownURL; }
    public void setShownURL(String value) { m_shownURL = value; }

    protected String m_url = "/workplacehelp/de/lastName.html";
    public void setUrl(String value) { m_url = value; }
    public String getUrl() { return m_url; }

    public void onShowURL(ActionEvent event) 
    {
        m_shownURL = m_url;
    }
}
