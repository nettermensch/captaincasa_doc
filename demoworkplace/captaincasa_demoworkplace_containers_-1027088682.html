The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • Containers
  • Adaptive Containers - Controls
  • Adaptive split area

Resize this demo in vertical and/or horizontal direction and observe what happens.

The ADAPTIVESPLITAREA renders two content areas - one on the left, one on the right if used with horizontal orientation, and one on the top, one on the bottom if used with vertical orientation.

If the available space is sufficient then both areas are rendered in parallel. If the space is not sufficient, then the first are will be shrinked automatically and is only shown if the user explicitly opens it.

The width when the conversion from two visible areas to one visible area is executed is defined by the minimum size of the second area. If the minimum size of this area is reached then the area before is shrinked. You also may directly set this with by using attribute SPLITSIZE.

package workplace;

import java.io.Serializable;

import org.eclnt.editor.annotations.CCGenClass;
import org.eclnt.jsfserver.elements.util.Trigger;
import org.eclnt.workplace.IWorkpageDispatcher;
import org.eclnt.workplace.WorkpageDispatchedPageBean;

import javax.faces.event.ActionEvent;

@CCGenClass (expressionBase="#{d.DemoAdaptiveSplitArea}")

public class DemoAdaptiveSplitArea
    extends PageBean 
    implements Serializable
{
    // ------------------------------------------------------------------------
    // inner classes
    // ------------------------------------------------------------------------

    // ------------------------------------------------------------------------
    // members
    // ------------------------------------------------------------------------

    String m_orientation = "horizontal";
    Trigger m_closeTrigger = new Trigger();

    // ------------------------------------------------------------------------
    // constructors & initialization
    // ------------------------------------------------------------------------

    public DemoAdaptiveSplitArea(IWorkpageDispatcher workpageDispatcher)
    {
        super(workpageDispatcher);        
        initRefComponent("adaptivesplitarea"); 
    }

    public String getPageName() { return "/workplace/demoadaptivesplitarea.jsp"; }
    public String getRootExpressionUsedInPage() { return "#{d.DemoAdaptiveSplitArea}"; }

    // ------------------------------------------------------------------------
    // public usage
    // ------------------------------------------------------------------------

    public String getOrientation() { return m_orientation; }
    public void setOrientation(String value) { this.m_orientation = value; }

    public Trigger getCloseTrigger() { return m_closeTrigger; }

    public void onNavigationAction(ActionEvent event) 
    {
        // ... do some navigation action...
        m_closeTrigger.trigger();
    }

    // ------------------------------------------------------------------------
    // private usage
    // ------------------------------------------------------------------------
}
