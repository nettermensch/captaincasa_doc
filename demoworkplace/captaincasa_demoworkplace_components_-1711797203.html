The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • Components
  • Straight Controls
  • Invokers
  • Avatar icon

The AVATORICON is a special icon that is used to render icons showing a person's image or - if not available - its initials. Of course it can be used for any similar purposes as well.

If no image is available and no explicit background color is defined then some background color is automatically chosen, which is derived from the TEXT that is passed.

This is the Java code:

package workplace;

import java.io.Serializable;

import org.eclnt.editor.annotations.CCGenClass;
import org.eclnt.jsfserver.defaultscreens.Statusbar;
import org.eclnt.workplace.IWorkpageDispatcher;

@CCGenClass (expressionBase="#{d.DemoAvatarIcon}")

public class DemoAvatarIcon
    extends PageBean 
    implements Serializable
{
    public DemoAvatarIcon(IWorkpageDispatcher workpageDispatcher)
    {
        super(workpageDispatcher);        
    }

    public String getPageName() { return "/workplace/demoavataricon.jsp"; }
    public String getRootExpressionUsedInPage() { return "#{d.DemoAvatarIcon}"; }

    public void onIconAction(javax.faces.event.ActionEvent event) 
    {
        Statusbar.outputSuccess("Icon was pressed. (" + System.currentTimeMillis() + ")");
    }
}
