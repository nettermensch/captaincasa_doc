The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • Containers
  • Foldable / Titled Pane
  • Basic Examples

The FOLDABLEPANE allows the user to open/close the content area by either clicking onto the FOLDABLEPANE's title text or by pressing the icon on the top right.

There are multiple ways of formatting the header line and the content area.

You can place exactly one FOLDABLEPANEHEADERROW into the FOLDABLEPANE. The content of the row is placed between the title text on the left and the toggle button on the right.

This is the Java code:

package workplace;

import javax.faces.event.ActionEvent;

import org.eclnt.editor.annotations.CCGenClass;
import org.eclnt.jsfserver.defaultscreens.Statusbar;
import org.eclnt.workplace.IWorkpageDispatcher;


@CCGenClass(expressionBase="#{d.demoFoldable}")

public class DemoFoldable extends PageBean
{
    boolean m_opened = true;
    
    public DemoFoldable(IWorkpageDispatcher dispatcher)
    {
        super(dispatcher);
    }
    
    public void setOpened(boolean value) 
    { 
        m_opened = value; 
    }
    public boolean getOpened() { return m_opened; }
    
    public void onCar(ActionEvent event) 
    {
        Statusbar.outputSuccess("onCar was called!");
    }

    public void onFind(ActionEvent event) 
    {
        Statusbar.outputSuccess("onFind was called!");
    }

    public void onFoldableAction(ActionEvent event) 
    {
        Statusbar.outputSuccess("onFoldableAction was called!");
    }
}
