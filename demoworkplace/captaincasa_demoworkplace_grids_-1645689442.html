The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • Grids & Trees
  • Grids and Trees
  • Special Issues
  • Individual column borders

By using attributes GRIDCOL-BORDERWIDTH and GRIDCOL-BORDERCOLOR you can explicitly define the cell border between the columns - and by this override the definitions that are done on FIXGRID level.

Please be aware of: the border is always between two columns! There is explicitly no border drawn for the right most column. Nevertheless you may define it - the user may change the column sequence, so that the most right column will be moved into grid...

This is the XML layout definition:

<?xml version="1.0" encoding="utf-8"?>
<t:layoutdefinition xmlns:t="http://www.CaptainCasa.com/controllibrary/t" >
<t:beanprocessing id="g_1" beanbinding="#{d.DemoFixgridCellBorder}" />
<t:rowdemobodypane id="g_4" objectbinding="#{d.DemoFixgridCellBorder}" >
<t:row id="g_6" >
<t:fixgrid id="g_7" height="100%" objectbinding="#{d.DemoFixgridCellBorder.grid}" width="100%" >
<t:gridcol id="g_8" bordercolor="#800000" borderwidth="1" text="First name" width="100%;100" >
<t:label id="g_10" text=".{person.firstName}" />
</t:gridcol>
<t:gridcol id="g_12" bordercolor="#008000" borderwidth="2" text="Last name" width="100%;100" >
<t:label id="g_13" text=".{person.lastName}" />
</t:gridcol>
<t:gridcol id="g_14" bordercolor="#000080" borderwidth="3" text="Birthday" width="100%;100" >
<t:label id="g_15" format="date" text=".{person.birthDay}" />
</t:gridcol>
<t:gridcol id="g_17" bordercolor="#808080" borderwidth="1" text="Id" width="100%;100" >
<t:label id="g_18" text=".{person.id}" />
</t:gridcol>
</t:fixgrid>
</t:row>
</t:rowdemobodypane>
</t:layoutdefinition>

This is the Java code:

package workplace;

import java.io.Serializable;

import org.eclnt.editor.annotations.CCGenClass;
import org.eclnt.jsfserver.elements.impl.FIXGRIDItem;
import org.eclnt.jsfserver.elements.impl.FIXGRIDListBinding;
import org.eclnt.workplace.IWorkpageDispatcher;

import workplace.logic.Person;
import workplace.logic.RandomLogic;

@CCGenClass (expressionBase="#{d.DemoFixgridCellBorder}")

public class DemoFixgridCellBorder
    extends PageBean 
    implements Serializable
{
    // ------------------------------------------------------------------------
    // inner classes
    // ------------------------------------------------------------------------

    public class GridItem extends FIXGRIDItem implements java.io.Serializable
    {
        Person i_person;
        public GridItem(Person person)
        {
            i_person = person;
        }
        public Person getPerson() { return i_person; }
        public void onRowSelect() { }
        public void onRowExecute() { }
    }

    // ------------------------------------------------------------------------
    // members
    // ------------------------------------------------------------------------

    FIXGRIDListBinding<GridItem> m_grid = new FIXGRIDListBinding<GridItem>();

    // ------------------------------------------------------------------------
    // constructors & initialization
    // ------------------------------------------------------------------------

    public DemoFixgridCellBorder(IWorkpageDispatcher workpageDispatcher)
    {
        super(workpageDispatcher);    
        fillGrid();
    }

    public String getPageName() { return "/workplace/demofixgridcellborder.xml"; }
    public String getRootExpressionUsedInPage() { return "#{d.DemoFixgridCellBorder}"; }

    // ------------------------------------------------------------------------
    // public usage
    // ------------------------------------------------------------------------

    public FIXGRIDListBinding<GridItem> getGrid() { return m_grid; }

    // ------------------------------------------------------------------------
    // private usage
    // ------------------------------------------------------------------------
    
    private void fillGrid()
    {
        m_grid.getItems().clear();
        for (int i=0; i<100; i++)
            m_grid.getItems().add(new GridItem(RandomLogic.createPerson()));
    }

}
