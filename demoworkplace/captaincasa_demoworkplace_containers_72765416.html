The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • Containers
  • Transfer of actual Size
  • Pixel size

The component SIZETRANSFER can be arranged inside a PANE. With every roundtrip it reports to the server the current size of its surrounding pane.

You may optionally set the FLUSH attribute to "true". In this case the size will be transferred if it changes. When flush is defined to be "true", then (and only then) also a corresponding actionListener will be called on server side.
There is a certain delay for sending over changes of the screen size, so that there is not too much traffic generated if the user is resizing the screen.

In case of client-scaling (e.g. client rendering scaled to "125%") the size is "down-scaled" accordingly automatically before being sent to the server side. So on server side you will always receive (scaled) pixel values.

This is the Java code:

package workplace;

import java.io.Serializable;
import org.eclnt.editor.annotations.CCGenClass;
import org.eclnt.jsfserver.defaultscreens.Statusbar;
import org.eclnt.workplace.IWorkpageDispatcher;
import org.eclnt.workplace.WorkpageDispatchedBean;

import javax.faces.event.ActionEvent;

@CCGenClass (expressionBase="#{d.DemoSizeTransfer}")

public class DemoSizeTransfer
    extends PageBean 
    implements Serializable
{
    Integer m_flushDuration = null;
    public Integer getFlushDuration() { return m_flushDuration; }
    public void setFlushDuration(Integer value) { this.m_flushDuration = value; }

    boolean m_flush = false;
    protected String m_size;
    
    public DemoSizeTransfer(IWorkpageDispatcher workpageDispatcher)
    {
        super(workpageDispatcher);        
        initRefComponent("sizetransfer"); 
    }

    public String getSize() { return m_size; }
    public void setSize(String value) { m_size = value; }

    public boolean getFlush() { return m_flush; }
    public void setFlush(boolean value) { this.m_flush = value; }

    public void onButtonAction(ActionEvent event) 
    {
        Statusbar.outputSuccess("BUTTON: The size of the gray client area is: " + m_size);
    }
    
    public void onSizeTransferAction(ActionEvent event)
    {
        Statusbar.outputSuccess("SIZETRANSFER: The size of the gray client area is: " + m_size);
    }
}
