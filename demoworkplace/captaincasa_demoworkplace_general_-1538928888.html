The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • General
  • Mouse over events
  • Example

When setting attribute PANE-WITHMOUSEOVEREVENT to "true" then the pane will send two events to the server side processing:

1. "mouseover" - this is event is not immediately thrown if the user moves the mouse over the pane, but it is thrown if the user stays with the mouse on top of the pane for a certain duration.

2. "mouseout" - this event is thrown if the user leaves the pane AND if there was a "mouseover" event before.

This is the Java code:

package workplace;

import java.io.Serializable;

import org.eclnt.editor.annotations.CCGenClass;
import org.eclnt.jsfserver.elements.events.BaseActionEventMouseOut;
import org.eclnt.jsfserver.elements.events.BaseActionEventMouseOver;
import org.eclnt.workplace.IWorkpageDispatcher;

@CCGenClass (expressionBase="#{d.DemoMouseOver}")

public class DemoMouseOver
    extends PageBean 
    implements Serializable
{
    // ------------------------------------------------------------------------
    // members
    // ------------------------------------------------------------------------

    String m_selectedImage;

    // ------------------------------------------------------------------------
    // constructors & initialization
    // ------------------------------------------------------------------------

    public DemoMouseOver(IWorkpageDispatcher workpageDispatcher)
    {
        super(workpageDispatcher);        
    }

    public String getPageName() { return "/workplace/demomouseover.jsp"; }
    public String getRootExpressionUsedInPage() { return "#{d.DemoMouseOver}"; }

    // ------------------------------------------------------------------------
    // public usage
    // ------------------------------------------------------------------------

    public void onPane1Action(javax.faces.event.ActionEvent event) 
    {
        m_selectedImage = "/images/pictures/riscBackground2.jpg";
    }

    public void onPane2Action(javax.faces.event.ActionEvent event) 
    {
        m_selectedImage = "/images/pictures/riscBackground1.jpg";
    }

    public String getSelectedImage() { return m_selectedImage; }
    // ------------------------------------------------------------------------
    // private usage
    // ------------------------------------------------------------------------
}
