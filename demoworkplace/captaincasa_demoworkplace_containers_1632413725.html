The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • Containers
  • Structuring page content
  • Containers to structure a dialog

This demo shows three useful container components that you may use for structuring the content of pages:

1. BOX is an empty rectangular container
2. FOLDABLEPANE is a container with a title text
3. TABBEDPANE is an arrangement of several content areas, one of them being selected

All three of them are harmonized from styling point of view.

package workplace;

import org.eclnt.editor.annotations.CCGenClass;
import org.eclnt.workplace.IWorkpageDispatcher;
import org.eclnt.workplace.WorkpageDispatchedPageBean;

import javax.faces.event.ActionEvent;

@CCGenClass (expressionBase="#{d.DemoBoxing}")

public class DemoBoxing
    extends WorkpageDispatchedPageBean 
    implements Serializable
{
    public DemoBoxing(IWorkpageDispatcher workpageDispatcher)
    {
        super(workpageDispatcher);        
    }

    public String getPageName() { return "/workplace/demoboxing.jsp"; }
    public String getRootExpressionUsedInPage() { return "#{d.DemoBoxing}"; }
}
