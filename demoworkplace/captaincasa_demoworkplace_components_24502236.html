The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • Components
  • Animation & Graphics & Sound
  • Paint Area
  • Line drawing
  • Position of starting/ending points

For certain arrow types there are two variants that handle the starting/ending point in a different way.

All the lines you see have a length of 100 pixels.

Some of the starting/ending points are shifted "into" the line, so that the resulting line actually keeps the length.

Some of the starting/ending points are positioned with their center on top of the starting/ending point so that as result the line i a bit longer.
