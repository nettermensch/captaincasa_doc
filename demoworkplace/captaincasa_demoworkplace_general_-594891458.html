The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • General
  • Arranging Content
  • Popup Dialogs
  • Own style class for popup dialogs

By default the popup dialog is styled using the style class "riscdialog_popup" - which in the default style is defined as area with white background and with a certain shading.

But you can also extend this style class by own style classes - and then pass the name of the style class by calling method Modal/ModelessPopup.setStyleseq(...).

There is a default style "riscdialog_popup_transparent" which defined the background with transparent color and with no shading.

This is the XML layout definition:

<?xml version="1.0" encoding="utf-8"?>
<t:layoutdefinition xmlns:t="http://www.CaptainCasa.com/controllibrary/t" xmlns:tpbc="http://www.CaptainCasa.com/controllibrary/tpbc" >
<t:rowdemobodypane id="g_1" objectbinding="#{d.DemoPopupStyleSeq}" rowdistance="#{ccstylevalue['@defaultrowdistance@']}" >
<t:row id="g_2" >
<t:button id="g_3" actionListener="#{d.DemoPopupStyleSeq.onOpenActionDefault}" text="Default style" width="150+" />
</t:row>
<t:row id="g_4" >
<t:button id="g_5" actionListener="#{d.DemoPopupStyleSeq.onOpenActionTransparent}" text="Style &#34;transparent&#34;" width="150+" />
</t:row>
<t:row id="g_7" >
<t:button id="g_8" actionListener="#{d.DemoPopupStyleSeq.onOpenActionBorder}" text="Style &#34;border&#34;" width="150+" />
</t:row>
</t:rowdemobodypane>
<t:pageaddons id="g_pa"/>
</t:layoutdefinition>

This is the Java code:

package workplace;

import java.io.Serializable;
import org.eclnt.editor.annotations.CCGenClass;
import org.eclnt.jsfserver.defaultscreens.ModalPopup;
import org.eclnt.jsfserver.elements.util.DefaultModalPopupListener;
import org.eclnt.workplace.IWorkpageDispatcher;
import org.eclnt.workplace.WorkpageDispatchedPageBean;

import javax.faces.event.ActionEvent;

@CCGenClass (expressionBase="#{d.DemoPopupStyleSeq}")

public class DemoPopupStyleSeq
    extends PageBean 
    implements Serializable
{
    // ------------------------------------------------------------------------
    // constructors & initialization
    // ------------------------------------------------------------------------

    public DemoPopupStyleSeq(IWorkpageDispatcher workpageDispatcher)
    {
        super(workpageDispatcher);        
    }

    public String getPageName() { return "/workplace/demopopupstyleseq.jsp"; }
    public String getRootExpressionUsedInPage() { return "#{d.DemoPopupStyleSeq}"; }

    // ------------------------------------------------------------------------
    // public usage
    // ------------------------------------------------------------------------
    
    public void onOpenActionDefault(ActionEvent event)
    {
        openPopup();
    }
    
    public void onOpenActionTransparent(ActionEvent event)
    {
        ModalPopup mp = openPopup();
        mp.setStyleseq("riscdialog_popup_transparent");
    }

    public void onOpenActionBorder(ActionEvent event)
    {
        ModalPopup mp = openPopup();
        mp.setStyleseq("riscdialog_popup_border");
    }

    // ------------------------------------------------------------------------
    // private usage
    // ------------------------------------------------------------------------
    
    private ModalPopup openPopup()
    {
        final DemoPopupStyleSeqPopup ui = new DemoPopupStyleSeqPopup();
        ui.prepare(new DemoPopupStyleSeqPopup.IListener()
        {
            public void reactOnClosed()
            {
                closePopup(ui);
            }
        });
        ModalPopup result = openModalPopup(ui,"Title",0,0,new DefaultModalPopupListener(this,ui));
        result.setUndecorated(true);
        return result;
    }
}
