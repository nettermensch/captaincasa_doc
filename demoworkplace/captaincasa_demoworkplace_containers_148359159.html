The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • Containers
  • Slide Container
  • Triple content

Uuuh, this usage type of the SLIDECONTAINER is not really simple to understand... - but it provides some nice features!

By setting attribute TRIPLESIZECONTAINER to "true" some inner issues happen:

1. the content is automatically sized so that its width is three times the width of the container. As consequence you may always slide the content to both sides and you automatically have "enough content" to be shown to the user.

2. the meaning of PIXELSHIFT is shifted as well - assuming that "0" is exactly the position to show the middle part of the content.

As consequence you can easily define scenarios with "infinite content" to the left and right.

Please note: at server side your application must reset the pixel shift to "0" every time the content is recalculated so that the client switches back to show the middle part of the content.

(In the text above we assumed that the SLIDECONTAINER is used with ORIENTATION "horizontal". The triple-size-feature is available with vertical orientation as well, of course. In this case the height of the content is the one to be explicitly sized.)

This is the Java code:

package workplace;

import java.io.Serializable;
import java.util.ArrayList;
import java.util.List;

import org.eclnt.editor.annotations.CCGenClass;
import org.eclnt.jsfserver.elements.events.BaseActionEventSlideContainer;
import org.eclnt.workplace.IWorkpageDispatcher;
import org.eclnt.workplace.WorkpageDispatchedPageBean;

@CCGenClass (expressionBase="#{d.DemoSlideContainerTriple}")

public class DemoSlideContainerTriple
    extends WorkpageDispatchedPageBean 
    implements Serializable
{
    // ------------------------------------------------------------------------
    // inner classes
    // ------------------------------------------------------------------------

    public class NumbersItem extends Object implements java.io.Serializable
    {
        int m_index;
        public int getValue() { return m_index+m_numberShift; }
    }

    // ------------------------------------------------------------------------
    // members
    // ------------------------------------------------------------------------

    List<NumbersItem> m_numbers = new ArrayList<NumbersItem>();
    List<NumbersItem> m_numbersLeft = new ArrayList<NumbersItem>();
    List<NumbersItem> m_numbersRight = new ArrayList<NumbersItem>();
    int m_pixelShift = 0;
    int m_numberShift = 0;

    // ------------------------------------------------------------------------
    // constructors & initialization
    // ------------------------------------------------------------------------

    public DemoSlideContainerTriple(IWorkpageDispatcher workpageDispatcher)
    {
        super(workpageDispatcher);        
        for (int i=0; i<10; i++)
        {
            {
                NumbersItem ni = new NumbersItem();
                ni.m_index = i-10;
                m_numbersLeft.add(ni);
            }
            {
                NumbersItem ni = new NumbersItem();
                ni.m_index = i;
                m_numbers.add(ni);
            }
            {
                NumbersItem ni = new NumbersItem();
                ni.m_index = i+10;
                m_numbersRight.add(ni);
            }
        }
    }

    public String getPageName() { return "/workplace/demoslidecontainertriple.jsp"; }
    public String getRootExpressionUsedInPage() { return "#{d.DemoSlideContainerTriple}"; }

    // ------------------------------------------------------------------------
    // public usage
    // ------------------------------------------------------------------------

    public List<NumbersItem> getNumbers() { return m_numbers; }
    public List<NumbersItem> getNumbersLeft() { return m_numbersLeft; }
    public List<NumbersItem> getNumbersRight() { return m_numbersRight; }

    public int getPixelShift() { return m_pixelShift; }
    public void setPixelShift(int value) { this.m_pixelShift = value; }

    public int getStartValue() { return m_numberShift; }
    public int getEndValue() { return m_numberShift+10; }

    public void onSlideAction(javax.faces.event.ActionEvent event) 
    {
        if (event instanceof BaseActionEventSlideContainer)
        {
            BaseActionEventSlideContainer e = (BaseActionEventSlideContainer)event;
            m_numberShift = m_numberShift + (int)Math.round( (double)m_pixelShift/(double)e.getPixelWidth() * 10);
            m_pixelShift = 0; // reset is important!
        }
    }

    // ------------------------------------------------------------------------
    // private usage
    // ------------------------------------------------------------------------
}
