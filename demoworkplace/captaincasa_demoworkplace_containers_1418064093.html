The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • Containers
  • Tiled Area

The TILEDAREA component arranges page bean instances in so called tiles. The tiles can be moved by the user by dragging them within their title area.

The information about a concrete tile arrangement is persisted.

The counter part of the TILEDAREA component is the binding class TILEDAREABinding on server side. The class provides corresponding methods to add tiles. Each tile is represented by an id, a title and a corresponding page bean object.

This is the XML layout definition:

<?xml version="1.0" encoding="utf-8"?>
<t:layoutdefinition xmlns:t="http://www.CaptainCasa.com/controllibrary/t" xmlns:tpbc="http://www.CaptainCasa.com/controllibrary/tpbc" >
<t:rowheader id="g_1" >
<t:coldistance id="g_2" width="100%" />
<t:button id="g_3" actionListener="#{d.DemoTiledArea.onClearHomeAddress}" text="Remove Home Address Tile" />
<t:coldistance id="g_4" width="50" />
</t:rowheader>
<t:rowdemobodypane id="g_5" objectbinding="#{d.DemoTiledArea}" >
<t:row id="g_6" >
<t:tiledarea id="g_7" objectbinding="#{d.DemoTiledArea.tiledAreaBinding}" persistid="demotiledarea" />
</t:row>
</t:rowdemobodypane>
<t:pageaddons id="g_pa"/>
</t:layoutdefinition>

This is the Java code:

package workplace;

import java.io.Serializable;

import javax.faces.event.ActionEvent;

import org.eclnt.editor.annotations.CCGenClass;
import org.eclnt.jsfserver.elements.impl.TILEDAREABinding;
import org.eclnt.workplace.IWorkpageDispatcher;

@CCGenClass (expressionBase="#{d.DemoTiledArea}")

public class DemoTiledArea
    extends PageBean 
    implements Serializable
{
    TILEDAREABinding m_tiledAreaBinding = new TILEDAREABinding();
    
    DemoPageBeanAddress m_homeAddress = new DemoPageBeanAddress();
    DemoPageBeanAddress m_businessAddress = new DemoPageBeanAddress();
    DemoPageBeanAddress m_vacationAddress = new DemoPageBeanAddress();

    public DemoTiledArea(IWorkpageDispatcher workpageDispatcher)
    {
        super(workpageDispatcher);        
        m_homeAddress.setStreet1("Home Street 1");
        m_homeAddress.setStreet2("Home Street 2");
        m_businessAddress.setStreet1("Business Street 1");
        m_businessAddress.setStreet2("Business Street 2");
        m_vacationAddress.setStreet1("Vacation Street 1");
        m_vacationAddress.setStreet2("Vacation Street 2");
        // add tiles
        m_tiledAreaBinding.setDragTitleOnly(true);
        m_tiledAreaBinding.addTile("HOME","Home Address",m_homeAddress,"400");
        m_tiledAreaBinding.addTile("BUSINESS","Business Address",m_businessAddress,"100%");
        m_tiledAreaBinding.addTile("VACATION","Vacation Address",m_vacationAddress);
    }

    public TILEDAREABinding getTiledAreaBinding() { return m_tiledAreaBinding; }
    
    public void onClearHomeAddress(ActionEvent event)
    {
        final String persistId = m_tiledAreaBinding.getPersistid();
        m_tiledAreaBinding.clearTiles();
        m_tiledAreaBinding.addTile("BUSINESS","Business Address",m_businessAddress,"100%");
        m_tiledAreaBinding.addTile("VACATION","Vacation Address",m_vacationAddress);
    }

}
