The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • Containers
  • Adaptive Containers - Controls
  • Flexible lines

The special row component ROWFLEXLINECONTAINER holds components which are distributed to one or more optical rows - according to the size of the components.

Only use absolute sizes for sizing the contained components - not percentage values.

package workplace;

import java.io.Serializable;
import org.eclnt.editor.annotations.CCGenClass;
import org.eclnt.workplace.IWorkpageDispatcher;
import org.eclnt.workplace.WorkpageDispatchedBean;

import javax.faces.event.ActionEvent;

@CCGenClass (expressionBase="#{d.DemoRowFlexLineContainer}")

public class DemoRowFlexLineContainer
    extends PageBean 
    implements Serializable
{
    String m_alignment = "left";
    public DemoRowFlexLineContainer(IWorkpageDispatcher workpageDispatcher)
    {
        super(workpageDispatcher);        
    }
    
    public String getAlignment() { return m_alignment; }
    public void setAlignment(String value) { this.m_alignment = value; }

}
