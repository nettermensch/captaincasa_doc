The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • General
  • Adapter Binding
  • By annotation, cleaned up

This is from functional point of view the same demo as the previous demo "Democabreflection4" - but now the logic is not melted in one class, but separated into one base generic base class and one extending class for the object "Person".

You see: there is not a lot left inside the extending class...!

This is the Java code:

package workplace;

import org.eclnt.editor.annotations.CCGenClass;
import org.eclnt.workplace.IWorkpageDispatcher;

import workplace.logic.Person;

@CCGenClass (expressionBase="#{d.Democabreflection5}")

public class Democabreflection5
    extends DemocabreflectionBase<Person> 
{
    // ------------------------------------------------------------------------
    // constructors & initialization
    // ------------------------------------------------------------------------

    public Democabreflection5(IWorkpageDispatcher workpageDispatcher)
    {
        super(workpageDispatcher);
        m_adapters.get("firstName").setMandatory(true);
        m_adapters.get("lastName").setMandatory(true);
        m_adapters.get("department").setMandatory(true);
        m_adapters.get("department").getVvb().addValidValue("0001","Sales");
        m_adapters.get("department").getVvb().addValidValue("0002","Marketing");
        m_adapters.get("department").getVvb().addValidValue("0003","Production");
        m_adapters.get("department").getVvb().addValidValue("0004","Human resources");
        m_adapters.get("title").getVvb().addValidValue("DR","Dr");
        m_adapters.get("title").getVvb().addValidValue("PROF","Prof");
        // create some bean
        Person p = new Person();
        p.setFirstName("Captain");
        p.setLastName("Casa");
        p.setTitle("DR");
        setBean(p);
    }

    public String getPageName() { return "/workplace/democabreflection5.jsp"; }
    public String getRootExpressionUsedInPage() { return "#{d.DemoCabReflection5}"; }

    // ------------------------------------------------------------------------
    // public usage
    // ------------------------------------------------------------------------

    protected void processChecks()
    {
        Person p = getBean();
        if (p.getFirstName() == null || p.getFirstName().trim().length() == 0)
            indicateError("First name does not contain any value.","firstName");
        if (p.getLastName() == null || p.getLastName().trim().length() == 0)
            indicateError("Last name does not contain any value.","lastName");
        if (p.getDepartment() == null || p.getDepartment().trim().length() == 0)
            indicateError("Department does not contain any value.","department");
    }
    
}
