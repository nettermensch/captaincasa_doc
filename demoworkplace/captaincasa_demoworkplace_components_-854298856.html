The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • Components
  • Straight Controls
  • Text Input/Output
  • Vertical Text Pane

The TEXTPANEVERTICAL renders multi line text content vertically. It's the vertical version of the normal TEXTPANE component.

The sizing is managed in the following way:

1. you may define both a HEIGHT and a WIDTH. In this case the component takes over exactly the size that you pass.

2. you may define only some HEIGHT. In this case the component calculates the desired width automatically.

This is the XML layout definition:

<?xml version="1.0" encoding="utf-8"?>
<t:layoutdefinition xmlns:t="http://www.CaptainCasa.com/controllibrary/t" xmlns:tpbc="http://www.CaptainCasa.com/controllibrary/tpbc" >
<t:rowdemobodypane id="g_1" objectbinding="#{d.DemoTextPaneVertical}" >
<t:row id="g_2" coldistance="10" >
<t:textpanevertical id="g_3" height="100%" text="#{d.DemoTextPaneVertical.text}" />
<t:pane id="g_4" height="100%" rowdistance="#{ccstylevalue['@defaultrowdistance@']}" width="100%" >
<t:row id="g_5" >
<t:textarea id="g_6" height="100%" text="#{d.DemoTextPaneVertical.text}" width="100%;200" />
</t:row>
<t:row id="g_7" >
<t:button id="g_8" text="Sync" width="100+" />
</t:row>
</t:pane>
</t:row>
</t:rowdemobodypane>
<t:pageaddons id="g_pa"/>
</t:layoutdefinition>
