The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • Grids & Trees
  • Grids and Trees
  • Selection Issues
  • Item selection and scrolling

This demos does not show a spectacular grid... - but shows that you can control the grid from server side:

(.) You can scroll the grid to any position using the setSbvalue() method
(.) You can select/deselect items as desired
(.) You can ensure that a certain item is guaranteed to be visible

Always keep in mind: the server side grid management may know hundreds of items - the client only receives these items (+ scroll bar location information), that are the visible ones.

There's one special thing built in as well: try to select item 13. The server side logic will not allow you to do so - and will immediately select item 14.

This is the XML layout definition:

This is the Java code:

package workplace;

import java.io.Serializable;

import javax.faces.event.ActionEvent;

import org.eclnt.jsfserver.elements.impl.FIXGRIDItem;
import org.eclnt.jsfserver.elements.impl.FIXGRIDListBinding;
import org.eclnt.workplace.IWorkpageDispatcher;

public class DemoGridselection 
    extends PageBean 
    implements Serializable
{
    public class MyRow extends FIXGRIDItem
    {
        String m_firstName;
        String m_lastName;
        public void setFirstName(String value) { m_firstName = value; }
        public String getFirstName() { return m_firstName; }
        public void onRowSelect()
        {
            super.onRowSelect();
            // special handling for item with index 12 (text says "13"...) this never allows
            // selection!!! Immediately jump to item 13
            if (m_rows.getItems().indexOf(this) == 12)
            {
                m_rows.deselectCurrentSelection();
                m_rows.selectItem(m_rows.getItems().get(13));
            }
        }
        
    }
    
    FIXGRIDListBinding<MyRow> m_rows = new FIXGRIDListBinding<MyRow>();

    public DemoGridselection(IWorkpageDispatcher dispatcher)
    {
        super(dispatcher);
        for (int i=0; i<100; i++)
        {
            MyRow row = new MyRow();
            row.setFirstName("First name " + (i+1));
            m_rows.getItems().add(row);
        }
        // select the first item to be selected from the beginning on
        m_rows.selectItem(m_rows.getItems().get(0));
    }
    
    public FIXGRIDListBinding<MyRow> getRows()
    {
        return m_rows;
    }

    /**
     * Moves selection to the next row. If the end of the items
     * is reached then selection is moved to the first row.
     */
    public void onSelectNextItem(ActionEvent ae)
    {
        MyRow selectedItem = m_rows.getSelectedItem();
        if (selectedItem == null)
            return;
        int index = m_rows.getItems().indexOf(selectedItem);
        index++;
        if (index >= m_rows.getItems().size())
            index = 0;  // start from the beginning
        MyRow newSelectedItem = m_rows.getItems().get(index);
        m_rows.deselectCurrentSelection();
        m_rows.selectItem(newSelectedItem);
        m_rows.ensureItemToBeDisplayed(newSelectedItem);
    }
    
    /**
     * Scroll the grid by one.
     */
    public void onScroll(ActionEvent ae)
    {
        int newSbvalue = m_rows.getSbvalue() + 1;
        int maxValue = m_rows.getItems().size() - m_rows.getClientvisibleamount();
        if (newSbvalue > maxValue)
            newSbvalue = 0; // back to top so that "rotating scrolling"
        m_rows.setSbvalue(""+newSbvalue);
    }
    
}
