The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • Containers
  • Background painting
  • BGPAINT command

There is a set of paint commands which you can use for pane components.

Using these paint commands you can...
(.) Draw colored rectangles
(.) Draw colored rectangles with a vetical or horizontal color change
(.) Draw rounded rectangles in the same way
(.) Paint images

Note: all this is the background of the pane, i.e. controls are placed on top of it.


Paint commands can be concatenated by building a semicolon separated list.

This is the XML layout definition:

<?xml version="1.0" encoding="utf-8"?>
<t:layoutdefinition xmlns:t="http://www.CaptainCasa.com/controllibrary/t" xmlns:tpbc="http://www.CaptainCasa.com/controllibrary/tpbc" >
<t:rowdemobodypane id="g_1" objectbinding="#{d.demoBgpaint}" rowdistance="#{ccstylevalue['@defaultrowdistance@']}" >
<t:row id="g_2" >
<t:label id="g_3" text="Vertical" width="150" />
<t:pane id="g_4" bgpaint="bgbackground(#00000000,#00000030,vertical)" height="100" width="100%" />
</t:row>
<t:row id="g_5" >
<t:label id="g_6" text="Horizontal" width="150" />
<t:pane id="g_7" bgpaint="bgbackground(#00000000,#00000030,horizontal)" height="100" width="100%" />
</t:row>
<t:row id="g_8" >
<t:label id="g_9" text="Command concatenation" width="150" />
<t:pane id="g_10" bgpaint="bgbackground(#00000000,#00000030,horizontal);bgbackground(#00000000,#00000030,vertical)" height="100" width="100%" />
</t:row>
<t:row id="g_11" >
<t:label id="g_12" text="With image" width="150" />
<t:pane id="g_13" bgpaint="bgbackground(#680000,#F0F0F0,horizontal);bgimage(left,/images/icons/bigspeakerpressed.png);bgimage(right,/images/icons/bigspeakerpressed.png);" height="100" width="100%" />
</t:row>
<t:row id="g_14" >
<t:label id="g_15" text="Text output" width="150" />
<t:pane id="g_16" bgpaint="bgbackground(#680000,#F0F0F0,horizontal);bgwrite(center,Hello world!);bgwrite(bottom,Hello world!,#FFFFFF80,20)" height="100" width="100%" />
</t:row>
</t:rowdemobodypane>
<t:pageaddons id="g_pa"/>
</t:layoutdefinition>
