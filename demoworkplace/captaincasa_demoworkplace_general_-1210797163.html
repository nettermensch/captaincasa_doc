The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • General
  • Client Integration
  • Mini Browser with Actions

The JBROWSER component allows to embed "special" links, that call a server side action listener.

These special links must be defined in the way "<a href='ACTION:anyfurtherinfo'>Text</a>", i.e. the reference must start with "ACTION:".

On server side an action listener is called, receiving a BaseActionEventLink event object.

This is the XML layout definition:

<?xml version="1.0" encoding="utf-8"?>
<t:layoutdefinition xmlns:t="http://www.CaptainCasa.com/controllibrary/t" xmlns:tpbc="http://www.CaptainCasa.com/controllibrary/tpbc" >
<t:rowdemobodypane id="g_1" objectbinding="#{d.DemoJbrowserAction}" >
<t:row id="g_2" >
<t:label id="g_3" text="URL" width="80" />
<t:field id="g_4" text="#{d.DemoJbrowserAction.url}" width="100%" />
<t:coldistance id="g_5" width="5" />
<t:button id="g_6" actionListener="#{d.DemoJbrowserAction.onShowURL}" text="Load" />
</t:row>
<t:row id="g_7" >
<t:jbrowser id="g_8" actionListener="#{d.DemoJbrowserAction.onBrowserAction}" height="100%" url="#{d.DemoJbrowserAction.shownURL}" width="100%" />
</t:row>
</t:rowdemobodypane>
<t:pageaddons id="g_pa"/>
</t:layoutdefinition>

This is the Java code:

package workplace;

import java.io.Serializable;

import javax.faces.event.ActionEvent;

import org.eclnt.editor.annotations.CCGenClass;
import org.eclnt.jsfserver.defaultscreens.Statusbar;
import org.eclnt.jsfserver.elements.events.BaseActionEventLink;
import org.eclnt.workplace.IWorkpageDispatcher;

@CCGenClass (expressionBase="#{d.DemoJbrowser}")

public class DemoJbrowserAction
    extends PageBean
    implements Serializable
{
    public DemoJbrowserAction(IWorkpageDispatcher dispatcher)
    {
        super(dispatcher);
    }

    protected String m_shownURL;
    public String getShownURL() { return m_shownURL; }
    public void setShownURL(String value) { m_shownURL = value; }

    protected String m_url = "/html/firstName.html";
    public void setUrl(String value) { m_url = value; }
    public String getUrl() { return m_url; }

    public void onShowURL(ActionEvent event) 
    {
        m_shownURL = m_url;
    }
    
    public void onBrowserAction(ActionEvent event)
    {
        if (event instanceof BaseActionEventLink)
        {
            BaseActionEventLink e = (BaseActionEventLink)event;
            Statusbar.outputSuccess("Action was sent to server side: " + e.getLinkId());
        }
    }
}
