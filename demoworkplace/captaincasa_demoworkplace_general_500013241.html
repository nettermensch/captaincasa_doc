The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • General
  • Arranging Content
  • Blockable Area
  • Basics

The BLOCKABLEAREA is a container component in which you can arrange exactly one other component as content.

When setting the attribute BLOCKED of the component to value "false" then the content of the BLOCKABLEAREA is completely blocked for user interaction. In addition some shading is arranged in front of the content so that the user receives some visual feed back.

BLOCKABLEAREA components can be nested.

This is the Java code:

package workplace;

import java.io.Serializable;
import org.eclnt.editor.annotations.CCGenClass;
import org.eclnt.jsfserver.pagebean.PageBean;
import org.eclnt.workplace.IWorkpageDispatcher;

import javax.faces.event.ActionEvent;

@CCGenClass (expressionBase="#{d.DemoBlockableArea}")

public class DemoBlockableArea
    extends PageBean 
    implements Serializable
{
    // ------------------------------------------------------------------------
    // inner classes
    // ------------------------------------------------------------------------
    
    // ------------------------------------------------------------------------
    // members
    // ------------------------------------------------------------------------
    
    boolean m_blocked = false;
    boolean m_blockedInner;
    
    // ------------------------------------------------------------------------
    // constructors & initialization
    // ------------------------------------------------------------------------

    public DemoBlockableArea(IWorkpageDispatcher dispatcher)
    {
        super(dispatcher);
    }

    public String getPageName() { return "/workplace/demoblockablearea.jsp"; }
    public String getRootExpressionUsedInPage() { return "#{d.DemoBlockableArea}"; }

    // ------------------------------------------------------------------------
    // public usage
    // ------------------------------------------------------------------------

    public boolean getBlocked() { return m_blocked; }
    public void setBlocked(boolean value) { this.m_blocked = value; }

    public boolean getBlockedInner() { return m_blockedInner; }
    public void setBlockedInner(boolean value) { this.m_blockedInner = value; }

    public void onBlockAction(javax.faces.event.ActionEvent event) 
    {
        m_blocked = !m_blocked;
    }
    
    public void onInnerBlockAction(javax.faces.event.ActionEvent event) 
    {
        m_blockedInner = !m_blockedInner;
    }

    
    // ------------------------------------------------------------------------
    // private usage
    // ------------------------------------------------------------------------
}
