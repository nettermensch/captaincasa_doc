The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • Containers
  • Adaptive Containers - Controls
  • Adaptive areas

Resize the screen horizontally and observer how the three blocks are arranged accordingly.

The component ROWADAPTIVEAREA arranges its content in one or several lines depending from the screen size. To do so it defines so called size categories according to a width which is available for the component.

The content is defined inside ADAPTIVEAREAITEM components, which themselves define their horizontal size depending from the size categories. In addition the component indicate if a line break should occur for which size category.

The size and position of each ADAPTAREAITEM component is defined by the attributes:
(.) ADAPTIVEWIDTHS ("<cat>:<width>;<cat>:<width>;...")
(.) ADAPTIVEBREAKS ("<cat>:<break true/false>;<cat>:<break true/false>;...")
