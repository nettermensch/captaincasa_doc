The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • Components
  • Animation & Graphics & Sound
  • 3D Animations
  • Cube rotation - simple example

The DEMOCUBEROTATOR is a powerful but very simple to use animation component:

When triggered then the current content is rotated into a certain direction - and is replaced with the updated content of the component.

This means: you do not have to deal with two content areas - the one to be rotated out, the one to be rotated in! You just work with one content area. The client side component will automatically clone the last version of the content area and will then use this clone to perform the "rotate-out" - and it will use the updated version of the "one" content area to perform the "rotate-in".

In the example the content of the component is exchanged with each click on a rotation-button - by selecting one page bean out of a list of page beans.

This is the Java code:

package workplace;

import java.io.Serializable;
import java.util.ArrayList;
import java.util.List;

import org.eclnt.editor.annotations.CCGenClass;
import org.eclnt.jsfserver.elements.BaseActionEvent;
import org.eclnt.jsfserver.elements.util.Trigger;
import org.eclnt.jsfserver.pagebean.IPageBean;
import org.eclnt.workplace.IWorkpageDispatcher;

@CCGenClass (expressionBase="#{d.DemoCubeRotator}")

public class DemoCubeRotator
    extends PageBean
    implements Serializable
{
    // ------------------------------------------------------------------------
    // inner classes
    // ------------------------------------------------------------------------

    // ------------------------------------------------------------------------
    // members
    // ------------------------------------------------------------------------

    List<IPageBean> m_pageBeans = new ArrayList<IPageBean>();
    int m_index = 0;
    String m_direction;
    Trigger m_trigger = new Trigger();

    // ------------------------------------------------------------------------
    // constructors & initialization
    // ------------------------------------------------------------------------

    public DemoCubeRotator(IWorkpageDispatcher workpageDispatcher)
    {
        super(workpageDispatcher);        
        m_pageBeans.add(new DemoHelloWorld(workpageDispatcher));
        m_pageBeans.add(new DemoMinispread(workpageDispatcher));
        m_pageBeans.add(new DemoBudget(workpageDispatcher));
    }

    public String getPageName() { return "/workplace/democuberotator.xml"; }
    public String getRootExpressionUsedInPage() { return "#{d.DemoCubeRotator}"; }

    // ------------------------------------------------------------------------
    // public usage
    // ------------------------------------------------------------------------

    public IPageBean getContentBean() 
    { 
        return m_pageBeans.get(m_index % m_pageBeans.size()); 
    }

    public void onRotateAction(javax.faces.event.ActionEvent event) 
    {
        m_direction = ((BaseActionEvent)event).getSourceReference();
        m_trigger.trigger();
        m_index++;
    }

    public String getDirection() { return m_direction; }
    public void setDirection(String value) { this.m_direction = value; }

    public Trigger getTrigger() { return m_trigger; }

    // ------------------------------------------------------------------------
    // private usage
    // ------------------------------------------------------------------------
}
