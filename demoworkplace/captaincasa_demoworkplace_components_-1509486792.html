The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • Components
  • Straight Controls
  • Field Control
  • Restricted keys only

By using attribute RESTRICTTOKEYS you can limit the keyboard input of the user to certain characters.

Please pay attention: the user still can copy paste other characters into the field!
Consequence: you still need to check values, either on server side by your backing bean logic, or by applying a regular expression in addition.

This is the XML layout definition:

<?xml version="1.0" encoding="utf-8"?>
<t:layoutdefinition xmlns:t="http://www.CaptainCasa.com/controllibrary/t" xmlns:tpbc="http://www.CaptainCasa.com/controllibrary/tpbc" >
<t:rowdemobodypane id="g_1" objectbinding="#{d.DemoFieldRestrictToKeys}" rowdistance="#{ccstylevalue['@defaultrowdistance@']}" >
<t:row id="g_2" >
<t:label id="g_3" text="FIELD, 0..9 only" width="180" />
<t:field id="g_4" restricttokeys="0123456789" text="#{d.DemoFieldRestrictToKeys.fieldValue}" width="100" />
</t:row>
<t:row id="g_5" >
<t:label id="g_6" text="FORMATTEDFIELD, 0..9 only" width="180" />
<t:formattedfield id="g_7" format="int" restricttokeys="0123456789" value="#{d.DemoFieldRestrictToKeys.formattedfieldValue}" width="100" />
</t:row>
<t:row id="g_8" >
<t:label id="g_9" text="COMBOFIELD, aAbBcC only" width="180" />
<t:combofield id="g_10" restricttokeys="aAbBcC" text="#{d.DemoFieldRestrictToKeys.fieldValue}" width="100" />
</t:row>
</t:rowdemobodypane>
<t:pageaddons id="g_pa"/>
</t:layoutdefinition>

This is the Java code:

package workplace;

import java.io.Serializable;
import org.eclnt.editor.annotations.CCGenClass;
import org.eclnt.workplace.IWorkpageDispatcher;
import org.eclnt.workplace.WorkpageDispatchedBean;

@CCGenClass (expressionBase="#{d.DemoFieldRestrictToKeys}")

public class DemoFieldRestrictToKeys
    extends PageBean 
    implements Serializable
{
    // ------------------------------------------------------------------------
    // constructors & initialization
    // ------------------------------------------------------------------------

    public DemoFieldRestrictToKeys(IWorkpageDispatcher workpageDispatcher)
    {
        super(workpageDispatcher);        
    }

    // ------------------------------------------------------------------------
    
    protected int m_formattedfieldValue;
    public int getFormattedfieldValue() { return m_formattedfieldValue; }
    public void setFormattedfieldValue(int value) { m_formattedfieldValue = value; }

    protected String m_combofieldValue;
    public String getCombofieldValue() { return m_combofieldValue; }
    public void setCombofieldValue(String value) { m_combofieldValue = value; }

    protected String m_fieldValue;
    public String getFieldValue() { return m_fieldValue; }
    public void setFieldValue(String value) { m_fieldValue = value; }

}
