The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • Components
  • Straight Controls
  • Text Input/Output
  • Text Pane - word wrapping

The same text is output to a TEXTPANE, one time without word-wrapping, the other time with word-wrapping.

By setting attribute TEXTPANE-WORDWRAP to "true", long words are wrapped if required. By default long words are cut.
