The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • Containers
  • Split Pane - Extended Functions
  • Divider Location Settings

This example shows different ways of using the attribute DIVDELOCATION.

By default it is interpreted as a (scaled) pixel or percentage value that is seen "from the left".

(.) 150 ==> 150 from the left
(.) 50% ==> 50% from the left

If you define the value lower than -10000, then the value is interpeted as (scaled) pixel value that is seen "from the right".

(.) -10150 ==> 150 from the right
