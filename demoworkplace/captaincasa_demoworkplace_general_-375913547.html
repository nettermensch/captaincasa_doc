The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • General
  • Asynchronous Processing
  • Call Server on Trigger

There are situations in which you want to asynchronously start some processing on client side. This is the purpose of the component CALLSERVERONTRIGGER - if triggered by the server, then a call is executed by the client as consequence, invoking the action listener that is defined with the component.

If no duration is defined then the asynchronous calling is executed on client side as fast possible. If a duration is defined, then the client waits correspondingly before calling the server.

Example: you want to show the user a certain screen for a certain duration - and then automatically guide him to the next screen.

This is the XML layout definition:

<?xml version="1.0" encoding="utf-8"?>
<t:layoutdefinition xmlns:t="http://www.CaptainCasa.com/controllibrary/t" xmlns:tpbc="http://www.CaptainCasa.com/controllibrary/tpbc" >
<t:beanprocessing id="g_1" >
<t:callserverontrigger id="g_2" actionListener="#{d.DemoCallServerByTrigger.onCallServerAction}" duration="1000" trigger="#{d.DemoCallServerByTrigger.trigger}" />
</t:beanprocessing>
<t:rowdemobodypane id="g_3" objectbinding="#{d.DemoCallServerByTrigger}" >
<t:row id="g_4" >
<t:button id="g_5" actionListener="#{d.DemoCallServerByTrigger.onTriggerAction}" text="Trigger!" />
</t:row>
</t:rowdemobodypane>
<t:pageaddons id="g_pa"/>
</t:layoutdefinition>

This is the Java code:

package workplace;

import java.io.Serializable;

import javax.faces.event.ActionEvent;

import org.eclnt.editor.annotations.CCGenClass;
import org.eclnt.jsfserver.defaultscreens.Statusbar;
import org.eclnt.jsfserver.elements.util.Trigger;
import org.eclnt.workplace.IWorkpageDispatcher;
import org.eclnt.workplace.WorkpageDispatchedPageBean;

@CCGenClass (expressionBase="#{d.DemoCallServerByTrigger}")

public class DemoCallServerByTrigger
    extends PageBean 
    implements Serializable
{
    // ------------------------------------------------------------------------
    // members
    // ------------------------------------------------------------------------
    
    Trigger m_trigger = new Trigger();
    
    // ------------------------------------------------------------------------
    // constructors & initialization
    // ------------------------------------------------------------------------

    public DemoCallServerByTrigger(IWorkpageDispatcher workpageDispatcher)
    {
        super(workpageDispatcher);        
    }

    public String getPageName() { return "/workplace/democallserverbytrigger.jsp"; }
    public String getRootExpressionUsedInPage() { return "#{d.DemoCallServerByTrigger}"; }

    // ------------------------------------------------------------------------
    // public usage
    // ------------------------------------------------------------------------

    public Trigger getTrigger() { return m_trigger; }
    public void setTrigger(Trigger value) { this.m_trigger = value; }

    public void onTriggerAction(ActionEvent event) 
    {
        m_trigger.trigger();
    }

    public void onCallServerAction(ActionEvent event) 
    {
        Statusbar.outputSuccessWithPopup("Server was called!");
    }

    // ------------------------------------------------------------------------
    // private usage
    // ------------------------------------------------------------------------
}
