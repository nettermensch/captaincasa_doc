The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • Components
  • Straight Controls
  • Invokers
  • Font icons

The FONTICON is a button/icon that uses an image font for rendering its image.

In the icon you need to reference the font-family and the character hex code of the character.

This is the Java code:

package workplace;

import java.io.Serializable;

import javax.faces.event.ActionEvent;

import org.eclnt.editor.annotations.CCGenClass;
import org.eclnt.jsfserver.defaultscreens.Statusbar;
import org.eclnt.workplace.IWorkpageDispatcher;

@CCGenClass (expressionBase="#{d.DemoFontIcon}")

public class DemoFontIcon
    extends PageBean 
    implements Serializable
{
    public DemoFontIcon(IWorkpageDispatcher workpageDispatcher)
    {
        super(workpageDispatcher);
        initRefComponent("fonticon"); 
    }

    public String getPageName() { return "/workplace/demofonticon.jsp"; }
    public String getRootExpressionUsedInPage() { return "#{d.DemoFontIcon}"; }
    
    public void onIconAction(ActionEvent event)
    {
        Statusbar.outputSuccessWithPopup("Icon was pressed!");
    }

}
