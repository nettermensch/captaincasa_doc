The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • Components
  • Straight Controls
  • Menus
  • Show default browser menu

Right click into each of the following text areas and you will see:

1. in the first text area the default browser menu pops up - this is due to the usage of the reserved word "CCBROWSER" as popup menu

2. in the second text area a normal RISC menu shows up - which has the id "NORMAL" and which is defined in the POPUPMENU definition

3. in the third text are no RISC menu shows up - because there is no popup definition at all.

The usage of the default browser popup menu is not activated by default - because it typically confuses the user - and it provides functions (like viewing the source of the page) which are not relevant fofr application users.

For the TEXTAREA, if switched to "check spelling" the default menu popup is very useful, because it shows proposals to replace words which are not spelled in a correct way.

This is the Java code:

package workplace;

import java.io.Serializable;
import org.eclnt.editor.annotations.CCGenClass;
import org.eclnt.jsfserver.elements.events.BaseActionEventDrop;
import org.eclnt.jsfserver.elements.events.BaseActionEventPopupMenuItem;
import org.eclnt.workplace.IWorkpageDispatcher;
import org.eclnt.workplace.WorkpageDispatchedPageBean;

import javax.faces.event.ActionEvent;

@CCGenClass (expressionBase="#{d.DemoPopupMenuBrowser}")

public class DemoPopupMenuBrowser
    extends PageBean 
    implements Serializable
{
    String m_text1 = "Some text to be editet.";
    String m_text2 = "Some text to be editet.";
    String m_text3 = "Some text to be editet.";

    public DemoPopupMenuBrowser(IWorkpageDispatcher workpageDispatcher)
    {
        super(workpageDispatcher);        
    }

    public String getPageName() { return "/workplace/demopopupmenubrowser.jsp"; }
    public String getRootExpressionUsedInPage() { return "#{d.DemoPopupMenuBrowser}"; }


    public String getText1() { return m_text1; }
    public void setText1(String value) { this.m_text1 = value; }
    public String getText2() { return m_text2; }
    public void setText2(String value) { this.m_text2 = value; }
    public String getText3() { return m_text3; }
    public void setText3(String value) { this.m_text3 = value; }
    
    public void onText3Action(javax.faces.event.ActionEvent event) 
    {
        if (event instanceof BaseActionEventPopupMenuItem)
        {
            BaseActionEventPopupMenuItem e = (BaseActionEventPopupMenuItem)event;
            if ("CLEAR".equals(e.getCommand()))
            {
                m_text3 = null;
            }
        }
    }
}
