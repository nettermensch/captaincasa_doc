The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • Components
  • Straight Controls
  • Invokers
  • Rounded Icons

This demo shows how to style rounded icons. The core attributes to use are:

1. WIDTH/HEIGHT - to explicitly define the dimension of the icon
2. ROUNDING - to define the rounding of edges
3. BACKGROUND - to define the background behind the icon

ROUNDING is available with a lot of components. The defaut rounding comes from the CSS definitions - but can be overridden, as show in this example.
