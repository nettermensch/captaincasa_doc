The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • Components
  • Scheduling
  • Light weight schedule items

The SCHEDULEITEMAREA component is a simple variant of the SCHEDULEITEM component.

It is just representing a container area that is positioned inside a SCHEDULE - without any further function. You can place any type of component inside.

It's purpose is to serve as much more lightweight component (compared to SCHEDULEITEM) in scenarios, in which e.g. sizing of schedule items is not required.

This is the Java code:

package workplace;

import java.io.Serializable;
import java.util.ArrayList;
import java.util.List;

import org.eclnt.editor.annotations.CCGenClass;
import org.eclnt.jsfserver.elements.componentnodes.SCHEDULEITEMAREANode;
import org.eclnt.jsfserver.elements.impl.DYNAMICCONTENTBinding;
import org.eclnt.jsfserver.elements.impl.ROWDYNAMICCONTENTBinding.ComponentNode;
import org.eclnt.workplace.IWorkpageDispatcher;
import org.eclnt.workplace.WorkpageDispatchedPageBean;

@CCGenClass (expressionBase="#{d.DemoScheduleItemArea}")

public class DemoScheduleItemArea
    extends PageBean 
    implements Serializable
{
    // ------------------------------------------------------------------------
    // inner classes
    // ------------------------------------------------------------------------

    // ------------------------------------------------------------------------
    // members
    // ------------------------------------------------------------------------

    DYNAMICCONTENTBinding m_dynContentSimple = new DYNAMICCONTENTBinding();

    // ------------------------------------------------------------------------
    // constructors & initialization
    // ------------------------------------------------------------------------

    public DemoScheduleItemArea(IWorkpageDispatcher workpageDispatcher)
    {
        super(workpageDispatcher);        
        renderContentSimple();
    }

    public String getPageName() { return "/workplace/demoscheduleitemarea.jsp"; }
    public String getRootExpressionUsedInPage() { return "#{d.DemoScheduleItemArea}"; }

    // ------------------------------------------------------------------------
    // public usage
    // ------------------------------------------------------------------------

    public DYNAMICCONTENTBinding getDynContentSimple() { return m_dynContentSimple; }

    // ------------------------------------------------------------------------
    // private usage
    // ------------------------------------------------------------------------
    
    private void renderContentSimple()
    {
        List<ComponentNode> l = new ArrayList<ComponentNode>();
        for (int i=0; i<25; i++)
        {
            String background;
            if (i%2 == 0)
                background = "#FFFFFF";
            else
                background = "#800000";
            SCHEDULEITEMAREANode sia = new SCHEDULEITEMAREANode()
                    .setScheduleleft(i*5)
                    .setSchedulewidth(4)
                    .setBackground(background);
            l.add(sia);
        }
        m_dynContentSimple.setContentNodes(l);
    }

}
