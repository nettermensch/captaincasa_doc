The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • General
  • General Issues
  • Long lasting Operations

On server side the processing is passed to a thread which runs aside the request thread. Consequence: a quick response is given to the user, the user does not have to wait for the execution of the processing thread.

The processing thread writes information into a log. Because all communication via http is a request based communication there is a polling - which is programmed to stop when the server side processing thread is finished.

The polling is done in the following way:

(.) There is the method AsynchMessageBus.addMessage("demoPolling()") on server side. This sends - as part of the repsonse - an event to the client. The event has a command (the name) and may have parameters.

(.) On client side there is a component MESSAGELISTENER that is part of the layout definition. Within the component the attribute COMMANDFILTER specifies the events that the component is interested in. There is the attribute DELAY that specifies a time (in ms) to wait until an actionListener is called on server side.

This is the Java code:

package workplace;

import javax.faces.event.ActionEvent;

import org.eclnt.editor.annotations.CCGenClass;
import org.eclnt.jsfserver.defaultscreens.AsynchMessageBus;
import org.eclnt.jsfserver.defaultscreens.Statusbar;
import org.eclnt.workplace.IWorkpageDispatcher;

@CCGenClass(expressionBase="#{d.demoPolling}")

public class DemoPolling extends WorkpageDispatchedPageBean
{
    // ------------------------------------------------------------------------
    // inner classes
    // ------------------------------------------------------------------------
    
    class LongLastingOperation extends Thread
    {
        boolean i_finished = false;
        public void run()
        {
            for (int i=0; i<8; i++)
            {
                try { Thread.currentThread().sleep(1000); } catch (Throwable t) {}
                m_log = m_log +     "Step " + (i+1) + " of 8 was executed.\n";
            }
            m_log = m_log + "...finished!";
            i_finished = true;
            m_blocked = false;
        }
    }
    
    // ------------------------------------------------------------------------
    // members
    // ------------------------------------------------------------------------
    
    String m_log = "";
    boolean m_blocked = false;
    LongLastingOperation m_llop;
    
    // ------------------------------------------------------------------------
    // constructors
    // ------------------------------------------------------------------------
    
    public DemoPolling(IWorkpageDispatcher dispatcher)
    {
        super(dispatcher);
    }
    
    // ------------------------------------------------------------------------
    // public usage
    // ------------------------------------------------------------------------

    public void setLog(String value) { m_log = value; }
    public String getLog() { return m_log; }
    
    public boolean getBlocked() { return m_blocked; }
    
    public void onStartOperation(ActionEvent ae)
    {
        // avoid double starting
        if (m_llop != null && m_llop.i_finished == false)
            return;
        // start
        m_llop = new LongLastingOperation();
        m_log = "Started operation...\n";
        m_llop.start();
        triggerNextPolling();
        m_blocked = true;
        Statusbar.outputMessage("Long running server side operation was started.");
    }
    
    public void onSynchronize(ActionEvent ae)
    {
        triggerNextPolling();
    }
    
    public boolean getButtonEnabled()
    {
        if (m_llop == null || m_llop.i_finished == true)
            return true;
        else
            return false;
    }
    
    public String getLogBackground()
    {
        if (m_llop == null || m_llop.i_finished == true)
            return "#FFFFFF";
        else
            return "#FFE0E0";
    }
    
    // ------------------------------------------------------------------------
    // private usage
    // ------------------------------------------------------------------------
    
    private void triggerNextPolling()
    {
        if (m_llop.i_finished == true)
        {
            Statusbar.outputSuccess("Long lasting operation was successfully executed.");
            return;
        }
        AsynchMessageBus.addMessage("demoPolling()");
    }

}
