The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • Containers
  • Focus Observation

Move the focus into one of the fields. The corresponding area will be highlighted - controlled by a roundtrip to the server.

When setting PANE-FOCUSSENSITIVE to true, then the PANE observes, if the focus is placed inside its content. If so, then a corresponding event is sent to server side (BaseActionEventAreaFocusGained).

Please note: the event is only passed if the user is focussing a component within the PANE "from outside". When the focus is moved within the PANE there is no event triggered.

This is the Java code:

package workplace;

import java.io.Serializable;

import javax.faces.event.ActionEvent;

import org.eclnt.editor.annotations.CCGenClass;
import org.eclnt.jsfserver.defaultscreens.Statusbar;
import org.eclnt.workplace.IWorkpageDispatcher;

@CCGenClass (expressionBase="#{d.DemoFocusSensitive}")

public class DemoFocusSensitive
    extends PageBean
    implements Serializable
{
    // ------------------------------------------------------------------------
    // inner classes
    // ------------------------------------------------------------------------
    
    // ------------------------------------------------------------------------
    // members
    // ------------------------------------------------------------------------
    
    int m_focusedArea = -1;

    // ------------------------------------------------------------------------
    // constructors & initialization
    // ------------------------------------------------------------------------

    public DemoFocusSensitive(IWorkpageDispatcher dispatcher)
    {
        super(dispatcher);
    }

    public String getPageName() { return "/workplace/demofocussensitive.jsp"; }
    public String getRootExpressionUsedInPage() { return "#{d.DemoFocusSensitive}"; }
    
    // ------------------------------------------------------------------------
    // public usage
    // ------------------------------------------------------------------------

    public String getLeftAreaColor() { return m_focusedArea == 0 ? "#FFC0C0" : null; }
    public String getCenterAreaColor() { return m_focusedArea == 1 ? "#FFC0C0" : null; }
    public String getRightAreaColor() { return m_focusedArea == 2 ? "#FFC0C0" : null; }

    public void onLeftAction(ActionEvent event) 
    { 
        m_focusedArea = 0; 
    }
    public void onCenterAction(ActionEvent event) 
    { 
        m_focusedArea = 1; 
    }
    public void onRightAction(ActionEvent event) 
    { 
        m_focusedArea = 2; 
    }

    public void onButtonAction(javax.faces.event.ActionEvent event) 
    {
        Statusbar.outputMessage("Button was pressed: " + System.currentTimeMillis());
    }


    // ------------------------------------------------------------------------
    // private usage
    // ------------------------------------------------------------------------
}
