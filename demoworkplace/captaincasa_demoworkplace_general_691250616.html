The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • General
  • Arranging Content
  • Popup Dialogs
  • Popup style selection

This example shows how to open a dialog that points to the component starting the dialog. Note, that the popup's styling is changing when clicking the button at the bottom.

The popup is opened with using style class "riscdialog_popup_pointer".

When taking a look into style class definitions then you find three style classes:

1. "riscdialog_popup_pointer"
2. "riscdialog_popup_pointer_alternative"
3. "riscdialog_popup_pointer_else"

The selection of styles is done in the following way:

A. A dialog popup is opened at a preferred location - which is normally directly below the control that triggers the creation of the popup. If this preferred location really is the one which is used then the "normal" style class is used, that is assigned to the dialog popup.

B. If the dialog does not fit at its preferred location due to missing screen space, then an alternative location is defined - which is normally directly above the triggering control. If a style with the extension "_alternative" is defined then this style is used.

C. If also the alternative location does not fit, then the popup is positioned "somewhere else" - and the style class with extension "_else" is used, if available.

This is the Java code:

package workplace;

import java.io.Serializable;

import org.eclnt.editor.annotations.CCGenClass;
import org.eclnt.jsfserver.defaultscreens.ModelessPopup;
import org.eclnt.jsfserver.elements.util.DefaultModelessPopupListener;
import org.eclnt.workplace.IWorkpageDispatcher;

@CCGenClass (expressionBase="#{d.DemoPopupWithArrow}")

public class DemoPopupWithArrow
    extends PageBean 
    implements Serializable
{
    // ------------------------------------------------------------------------
    // constructors & initialization
    // ------------------------------------------------------------------------

    public DemoPopupWithArrow(IWorkpageDispatcher workpageDispatcher)
    {
        super(workpageDispatcher);        
    }

    public String getPageName() { return "/workplace/demopopupwitharrow.jsp"; }
    public String getRootExpressionUsedInPage() { return "#{d.DemoPopupWithArrow}"; }

    // ------------------------------------------------------------------------
    // public usage
    // ------------------------------------------------------------------------

    public void onOpenPopupAction(javax.faces.event.ActionEvent event) 
    {
        final DemoPopupWithArrowPopup ui = new DemoPopupWithArrowPopup();
        ui.prepare(new DemoPopupWithArrowPopup.IListener()
        {
            @Override
            public void reactOnClose() { closePopup(ui); }
        });
        ModelessPopup p = openModelessPopup(ui,"",300,0,new DefaultModelessPopupListener(this,ui));
        p.setUndecorated(true);
        p.setCloseonclickoutside(true);
        p.setLeftTopReferenceBottom();
        p.setStyleseq("riscdialog_popup_pointer");
    }

    public void onOpenPopupRightAction(javax.faces.event.ActionEvent event) 
    {
        final DemoPopupWithArrowPopup ui = new DemoPopupWithArrowPopup();
        ui.prepare(new DemoPopupWithArrowPopup.IListener()
        {
            @Override
            public void reactOnClose() { closePopup(ui); }
        });
        ModelessPopup p = openModelessPopup(ui,"",300,0,new DefaultModelessPopupListener(this,ui));
        p.setUndecorated(true);
        p.setCloseonclickoutside(true);
        p.setLeftTopReferenceRight();
        p.setStyleseq("riscdialog_popup_pointer_right");
        p.setSizeAnimationtype("leftright");
    }

    public void onOpenPopupLeftAction(javax.faces.event.ActionEvent event) 
    {
        final DemoPopupWithArrowPopup ui = new DemoPopupWithArrowPopup();
        ui.prepare(new DemoPopupWithArrowPopup.IListener()
        {
            @Override
            public void reactOnClose() { closePopup(ui); }
        });
        ModelessPopup p = openModelessPopup(ui,"",300,0,new DefaultModelessPopupListener(this,ui));
        p.setUndecorated(true);
        p.setCloseonclickoutside(true);
        p.setLeftTopReferenceLeft();
        p.setStyleseq("riscdialog_popup_pointer_left");
        p.setSizeAnimationtype("rightleft");
    }

    public void onOpenPopupRightOfMouseAction(javax.faces.event.ActionEvent event) 
    {
        final DemoPopupWithArrowPopup ui = new DemoPopupWithArrowPopup();
        ui.prepare(new DemoPopupWithArrowPopup.IListener()
        {
            @Override
            public void reactOnClose() { closePopup(ui); }
        });
        ModelessPopup p = openModelessPopup(ui,"",300,0,new DefaultModelessPopupListener(this,ui));
        p.setUndecorated(true);
        p.setCloseonclickoutside(true);
        p.setLeftTopMousePosition(false);
        p.setStyleseq("riscdialog_popup_pointer_right");
        p.setSizeAnimationtype("leftright");
        p.setTop(-38);
    }

    // ------------------------------------------------------------------------
    // private usage
    // ------------------------------------------------------------------------
}
