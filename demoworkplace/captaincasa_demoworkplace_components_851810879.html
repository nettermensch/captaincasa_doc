The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • Components
  • Touch Screen
  • Keyboard scanner - without field

Either connect a barcode reader (or other device) and scan a value - or type come characters in quick order. When interrupting the keyboard input the key sequence will be sent to the server side.

The KEYBOARDSCANNER is a visible component that receives keyboard input and passes it to the server side if the keyboard input is interrupted for 400ms.

The component is useful for scnearios in which barcodes or other ids are scanned via a technical device that is internally transferring its data into keyboard events.

In order to receive keyboard events the component must be focused. As consequence it provides an attribute REQUSTFOCUSTIMER by which you can define a duration in ms, after which the component tries to request the focus in case of user inactivity.

This is the XML layout definition:

<?xml version="1.0" encoding="utf-8"?>
<t:layoutdefinition xmlns:t="http://www.CaptainCasa.com/controllibrary/t" xmlns:tpbc="http://www.CaptainCasa.com/controllibrary/tpbc" >
<t:rowdemobodypane id="g_1" objectbinding="#{d.DemoKeyboardScanner}" rowdistance="#{ccstylevalue[&#39;@defaultrowdistance@&#39;]}" >
<t:row id="g_2" >
<t:keyboardscanner id="g_3" actionListener="#{d.DemoKeyboardScanner.onScanAction}" requestfocus="creation" requestfocustimer="5000" />
</t:row>
<t:rowline id="g_4" />
<t:row id="g_5" >
<t:label id="g_6" text="Scanned Value" width="100" />
<t:field id="g_7" enabled="false" text="#{d.DemoKeyboardScanner.scannedValue}" width="100%" />
</t:row>
</t:rowdemobodypane>
<t:pageaddons id="g_pa"/>
</t:layoutdefinition>

This is the Java code:

package workplace;

import java.io.Serializable;

import org.eclnt.editor.annotations.CCGenClass;
import org.eclnt.jsfserver.elements.events.BaseActionEventScan;
import org.eclnt.workplace.IWorkpageDispatcher;
import org.eclnt.workplace.WorkpageDispatchedPageBean;

import javax.faces.event.ActionEvent;

@CCGenClass (expressionBase="#{d.DemoKeyboardScanner}")

public class DemoKeyboardScanner
    extends PageBean 
    implements Serializable
{
    // ------------------------------------------------------------------------
    // inner classes
    // ------------------------------------------------------------------------

    // ------------------------------------------------------------------------
    // members
    // ------------------------------------------------------------------------

    String m_scannedValue;

    // ------------------------------------------------------------------------
    // constructors & initialization
    // ------------------------------------------------------------------------

    public DemoKeyboardScanner(IWorkpageDispatcher workpageDispatcher)
    {
        super(workpageDispatcher);        
    }

    public String getPageName() { return "/workplace/demokeyboardscanner.jsp"; }
    public String getRootExpressionUsedInPage() { return "#{d.DemoKeyboardScanner}"; }

    // ------------------------------------------------------------------------
    // public usage
    // ------------------------------------------------------------------------

    public void onScanAction(javax.faces.event.ActionEvent event) 
    {
        if (event instanceof BaseActionEventScan)
        {
            BaseActionEventScan e = (BaseActionEventScan)event;
            m_scannedValue = e.getScannedValue();
        }
    }

    public String getScannedValue() { return m_scannedValue; }
    public void setScannedValue(String value) { this.m_scannedValue = value; }

    // ------------------------------------------------------------------------
    // private usage
    // ------------------------------------------------------------------------
}
