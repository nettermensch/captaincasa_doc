The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • Containers
  • Container sizing
  • Cut percentage height/width

You can use the attribute CUTWIDTH and CUTHEIGHT in order to influence the PANE's behavior when being sized with percentage width/height.

By default the assignment of a percentage size (e.g. width) has the following consequences:

1. the controls receives the width according to the current screen size / page constellation
2. if the width is not sufficient then the width automatically is expanded to show the inner content

So a percentage size by default means, that the control will always be rendered with its minimum size, if the percentage size is not sufficient.

By setting CUTWIDTH / CUTHEIGHT to "true" you override this default behavior: the percentage size is not interpreted as strict size, so that the content of the pane is cut when exceeding the percentage size.
