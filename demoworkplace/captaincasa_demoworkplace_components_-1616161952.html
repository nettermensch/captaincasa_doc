The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • Components
  • Straight Controls
  • Combobox and List
  • Default / Null Value

In the first combo box a default value is assigned using the attribute DEFAULTVALUE. In case the value that is assigned to the combobox is null, the default value is displayed.

In the second combo box no default value is assigned.

What you see in addition: you can define null as valid id of a combo box item. As a result you can offer the user to set the combo box to an empty (null) value.

This is the XML layout definition:

This is the Java code:

package workplace;

import java.io.Serializable;
import org.eclnt.editor.annotations.CCGenClass;
import org.eclnt.jsfserver.defaultscreens.Statusbar;
import org.eclnt.jsfserver.managedbean.IDispatcher;

@CCGenClass (expressionBase="#{d.DemoComboDefaultValue}")

public class DemoComboDefaultValue
    extends PageBean
    implements Serializable
{
    public DemoComboDefaultValue(IDispatcher dispatcher)
    {
        super(dispatcher);
        initRefComponent("combobox"); 
    }
    
    protected String m_value;
    public String getValue() { return m_value; }
    public void setValue(String value) 
    {
        m_value = value; 
        Statusbar.outputSuccess("Value passed = " + m_value);
    }

}
