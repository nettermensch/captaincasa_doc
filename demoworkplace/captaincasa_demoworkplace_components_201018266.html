The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • Components
  • Touch Screen
  • Keyboard scanner - without field - multiple scans

The KEYBOARDSCANNERMULTISCAN component is able to register several scans on client side - which it collects and then sends to the client in one round-trip.

This component requires some sophisticated definition of time intervals!

(.) the attribute EVENTDELAY is the duration that the client waits after some keyboard input to recognize that a scan is ended.
(.) the attribute MULTISCANDELAY is the duration that the client then waits to check if there is no further, continuous scan.

This is the XML layout definition:

<?xml version="1.0" encoding="utf-8"?>
<t:layoutdefinition xmlns:t="http://www.CaptainCasa.com/controllibrary/t" >
<t:beanprocessing id="g_1" beanbinding="#{d.DemoKeyboardScannerMultiScan}" />
<t:rowdemobodypane id="g_4" objectbinding="#{d.DemoKeyboardScannerMultiScan}" rowdistance="#{ccstylevalue[&#39;@defaultrowdistance@&#39;]}" >
<t:row id="g_2" >
<t:keyboardscannermultiscan id="g_3" actionListener="#{d.DemoKeyboardScannerMultiScan.onScanAction}" eventdelay="200" multiscandelay="1000" requestfocus="creation" requestfocustimer="5000" />
</t:row>
<t:rowline id="g_9" />
<t:row id="g_5" >
<t:label id="g_6" text="Scanned Value" width="100" />
<t:field id="g_7" enabled="false" text="#{d.DemoKeyboardScannerMultiScan.scannedValue}" width="100%" />
</t:row>
</t:rowdemobodypane>
</t:layoutdefinition>

This is the Java code:

package workplace;

import java.io.Serializable;

import org.eclnt.editor.annotations.CCGenClass;
import org.eclnt.jsfserver.elements.events.BaseActionEventMultiScan;
import org.eclnt.workplace.IWorkpageDispatcher;

@CCGenClass (expressionBase="#{d.DemoKeyboardScannerMultiScan}")

public class DemoKeyboardScannerMultiScan
    extends PageBean 
    implements Serializable
{
    // ------------------------------------------------------------------------
    // inner classes
    // ------------------------------------------------------------------------

    // ------------------------------------------------------------------------
    // members
    // ------------------------------------------------------------------------

    String m_scannedValue;

    // ------------------------------------------------------------------------
    // constructors & initialization
    // ------------------------------------------------------------------------

    public DemoKeyboardScannerMultiScan(IWorkpageDispatcher workpageDispatcher)
    {
        super(workpageDispatcher);        
    }

    public String getPageName() { return "/workplace/demokeyboardscanner.jsp"; }
    public String getRootExpressionUsedInPage() { return "#{d.DemoKeyboardScanner}"; }

    // ------------------------------------------------------------------------
    // public usage
    // ------------------------------------------------------------------------

    public void onScanAction(javax.faces.event.ActionEvent event) 
    {
        if (event instanceof BaseActionEventMultiScan)
        {
            BaseActionEventMultiScan e = (BaseActionEventMultiScan)event;
            m_scannedValue = e.getScannedValuesAsCSV();
            // or use: e.getScannedValues(), that returns String[]
        }
    }

    public String getScannedValue() { return m_scannedValue; }
    public void setScannedValue(String value) { this.m_scannedValue = value; }

    // ------------------------------------------------------------------------
    // private usage
    // ------------------------------------------------------------------------
}
