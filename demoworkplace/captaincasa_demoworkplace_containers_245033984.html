The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • Containers
  • Tab Pane
  • Tab Pane

The tab pane allows to set a title and an image per tab. Both attributes can be updated during runtime.

The value of the currently opened tab is transferred to the server - if defining the VALUE attribute. In addition you can set the value of attribute FLUSH to "true", then the value will be transferred immediately when the user selects a tab. - As consequence you can read in some data which is required to be displayed within the tab that the user just opened.

This is the Java code:

package workplace;

import java.io.Serializable;

import javax.faces.event.ActionEvent;

import org.eclnt.editor.annotations.CCGenClass;
import org.eclnt.workplace.IWorkpageDispatcher;

@CCGenClass (expressionBase="#{d.DemoTabbedPane}")

public class DemoTabbedPane
    extends PageBean
    implements Serializable
{
    protected int m_selectedTabIndex;
    public int getSelectedTabIndex() { return m_selectedTabIndex; }
    public void setSelectedTabIndex(int value) { m_selectedTabIndex = value; }

    public DemoTabbedPane(IWorkpageDispatcher dispatcher)
    {
        super(dispatcher);
    }

    public void onUpdateTabTextImage(ActionEvent event) 
    {
        m_leftTabText += "!";
        m_rightTabText += "!";
        String temp;
        temp = m_leftTabImage;
        m_leftTabImage = m_rightTabImage;
        m_rightTabImage = temp;
    }

    protected String m_rightTabText = "Right";
    public String getRightTabText() { return m_rightTabText; }
    public void setRightTabText(String value) { m_rightTabText = value; }

    protected String m_rightTabImage = "/images/icons/car.png";
    public String getRightTabImage() { return m_rightTabImage; }
    public void setRightTabImage(String value) { m_rightTabImage = value; }

    protected String m_leftTabImage = "/images/icons/disk.png";
    public String getLeftTabImage() { return m_leftTabImage; }
    public void setLeftTabImage(String value) { m_leftTabImage = value; }

    protected String m_leftTabText = "Left";
    public String getLeftTabText() { return m_leftTabText; }
    public void setLeftTabText(String value) { m_leftTabText = value; }

}
