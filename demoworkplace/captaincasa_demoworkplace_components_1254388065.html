The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • Components
  • Straight Controls
  • Value Representation
  • Slider - From/to range

The SLIDERRANGE manages two values: a FROMVALUE and a TOVALUE. Both are read and updated when using the component.

The values can be changed by the user in the following ways:

1. the user can move the from- and to-part individually
2. the user can move the range-part between the from- and the to-part
3. the user can click into the component - the most probable value will be changed as result

As with any other component, changes to the data by default do not immediately trigger a roundtrip to the server-side. For the default scenario you need some other activity - e.g. pressing the OK button.

When setting FLUSH to "true" then the component triggers the roundtrip (and an action-event) when releasing the mouse-button.

When setting FLUSHTIMER in addition then the component also sends events with the keeping the mouse button pressed - after the user not moving the mouse for a defined duration.

This is the Java code:

package workplace;

import java.io.Serializable;

import org.eclnt.editor.annotations.CCGenClass;
import org.eclnt.workplace.IWorkpageDispatcher;

@CCGenClass (expressionBase="#{d.DemoSliderRange1}")

public class DemoSliderRange1
    extends PageBean 
    implements Serializable
{
    // ------------------------------------------------------------------------
    // members
    // ------------------------------------------------------------------------

    int m_to = 75;
    int m_from = 25;

    // ------------------------------------------------------------------------
    // constructors & initialization
    // ------------------------------------------------------------------------

    public DemoSliderRange1(IWorkpageDispatcher workpageDispatcher)
    {
        super(workpageDispatcher);        
    }

    public String getPageName() { return "/workplace/demosliderrange1.xml"; }
    public String getRootExpressionUsedInPage() { return "#{d.DemoSliderRange1}"; }

    // ------------------------------------------------------------------------
    // public usage
    // ------------------------------------------------------------------------

    public int getTo() { return m_to; }
    public void setTo(int value) { this.m_to = value; }

    public int getFrom() { return m_from; }
    public void setFrom(int value) { this.m_from = value; }

    public void onSliderAction(javax.faces.event.ActionEvent event) {}
}
