The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • Containers
  • Foldable / Titled Pane
  • Drag Drop in Title Area only

When defining DRAGSEND with a FOLDABLEPANE then by default the user can start drag operations from the whole area of the FOLDABLEPANE.

By defining attribute DRAGTITLEONLY as "true", drag activities can only be started from the title area.
