The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • General
  • RISC Client Integration
  • Subpage

A CaptainCasa dialog can be embedded either by using the general BROWSER components - or by using the SUBPAGE component.

In the SUBPAGE component you directly pass the URL of the layout definition that you want to embed.

The embedding of the page is done using the HTML-iframe element. Please pay attention: some configuration is required to embed pages that are not loaded from the same origin (protocol + server + port) than the outside page. Please check the document "CaptainCasa Security Guide", section "Content Security Protocol / CSP".

This is the XML layout definition:

<?xml version="1.0" encoding="utf-8"?>
<t:layoutdefinition xmlns:t="http://www.CaptainCasa.com/controllibrary/t" xmlns:tpbc="http://www.CaptainCasa.com/controllibrary/tpbc" >
<t:rowdemobodypane id="g_1" objectbinding="#{d.DemoSubpage}" rowdistance="10" >
<t:row id="g_2" coldistance="5" >
<t:combobox id="g_4" editable="true" labeltext="Layout URL of contained sub-page" validvaluesbinding="#{d.DemoSubpage.subpageURLVVS}" value="#{d.DemoSubpage.subpageURL}" width="100%" />
<t:button id="g_5" text="Show" width="100+" />
</t:row>
<t:rowdistance id="g_7" height="30" />
<t:row id="g_15" >
<t:label id="g_6" stylevariant="ccbig" text="Embedded layout" />
</t:row>
<t:row id="g_17" >
<t:textpane id="g_18" font="weight:normal" stylevariant="ccsmall" text="The result might not be shown due to security settings or non-availability of servers behind the selected URL!" width="100%" />
</t:row>
<t:row id="g_10" >
<t:subpage id="g_11" height="100%" subpagesessionid="#{d.DemoSubpage.subPageSessionId}" url="#{d.DemoSubpage.subpageURL}" width="100%" />
</t:row>
</t:rowdemobodypane>
<t:pageaddons id="g_pa"/>
</t:layoutdefinition>

This is the Java code:

package workplace;

import java.io.Serializable;

import org.eclnt.editor.annotations.CCGenClass;
import org.eclnt.jsfserver.elements.util.ValidValuesBinding;
import org.eclnt.jsfserver.util.HttpSessionAccess;
import org.eclnt.workplace.IWorkpageDispatcher;

@CCGenClass (expressionBase="#{d.DemoSubpage}")

public class DemoSubpage
    extends WorkpageDispatchedPageBean 
    implements Serializable
{
    // ------------------------------------------------------------------------
    // inner classes
    // ------------------------------------------------------------------------
    // ------------------------------------------------------------------------
    // members
    // ------------------------------------------------------------------------

    static final String PORT = "50000"; 
    ValidValuesBinding m_subpageURLVVS = new ValidValuesBinding();
    String m_subpageURL;

    // ------------------------------------------------------------------------
    // constructors & initialization
    // ------------------------------------------------------------------------

    public DemoSubpage(IWorkpageDispatcher workpageDispatcher)
    {
        super(workpageDispatcher);
        if (HttpSessionAccess.checkIfCurrentClientJavaTypeIsRisc())
            this.m_subpageURL = "http://localhost:"+PORT+"/demos/faces/workplace/demohelloworld.jsp";
        else if (HttpSessionAccess.checkIfCurrentClientJavaTypeIsFX())
            this.m_subpageURL = "http://localhost:"+PORT+"/demos/faces/workplace/demohelloworld.jsp?ccstyle=ccmetrofx";
        else if (HttpSessionAccess.checkIfCurrentClientJavaTypeIsSwing())
            this.m_subpageURL = "http://localhost:"+PORT+"/demos/faces/workplace/demohelloworld.jsp?ccstyle=default";
        m_subpageURLVVS.addValidValue(this.m_subpageURL,this.m_subpageURL);
        m_subpageURLVVS.addValidValue("http://www.captaincasa.online/ccdemos/faces/workplace/demohelloworld.jsp",
                                      "http://www.captaincasa.online/ccdemos/faces/workplace/demohelloworld.jsp");
        m_subpageURLVVS.addValidValue("https://www.captaincasa.online/ccdemos/faces/workplace/demohelloworld.jsp",
                                      "https://www.captaincasa.online/ccdemos/faces/workplace/demohelloworld.jsp");
    }

    public String getPageName() { return "/workplace/demosubpage.jsp"; }
    public String getRootExpressionUsedInPage() { return "#{d.DemoSubpage}"; }

    // ------------------------------------------------------------------------
    // public usage
    // ------------------------------------------------------------------------

    public ValidValuesBinding getSubpageURLVVS() { return m_subpageURLVVS; }

    public String getSubpageURL() { return m_subpageURL; }
    public void setSubpageURL(String value) { this.m_subpageURL = value; }

    // ------------------------------------------------------------------------
    // private usage
    // ------------------------------------------------------------------------
}
