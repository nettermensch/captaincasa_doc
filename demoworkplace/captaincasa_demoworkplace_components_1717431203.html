The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • Components
  • Touch Screen
  • Keyboard scanner - with field

The SCANFIELD is a special field variant that is specialized to receive values that are coming from a scan devices - and that in addition provide the direct input of the value for the user.

The additional functions are:

1. the focus is requested for this field after a certain duration of user-inactivity.
2. you can define a SCANHOTKEY which is the special key that may be sent by a scan device to indicate that the scan input is finished.
3. you can define a SCANPREFIX and a SCANPOSTFIX - special characters that may be prepended or appended by the scanner. The scanned value is cut correspondingly before being sent to the server-side.

Other typical functions are normal field functions:

1. by setting FLUSH to true, user-input is sent to the server after leaving the field
2. by settting FLUSHTIMER to a certain duration, user input is automatically sent to the server if the user finished his/her input.

This is the Java code:

package workplace;

import java.io.Serializable;

import org.eclnt.editor.annotations.CCGenClass;
import org.eclnt.jsfserver.defaultscreens.Statusbar;
import org.eclnt.jsfserver.elements.events.BaseActionEventFlush;
import org.eclnt.jsfserver.elements.events.BaseActionEventScan;
import org.eclnt.jsfserver.pagebean.PageBean;
import org.eclnt.workplace.IWorkpageDispatcher;

@CCGenClass (expressionBase="#{d.DemoScanField}")

public class DemoScanField
    extends PageBean 
    implements Serializable
{
    // ------------------------------------------------------------------------
    // inner classes
    // ------------------------------------------------------------------------
    
    // ------------------------------------------------------------------------
    // members
    // ------------------------------------------------------------------------
    
    String m_scannedValue;
    String m_normal1;
    String m_normal2;
    
    // ------------------------------------------------------------------------
    // constructors & initialization
    // ------------------------------------------------------------------------

    public DemoScanField(IWorkpageDispatcher dispatcher)
    {
        super(dispatcher);
    }

    public String getPageName() { return "/workplace/demoscanfield.jsp"; }
    public String getRootExpressionUsedInPage() { return "#{d.DemoScanField}"; }

    // ------------------------------------------------------------------------
    // public usage
    // ------------------------------------------------------------------------

    public String getNormal2() { return m_normal2; }
    public void setNormal2(String value) { this.m_normal2 = value; }

    public String getNormal1() { return m_normal1; }
    public void setNormal1(String value) { this.m_normal1 = value; }

    public String getScannedValue() { return m_scannedValue; }
    public void setScannedValue(String value) { this.m_scannedValue = value; }

    public void onScannedValueAction(javax.faces.event.ActionEvent event) 
    {
        if (event instanceof BaseActionEventScan)
        {
            BaseActionEventScan e = (BaseActionEventScan)event;
            Statusbar.outputSuccess("Value was scanned: " + m_scannedValue + "/" + e.getScannedValue());
        }
        else if (event instanceof BaseActionEventFlush)
        {
            Statusbar.outputSuccess("Value was flushed: " + m_scannedValue);
        }
    }

    // ------------------------------------------------------------------------
    // private usage
    // ------------------------------------------------------------------------
}
