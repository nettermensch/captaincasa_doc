The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • Components
  • Straight Controls
  • Label Control
  • Label

This demo shows the usage of the LABEL component.

When it comes to rendering images with more features (e.g. explicit sizing of height/width): please take a look onto the IMAGE component as well.
