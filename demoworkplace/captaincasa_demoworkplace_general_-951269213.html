The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • General
  • Tool Tips / User Hints
  • Programmed tooltip

You may explicitly take over the tooltip management by a server side implementation. To do so define the attribute TOOLTIP as "server:true". As result you will receive two corresponding events for opening and closing the tooltip.

The popup dialogs that you may open up, can be designed without any restrictions. - Of course you may not open up a popup at all, but e.g. also show some information in another area of the screen as reaction on the tooltip processing.

This is the Java code:

package workplace;

import java.io.Serializable;
import org.eclnt.editor.annotations.CCGenClass;
import org.eclnt.jsfserver.defaultscreens.ModalPopup;
import org.eclnt.jsfserver.defaultscreens.ModelessPopup;
import org.eclnt.jsfserver.defaultscreens.Statusbar;
import org.eclnt.jsfserver.elements.BaseActionEventTooltipEnded;
import org.eclnt.jsfserver.elements.BaseActionEventTooltipStarted;
import org.eclnt.jsfserver.pagebean.PageBean;
import org.eclnt.workplace.IWorkpageDispatcher;

import javax.faces.event.ActionEvent;

@CCGenClass (expressionBase="#{d.DemoToolTipByServer}")

public class DemoToolTipByServer
    extends WorkpageDispatchedPageBean 
    implements Serializable
{
    // ------------------------------------------------------------------------
    // inner classes
    // ------------------------------------------------------------------------
    
    // ------------------------------------------------------------------------
    // members
    // ------------------------------------------------------------------------

    DemoToolTipByServerPopup m_tooltipPopup = null;
    
    // ------------------------------------------------------------------------
    // constructors & initialization
    // ------------------------------------------------------------------------

    public DemoToolTipByServer(IWorkpageDispatcher dispatcher)
    {
        super(dispatcher);
    }

    public String getPageName() { return "/workplace/demotooltipbyserver.jsp"; }
    public String getRootExpressionUsedInPage() { return "#{d.DemoToolTipByServer}"; }

    // ------------------------------------------------------------------------
    // public usage
    // ------------------------------------------------------------------------

    public void onFieldAction(ActionEvent event) 
    {
        if (event instanceof BaseActionEventTooltipStarted)
        {
            Statusbar.outputMessage("Tooltip started");
            if (m_tooltipPopup == null)
            {
                m_tooltipPopup = new DemoToolTipByServerPopup();
                ModelessPopup mp = openModelessPopup(m_tooltipPopup,"",0,0,new ModelessPopup.IModelessPopupListener()
                {
                    public void reactOnPopupClosedByUser()
                    {
                        closePopup(m_tooltipPopup);
                        m_tooltipPopup = null;
                    }
                });
                mp.setUndecorated(true);
                mp.setCloseonclickoutside(true);
                mp.setCloseonmouseexit(true);
                mp.setLeftTopMousePosition(true);
            }
        }
        else if (event instanceof BaseActionEventTooltipEnded)
        {
            Statusbar.outputMessage("Tooltip ended");
            if (m_tooltipPopup != null)
            {
                closePopup(m_tooltipPopup);
                m_tooltipPopup = null;
            }
        }
    }

    // ------------------------------------------------------------------------
    // private usage
    // ------------------------------------------------------------------------
}
