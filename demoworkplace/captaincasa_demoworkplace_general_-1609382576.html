The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • General
  • RISC Client Integration
  • Update browser URL

By changing the so called "reloadurl" you can update the URL that is currently shown on the top of the page within your browser.

The direct interface to change the URL is "Client.instance().setRelaodurl(..)" which internally is using the component CLIENTCONFIG with its attribute RELOADURL.

The actual URL that was used to start the page with can be picked from the request header attribute "eclnt-locationhref".

This is the XML layout definition:

<?xml version="1.0" encoding="utf-8"?>
<t:layoutdefinition xmlns:t="http://www.captaincasa.com/controllibrary/t" >

<t:rowdemobodypane id="g_4" objectbinding="#{d.DemoUpdateURL}" rowdistance="#{ccstylevalue[&#39;@defaultrowdistance@&#39;]}" >
<t:row id="g_5" >
<t:label id="g_6" cutwidth="true" labeltext="Current URL" text="#{d.DemoUpdateURL.currentUrl}" width="100%" />
</t:row>
<t:row id="g_14" >

</t:row>
<t:row id="g_10" >
<t:button id="g_12" actionListener="#{d.DemoUpdateURL.onUpdateAction}" text="Update browser URL" />

</t:rowdemobodypane>
</t:layoutdefinition>

This is the Java code:

package workplace;

import java.io.Serializable;

import org.eclnt.editor.annotations.CCGenClass;
import org.eclnt.jsfserver.defaultscreens.Client;
import org.eclnt.jsfserver.util.HttpSessionAccess;
import org.eclnt.workplace.IWorkpageDispatcher;

@CCGenClass (expressionBase="#{d.DemoUpdateURL}")

public class DemoUpdateURL
    extends PageBean 
    implements Serializable
{
    // ------------------------------------------------------------------------
    // inner classes
    // ------------------------------------------------------------------------

    // ------------------------------------------------------------------------
    // members
    // ------------------------------------------------------------------------

    String m_currentUrl;
    String m_urlQuery = "";
    
    // ------------------------------------------------------------------------
    // constructors & initialization
    // ------------------------------------------------------------------------

    public DemoUpdateURL(IWorkpageDispatcher workpageDispatcher)
    {
        super(workpageDispatcher);        
        m_currentUrl = HttpSessionAccess.getCurrentRequest().getHeader("eclnt-locationhref");
    }

    public String getPageName() { return "/workplace/demoupdateurl.xml"; }
    public String getRootExpressionUsedInPage() { return "#{d.DemoUpdateURL}"; }

    // ------------------------------------------------------------------------
    // public usage
    // ------------------------------------------------------------------------

    public void onUpdateAction(javax.faces.event.ActionEvent event) 
    {
        String newUrl = null;
        if (m_currentUrl.indexOf("?") >= 0)
            newUrl = m_currentUrl + "&" + m_urlQuery;
        else
            newUrl = m_currentUrl + "?" + m_urlQuery;
        Client.instance().setReloadUrl(newUrl);
    }

    public String getCurrentUrl() { return m_currentUrl; }
    public void setCurrentUrl(String value) { this.m_currentUrl = value; }

    public String getUrlQuery() { return m_urlQuery; }
    public void setUrlQuery(String value) { this.m_urlQuery = value; }

    // ------------------------------------------------------------------------
    // private usage
    // ------------------------------------------------------------------------
}
