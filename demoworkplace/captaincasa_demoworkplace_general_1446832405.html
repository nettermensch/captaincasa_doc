The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • General
  • Arranging Content
  • Popup Dialogs
  • Moving window by titlebar

This example shows a window that is popped up when pressing the button. The window is an undecorated window - but sill can be moved by the user dragging & dropping the mouse onto the titlebar of the window.

In the window that is popped up, the titlebar is defined to be one that is abled to move the whole window. Use the ROWTITLEBAR-ISWINDOWMOVER attribute correspondingly.

This is the XML layout definition:

<?xml version="1.0" encoding="utf-8"?>
<t:layoutdefinition xmlns:t="http://www.CaptainCasa.com/controllibrary/t" xmlns:tpbc="http://www.CaptainCasa.com/controllibrary/tpbc" >
<t:rowdemobodypane id="g_1" objectbinding="#{d.DemoPopupTitlebar}">
<t:row id="g_2" >
<t:label id="g_3" text="Open Popup" width="100" />
<t:button id="g_4" actionListener="#{d.DemoPopupTitlebar.onStartPopup}" text="Open..." />
</t:row>
</t:rowdemobodypane>
<t:pageaddons id="g_pa"/>
</t:layoutdefinition>

This is the Java code:

package workplace;

import java.io.Serializable;

import javax.faces.event.ActionEvent;

import org.eclnt.editor.annotations.CCGenClass;
import org.eclnt.jsfserver.defaultscreens.ModalPopup;
import org.eclnt.jsfserver.elements.util.DefaultModalPopupListener;
import org.eclnt.workplace.IWorkpageDispatcher;

@CCGenClass (expressionBase="#{d.DemoPopupTitlebar}")

public class DemoPopupTitlebar
    extends PageBean
    implements Serializable
{
    public DemoPopupTitlebar(IWorkpageDispatcher dispatcher)
    {
        super(dispatcher);
    }

    public void onStartPopup(ActionEvent event) 
    {
        final DemoPopupTitlebarPopup bean = new DemoPopupTitlebarPopup(getOwningDispatcher());
        bean.prepare(new DemoPopupTitlebarPopup.IListener()
        {
            public void reactOnClose()
            {
                closePopup(bean);
            }
        });
        ModalPopup popup = openModalPopup(bean,"Moveable titlebar",400,300,new DefaultModalPopupListener(this,bean));
        popup.setUndecorated(true);
    }
}
