The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • Containers
  • Layering and absolute postioning
  • Show content on mouse over only

Certain content is only shown if the user moves with the mouse in top of the component.

The component to be used is the OVERLAYAREA - internally showing OVERLAYAREAITEMS. For each item you can set a flag SHOWONAREAMOUSEOVERONLY to "true" - as consequence this item will only be shown, if the user moves the mouse on top of the area on top.
