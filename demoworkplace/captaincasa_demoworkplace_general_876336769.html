The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • General
  • Client Integration
  • Embedded Browser

The BROWSER component embeds some page within your dialog. Internally an IFRAME is used to embed the page.

This is the XML layout definition:

<?xml version="1.0" encoding="utf-8"?>
<t:layoutdefinition xmlns:t="http://www.CaptainCasa.com/controllibrary/t" xmlns:tpbc="http://www.CaptainCasa.com/controllibrary/tpbc" >
<t:rowdemobodypane id="g_1" objectbinding="#{d.demoBrowser}" >
<t:row id="g_2" >
<t:field id="g_3" text="#{d.demoBrowser.url}" width="100%" />
<t:coldistance id="g_4" width="2" />
<t:button id="g_5" actionListener="#{d.demoBrowser.onLoad}" text="Load Page" />
</t:row>
<t:row id="g_6" >
<t:browser id="g_7" height="100%" url="#{d.demoBrowser.browserUrl}" width="100%" />
</t:row>
</t:rowdemobodypane>
<t:pageaddons id="g_pa"/>
</t:layoutdefinition>

This is the Java code:

package workplace;

import javax.faces.event.ActionEvent;

import org.eclnt.workplace.IWorkpageDispatcher;

public class DemoBrowser extends PageBean
{
    public DemoBrowser(IWorkpageDispatcher dispatcher)
    {
        super(dispatcher);
    }

//    String m_url = "/pdf/pdftest1.pdf";
    String m_url = "http://www.captaincasa.com";
//    String m_url = "http://www.poksundo.com";
    String m_browserUrl = m_url;
    
    public void setUrl(String value) { m_url = value; }
    public String getUrl() { return m_url; }
    
    public void setBrowserUrl(String value) { m_browserUrl = value; }
    public String getBrowserUrl() { return m_browserUrl; }
    
    public void onLoad(ActionEvent ae)
    {
        m_browserUrl = m_url;
    }
}
