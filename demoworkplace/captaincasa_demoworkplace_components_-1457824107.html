The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • Components
  • Straight Controls
  • Combobox and List
  • Fix definition of items

The COMBOBOX is bound to a VALUE-property on server-side.

By selecting a combobox item the corresponding ID of the item is passed into the VALUE-property.

In this demo the items are hard-coded as part of the layout. For each item there is a COMBOBOXITEM instance in the layout definition.

Please note: as any other input component a user selection is not immediately sent to the server-side but is queued within the client. You may switch attribute FLUSH to true if you want to immediately trigger a round-trip after user-selection.

This is the XML layout definition:

<?xml version="1.0" encoding="utf-8"?>
<t:layoutdefinition xmlns:t="http://www.CaptainCasa.com/controllibrary/t" >
<t:beanprocessing id="g_1" beanbinding="#{d.DemoComboboxFix}" />
<t:rowdemobodypane id="g_4" objectbinding="#{d.DemoComboboxFix}" >
<t:row id="g_6" >
<t:combobox id="g_7" labeltext="Department" value="#{d.DemoComboboxFix.department}" width="200" >
<t:comboboxitem id="g_3" text="Sales" value="0001" />
<t:comboboxitem id="g_9" text="Production" value="0002" />
<t:comboboxitem id="g_10" text="Human Resources" value="0003" />
<t:comboboxitem id="g_11" text="Development" value="0004" />
</t:combobox>
</t:row>
</t:rowdemobodypane>
</t:layoutdefinition>

package workplace;

import java.io.Serializable;

import org.eclnt.editor.annotations.CCGenClass;
import org.eclnt.workplace.IWorkpageDispatcher;

@CCGenClass (expressionBase="#{d.DemoComboboxFix}")

public class DemoComboboxFix
    extends PageBean 
    implements Serializable
{
    // ------------------------------------------------------------------------
    // inner classes
    // ------------------------------------------------------------------------

    // ------------------------------------------------------------------------
    // members
    // ------------------------------------------------------------------------

    String m_department;

    // ------------------------------------------------------------------------
    // constructors & initialization
    // ------------------------------------------------------------------------

    public DemoComboboxFix(IWorkpageDispatcher workpageDispatcher)
    {
        super(workpageDispatcher);        
    }

    public String getPageName() { return "/workplace/democomboboxfix.xml"; }
    public String getRootExpressionUsedInPage() { return "#{d.DemoComboboxFix}"; }

    // ------------------------------------------------------------------------
    // public usage
    // ------------------------------------------------------------------------

    public String getDepartment() { return m_department; }
    public void setDepartment(String value) { this.m_department = value; }

    // ------------------------------------------------------------------------
    // private usage
    // ------------------------------------------------------------------------
}
