The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • General
  • RISC Client Integration
  • Insert JavaScript

This component is usabled within the RISC Client only.

You may pass a JavaScript text which is executed when the component is triggered.

This is the XML layout definition:

<?xml version="1.0" encoding="utf-8"?>
<t:layoutdefinition xmlns:t="http://www.CaptainCasa.com/controllibrary/t" xmlns:tpbc="http://www.CaptainCasa.com/controllibrary/tpbc" >
<t:beanprocessing id="g_1" >
<t:scriptinsert id="g_2" scripttext="#{d.DemoScriptInsert.scriptText}" trigger="#{d.DemoScriptInsert.scriptTrigger}" />
</t:beanprocessing>
<t:rowdemobodypane id="g_3" objectbinding="#{d.DemoScriptInsert}" rowdistance="#{ccstylevalue['@defaultrowdistance@']}" >
<t:row id="g_4" >
<t:textarea id="g_5" height="200" text="#{d.DemoScriptInsert.scriptText}" width="100%" />
</t:row>
<t:row id="g_6" >
<t:button id="g_7" actionListener="#{d.DemoScriptInsert.onButtonAction}" text="Trigger" />
</t:row>
</t:rowdemobodypane>
<t:pageaddons id="g_pa"/>
</t:layoutdefinition>

This is the Java code:

package workplace;

import java.io.Serializable;

import javax.faces.event.ActionEvent;

import org.eclnt.editor.annotations.CCGenClass;
import org.eclnt.jsfserver.elements.util.Trigger;
import org.eclnt.workplace.IWorkpageDispatcher;
import org.eclnt.workplace.WorkpageDispatchedPageBean;

@CCGenClass (expressionBase="#{d.DemoScriptInsert}")

public class DemoScriptInsert
    extends PageBean 
    implements Serializable
{
    // ------------------------------------------------------------------------
    // inner classes
    // ------------------------------------------------------------------------

    // ------------------------------------------------------------------------
    // members
    // ------------------------------------------------------------------------

    Trigger m_scriptTrigger = new Trigger();
    String m_scriptText = "alert(\"Harry\");";

    // ------------------------------------------------------------------------
    // constructors & initialization
    // ------------------------------------------------------------------------

    public DemoScriptInsert(IWorkpageDispatcher workpageDispatcher)
    {
        super(workpageDispatcher);     
        onButtonAction(null);
    }

    public String getPageName() { return "/workplace/demoscriptinsert.jsp"; }
    public String getRootExpressionUsedInPage() { return "#{d.DemoScriptInsert}"; }

    // ------------------------------------------------------------------------
    // public usage
    // ------------------------------------------------------------------------

    public void onButtonAction(ActionEvent event) 
    {
        m_scriptTrigger.trigger();
    }

    public Trigger getScriptTrigger() { return m_scriptTrigger; }
    public void setScriptTrigger(Trigger value) { this.m_scriptTrigger = value; }

    public String getScriptText() { return m_scriptText; }
    public void setScriptText(String value) { this.m_scriptText = value; }

    // ------------------------------------------------------------------------
    // private usage
    // ------------------------------------------------------------------------
}
