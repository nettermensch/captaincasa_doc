The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • General
  • Workplace Management
  • Passing Data
  • John McEnroe

You can pass parameters from the starting component (e.g. node within the function tree of the workplace) to the workpage instance that is opened.

In the function tree you can pass parameters using a setParam(name,value) interface - in the workpage there is the counter part: a getParam(name) method, that typcically is called in the constructor of a UI processing bean.

This is the XML layout definition:

<?xml version="1.0" encoding="utf-8"?>
<t:layoutdefinition xmlns:t="http://www.CaptainCasa.com/controllibrary/t" xmlns:tpbc="http://www.CaptainCasa.com/controllibrary/tpbc" >
<t:rowdemobodypane id="g_1" rowdistance="#{ccstylevalue['@defaultrowdistance@']}" objectbinding="#{d.DemoWpPassData}">
  <t:row id="g_2" >
    <t:label id="g_3" text="First Name" width="120" />
    <t:field id="g_4" text="#{d.DemoWpPassData.firstName}" width="150" />
  </t:row>
  <t:row id="g_5" >
    <t:label id="g_6" text="Last Name" width="120" />
    <t:field id="g_7" text="#{d.DemoWpPassData.lastName}" width="150" />
  </t:row>
</t:rowdemobodypane>
<t:pageaddons id="g_pa"/>
</t:layoutdefinition>

This is the Java code:

package workplace;

import java.io.Serializable;
import org.eclnt.editor.annotations.CCGenClass;
import org.eclnt.jsfserver.managedbean.IDispatcher;
import org.eclnt.util.log.CLog;
import org.eclnt.workplace.IPrepareByWorkpageStartInfo;
import org.eclnt.workplace.IWorkpageStartInfo;

@CCGenClass (expressionBase="#{d.DemoWpPassData}")

public class DemoWpPassData
    extends WorkpageDispatchedPageBean // DemoBase extends WorkpageDisptachedBean
    implements IPrepareByWorkpageStartInfo
{
    public DemoWpPassData(IDispatcher dispatcher)
    {
       super(dispatcher);
    }

    /**
     * This method is called by the workpage dispatcher just after
     * creating the object within the dispatcher context.
     */
    public void prepare(IWorkpageStartInfo workpageStartInfo)
    {
        CLog.L.log(CLog.LL_INF,"########## prepare was called in bean");
        m_firstName = workpageStartInfo.getParam("firstName");
        m_lastName = workpageStartInfo.getParam("lastName");
    }

    protected String m_lastName;
    public String getLastName() { return m_lastName; }
    public void setLastName(String value) { m_lastName = value; }

    protected String m_firstName;
    public String getFirstName() { return m_firstName; }
    public void setFirstName(String value) { m_firstName = value; }

}
