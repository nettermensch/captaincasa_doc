The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • Containers
  • Hideable Container
  • Default usage

Sometimes you want to hide components on client side - but without removing them on the client side.

In this case you may e.g. think about setting the size of the corresponding e.g. pane (0,0), so that it is not visible anymore. But: with setting the size the components are still part of the client component tree - and need to be processed by the client side user interface management.

The HIDEABLE component removes the node below the component itself from the UI management (this means it is taken out of the client side component tree) when its HIDDEN attribute is set to "true". And it re-arranges the component into the component tree when HIDDEN is set to "false".

Please pay attention: you must only add ONE component below the HIDEABLE component! You must not add two or more components!

This is the Java code:

package workplace;

import java.io.Serializable;
import org.eclnt.editor.annotations.CCGenClass;
import org.eclnt.workplace.IWorkpageDispatcher;
import org.eclnt.workplace.WorkpageDispatchedPageBean;

import javax.faces.event.ActionEvent;

@CCGenClass (expressionBase="#{d.DemoHideable}")

public class DemoHideable
    extends WorkpageDispatchedPageBean
    implements Serializable
{
    boolean m_rowRendered = true;
    public boolean getRowRendered() { return m_rowRendered; }
    public void setRowRendered(boolean value) { this.m_rowRendered = value; }

    public void onHideAction(ActionEvent event) 
    {
        m_hidden = !m_hidden;
    }

    boolean m_hidden = false;
    public boolean getHidden() { return m_hidden; }
    public void setHidden(boolean value) { this.m_hidden = value; }

    public DemoHideable(IWorkpageDispatcher workpageDispatcher)
    {
        super(workpageDispatcher);        
    }

    public String getPageName() { return "/workplace/demohideable.jsp"; }
    public String getRootExpressionUsedInPage() { return "#{d.DemoHideable}"; }

}
