The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • Components
  • Straight Controls
  • Hot Keys
  • Associated with invokers

Invoker components (buttons, icons, menus) allow to directly bind to a key by defining their HOTKEY attribute. The attribute's value contains the string "ctrl-", "alt-", "shift-" and the keycode definition of the corresponding key.

The hot key is available "globally" within the page - i.e. the corresponding actionListener will be executed whenever the user will press the corresponding key within the window showing the page.

Pay attention: there is no conflict resolution when you define multiple components using the same hotkey definition!

Components that are associated with a hotkey automatically append this information to their tooltip, so that the user can see what key to press.

This is the Java code:

package workplace;

import javax.faces.event.ActionEvent;

import org.eclnt.editor.annotations.CCGenClass;
import org.eclnt.jsfserver.defaultscreens.IdTextSelection;
import org.eclnt.jsfserver.defaultscreens.Statusbar;
import org.eclnt.workplace.IWorkpageDispatcher;


@CCGenClass (expressionBase="#{d.demoHotkeys}")

public class DemoHotkeys extends PageBean
{
    String m_protocol = "";
    public String getProtocol() { return m_protocol; }
    public void setProtocol(String protocol) { m_protocol = protocol; }

    public DemoHotkeys(IWorkpageDispatcher dispatcher)
    {
        super(dispatcher);
        initRefComponent("popupmenu");
    }

    public void onOK(ActionEvent event)
    {
        Statusbar.outputMessage("OK was executed.");
        m_protocol += "OK was executed.\n";
    }

    public void onEnter(ActionEvent event)
    {
        Statusbar.outputMessage("Enter was executed.");
        m_protocol += "Enter was executed.\n";
    }

    public void onIcon(ActionEvent ae)
    {
        Statusbar.outputMessage("Icon was pressed.");
        m_protocol += "Icon was pressed.\n";
    }

    public void onFind(ActionEvent ae)
    {
        Statusbar.outputMessage("Find was called.");
        m_protocol += "Find was called.\n";
    }

    public void onReplace(ActionEvent ae)
    {
        Statusbar.outputMessage("Replace was called.");
        m_protocol += "Replace was called.\n";
    }
    
    public void onDepartmentAction(javax.faces.event.ActionEvent event) 
    {
        IdTextSelection idts = IdTextSelection.createInstance();
        for (int i=0; i<10; i++)
            idts.addLine(""+i,"Value " + i);
        idts.getPopup().setParentHotkeysActive(true);
    }

}
