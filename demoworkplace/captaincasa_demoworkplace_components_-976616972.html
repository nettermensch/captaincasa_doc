The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • Components
  • Straight Controls
  • Text Input/Output
  • Text Pane

On the left you can define some text. The text is then "shifted" into the right TEXTPANE components when pressing the button "Update the right side".

This is the XML layout definition:

This is the Java code:

package workplace;

import org.eclnt.workplace.IWorkpageDispatcher;

public class DemoTextarea extends WorkpageDispatchedPageBean
{
    String m_plain = "Plain text, no formatting possible.";
    String m_html = "<span style=\"font-family:Arial;font-size:9px\">Some <b>HTML Text</b>.</span>";
    String m_rtf = "Hmmm. Is there some RTF expert...? ;-)";

    public DemoTextarea(IWorkpageDispatcher dispatcher)
    {
        super(dispatcher);
        initRefComponent("textpane"); 
    }
    
    public void setPlain(String value) { m_plain = value; }
    public String getPlain() { return m_plain; }
    
    public void setHtml(String value) { m_html = value; }
    public String getHtml() { return m_html; }
    
    public void setRtf(String value) { m_rtf = value; }
    public String getRtf() { return m_rtf; }
}
