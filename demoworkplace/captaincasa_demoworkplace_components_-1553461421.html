The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • Components
  • Straight Controls
  • Statusbar and Messages
  • Statusbar

The STATUSBAR component typically is placed ONE time into your pages. Typically it is placed in the page which is the "most outside" one.

From bean logic point of view it can be reached by calling static methods of the class Statusbar. Please have a look into the code view.

There are 4 levels of messages available:
(1) message (no hightlighting, just output)
(2) success (green highlighting)
(3) error (red highlighting)
(4) alert (red highlighting + confirmation dialog)

When placing a STATUSBAR two times into a page then the corresponding messages will show up in both status bar instances. This typically only makes sense when calling popups with own status bar.

This is the Java code:

package workplace;

import javax.faces.event.ActionEvent;

import org.eclnt.jsfserver.defaultscreens.Statusbar;
import org.eclnt.jsfserver.managedbean.IDispatcher;
import org.eclnt.workplace.IWorkpageDispatcher;

/*
 * When working with the workpalce framework there is an additional
 * statusbar management. Pleae check the corresponding examples.
 */
public class DemoStatusbar 
    extends PageBean
{
    public DemoStatusbar(IDispatcher dispatcher)
    {
        super(dispatcher);
        initRefComponent("rowstatusbar");
    }

    public void onOutputAlert(ActionEvent ae)
    {
        Statusbar.outputAlert("This message is output as ALERT message.");
    }
    
    public void onOutputError(ActionEvent ae)
    {
        Statusbar.outputError("This message is output as ERROR message..","You may pass a long text to the statusbar message as well. You may pass a long text to the statusbar message as well. You may pass a long text to the statusbar message as well. You may pass a long text to the statusbar message as well. You may pass a long text to the statusbar message as well. You may pass a long text to the statusbar message as well. You may pass a long text to the statusbar message as well. You may pass a long text to the statusbar message as well. You may pass a long text to the statusbar message as well. You may pass a long text to the statusbar message as well. ");
    }
    
    public void onOutputSuccess(ActionEvent ae)
    {
        Statusbar.outputSuccess("This message is output as SUCCESS message.");
    }
    
    public void onOutputSuccessWithPopup(ActionEvent ae)
    {
        Statusbar.outputSuccessWithPopup("This message is output as SUCCESS message.");
    }
    
    public void onOutputMessage(ActionEvent ae)
    {
        // several messages are output: as consequence a small icon is 
        // renedered inside the statusbar - when the user clicks on the
        // icon then all messages will be shown in a scrollable area.
        Statusbar.outputMessage("This message is output as normal message 1.");
        Statusbar.outputWarning("This message is output as warning message 2.");
        Statusbar.outputError("This message is output as error message 3.");
        Statusbar.outputMessage("This message is output as normal message 4.");
        Statusbar.outputMessage("This message is output as normal message 5.");
        Statusbar.outputSuccess("This message is output as success message 6.");
        Statusbar.outputMessage("This last message is output as normal message.");
    }
    
    public void onOutputMessageWithPopup(ActionEvent ae)
    {
        Statusbar.outputMessageWithPopup("This message is output as normal message.");
    }
    
    public void onOutputWarning(ActionEvent ae)
    {
        Statusbar.outputWarning("This message is output as WARNING message.");
    }
    
    public void onOutputWarningWithPopup(ActionEvent ae)
    {
        Statusbar.outputWarningWithPopup("This message is output as WARNING message.");
    }
    
    public boolean getImmediatelyShowAllMessages() { return Statusbar.getImmediatelyShowAllMessages(); }
    public void setImmediatelyShowAllMessages(boolean value) { Statusbar.setImmediatelyShowAllMessages(value); }
    
}
