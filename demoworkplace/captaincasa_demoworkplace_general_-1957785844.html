The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • General
  • Arranging Content
  • Popup Dialogs
  • Own popup title bar (PBC)

By calling a popup dialog and setting the property "undecorated" to "true", the dialog will not show its default title bar. You can design some own one, just using normal components.

In the addons-component-library (eclnt_pbc), there is a predefined extended title bar component "CCPopupDialogTitle":

The component renders a normal popup title bar, but allows to add additional content, for example additional icons, menus or other components.

This is the XML layout definition:

<?xml version="1.0" encoding="utf-8"?>
<t:layoutdefinition xmlns:t="http://www.CaptainCasa.com/controllibrary/t" xmlns:tpbc="http://www.CaptainCasa.com/controllibrary/tpbc" >
<t:beanprocessing id="g_1" beanbinding="#{d.DemoPBCCCPopupDialogTitleCaller}" />
<t:rowdemobodypane id="g_4" objectbinding="#{d.DemoPBCCCPopupDialogTitleCaller}" >
<t:row id="g_6" >
<t:button id="g_2" actionListener="#{d.DemoPBCCCPopupDialogTitleCaller.onOpenPopupAction}" text="Open popup" width="200+" />
</t:row>
</t:rowdemobodypane>
<t:pageaddons id="g_pa"/>
</t:layoutdefinition>

This is the Java code:

package workplace;

import java.io.Serializable;

import org.eclnt.editor.annotations.CCGenClass;
import org.eclnt.jsfserver.defaultscreens.ModalPopup;
import org.eclnt.workplace.IWorkpageDispatcher;

@CCGenClass (expressionBase="#{d.DemoPBCCCPopupDialogTitleCaller}")

public class DemoPBCCCPopupDialogTitleCaller
    extends PageBean 
    implements Serializable
{
    // ------------------------------------------------------------------------
    // constructors & initialization
    // ------------------------------------------------------------------------

    public DemoPBCCCPopupDialogTitleCaller(IWorkpageDispatcher workpageDispatcher)
    {
        super(workpageDispatcher);        
    }

    public String getPageName() { return "/workplace/demopbcccpopupdialogtitlecaller.jsp"; }
    public String getRootExpressionUsedInPage() { return "#{d.DemoPBCCCPopupDialogTitleCaller}"; }

    // ------------------------------------------------------------------------
    // public usage
    // ------------------------------------------------------------------------

    public void onOpenPopupAction(javax.faces.event.ActionEvent event) 
    {
        final DemoPBCCCPopupDialogTitle ui = new DemoPBCCCPopupDialogTitle();
        ModalPopup p =openModalPopup(ui,"This is a popup dialog",500,300,new ModalPopup.IModalPopupListener()
        {
            @Override
            public void reactOnPopupClosedByUser()
            {
                closePopup(ui);
            }
        });
        p.setUndecorated(true); // own titlebar!
        p.setSizeableIfUndecorated(true); // still size-able
    }

}
