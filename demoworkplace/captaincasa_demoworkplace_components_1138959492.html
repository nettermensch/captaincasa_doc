The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • Components
  • Straight Controls
  • Text Input/Output
  • HTML Editor
  • CKEditor

The component SIMPLEHTMLEDITOR is a really simple HTML editor that can be used to input HTML text that does require the typical "Wordpad-like" formatting.

In the RISC Client we internally use the CKEditor-component. Please check details about this embedded component via the URL https://ckeditor.com/ckeditor-4/. - We like this component because it is lightweight and configure-able, maybe you like it for any other purpose on your side!

Please carefully check license conditions of CKEditor prior to using this component within your projects!

This is the Java code:

package workplace;

import javax.faces.event.ActionEvent;

import org.eclnt.editor.annotations.CCGenClass;
import org.eclnt.jsfserver.defaultscreens.Statusbar;
import org.eclnt.workplace.IWorkpageDispatcher;

@CCGenClass (expressionBase="#{d.DemoSimpleHtmlEditor}")

public class DemoSimpleHtmlEditor 
    extends PageBean
{
    protected String m_htmlText = "<html><body style='font-family:Trebuchet MS'><p>This is some initial text!</p></body></html>";
    protected boolean m_enabled = false;

    public DemoSimpleHtmlEditor(IWorkpageDispatcher dispatcher)
    {
        super(dispatcher);
        initRefComponent("simplehtmleditor");
    }
    
    public String getHtmlText() { return m_htmlText; }
    public void setHtmlText(String value) { m_htmlText = value; }

    public boolean getEnabled() { return m_enabled; } 
    public void setEnabled(boolean enabled) { m_enabled = enabled; } 

    public void onLoadText(ActionEvent event) 
    {
        m_htmlText = "<html><body style='font-family:Trebuchet MS'><p>Hello world!</p></body></html>";
    }
    
    public void onSaveText(ActionEvent event)
    {
        System.out.println("==========");
        System.out.println(m_htmlText);
        System.out.println("==========");
        Statusbar.outputSuccess("Text was processed on server side.");
    }

}
