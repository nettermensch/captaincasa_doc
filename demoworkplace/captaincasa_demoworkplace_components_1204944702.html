The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • Components
  • Straight Controls
  • Structure Components
  • Tabbed Line - Vertical

The vertical tabbed line is a vertical arrangement of click-able areas. Its definition is very similar to the normal (horizontal) tabbed line.

The tabbed line maintains a VALUE, which is "0" when the first item is selected, "1" when the next one is selected, etc. In addition each tab item has an own action listener, so that you can directly react on user selections.

By using the different bgpaint-attributes of the component, you can achieve nice graphical effects with low effort.

This is the Java code:

package workplace;

import javax.faces.event.ActionEvent;

import org.eclnt.jsfserver.defaultscreens.Statusbar;
import org.eclnt.workplace.IWorkpageDispatcher;

public class DemoTabbedlineVertical extends PageBean
{
    int m_indexTL1 = 0;
    int m_indexTL2 = 2;
    
    public DemoTabbedlineVertical(IWorkpageDispatcher dispatcher)
    {
        super(dispatcher);
        initRefComponent("tabbedlinevertical");
    }

    public int getIndexTL1() { return m_indexTL1; }
    public void setIndexTL1(int indexTL1) { m_indexTL1 = indexTL1; }
    public void onTL1Action(ActionEvent event) { updateStatusbar(); }

    public int getIndexTL2() { return m_indexTL2; }
    public void setIndexTL2(int indexTL2) { m_indexTL2 = indexTL2; }
    public void onTL2Action(ActionEvent event) { updateStatusbar(); }
    
    public void updateStatusbar()
    {
        Statusbar.outputSuccess("Selected index 1: " + m_indexTL1 + "; selected index 2: " + m_indexTL2);
    }
}
