The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • Components
  • Straight Controls
  • Field Control
  • Date Input
  • Time picker

The time picker component allows the user to comfortably select a time.

As with all date/time-components, the server side can be...

1. a property of type "LocalTime", then select "LOCAL" as TIMEZONE inside the component
2. a property of type "java.util.Date", then select the TIMEZONE of the application (see Developers' Guide for details on how to manage java.util.Date)
3. a property of type "Long" or "long" which is handled in the same way as 2 (the long-value being the long-value that is kept within a Date-instance)

This is the XML layout definition:

This is the Java code:

package workplace;

import java.io.Serializable;
import java.time.LocalTime;

import org.eclnt.editor.annotations.CCGenClass;
import org.eclnt.workplace.IWorkpageDispatcher;

@CCGenClass (expressionBase="#{d.DemoTimePicker}")

public class DemoTimePicker
    extends PageBean 
    implements Serializable
{
    // ------------------------------------------------------------------------
    // inner classes
    // ------------------------------------------------------------------------

    // ------------------------------------------------------------------------
    // members
    // ------------------------------------------------------------------------

    LocalTime m_time = LocalTime.now();

    // ------------------------------------------------------------------------
    // constructors & initialization
    // ------------------------------------------------------------------------

    public DemoTimePicker(IWorkpageDispatcher workpageDispatcher)
    {
        super(workpageDispatcher);        
    }

    public String getPageName() { return "/workplace/demotimepicker.xml"; }
    public String getRootExpressionUsedInPage() { return "#{d.DemoTimePicker}"; }

    // ------------------------------------------------------------------------
    // public usage
    // ------------------------------------------------------------------------

    public LocalTime getTime() { return m_time; }
    public void setTime(LocalTime value) { this.m_time = value; }

    public void onSet1430Action(javax.faces.event.ActionEvent event) 
    {
        m_time = LocalTime.of(14,30);
    }

    public void onSet0855Action(javax.faces.event.ActionEvent event) 
    {
        m_time = LocalTime.of(8,55);
    }

    public void onSetNowAction(javax.faces.event.ActionEvent event) 
    {
        m_time = LocalTime.now();
    }

    // ------------------------------------------------------------------------
    // private usage
    // ------------------------------------------------------------------------
}
