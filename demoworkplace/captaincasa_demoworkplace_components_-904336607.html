The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • Components
  • Straight Controls
  • Field Control
  • Masked input
  • Masks with included numbers

Characters that you are using to define an input mask cannot be used by the user.

In certain use cases you want to define number-characters as fix content of the input mask - but at same point of time want the user to be allowed to key in numbers.

A solution is to use characters from the unicode character set which represent numbers, but which are not the default number-characters. There are a couple of parallel number representations, the ones listed below are not the only ones.

Use this characters inside your mask definition - and the mask characters will not conflict with the input of the user anymore.

This is the Java code:

package workplace;

import java.io.Serializable;

import org.eclnt.editor.annotations.CCGenClass;
import org.eclnt.workplace.IWorkpageDispatcher;

@CCGenClass (expressionBase="#{d.DemoMaskedInputNumbers}")

public class DemoMaskedInputNumbers
    extends PageBean 
    implements Serializable
{
    // ------------------------------------------------------------------------
    // members
    // ------------------------------------------------------------------------

    String m_phoneNumber;

    // ------------------------------------------------------------------------
    // constructors & initialization
    // ------------------------------------------------------------------------

    public DemoMaskedInputNumbers(IWorkpageDispatcher workpageDispatcher)
    {
        super(workpageDispatcher);        
    }

    public String getPageName() { return "/workplace/demomaskedinputnumbers.xml"; }
    public String getRootExpressionUsedInPage() { return "#{d.DemoMaskedInputNumbers}"; }

    // ------------------------------------------------------------------------
    // public usage
    // ------------------------------------------------------------------------

    public String getPhoneNumber() { return m_phoneNumber; }
    public void setPhoneNumber(String value) { this.m_phoneNumber = value; }
}
