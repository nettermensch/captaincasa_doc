The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • Containers
  • Foldable / Titled Pane
  • Additional header components

You can add additional header elements to the FOLDABLEPANE component by using its sub-component FOLDABLEPANEHEADERROW.

By default the additional elements are arranged on the right side, in front of the toggle icon. By setting attribute FOLDABLEPANE-HEADERFULLWIDTH to "true" the additional header area is defined to occupy the fully space between the title and the toggle icon - as consequence the additional components are arranged "on the left".

This is the Java code:

package workplace;

import java.io.Serializable;

import org.eclnt.editor.annotations.CCGenClass;
import org.eclnt.workplace.IWorkpageDispatcher;

@CCGenClass (expressionBase="#{d.DemoFoldablePaneHeader}")
public class DemoFoldablePaneHeader
    extends PageBean 
    implements Serializable
{
    public DemoFoldablePaneHeader(IWorkpageDispatcher workpageDispatcher)
    {
        super(workpageDispatcher);        
    }

    public String getPageName() { return "/workplace/demofoldablepaneheader.xml"; }
    public String getRootExpressionUsedInPage() { return "#{d.DemoFoldablePaneHeader}"; }
}
