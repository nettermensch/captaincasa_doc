The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • Components
  • Straight Controls
  • Menus
  • Popup Menu in Grid

On FIXGRID level there are two popup menu definitions that you can use:

(1) the POPUPMENU attribute is the one that is the reference to the POPUPMENU definition for the whole grid.
(2) the ROWPOPUPMENU attribute is the one that is the reference to the POPUPMENU definition for the grid items.

On FIXGRID level the popup events are automatically delegated to the item's or the grid's "onPopupMenuItem()" method. You need to override these methods in order to add your event processing.

In the example the menu for the whole grid only contains the "New Item" command, whereas the menu on item level allows to either call "New Item" or "Remove Item".

The event information that is available in the action listener method allows to check, where the user pressed the right mouse button in order to open the popup menu. Dependent from this information in the example the new item is either created in front or behind the corresponding item.

Of course you still can in addition define individual popup menus for each component that you place into the grid columns.

This is the XML layout definition:

<t:row id="g_2" >
<t:fixgrid id="g_3" bordercolor="#00000020" borderheight="1" borderwidth="1" height="100%" multiselect="true" objectbinding="#{d.DemoGridPopupMenu.rows}" popupmenu="GRIDPOPUPMENU" rowpopupmenu="GRIDROWPOPUPMENU" rowpopupmenuloadroundtrip="true" sbvisibleamount="25" width="100%" >

This is the Java code:

package workplace;

import java.io.Serializable;

import javax.faces.event.ActionEvent;

import org.eclnt.editor.annotations.CCGenClass;
import org.eclnt.jsfserver.defaultscreens.Statusbar;
import org.eclnt.jsfserver.elements.BaseActionEvent;
import org.eclnt.jsfserver.elements.events.BaseActionEventPopupMenuItem;
import org.eclnt.jsfserver.elements.impl.FIXGRIDItem;
import org.eclnt.jsfserver.elements.impl.FIXGRIDListBinding;
import org.eclnt.workplace.IWorkpageDispatcher;

@CCGenClass (expressionBase="#{d.DemoGridPopupMenu}")

public class DemoGridPopupMenu
    extends PageBean
    implements Serializable
{
    // ------------------------------------------------------------------------
    // inner classes
    // ------------------------------------------------------------------------
    
    public class MyRow extends FIXGRIDItem implements java.io.Serializable
    {    
        protected String m_lastName;
        public String getLastName() { return m_lastName; }
        public void setLastName(String value) { m_lastName = value; }

        protected String m_firstName;
        public String getFirstName() { return m_firstName; }
        public void setFirstName(String value) { m_firstName = value; }
        
        @Override
        public void onRowPopupMenuItem(BaseActionEventPopupMenuItem event)
        {
            if ("REMOVE".equals(event.getCommand()))
            {
                m_rows.getItems().remove(this);
            }
            else if ("NEW".equals(event.getCommand()))
            {
                boolean inFront = (event.getPercentageVertical() < 50);
                MyRow r = new MyRow();
                int insertIndex = m_rows.getItems().indexOf(this);
                if (inFront == false)
                    insertIndex++;
                m_rows.getItems().add(insertIndex,r);
                m_rows.ensureItemToBeDisplayed(r);
            }
        }
    }
    
    public class MyGridListBinding extends FIXGRIDListBinding<MyRow>
    {
        @Override
        protected void onPopupMenuItem(BaseActionEventPopupMenuItem event)
        {
            if ("NEW".equals(event.getCommand()))
            {
                MyRow r = new MyRow();
                m_rows.getItems().add(r);
                m_rows.ensureItemToBeDisplayed(r);
            }
            else if ("SELECTALL".equals(event.getCommand()))
            {
                m_rows.deselectCurrentSelection();
                for (MyRow row: getItems())
                    m_rows.selectItem(row);
            }
        }
    }

    // ------------------------------------------------------------------------
    // members
    // ------------------------------------------------------------------------
    
    protected MyGridListBinding m_rows = new MyGridListBinding();
    
    // ------------------------------------------------------------------------
    // constructors
    // ------------------------------------------------------------------------
    
    public DemoGridPopupMenu(IWorkpageDispatcher dispatcher)
    {
        super(dispatcher);
        initRefComponent("fixgrid");
        for (int i=0; i<10; i++)
        {
            MyRow r = new MyRow();
            r.setFirstName("FN " + i);
            r.setLastName("LN " + i);
            m_rows.getItems().add(r);
        }
    }
    
    // ------------------------------------------------------------------------
    // public usage
    // ------------------------------------------------------------------------
    
    public FIXGRIDListBinding<MyRow> getRows() { return m_rows; }
    
    public void onWholePageAction(ActionEvent event) 
    {
        if (event instanceof BaseActionEventPopupMenuItem)
        {
            BaseActionEventPopupMenuItem e = (BaseActionEventPopupMenuItem)event;
            if ("COPY".equals(e.getCommand()))
            {
                Statusbar.outputAlert("Number of items: " + m_rows.getSelectedItems().size());
            }
        }
    }

}
