The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • Grids & Trees
  • Grids and Trees
  • Selection Issues
  • Re-selection of items

Every time a selection is sent to the server side, the demo application outputs some information to the status bar - including some timestamp. You will notice that there is a different behavior of sending events to the server side if re-clicking an already selected item, dependent on the definition of the RESELETABLE attribute.

By default a grid only sends item selection events if the item is switching from unselected to selected.

By defining attribute RESELECTABLE to "true" you can override this behavior. In this case any click onto the item is causing a selection event which is tranferred to the server side.

In most cases you do not require a re-selection event, so the default is set to "false". You may for example require to define it as "true" if you e.g. want to get information about the currently selected column - or if you just always want to re-read certain data with every click event.-

This is the Java code:

package workplace;

import java.io.Serializable;

import org.eclnt.editor.annotations.CCGenClass;
import org.eclnt.jsfserver.defaultscreens.Statusbar;
import org.eclnt.jsfserver.elements.impl.FIXGRIDItem;
import org.eclnt.jsfserver.elements.impl.FIXGRIDListBinding;
import org.eclnt.workplace.IWorkpageDispatcher;
import org.eclnt.workplace.WorkpageDispatchedPageBean;

@CCGenClass (expressionBase="#{d.DemoGridReselectable}")

public class DemoGridReselectable
    extends PageBean 
    implements Serializable
{
    // ------------------------------------------------------------------------
    // inner classes
    // ------------------------------------------------------------------------

    public class GridItem extends FIXGRIDItem implements java.io.Serializable
    {
        String i_lastName;
        String i_firstName;
        public String getLastName() { return i_lastName; }
        public void setLastName(String value) { this.i_lastName = value; }
        public String getFirstName() { return i_firstName; }
        public void setFirstName(String value) { this.i_firstName = value; }
        public void onRowSelect()
        {
            Statusbar.outputSuccess("Grid item selection: " + i_lastName + ", " + i_firstName + " - " + System.currentTimeMillis());
        }
    }

    // ------------------------------------------------------------------------
    // members
    // ------------------------------------------------------------------------

    FIXGRIDListBinding<GridItem> m_grid = new FIXGRIDListBinding<GridItem>();
    boolean m_reselectable = false;

    // ------------------------------------------------------------------------
    // constructors & initialization
    // ------------------------------------------------------------------------

    public DemoGridReselectable(IWorkpageDispatcher workpageDispatcher)
    {
        super(workpageDispatcher);      
        for (int i=0; i<100; i++)
        {
            GridItem gi = new GridItem();
            gi.setFirstName("First " + i);
            gi.setLastName("Last " + i);
            m_grid.getItems().add(gi);
        }
    }

    public String getPageName() { return "/workplace/demogridreselectable.jsp"; }
    public String getRootExpressionUsedInPage() { return "#{d.DemoGridReselectable}"; }

    // ------------------------------------------------------------------------
    // public usage
    // ------------------------------------------------------------------------

    public FIXGRIDListBinding<GridItem> getGrid() { return m_grid; }

    public boolean getReselectable() { return m_reselectable; }
    public void setReselectable(boolean value) { this.m_reselectable = value; }

    // ------------------------------------------------------------------------
    // private usage
    // ------------------------------------------------------------------------
}
