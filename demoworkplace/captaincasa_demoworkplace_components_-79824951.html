The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • Components
  • Straight Controls
  • Label Control
  • Vertical Label

The component LABELVERTICAL is a normal label - the text is output in vertical direction instead of horizontal direction. You can set font, foreground, background, etc. just as with the normal LABEL component.

This is the XML layout definition:

<?xml version="1.0" encoding="utf-8"?>
<t:layoutdefinition xmlns:t="http://www.CaptainCasa.com/controllibrary/t" xmlns:tpbc="http://www.CaptainCasa.com/controllibrary/tpbc" >
<t:rowdemobodypane id="g_1" objectbinding="#{d.DemoLabelVertical}" >
<t:row id="g_2" >
<t:label id="g_3" rowalignmenty="top" text="CaptainCasa" />
<t:coldistance id="g_4" width="10" />
<t:label id="g_5" font="size:20;weight:bold" foreground="#800000" text="Bob the Builder" />
</t:row>
<t:row id="g_6" >
<t:labelvertical id="g_7" rowalignmenty="top" text="CaptainCasa" />
<t:coldistance id="g_8" width="10" />
<t:labelvertical id="g_9" font="size:20;weight:bold" foreground="#800000" text="Bob the Builder" />
</t:row>
</t:rowdemobodypane>
<t:pageaddons id="g_pa"/>
</t:layoutdefinition>
