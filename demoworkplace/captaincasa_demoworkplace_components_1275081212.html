The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • Components
  • Animation & Graphics & Sound
  • Image Capturer & Animator (1)

The CAPTUREANIMATOR component creates an internal screenshot of the component, in which it is placed. This screenshot then is passed to an animation that e.g. hides the screenshot into a certain direction. All this happens if a trigger from the server side is activated.

There are different animation types that you can select from.

With the CPATUREANIMATOR you can create nice optical effects to tell the user that certain data has changed on the screen. Although it seems that during the animation phase, there are two page areas painted in parallel: the bottom one is the "real one", the animated top one is the internal screen shot image of the area as it was before the update.

This is the Java code:

package workplace;

import java.io.Serializable;

import javax.faces.event.ActionEvent;

import org.eclnt.editor.annotations.CCGenClass;
import org.eclnt.jsfserver.defaultscreens.ModalPopup;
import org.eclnt.jsfserver.elements.util.DefaultModalPopupListener;
import org.eclnt.jsfserver.elements.util.Trigger;
import org.eclnt.jsfserver.pagebean.IPageBean;
import org.eclnt.workplace.IWorkpageDispatcher;

@CCGenClass (expressionBase="#{d.DemoCaptureAnimator2}")

public class DemoCaptureAnimator2
    extends WorkpageDispatchedPageBean
    implements Serializable
{
    public DemoCaptureAnimator2(IWorkpageDispatcher dispatcher)
    {
        super(dispatcher);
        initRefComponent("captureanimator");
    }

    protected Trigger m_animationTrigger = new Trigger();
    protected String m_animationType = "hidecontenttotop";
    int m_animationStepCount = 8;
    
    public int getAnimationStepCount() { return m_animationStepCount; }
    public void setAnimationStepCount(int value) { this.m_animationStepCount = value; }
    
    public Trigger getAnimationTrigger() { return m_animationTrigger; }
    public void setAnimationTrigger(Trigger value) { m_animationTrigger = value; }

    public String getAnimationType() { return m_animationType; }
    public void setAnimationType(String value) { m_animationType = value; }

    public void onAnimate(ActionEvent event) 
    {
        m_animationTrigger.trigger();
    }
    
    public void onAnimateDialog(javax.faces.event.ActionEvent event) 
    {
        final IPageBean ui = new DemoHelloWorld(getOwningDispatcher());
        ModalPopup mp = openModalPopup(ui,"Demo Hello World!",750,400,new DefaultModalPopupListener(this,ui));
        mp.setAnimationtype(m_animationType);
    }

}
