The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • Components
  • Animation & Graphics & Sound
  • Image Selector

The IMAGESHAKER component is a nice one! It outputs a list of images and lets the user select. There is a certain animation involved with selecting the image.

Each image selection is passed to server side so that your program can react.

This is the XML layout definition:

<?xml version="1.0" encoding="utf-8"?>

This is the Java code:

package workplace;

import java.io.Serializable;

import org.eclnt.editor.annotations.CCGenClass;
import org.eclnt.jsfserver.defaultscreens.Statusbar;
import org.eclnt.jsfserver.elements.impl.IMAGESHAKERBinding;
import org.eclnt.jsfserver.managedbean.IDispatcher;

@CCGenClass (expressionBase="#{d.DemoImageShaker}")

public class DemoImageShaker
    extends PageBean
    implements Serializable
{
    // ------------------------------------------------------------------------
    // inner classes
    // ------------------------------------------------------------------------
    
    public class MyIMAGESHAKERBinding extends IMAGESHAKERBinding
    {
        @Override
        protected void reactOnImageSelect(int selectedIndex)
        {
            Statusbar.outputMessage("Image selection, index: " + selectedIndex);
        }
    }
    
    // ------------------------------------------------------------------------
    // members
    // ------------------------------------------------------------------------
    
    protected MyIMAGESHAKERBinding m_shaker = new MyIMAGESHAKERBinding();
    
    // ------------------------------------------------------------------------
    // constructors
    // ------------------------------------------------------------------------
    
    public DemoImageShaker(IDispatcher dispatcher)
    {
        super(dispatcher);
        initRefComponent("imageshaker");
        m_shaker.addImage("/images/pictures/photo0.gif","Photo 0"); // with tooltip
        m_shaker.addImage("/images/pictures/photo1.gif","Photo 1"); // with tooltip
        m_shaker.addImage("/images/pictures/photo2.gif","Photo 2"); // with tooltip
        m_shaker.addImage("/images/pictures/photo3.gif"); // without tooltip
        m_shaker.addImage("/images/pictures/photo4.gif"); // without tooltip
    }
    
    // ------------------------------------------------------------------------
    // public usage
    // ------------------------------------------------------------------------
    
    public MyIMAGESHAKERBinding getShaker() { return m_shaker; }
    
    public int getSelection() 
    { 
        return m_shaker.getSelectedIndex(); 
    }
    
    // ------------------------------------------------------------------------
    // private usage
    // ------------------------------------------------------------------------
    
}
