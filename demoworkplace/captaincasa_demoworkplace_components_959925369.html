The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • Components
  • Straight Controls
  • Menus
  • Dynamic loading in Button Menu

Press the "Show Menu" button: the menu below is different every time you press the button.

There is an attribute BUTONMENU-MENULOADROUNDTRIP. If set to "true" then an action listener will be called prior to building the menu on client side. Within the action listener you may dynamically create the menu.

Please pay attention: you must define at least one menu item in order to be able to press the menu-button! The menu-button is disabled if there are no items defined.

This is the XML layout definition:

<?xml version="1.0" encoding="utf-8"?>
<t:layoutdefinition xmlns:t="http://www.CaptainCasa.com/controllibrary/t" xmlns:tpbc="http://www.CaptainCasa.com/controllibrary/tpbc" >
<t:rowdemobodypane id="g_1" objectbinding="#{d.DemoButtonMenuLoad}" >
<t:row id="g_2" >
<t:buttonmenu id="g_3" actionListener="#{d.DemoButtonMenuLoad.onButtonMenuAction}" menuloadroundtrip="true" text="Show Menu" >
<t:dynamiccontent id="g_4" contentbinding="#{d.DemoButtonMenuLoad.dynContent}" />
</t:buttonmenu>
</t:row>
</t:rowdemobodypane>
<t:pageaddons id="g_pa"/>
</t:layoutdefinition>

This is the Java code:

package workplace;

import java.io.Serializable;
import java.util.ArrayList;
import java.util.List;

import javax.faces.event.ActionEvent;

import org.eclnt.editor.annotations.CCGenClass;
import org.eclnt.jsfserver.defaultscreens.Statusbar;
import org.eclnt.jsfserver.elements.BaseActionEvent;
import org.eclnt.jsfserver.elements.componentnodes.MENUITEMNode;
import org.eclnt.jsfserver.elements.events.BaseActionEventMenuLoad;
import org.eclnt.jsfserver.elements.impl.DYNAMICCONTENTBinding;
import org.eclnt.jsfserver.elements.impl.MENUITEMComponent;
import org.eclnt.jsfserver.elements.impl.ROWDYNAMICCONTENTBinding.ComponentNode;
import org.eclnt.workplace.IWorkpageDispatcher;
import org.eclnt.workplace.WorkpageDispatchedBean;

@CCGenClass (expressionBase="#{d.DemoButtonMenuLoad}")

public class DemoButtonMenuLoad
    extends PageBean
    implements Serializable
{
    public void onButtonMenuAction(ActionEvent event) 
    {
        if (event instanceof BaseActionEventMenuLoad)
        {
            render();
        }
    }

    DYNAMICCONTENTBinding m_dynContent = new DYNAMICCONTENTBinding();
    public DYNAMICCONTENTBinding getDynContent() { return m_dynContent; }

    // ------------------------------------------------------------------------
    // constructors & initialization
    // ------------------------------------------------------------------------

    public DemoButtonMenuLoad(IWorkpageDispatcher workpageDispatcher)
    {
        super(workpageDispatcher);
        initRefComponent("buttonmenu");
        render();
    }

    public void onMenuItemAction(ActionEvent event)
    {
        BaseActionEvent e = (BaseActionEvent)event;
        Statusbar.outputSuccess("Menu Item was pressed: " + e.getSourceReference());
    }
    
    private void render()
    {
        List<ComponentNode> nodes = new ArrayList<ComponentNode>();
        for (int i=0; i<10; i++)
        {
            MENUITEMNode mi = new MENUITEMNode();
            mi.setText("Hello " + System.currentTimeMillis() + " " + i);
            mi.setActionListener(getRootExpressionUsedInPage().replace("}",".onMenuItemAction}"));
            mi.setReference(""+i);
            nodes.add(mi);
        }
        m_dynContent.setContentNodes(nodes);
    }
}
