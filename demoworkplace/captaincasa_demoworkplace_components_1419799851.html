The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • Components
  • Touch Screen
  • Touch Selector

The TOUCHSELECTOR is a control that allows to select one value out of a defined set of values. The user sees the currently selected value and can navigate through the available values by using the up down keys.

From programming point of view the touch selector is "the same" as programming a combo box: you need to bind the value that receives the selected id and you need to bind an instance of ValidValuesBinding that provides the set of select-ale values.

When the control is used on a desktop browser then the user can also use the mouse wheel for navigating through the valid values.

This is the Java code:

package workplace;

import java.io.Serializable;

import org.eclnt.editor.annotations.CCGenClass;
import org.eclnt.jsfserver.elements.util.ValidValuesBinding;
import org.eclnt.jsfserver.pagebean.PageBean;
import org.eclnt.workplace.IWorkpageDispatcher;

@CCGenClass (expressionBase="#{d.DemoTouchSelector}")

public class DemoTouchSelector
    extends PageBean
    implements Serializable
{
    // ------------------------------------------------------------------------
    // members
    // ------------------------------------------------------------------------
    
    ValidValuesBinding m_departmentVVS = new ValidValuesBinding();
    String m_department = "0001";
    int m_value = 0;
    ValidValuesBinding m_valueVVS = new ValidValuesBinding();

    // ------------------------------------------------------------------------
    // constructors & initialization
    // ------------------------------------------------------------------------

    public DemoTouchSelector(IWorkpageDispatcher dispatcher)
    { 
        super(dispatcher);
        // valid values for "value"
        for (int i=0; i<10; i++)
            m_valueVVS.addValidValue(""+i,""+i);
        // valid values for department
        m_departmentVVS.addValidValue("0001","Sales");
        m_departmentVVS.addValidValue("0002","Production");
        m_departmentVVS.addValidValue("0003","Management");
        m_departmentVVS.addValidValue("0004","Human Resources");
    }

    public String getPageName() { return "/workplace/demotouchselector.jsp"; }
    public String getRootExpressionUsedInPage() { return "#{d.DemoTouchSelector}"; }

    // ------------------------------------------------------------------------
    // public usage
    // ------------------------------------------------------------------------

    public String getDepartment() { return m_department; }
    public void setDepartment(String value) { this.m_department = value; }
    
    public ValidValuesBinding getDepartmentVVS() { return m_departmentVVS; }
    
    public int getValue() { return m_value; }
    public void setValue(int value) { this.m_value = value; }

    public ValidValuesBinding getValueVVS() { return m_valueVVS; }

    // ------------------------------------------------------------------------
    // private usage
    // ------------------------------------------------------------------------
}
