The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • Components
  • Straight Controls
  • Field Control
  • Additional info in field
  • Field with User Info

There are many ways of presenting additional information about what a user needs to input within a certain field.

The USERHINT is a text that shows up when the user steps into a field - and disappears when the user leaves the field.

The LABELTEXT is a text which is shown in the background if the field is empty and which then moves up into the top area during input.

The PLACEHOLDER is a background text which only appears on empty content.

The UNITTEXT is a text which is placed at the right of the field, typically showing a unit of measure. Of course it can be used for any other purpose as well.
