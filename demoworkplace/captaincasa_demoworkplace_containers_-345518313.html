The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • Containers
  • Slide Container
  • Synchronous sliding

In the demo the master container position is propagated to the slide containers below.

The coupling is done via the attributes MASTERID and MASTERIDTOFOLLOW: the MASTERID is defined with the leading slide container, the other ones reference the master by defining the same id in the attribute MASTERIDTOFOLLOW.

Only the master slide container is the active one - the other ones can not be slided on their owns anymore. And they will also not send or receive any data from the server side.

This is the Java code:

package workplace;

import java.io.Serializable;
import org.eclnt.editor.annotations.CCGenClass;
import org.eclnt.jsfserver.pagebean.PageBean;
import org.eclnt.workplace.IWorkpageDispatcher;

import javax.faces.event.ActionEvent;

@CCGenClass (expressionBase="#{d.DemoSlideContainerMasterSlave}")

public class DemoSlideContainerMasterSlave
    extends WorkpageDispatchedPageBean
    implements Serializable
{
    // ------------------------------------------------------------------------
    // inner classes
    // ------------------------------------------------------------------------
    
    // ------------------------------------------------------------------------
    // members
    // ------------------------------------------------------------------------
    
    int m_position = 0;
    
    // ------------------------------------------------------------------------
    // constructors & initialization
    // ------------------------------------------------------------------------

    public DemoSlideContainerMasterSlave(IWorkpageDispatcher d)
    {
        super(d);
    }

    public String getPageName() { return "/workplace/demoslidecontainermasterslave.jsp"; }
    public String getRootExpressionUsedInPage() { return "#{d.DemoSlideContainerMasterSlave}"; }

    // ------------------------------------------------------------------------
    // public usage
    // ------------------------------------------------------------------------

    public void onSliderAction(javax.faces.event.ActionEvent event) 
    {
        m_position = 0;
    }

    public int getPosition() { return m_position; }
    public void setPosition(int value) { this.m_position = value; }

    // ------------------------------------------------------------------------
    // private usage
    // ------------------------------------------------------------------------
}
