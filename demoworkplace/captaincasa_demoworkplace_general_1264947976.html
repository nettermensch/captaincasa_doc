The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • General
  • Arranging Content
  • Popup Dialogs
  • Positioning on top of component

When pressing the button "Open popup" a modal dialog is opened. It is placed directly on top of the "Some content"-area.

Please check the JavaCode - there are two methods which cause this behavior:

1. the method "setLefTopReferenceComponentIdCentered("<id>")" positions the popup on to of the center of the referenced component.

2. the method "setWidthHeightToSizeOfReference()" automatically sets the size of the popup to the size of the referenced area.

This is the Java code:

package workplace;

import java.io.Serializable;
import org.eclnt.editor.annotations.CCGenClass;
import org.eclnt.jsfserver.defaultscreens.ModalPopup;
import org.eclnt.jsfserver.defaultscreens.OKPopup;
import org.eclnt.workplace.IWorkpageDispatcher;
import org.eclnt.workplace.WorkpageDispatchedPageBean;

import javax.faces.event.ActionEvent;

@CCGenClass (expressionBase="#{d.DemoPopupSizingByReference}")

public class DemoPopupSizingByReference
    extends PageBean 
    implements Serializable
{
    // ------------------------------------------------------------------------
    // constructors & initialization
    // ------------------------------------------------------------------------

    public DemoPopupSizingByReference(IWorkpageDispatcher workpageDispatcher)
    {
        super(workpageDispatcher);        
    }

    public String getPageName() { return "/workplace/demopopupsizingbyreference.jsp"; }
    public String getRootExpressionUsedInPage() { return "#{d.DemoPopupSizingByReference}"; }

    // ------------------------------------------------------------------------
    // public usage
    // ------------------------------------------------------------------------

    public void onOpenPopupAction(javax.faces.event.ActionEvent event) 
    {
        final DemoHelloWorld ui = new DemoHelloWorld(getOwningDispatcher());
        ModalPopup mp = openModalPopup(ui,"Hello world",0,0,new ModalPopup.IModalPopupListener()
        {
            public void reactOnPopupClosedByUser()
            {
                closePopup(ui);
            }
        });
        mp.setLeftTopReferenceComponentIdCentered("REF1");
        mp.setWidthHeightToSizeOfReference();
        mp.setCloseonclickoutside(true);
    }
}
