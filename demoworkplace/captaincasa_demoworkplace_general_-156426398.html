The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • General
  • General Issues
  • Change of http session id

In some security-related scenarios it is required to update the http session id during the runtime of the session.

This demo shows how to do this on server-side - and tests that the client-server configuration is automatically adapted to the new session id.

This is the XML layout definition:

<?xml version="1.0" encoding="utf-8"?>
<t:layoutdefinition xmlns:t="http://www.CaptainCasa.com/controllibrary/t" >
<t:beanprocessing id="g_1" beanbinding="#{d.DemoChangeHttpSessionId}" />
<t:rowdemobodypane id="g_4" objectbinding="#{d.DemoChangeHttpSessionId}" rowdistance="10" >
<t:row id="g_3" >
<t:label id="g_8" labeltext="Current session id" text="#{d.DemoChangeHttpSessionId.currentSessionId}" />
</t:row>
<t:row id="g_6" >
<t:button id="g_2" actionListener="#{d.DemoChangeHttpSessionId.onChangeSessionIdAction}" text="Change Session Id" width="200+" />
</t:row>
<t:row id="g_5" >
<t:textarea id="g_10" enabled="false" height="100%" labeltext="Protocol" stylevariant="cc_code" text="#{d.DemoChangeHttpSessionId.protocol}" width="100%" />
</t:row>
</t:rowdemobodypane>
</t:layoutdefinition>

This is the Java code:

package workplace;

import java.io.Serializable;

import javax.servlet.http.HttpServletRequest;

import org.eclnt.editor.annotations.CCGenClass;
import org.eclnt.jsfserver.util.HttpSessionAccess;
import org.eclnt.workplace.IWorkpageDispatcher;

@CCGenClass (expressionBase="#{d.DemoChangeHttpSessionId}")

public class DemoChangeHttpSessionId
    extends PageBean 
    implements Serializable
{
    // ------------------------------------------------------------------------
    // inner classes
    // ------------------------------------------------------------------------

    // ------------------------------------------------------------------------
    // members
    // ------------------------------------------------------------------------

    String m_protocol = "";
    
    // ------------------------------------------------------------------------
    // constructors & initialization
    // ------------------------------------------------------------------------

    public DemoChangeHttpSessionId(IWorkpageDispatcher workpageDispatcher)
    {
        super(workpageDispatcher);        
    }

    public String getPageName() { return "/workplace/demochangehttpsessionid.xml"; }
    public String getRootExpressionUsedInPage() { return "#{d.DemoChangeHttpSessionId}"; }

    // ------------------------------------------------------------------------
    // public usage
    // ------------------------------------------------------------------------

    public String getCurrentSessionId() 
    { 
        return HttpSessionAccess.getCurrentHttpSession().getId();
    }
    
    public String getProtocol() { return m_protocol; } 
    public void setProtocol(String protocol) { m_protocol = protocol; }

    public void onChangeSessionIdAction(javax.faces.event.ActionEvent event) 
    {
        String from = getCurrentSessionId();
        HttpServletRequest req = HttpSessionAccess.getCurrentRequest();
        req.changeSessionId();
        String to = getCurrentSessionId();
        m_protocol += "Changed session id from " + from + " to " + to + "\n";
    }

    // ------------------------------------------------------------------------
    // private usage
    // ------------------------------------------------------------------------
}
