The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • General
  • Arranging Content
  • Popup Dialogs
  • Auto sizing of popups
  • Dealing with SCROLLPANEs

When passing a size of "0" as width and/or height of a popup dialog then the dialog is automatically sized by measuring the required height of its contained components.

If the content contains a SCROLLPANE (which also is implicitly contained in the ROWBODYPANE component) then there is a problem: it's exactly the task of a SCROLLPANE to hide the size of its contained content - by letting the user scroll through the content. So the required content of a SCROLLPANE is very small indeed!

To overcome there is an attribute SIZETOCONTENTINPOPUP. If defining as "true" then the SCROLLPANE indicates to take the full size of its content in case of being part of popup dialog.

Of course: if there is no sufficient screen space, then the SCROLLPANE will be shrinked accordingly - and then shows corresponding scroll bars.

This is the Java code:

package workplace;

import java.io.Serializable;
import org.eclnt.editor.annotations.CCGenClass;
import org.eclnt.jsfserver.elements.util.DefaultModalPopupListener;
import org.eclnt.workplace.IWorkpageDispatcher;
import org.eclnt.workplace.WorkpageDispatchedPageBean;

import javax.faces.event.ActionEvent;

@CCGenClass (expressionBase="#{d.DemoPopupScrollpaneCaller}")

public class DemoPopupScrollpaneCaller
    extends PageBean 
    implements Serializable
{
    // ------------------------------------------------------------------------
    // constructors & initialization
    // ------------------------------------------------------------------------

    public DemoPopupScrollpaneCaller(IWorkpageDispatcher workpageDispatcher)
    {
        super(workpageDispatcher);        
    }

    public String getPageName() { return "/workplace/demopopupscrollpanecaller.jsp"; }
    public String getRootExpressionUsedInPage() { return "#{d.DemoPopupScrollpaneCaller}"; }

    // ------------------------------------------------------------------------
    // public usage
    // ------------------------------------------------------------------------

    public void onOpenPopupAction(javax.faces.event.ActionEvent event) 
    {
        final DemoPopupScrollpane ui = new DemoPopupScrollpane();
        openModalPopup(ui,"Popup containing scroll pane",0,0,new DefaultModalPopupListener(this,ui));
    }

    public void onOpenPopup2Action(javax.faces.event.ActionEvent event) 
    {
        final DemoPopupScrollpane2 ui = new DemoPopupScrollpane2();
        openModalPopup(ui,"Popup containing scroll pane",0,0,new DefaultModalPopupListener(this,ui));
    }

}
