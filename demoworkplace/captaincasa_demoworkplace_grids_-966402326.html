The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • Grids & Trees
  • Grids and Trees
  • Flexible row heights
  • For header and footer

The grid can be configured so that the header and footer rows may define a row height on their owns. To do so, use the following attributes:

(.) FIXGRID-HEADLINEROWHEIGHT: this is the row height that is assigned to the headline, containing the column titles.

(.) GRIDHEADER-ROWHEIGHT and GRIDFOOTER-ROWHEIGHT: this is the row height of the corresponding header/footer row.

If not explicitly defined then the FIXGRID-ROWHEIGHT definition is automatically applied.

This is the Java code:

package workplace;

import java.io.Serializable;

import org.eclnt.editor.annotations.CCGenClass;
import org.eclnt.jsfserver.elements.impl.FIXGRIDItem;
import org.eclnt.jsfserver.elements.impl.FIXGRIDListBinding;
import org.eclnt.workplace.IWorkpageDispatcher;

@CCGenClass (expressionBase="#{d.DemoGridRowHeight}")

public class DemoGridRowHeight
    extends PageBean
    implements Serializable
{
    public class GridItem extends FIXGRIDItem implements java.io.Serializable
    {
        protected String m_lastName;
        public String getLastName() { return m_lastName; }
        public void setLastName(String value) { m_lastName = value; }

        protected String m_firstName;
        public String getFirstName() { return m_firstName; }
        public void setFirstName(String value) { m_firstName = value; }

    }

    protected FIXGRIDListBinding<GridItem> m_grid = new FIXGRIDListBinding<GridItem>();
    
    public DemoGridRowHeight(IWorkpageDispatcher dispatcher)
    {
        super(dispatcher);
        for (int i=0; i<10; i++)
        {
            GridItem gi = new GridItem();
            gi.setFirstName("FN " + i);
            gi.setLastName("LN " + i);
            m_grid.getItems().add(gi);
        }
    }
    
    public FIXGRIDListBinding<GridItem> getGrid() { return m_grid; }
    public void setGrid(FIXGRIDListBinding<GridItem> value) { m_grid = value; }
    
}
