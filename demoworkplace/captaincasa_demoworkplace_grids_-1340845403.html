The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • Grids & Trees
  • Grids and Trees
  • Special Issues
  • Grid scroll bar style

The grid's own style (attribute STYLESEQ) points to the style class that defines the scroll container that manages the vertical scrolling of the grid items.

The default style class is "riscscrollcontainer". There is an alternative "riscscrollcontainer_dark" in which the rendering of the scroll bar is done in dark colors.

Of course you can set up own styles for the scroll container and the included scroll bar by extending style "riscscrollcontainer".

This is the Java code:

package workplace;

import java.io.Serializable;

import org.eclnt.editor.annotations.CCGenClass;
import org.eclnt.jsfserver.elements.impl.FIXGRIDItem;
import org.eclnt.jsfserver.elements.impl.FIXGRIDListBinding;
import org.eclnt.workplace.IWorkpageDispatcher;

import workplace.logic.RandomLogic;

@CCGenClass (expressionBase="#{d.DemoGridStyle}")

public class DemoGridStyle
    extends PageBean 
    implements Serializable
{
    // ------------------------------------------------------------------------
    // inner classes
    // ------------------------------------------------------------------------

    public class GridItem extends FIXGRIDItem implements java.io.Serializable
    {
        String i_lastName;
        String i_firstName;
        public String getLastName() { return i_lastName; }
        public void setLastName(String value) { this.i_lastName = value; }
        public String getFirstName() { return i_firstName; }
        public void setFirstName(String value) { this.i_firstName = value; }
        public void onRowSelect()
        {
        }
        public void onRowExecute()
        {
        }
    }

    // ------------------------------------------------------------------------
    // members
    // ------------------------------------------------------------------------

    FIXGRIDListBinding<GridItem> m_grid = new FIXGRIDListBinding<GridItem>();

    // ------------------------------------------------------------------------
    // constructors & initialization
    // ------------------------------------------------------------------------

    public DemoGridStyle(IWorkpageDispatcher workpageDispatcher)
    {
        super(workpageDispatcher);
        initGridData();
    }

    public String getPageName() { return "/workplace/demogridstyle.xml"; }
    public String getRootExpressionUsedInPage() { return "#{d.DemoGridStyle}"; }

    // ------------------------------------------------------------------------
    // public usage
    // ------------------------------------------------------------------------

    public FIXGRIDListBinding<GridItem> getGrid() { return m_grid; }
    
    // ------------------------------------------------------------------------
    // private usage
    // ------------------------------------------------------------------------
    
    private void initGridData()
    {
        for (int i=0; i<100; i++)
        {
            GridItem gi = new GridItem();
            gi.setFirstName(RandomLogic.getFirstName());
            gi.setLastName(RandomLogic.getLastName());
            m_grid.getItems().add(gi);
        }
    }

}
