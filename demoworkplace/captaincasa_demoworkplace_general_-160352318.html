The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • General
  • Arranging Content
  • Includes
  • Dynamic Page Includes

The demo shows an outer page showing some inner pages.

The component for including one page within another page is the ROWPAGEBEANINCLUDE component.

In the page the ROWPAGEBEANINCLUDE is bound to the "contentBean" property. This property either holds an instance of the "Mini Spreadsheet" page bean or of the "Budget" page bean.

This is the XML layout definition:

<?xml version="1.0" encoding="utf-8"?>
<t:layoutdefinition xmlns:t="http://www.CaptainCasa.com/controllibrary/t" xmlns:tpbc="http://www.CaptainCasa.com/controllibrary/tpbc" >
<t:rowdemobodypane id="g_3" objectbinding="#{d.DemoInclude}" >
<t:row id="g_4" coldistance="5" >
<t:button id="g_5" actionListener="#{d.demoInclude.onSetPageSpread}" text="Mini Spreadsheet" />
<t:button id="g_6" actionListener="#{d.demoInclude.onSetPageBudget}" text="Budget Calculation" />
</t:row>
<t:rowdistance id="g_7" height="10" />
<t:row id="g_8" >
<t:pane id="g_9" background="#C0C0C0" height="1" width="100%" />
</t:row>
<t:rowdistance id="g_10" height="10" />
<t:rowpagebeaninclude id="g_11" pagebeanbinding="#{d.DemoInclude.contentBean}" />
</t:rowdemobodypane>
<t:pageaddons id="g_pa"/>
</t:layoutdefinition>

This is the Java code:

package workplace;

import javax.faces.event.ActionEvent;

import org.eclnt.jsfserver.pagebean.IPageBean;
import org.eclnt.workplace.IWorkpageDispatcher;

public class DemoInclude extends PageBean
{
    IPageBean m_contentBean = null;
    
    DemoMinispread m_miniSpread;
    DemoBudget m_budget;
    
    public DemoInclude(IWorkpageDispatcher dispatcher)
    {
        super(dispatcher);
        m_miniSpread = new DemoMinispread(getOwningDispatcher());
        m_budget = new DemoBudget(getOwningDispatcher());
        m_contentBean = m_miniSpread;
    }
    
    public IPageBean getContentBean() { return m_contentBean; }

    public void onSetPageSpread(ActionEvent ae)
    {
        m_contentBean = m_miniSpread;
    }
    
    public void onSetPageBudget(ActionEvent ae)
    {
        m_contentBean = m_budget;
    }
}
