The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • General
  • Arranging Content
  • Popup Dialogs
  • Adapting Popup Size

This example opens a popup with a certain height.

The popup shows information about a Java error - both a short text and a long text (the stack trace).

Within the popup a FOLDABLEPANE is contained. When opening this FOLDABLEPANE then the popup height is adapted accordingly.

This is the XML layout definition:

<?xml version="1.0" encoding="utf-8"?>
<t:layoutdefinition xmlns:t="http://www.CaptainCasa.com/controllibrary/t" xmlns:tpbc="http://www.CaptainCasa.com/controllibrary/tpbc" >
<t:rowdemobodypane id="g_1" objectbinding="#{d.DemoMessagePopupStarter}" >
<t:row id="g_2" >
<t:button id="g_3" actionListener="#{d.DemoMessagePopupStarter.onShowMessageAction}" text="Show Message" />
</t:row>
</t:rowdemobodypane>
<t:pageaddons id="g_pa"/>
</t:layoutdefinition>

This is the Java code:

package workplace;

import java.io.Serializable;

import javax.faces.event.ActionEvent;

import org.eclnt.editor.annotations.CCGenClass;
import org.eclnt.jsfserver.defaultscreens.ModalPopup;
import org.eclnt.workplace.IWorkpageDispatcher;

@CCGenClass (expressionBase="#{d.DemoMessagePopupStarter}")

public class DemoMessagePopupStarter
    extends PageBean 
    implements Serializable
{
    public DemoMessagePopupStarter(IWorkpageDispatcher workpageDispatcher)
    {
        super(workpageDispatcher);        
    }

    public String getPageName() { return "/workplace/demomessagepopupstarter.jsp"; }
    public String getRootExpressionUsedInPage() { return "#{d.DemoMessagePopupStarter}"; }
    
    public void onShowMessageAction(ActionEvent event) 
    {
        final DemoMessagePopup dmp = new DemoMessagePopup();
        ModalPopup popup = openModalPopup(dmp,"Message",0,0,new ModalPopup.IModalPopupListener()
        {
            public void reactOnPopupClosedByUser()
            {
                closePopup(dmp);
            }
        });
        dmp.prepare(new Error(),popup);
        popup.setLeftTopReferenceCentered();
    }

}
