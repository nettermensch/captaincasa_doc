The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • Components
  • Straight Controls
  • Field Control
  • Long Text Popup - lazy loading

This demo shows the possibility to lazily load the text at the point of time when the user opens the long text popup dialog.

There are three issues that you have to do:

1. Set LONGTEXTFIELD-LAZYLAODING to "true".

2. The text value requires to end with "..." if it is not fully loaded.

3. Implement the BaseActionEventLazyLoading event

Lazy loading automatically will switch the field component to non-editable - the user can only edit the text via the popup dialog.

This is the XML layout definition:

<?xml version="1.0" encoding="utf-8"?>
<t:layoutdefinition xmlns:t="http://www.CaptainCasa.com/controllibrary/t" xmlns:tpbc="http://www.CaptainCasa.com/controllibrary/tpbc" >
<t:beanprocessing id="g_1" beanbinding="#{d.DemoLongTextFieldLazyLoading}" />
<t:rowdemobodypane id="g_2" objectbinding="#{d.DemoLongTextFieldLazyLoading}" >
<t:row id="g_3" >
<t:longtextfield id="g_4" actionListener="#{d.DemoLongTextFieldLazyLoading.onFieldAction}" flush="true" labeltext="Lazily loaded text" lazyloading="true" text="#{d.DemoLongTextFieldLazyLoading.text}" width="100%" />
</t:row>
<t:rowdistance id="g_5" height="50" />
<t:row id="g_6" >
<t:textpane id="g_7" labeltext="Same text rendered in text pane" text="#{d.DemoLongTextFieldLazyLoading.text}" width="100%" />
</t:row>
</t:rowdemobodypane>
<t:pageaddons id="g_pa"/>
</t:layoutdefinition>

public class DemoLongTextFieldLazyLoading
    extends PageBean

{

    {

    }

    public String getPageName() { return "/workplace/demolongtextfieldlazyloading.jsp"; }
    public String getRootExpressionUsedInPage() { return "#{d.DemoLongTextFieldLazyLoading}"; }

    // ------------------------------------------------------------------------
    // public usage
    // ------------------------------------------------------------------------

    public String getText() { return m_text; }
    public void setText(String value) { this.m_text = value; }

    public void onFieldAction(javax.faces.event.ActionEvent event) 
    {
        if (event instanceof BaseActionEventLazyLoading)
        {
            m_text = "Lorem ipsum dolor sit amet, consetetur sadipscing elitr, sed diam nonumy eirmod tempor "

                    + "Lorem ipsum dolor sit amet, consetetur sadipscing elitr, sed diam nonumy eirmod tempor "
                    + "invidunt ut labore et dolore magna aliquyam erat, sed diam voluptua. "
