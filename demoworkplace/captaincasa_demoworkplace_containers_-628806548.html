The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • Containers
  • Scroll Pane - Extended Functions
  • Scroll pane navigator

The component SCROLLPANENVIGATOR renders the content of a SCROLLPANE component and allows to smartly navigate within the scroll pane.

Pleas note:

(.) The scroll pane is referenced by attribute REFSCROLLPANEID - pointing to the ID of the SCROLLPANE

(.) The component can be placed "everywhere". In the example the SCROLLPANE is the base item of an OVERLAYAREA and the SCROLLPANENAVIGATOR is arranged on top. But you may place it anywhere else, too.

(.) The component takes over the DOM tree of the original SCROLLPANE and renders it internally. So it actually operated on a cloned DOM tree. The cloning is executed at the beginning and then need to be triggered: whenever the rendering of the content is updated, you also have to trigger the cloning so that the navigator area is kept up to date.

This is the Java code:

package workplace;

import java.io.Serializable;
import java.util.ArrayList;
import java.util.List;

import org.eclnt.editor.annotations.CCGenClass;
import org.eclnt.jsfserver.elements.componentnodes.LABELNode;
import org.eclnt.jsfserver.elements.componentnodes.PAINTAREAITEMNode;
import org.eclnt.jsfserver.elements.componentnodes.PAINTAREALINEITEMNode;
import org.eclnt.jsfserver.elements.impl.DYNAMICCONTENTBinding;
import org.eclnt.jsfserver.elements.impl.ROWDYNAMICCONTENTBinding;
import org.eclnt.jsfserver.elements.impl.ROWDYNAMICCONTENTBinding.ComponentNode;
import org.eclnt.workplace.IWorkpageDispatcher;

@CCGenClass (expressionBase="#{d.DemoScrollPaneNavigator2}")

public class DemoScrollPaneNavigator2
    extends PageBean 
    implements Serializable
{
    // ------------------------------------------------------------------------
    // inner classes
    // ------------------------------------------------------------------------

    // ------------------------------------------------------------------------
    // members
    // ------------------------------------------------------------------------

    DYNAMICCONTENTBinding m_dynContent = new DYNAMICCONTENTBinding();

    // ------------------------------------------------------------------------
    // constructors & initialization
    // ------------------------------------------------------------------------

    public DemoScrollPaneNavigator2(IWorkpageDispatcher workpageDispatcher)
    {
        super(workpageDispatcher);       
        renderContent();
    }

    public String getPageName() { return "/workplace/demoscrollpanenavigator2.xml"; }
    public String getRootExpressionUsedInPage() { return "#{d.DemoScrollPaneNavigator2}"; }

    // ------------------------------------------------------------------------
    // public usage
    // ------------------------------------------------------------------------

    public DYNAMICCONTENTBinding getDynContent() { return m_dynContent; }

    // ------------------------------------------------------------------------
    // private usage
    // ------------------------------------------------------------------------
    
    private void renderContent()
    {
        List<ComponentNode> l = new ArrayList<ROWDYNAMICCONTENTBinding.ComponentNode>();
        int w = 120;
        int h = 80;
        int d = 100;
        for (int i=0; i<10; i++)
        {
            for (int j=0; j<10; j++)
            {
                PAINTAREAITEMNode pin = new PAINTAREAITEMNode()
                    .setBounds((i*w+(i+1)*d)+";" +
                               (j*h+(j+1)*d)+";" +
                               w + ";" +
                               h)
                    .setBorder("#C0C0C0")
                    .setBackground("#F0F0F0")
                    .setRounding(10);
                LABELNode lab = new LABELNode()
                    .setText(i+"/"+j)
                    .setAlign("center");
                pin.addSubNode(lab);
                l.add(pin);
                if (i > 0)
                {
                    PAINTAREALINEITEMNode lin = new PAINTAREALINEITEMNode()
                        .setBounds(((i)*w+(i)*d)+";" +
                                   (j*h+(j+1)*d+h/2)+";" +
                                   d + ";" +
                                   0)
                        .setArrowto(1);
                    l.add(lin);
                }
            }
        }
        m_dynContent.setContentNodes(l);
    }

    
}
