The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • Grids & Trees
  • Repeat Processing
  • Server side scrolling

"Normally" the REPEAT binds to a server side list of items. This may have the reulst of long lists being sent from the server to the client which means a problem both from performance and from end user point of view.

As consequence the server side scrolling that you may know from the FIXGRID component already is used for the REPEAT component as well:

On server side you do NOT manage the items within a straigh list, but within a "ServerSideScrollList". This list always provides these items to the client which are currently displayed. You can control the number of items by using the method setNumberOfItems() within the instance of the "special" list.

Within the demo a simple SCROLLBAR component is attached to the right of the pane. The scrollbar binds to the properties of the ServerSideScrollList-instance.

This is the Java code:

package workplace;

import java.io.Serializable;

import org.eclnt.editor.annotations.CCGenClass;
import org.eclnt.jsfserver.pagebean.PageBean;
import org.eclnt.jsfserver.util.ServerSideScrollList;
import org.eclnt.workplace.IWorkpageDispatcher;

import javax.faces.event.ActionEvent;

@CCGenClass (expressionBase="#{d.DemoRepeatServerSideScrolling}")

public class DemoRepeatServerSideScrolling
    extends PageBean 
    implements Serializable
{
    // ------------------------------------------------------------------------
    // inner classes
    // ------------------------------------------------------------------------
    
    public class Item
    {
        String m_firstName;
        String m_lastName;
        public String getFirstName() { return m_firstName; }
        public void setFirstName(String firstName) { m_firstName = firstName; }
        public String getLastName() { return m_lastName; }
        public void setLastName(String lastName) { m_lastName = lastName; }
    }
    
    // ------------------------------------------------------------------------
    // members
    // ------------------------------------------------------------------------
    
    ServerSideScrollList<Item> m_items = new ServerSideScrollList<Item>(); 
    
    // ------------------------------------------------------------------------
    // constructors & initialization
    // ------------------------------------------------------------------------

    public DemoRepeatServerSideScrolling(IWorkpageDispatcher dispatcher)
    {
        super(dispatcher);
        for (int i=0; i<45; i++)
        {
            Item it = new Item();
            it.setFirstName("First " + i);
            it.setLastName("Last " + i);
            m_items.getItems().add(it);
        }
    }

    public String getPageName() { return "/workplace/demorepeatserversidescrolling.jsp"; }
    public String getRootExpressionUsedInPage() { return "#{d.DemoRepeatServerSideScrolling}"; }

    // ------------------------------------------------------------------------
    // public usage
    // ------------------------------------------------------------------------

    public ServerSideScrollList<Item> getItems() { return m_items; }
    
    // ------------------------------------------------------------------------
    // private usage
    // ------------------------------------------------------------------------
}
