The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • Containers
  • Adaptive Containers - Controls
  • Adaptive matrix area
  • Always occupying full width

A component inside ROWPSANMATRIXAREA may use its COLSPAN attribute to indicate that it wants to occupy more than one matrix column - if the number of columns is available.

Take a look into the last component, occupying a COLSPAN of 100. This basically means: the component always occupies the full width of the matrix area.

This is the Java code:

package workplace;

import java.io.Serializable;

import org.eclnt.editor.annotations.CCGenClass;
import org.eclnt.workplace.IWorkpageDispatcher;

@CCGenClass (expressionBase="#{d.DemoAdaptiveSpans01}")

public class DemoAdaptiveSpans04
    extends WorkpageDispatchedPageBean 
    implements Serializable
{
    // ------------------------------------------------------------------------
    // inner classes
    // ------------------------------------------------------------------------

    // ------------------------------------------------------------------------
    // members
    // ------------------------------------------------------------------------

    int m_scaleInt = 60;

    // ------------------------------------------------------------------------
    // constructors & initialization
    // ------------------------------------------------------------------------

    public DemoAdaptiveSpans04(IWorkpageDispatcher workpageDispatcher)
    {
        super(workpageDispatcher);        
    }

    public String getPageName() { return "/workplace/demoadaptivespans01.xml"; }
    public String getRootExpressionUsedInPage() { return "#{d.DemoAdaptiveSpans01}"; }

    // ------------------------------------------------------------------------
    // public usage
    // ------------------------------------------------------------------------

    public int getScaleInt() { return m_scaleInt; }
    public void setScaleInt(int value) { m_scaleInt = value; }
    
    public double getScaleDouble() 
    {
        if (m_scaleInt == 100) 
            return 1d;
        else
            return (double)m_scaleInt/100d;
    }

    public void onScaleIntAction(javax.faces.event.ActionEvent event) 
    {
    }

    // ------------------------------------------------------------------------
    // private usage
    // ------------------------------------------------------------------------
}
