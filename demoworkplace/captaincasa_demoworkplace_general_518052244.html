The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • General
  • Workplace Management
  • Export / Import current workplace

For various reasons you might export the current workplace settings - and import them at a later point of time.

Example: you want that the user sees the same workplace that he/she left when logging on next time to the system.

This is the XML layout definition:

<?xml version="1.0" encoding="utf-8"?>
<t:layoutdefinition xmlns:t="http://www.CaptainCasa.com/controllibrary/t" >
<t:beanprocessing id="g_1" beanbinding="#{d.DemoWorkplaceExImport}" />
<t:rowdemobodypane id="g_4" objectbinding="#{d.DemoWorkplaceExImport}" rowdistance="#{ccstylevalue[&#39;@defaultrowdistance@&#39;]}" >
<t:row id="g_6" >
<t:button id="g_7" actionListener="#{d.DemoWorkplaceExImport.onExportAction}" text="Export" width="100+" />
</t:row>
<t:row id="g_9" >
<t:button id="g_10" actionListener="#{d.DemoWorkplaceExImport.onImportAction}" text="Import" width="100+" />
</t:row>

This is the Java code:

package workplace;

import java.io.Serializable;

import org.eclnt.editor.annotations.CCGenClass;
import org.eclnt.jsfserver.streamstore.StreamStore;
import org.eclnt.workplace.IWorkpageDispatcher;
import org.eclnt.workplace.WorkpageContainer;
import org.eclnt.workplace.eximport.WorkplaceExporter;

@CCGenClass (expressionBase="#{d.DemoWorkplaceExImport}")

public class DemoWorkplaceExImport
    extends WorkpageDispatchedPageBean 
    implements Serializable
{
    // ------------------------------------------------------------------------
    // inner classes
    // ------------------------------------------------------------------------

    // ------------------------------------------------------------------------
    // members
    // ------------------------------------------------------------------------

    String m_xml;
    // ------------------------------------------------------------------------
    // constructors & initialization
    // ------------------------------------------------------------------------

    public DemoWorkplaceExImport(IWorkpageDispatcher workpageDispatcher)
    {
        super(workpageDispatcher);        
    }

    public String getPageName() { return "/workplace/demoworkplaceeximport.xml"; }
    public String getRootExpressionUsedInPage() { return "#{d.DemoWorkplaceExImport}"; }

    // ------------------------------------------------------------------------
    // public usage
    // ------------------------------------------------------------------------

    public String getXml() { return m_xml; }
    public void setXml(String value) { this.m_xml = value; }

    public void onImportAction(javax.faces.event.ActionEvent event) 
    {
        m_xml = StreamStore.getInstance().readUTF8("ccdemos/workplaceeximport/workplace.xml",false);
        if (m_xml != null)
            new WorkplaceExporter().importWorkplaceFromXML((WorkpageContainer)getWorkpageContainer(),m_xml);
    }

    public void onExportAction(javax.faces.event.ActionEvent event) 
    {
        m_xml = new WorkplaceExporter().exportCurrentWorkplaceToXML((WorkpageContainer)getWorkpageContainer());
        StreamStore.getInstance().writeUTF8("ccdemos/workplaceeximport/workplace.xml",m_xml,true);
    }


    // ------------------------------------------------------------------------
    // private usage
    // ------------------------------------------------------------------------
}
