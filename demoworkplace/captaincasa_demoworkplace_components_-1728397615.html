The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • Components
  • Straight Controls
  • Field Control
  • Field Value Help - Details
  • Image Selection

Together with the COMBOFIELD you can provide value help, combining an image, a description and the id/value to be selected.

This is the XML layout definition:

<?xml version="1.0" encoding="utf-8"?>
<t:layoutdefinition xmlns:t="http://www.CaptainCasa.com/controllibrary/t" xmlns:tpbc="http://www.CaptainCasa.com/controllibrary/tpbc" >
<t:rowdemobodypane id="g_1" objectbinding="#{d.DemoComboFieldImageSelection}" >
<t:row id="g_2" >
<t:combofield id="g_3" actionListener="#{d.DemoComboFieldImageSelection.onCostCenterAction}" labeltext="Cost center" text="#{d.DemoComboFieldImageSelection.costCenter}" width="200" />
</t:row>
</t:rowdemobodypane>
<t:pageaddons id="g_pa"/>
</t:layoutdefinition>

This is the Java code:

package workplace;

import java.io.Serializable;

import javax.faces.event.ActionEvent;

import org.eclnt.editor.annotations.CCGenClass;
import org.eclnt.jsfserver.defaultscreens.ISetId;
import org.eclnt.jsfserver.defaultscreens.IdTextSelection;
import org.eclnt.jsfserver.elements.events.BaseActionEventValueHelp;
import org.eclnt.workplace.IWorkpageDispatcher;

@CCGenClass (expressionBase="#{d.DemoComboFieldImageSelection}")

public class DemoComboFieldImageSelection
    extends PageBean
    implements Serializable
{
    protected String m_costCenter;

    public DemoComboFieldImageSelection(IWorkpageDispatcher dispatcher)
    {
        super(dispatcher);
        initRefComponent("combofield"); 
    }
    
    public String getCostCenter() { return m_costCenter; }
    public void setCostCenter(String value) { m_costCenter = value; }

    public void onCostCenterAction(ActionEvent event) 
    {
        if (event instanceof BaseActionEventValueHelp)
        {
            IdTextSelection idts = IdTextSelection.createInstanceWithImageLine();
            idts.addLine("BOARD","Executive Board","/images/pictures/ph0.jpg");
            idts.addLine("DEV","Development","/images/pictures/ph1.jpg");
            idts.addLine("PROD","Production","/images/pictures/ph2.jpg");
            idts.addLine("SALES","Sales","/images/pictures/ph3.jpg");
            idts.addLine("HR","Human Resources","/images/pictures/ph4.jpg");
            idts.addLine("QA1","Quality Assurance 1","/images/pictures/ph5.jpg");
            idts.addLine("QA2","Quality Assurance 2","/images/pictures/ph6.jpg");
            idts.setCallBack(new ISetId() 
            {
                public void setId(String id)
                {
                    m_costCenter = id;
                }
            });
        }
    }
    
}
