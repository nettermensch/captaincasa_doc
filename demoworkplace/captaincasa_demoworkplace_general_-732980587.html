The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • General
  • Client Integration
  • Client File API (Java Client only)
  • USB File Reader

If triggered, the component scans all available client root directories and checks if a file with the name defined in FILENAMEINROOT exists.

If it exists then an event is sent to the server side - providing the full client name of the file and providing the content of the file.

If it does not exist then also an evnet is sent up to the server side, this time indicating that there is no corresponding file on client side.

Typical usage case: you want to check and read a certain file that resides on a USB stick (e.g. containing some certificate information). You do not know the drive name of the file, because this is dynamically assigned by the client operating system.

With ROOTSTOEXCLUDE you may optionally define roots in which you do not want to look for the file, e.g. you may exclude "C:\;D:\" in order bypass the client side hard disk. You may also pass a ROOTSTOPREFER value, a semicolon separated list of roots, that the component should check first for the file. This makes sense, if your USB sticks always are bound to a certain drive letter.

This is the Java code:

package workplace;

import java.io.Serializable;

import javax.faces.event.ActionEvent;

import org.eclnt.editor.annotations.CCGenClass;
import org.eclnt.jsfserver.elements.events.BaseActionEventClientUsbFileReader;
import org.eclnt.jsfserver.elements.util.Trigger;
import org.eclnt.workplace.IWorkpageDispatcher;
import org.eclnt.workplace.WorkpageDispatchedBean;

@CCGenClass (expressionBase="#{d.DemoClientUsbFileReader}")

public class DemoClientUsbFileReader
    extends PageBean 
    implements Serializable
{
    String m_filenameInRoot = "temp/test.png";
    public String getFilenameInRoot() { return m_filenameInRoot; }
    public void setFilenameInRoot(String value) { this.m_filenameInRoot = value; }

    Trigger m_trigger = new Trigger();
    public Trigger getTrigger() { return m_trigger; }
    public void setTrigger(Trigger value) { this.m_trigger = value; }

    String m_result;
    public String getResult() { return m_result; }
    public void setResult(String value) { this.m_result = value; }

    // ------------------------------------------------------------------------
    // constructors & initialization
    // ------------------------------------------------------------------------

    public DemoClientUsbFileReader(IWorkpageDispatcher workpageDispatcher)
    {
        super(workpageDispatcher);        
    }

    public void onTriggerUsbFileRead(ActionEvent event)
    {
        m_trigger.trigger();
    }
    
    public void onClientUsbFileAction(ActionEvent event) 
    {
        if (event instanceof BaseActionEventClientUsbFileReader)
        {
            BaseActionEventClientUsbFileReader e = (BaseActionEventClientUsbFileReader)event;
            if (e.getFileFound() == true)
            {
                StringBuffer sb = new StringBuffer();
                sb.append("File was found on client side." +
                          "\n\nClient file name: " +
                          e.getClientFileName() + 
                          "\n\nContent:\n" +
                          e.getFileContentAsHexString());
                m_result = sb.toString();
            }
            else
            {
                StringBuffer sb = new StringBuffer();
                sb.append("File was NOT found on client side."); 
                m_result = sb.toString();
            }
        }
    }

}
