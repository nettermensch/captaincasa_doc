The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • General
  • Dynamic Layout
  • Tuning - Stable Ids
  • Simple example

Click the "Rebuild" buttons: on the left side the dynamically content will flicker because it is rebuilt from the scratch. On the right side there is no flickering because components are re-used.

When building dynamic content that is updated frequently then explicitly assign stable ids to the components.

Both the server processing and the client management of controls are "id-aware": if a component at a certain position is already created, then it will not be re-created, but the existing component is used.

Please pay attention: you need to assign on any every level of the component hierarchy! - And of course: you are responsible to ensure that each id is unique within the scope of the corresponding ROWDYNAMICCONTENT component.

This is the Java code:

package workplace;

import java.io.Serializable;
import java.util.HashMap;
import java.util.Map;

import org.eclnt.editor.annotations.CCGenClass;
import org.eclnt.jsfserver.elements.componentnodes.FIELDNode;
import org.eclnt.jsfserver.elements.componentnodes.PANENode;
import org.eclnt.jsfserver.elements.componentnodes.ROWNode;
import org.eclnt.jsfserver.elements.impl.ROWDYNAMICCONTENTBinding;
import org.eclnt.workplace.IWorkpageDispatcher;

@CCGenClass (expressionBase="#{d.DemoRowDynamicContentIds}")

public class DemoRowDynamicContentIds
    extends PageBean 
    implements Serializable
{
    // ------------------------------------------------------------------------
    // members
    // ------------------------------------------------------------------------

    ROWDYNAMICCONTENTBinding m_dynContent1 = new ROWDYNAMICCONTENTBinding();
    ROWDYNAMICCONTENTBinding m_dynContent2 = new ROWDYNAMICCONTENTBinding();
    Map<String,String> m_data = new HashMap<String,String>(); 

    // ------------------------------------------------------------------------
    // constructors & initialization
    // ------------------------------------------------------------------------

    public DemoRowDynamicContentIds(IWorkpageDispatcher workpageDispatcher)
    {
        super(workpageDispatcher);     
        onRebuild1Action(null);
        onRebuild2Action(null);
    }

    public String getPageName() { return "/workplace/demorowdynamiccontentids.jsp"; }
    public String getRootExpressionUsedInPage() { return "#{d.DemoRowDynamicContentIds}"; }

    // ------------------------------------------------------------------------
    // public usage
    // ------------------------------------------------------------------------

    public ROWDYNAMICCONTENTBinding getDynContent1() { return m_dynContent1; }
    public ROWDYNAMICCONTENTBinding getDynContent2() { return m_dynContent2; }
    public Map<String, String> getData() { return m_data; }

    public void onRebuild1Action(javax.faces.event.ActionEvent event) 
    {
        PANENode p = new PANENode().setWidth("100%");
        for (int i=0; i<20; i++)
        {
            ROWNode r = new ROWNode();
            p.addSubNode(r);
            FIELDNode f = new FIELDNode();
            f.setLabeltext("Name " + i);
            f.setText(pbx("data.name"+i)).setWidth("100%");
            r.addSubNode(f);
        }
        m_dynContent1.setContentNode(p);
    }
    
    public void onRebuild2Action(javax.faces.event.ActionEvent event) 
    {
        PANENode p = new PANENode().setWidth("100%").setId("PANE");
        for (int i=0; i<20; i++)
        {
            ROWNode r = new ROWNode().setId("ROW"+i);
            p.addSubNode(r);
            FIELDNode f = new FIELDNode().setId("FIELD"+i);
            f.setLabeltext("Name " + i);
            f.setText(pbx("data.name"+i)).setWidth("100%");
            r.addSubNode(f);
        }
        m_dynContent2.setContentNode(p);
    }

    // ------------------------------------------------------------------------
    // private usage
    // ------------------------------------------------------------------------
}
