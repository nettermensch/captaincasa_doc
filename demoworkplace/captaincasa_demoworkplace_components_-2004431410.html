The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • Components
  • Animation & Graphics & Sound
  • Paint Area
  • WYSIWYG Editing

The PAINTAREA and PAINTAREAITEM components are prepared to serve as base for wysiwyg editor processing (what you see is what you get).

PAINTAREAITEM components can be defined to be move-able (MOVINGENABLED) and size-able (RESIZINGENABLED). In this case small areas are painted on top of the component - when dragging these areas the corresponding component is moved or resized.

When updating a component's position and/or size then the BOUNDS are updated accordingly. You may set FLUSH to true in order to immediately transfer these changed to the server.

...what else?

PAINTAREAITEM components may hold (exactly!) one subcomponent which is rendered to fit inside the component.

PAINTAREAITEM provides an attribute EDITGRID. When setting to a certain integer value then moving and sizing a PAINTAREAITEM will snap to the value defined. In addition a grid will be painted as background.

This is the Java code:

package workplace;

import java.io.Serializable;

import javax.faces.event.ActionEvent;

import org.eclnt.editor.annotations.CCGenClass;
import org.eclnt.jsfserver.defaultscreens.Statusbar;
import org.eclnt.jsfserver.elements.events.BaseActionEventRangeSelection;
import org.eclnt.workplace.IWorkpageDispatcher;

@CCGenClass (expressionBase="#{d.DemoWysiwyg}")

public class DemoWysiwyg
    extends PageBean 
    implements Serializable
{
    int m_scaleInt = 100;
    public int getScaleInt() { return m_scaleInt; }
    public void setScaleInt(int value) { this.m_scaleInt = value; }

    double m_scale = 1d;
    
    String m_boundsLabel = "20;20;100;25";
    String m_boundsField = "120;20;100;25";
    String m_boundsButton = "260;20;80;25";
    
    public DemoWysiwyg(IWorkpageDispatcher workpageDispatcher)
    {
        super(workpageDispatcher);        
    }

    public String getBoundsField() { return m_boundsField; }
    public void setBoundsField(String bounds) { m_boundsField = bounds; }

    public String getBoundsLabel() { return m_boundsLabel; }
    public void setBoundsLabel(String boundsLabel) { m_boundsLabel = boundsLabel; }

    public String getBoundsButton() { return m_boundsButton; }
    public void setBoundsButton(String boundsButton) { m_boundsButton = boundsButton; }

    public double getScale() { return m_scale; }
    public void setScale(double scale) { m_scale = scale; }

    public void onScaleIntAction(ActionEvent event) 
    {
        m_scale = (double)m_scaleInt / 100d;
    }

    public void onScaleAction(ActionEvent event) 
    {
        m_scaleInt = (int)Math.round(m_scale * 100d);
    }

    public void onBoundsAction(ActionEvent event) 
    {
        Statusbar.outputSuccess("Bounds Update: " + m_boundsField);
    }

    public void onPaintAreaAction(ActionEvent event) 
    {
        if (event instanceof BaseActionEventRangeSelection)
        {
            BaseActionEventRangeSelection e = (BaseActionEventRangeSelection)event;
            Statusbar.outputSuccessWithPopup(e.getLeft() + "/" + e.getTop() + "/" + e.getWidth() + "/" + e.getHeight());
        }
    }

}
