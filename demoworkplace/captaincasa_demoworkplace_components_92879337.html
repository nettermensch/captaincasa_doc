The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • Components
  • Straight Controls
  • Invokers
  • Vertical button

The component BUTTONVERTICAL is just a normal button, but rotated.

This is the XML layout definition:

This is the Java code:

package workplace;

import java.io.Serializable;
import org.eclnt.editor.annotations.CCGenClass;
import org.eclnt.jsfserver.managedbean.IDispatcher;
import org.eclnt.jsfserver.pagebean.PageBean;
import org.eclnt.workplace.IWorkpageDispatcher;

import javax.faces.event.ActionEvent;

@CCGenClass (expressionBase="#{d.DemoButtonVertical}")

public class DemoButtonVertical
    extends WorkpageDispatchedPageBean
    implements Serializable
{
    public DemoButtonVertical(IWorkpageDispatcher dispatcher)
    {
        super(dispatcher);
    }

    public String getPageName() { return "/workplace/demobuttonvertical.jsp"; }
    public String getRootExpressionUsedInPage() { return "#{d.DemoButtonVertical}"; }
}
