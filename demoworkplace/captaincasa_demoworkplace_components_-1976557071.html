The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • Components
  • Straight Controls
  • Text Input/Output
  • Text Area with Line Numbers

The component TEXTAREAWITHCOMMENT shows a normal text area on the right.

On the left there is a parallel text area which renders a comment. The comment typically is some numbering, but may also be something different.

This is the XML layout definition:

<?xml version="1.0" encoding="utf-8"?>
<t:layoutdefinition xmlns:t="http://www.CaptainCasa.com/controllibrary/t" xmlns:tpbc="http://www.CaptainCasa.com/controllibrary/tpbc" >
<t:rowdemobodypane id="g_1" objectbinding="#{d.DemoTextAreaWithComment}" rowdistance="#{ccstylevalue['@defaultrowdistance@']}" >
<t:row id="g_2" >
<t:textareawithcomment id="g_3" commenttext="#{d.DemoTextAreaWithComment.comment}" flush="true" flushtimer="1000" font="family:Courier New" height="100%" text="#{d.DemoTextAreaWithComment.text}" userhint="Hello User Hint" width="100%" />
</t:row>
<t:row id="g_4" coldistance="5" >
<t:button id="g_5" actionListener="#{d.DemoTextAreaWithComment.onDefaultCommentAction}" text="Default numbering" />
<t:button id="g_6" actionListener="#{d.DemoTextAreaWithComment.onSpecialCommentAction}" text="Own numbering" />
</t:row>
</t:rowdemobodypane>
<t:pageaddons id="g_pa"/>
</t:layoutdefinition>

This is the Java code:

package workplace;

import java.io.Serializable;
import org.eclnt.editor.annotations.CCGenClass;
import org.eclnt.jsfserver.pagebean.PageBean;
import org.eclnt.workplace.IWorkpageDispatcher;

import javax.faces.event.ActionEvent;

@CCGenClass (expressionBase="#{d.DemoTextAreaWithComment}")

public class DemoTextAreaWithComment
    extends DemoPageBean 
    implements Serializable
{
    // ------------------------------------------------------------------------
    // inner classes
    // ------------------------------------------------------------------------
    
    // ------------------------------------------------------------------------
    // members
    // ------------------------------------------------------------------------
    
    String m_text = "a\nb\nc\na\nb\nc\na\nb\nc\na\nb\nc\na\nb\nc\n";
    String m_comment = null;
    
    // ------------------------------------------------------------------------
    // constructors & initialization
    // ------------------------------------------------------------------------

    public DemoTextAreaWithComment(IWorkpageDispatcher dispatcher)
    {
        super(dispatcher);
    }

    public String getPageName() { return "/workplace/demotextareawithcomment.jsp"; }
    public String getRootExpressionUsedInPage() { return "#{d.DemoTextAreaWithComment}"; }

    // ------------------------------------------------------------------------
    // public usage
    // ------------------------------------------------------------------------

    public String getText() { return m_text; }
    public void setText(String value) { this.m_text = value; }
    
    public String getComment() { return m_comment; }

    public void onDefaultCommentAction(ActionEvent event)
    {
        m_comment = null;
    }
    
    public void onSpecialCommentAction(ActionEvent event)
    {
        StringBuffer sb = new StringBuffer();
        for (int i=1; i<200; i++)
        {
            sb.append("\n\n\n\n\n\n\n\n\n"+(i*10)+"\n");
        }
        m_comment = sb.toString();
    }
    

    // ------------------------------------------------------------------------
    // private usage
    // ------------------------------------------------------------------------
}
