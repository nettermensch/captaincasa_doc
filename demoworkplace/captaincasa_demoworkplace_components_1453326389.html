The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • Components
  • Straight Controls
  • Value Representation
  • Slider - From to time range

As with the normal SLIDER, you can also assign any meaning to the integer values that are maintained by the SLIDERRANGE component.

In this demo the slider component is used to define some time range between 08:00h and 16:00h.

Take a look into the definition of the attribute VALUESTRINCONVERSION which binds the inner integer values to some semantic definition.

This is the XML layout definition:

This is the Java code:

package workplace;

import java.io.Serializable;
import java.time.LocalTime;

import org.eclnt.editor.annotations.CCGenClass;
import org.eclnt.workplace.IWorkpageDispatcher;

@CCGenClass (expressionBase="#{d.DemoSliderRange2}")

public class DemoSliderRange2
    extends PageBean 
    implements Serializable
{
    // ------------------------------------------------------------------------
    // inner classes
    // ------------------------------------------------------------------------

    // ------------------------------------------------------------------------
    // members
    // ------------------------------------------------------------------------

    LocalTime m_toTime = LocalTime.of(12,0);
    LocalTime m_fromTime = LocalTime.of(10,0);

    // ------------------------------------------------------------------------
    // constructors & initialization
    // ------------------------------------------------------------------------

    public DemoSliderRange2(IWorkpageDispatcher workpageDispatcher)
    {
        super(workpageDispatcher);        
    }

    public String getPageName() { return "/workplace/demosliderrange2.xml"; }
    public String getRootExpressionUsedInPage() { return "#{d.DemoSliderRange2}"; }

    // ------------------------------------------------------------------------
    // public usage
    // ------------------------------------------------------------------------

    public int getToAsInt() { return convertLocalTimeToInt(m_toTime); }
    public void setToAsInt(int value) { m_toTime = convertIntToLocalTime(value); }

    public int getFromAsInt() { return convertLocalTimeToInt(m_fromTime); }
    public void setFromAsInt(int value) { m_fromTime = convertIntToLocalTime(value); }

    public LocalTime getToTime() { return m_toTime; }
    public void setToTime(LocalTime value) { this.m_toTime = value; }

    public LocalTime getFromTime() { return m_fromTime; }
    public void setFromTime(LocalTime value) { this.m_fromTime = value; }

    // ------------------------------------------------------------------------
    // private usage
    // ------------------------------------------------------------------------
    
    private int convertLocalTimeToInt(LocalTime lt)
    {
        int hour = (lt.getHour()-8) * 4;
        int quarter = Math.floorDiv(lt.getMinute(),15);
        return hour + quarter;
    }
    
    private LocalTime convertIntToLocalTime(int value)
    {
        int hour = 8 + value/4;
        int minute = value%4 * 15;
        return LocalTime.of(hour,minute);
    }
}
