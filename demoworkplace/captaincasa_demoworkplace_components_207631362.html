The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • Components
  • Animation & Graphics & Sound
  • Paint Area
  • Performance considerations
  • Creating large 1:n relations

In this example all FIELD instances are rendered within one PAINTAREA component.

This means: on client side a DOM element tree is managed in which one parent node manages a all the sub nodes as children. The performance of any operation on server and client side, which affects this 1:n relationship, as consequence linearly depends on the number of children.

Examples for such operations are:

1. adding / rearranging components
2. focus management (e.g. "tabbing" between components)

We recommend to not build up parent-child relationships with more than 500 children. ...and we recommend to test your scenarios in which a lot of nodes may be created!

The next example in the demo workplace (DemoPaintAreaManyFields2) shows a way to overcome.

This is the Java code:

package workplace;

import java.io.Serializable;
import java.util.ArrayList;
import java.util.HashMap;
import java.util.List;
import java.util.Map;

import org.eclnt.editor.annotations.CCGenClass;
import org.eclnt.jsfserver.elements.componentnodes.FIELDNode;
import org.eclnt.jsfserver.elements.componentnodes.PAINTAREAITEMNode;
import org.eclnt.jsfserver.elements.impl.DYNAMICCONTENTBinding;
import org.eclnt.jsfserver.elements.impl.ROWDYNAMICCONTENTBinding.ComponentNode;
import org.eclnt.workplace.IWorkpageDispatcher;

@CCGenClass (expressionBase="#{d.DemoPaintAreaManyFields}")

public class DemoPaintAreaManyFields
    extends PageBean 
    implements Serializable
{
    // ------------------------------------------------------------------------
    // inner classes
    // ------------------------------------------------------------------------

    // ------------------------------------------------------------------------
    // members
    // ------------------------------------------------------------------------

    private Map<String,Object> m_data = new HashMap<String,Object>();
    int m_numberOfFields = 2000;
    DYNAMICCONTENTBinding m_dynContent = new DYNAMICCONTENTBinding();

    // ------------------------------------------------------------------------
    // constructors & initialization
    // ------------------------------------------------------------------------

    public DemoPaintAreaManyFields(IWorkpageDispatcher workpageDispatcher)
    {
        super(workpageDispatcher);        
    }

    public String getPageName() { return "/workplace/demopaintareamanyfields.xml"; }
    public String getRootExpressionUsedInPage() { return "#{d.DemoPaintAreaManyFields}"; }

    // ------------------------------------------------------------------------
    // public usage
    // ------------------------------------------------------------------------

    public Map<String, Object> getData() { return m_data; } 

    public void onRenderAction(javax.faces.event.ActionEvent event) { render(); }

    public int getNumberOfFields() { return m_numberOfFields; }
    public void setNumberOfFields(int value) { this.m_numberOfFields = value; }

    public DYNAMICCONTENTBinding getDynContent() { return m_dynContent; }

    // ------------------------------------------------------------------------
    // private usage
    // ------------------------------------------------------------------------
    
    private void render()
    {
        int x = 0;
        int y = -30;
        List<ComponentNode> nodes = new ArrayList<ComponentNode>();
        for (int i=0; i<m_numberOfFields; i++)
        {
            x += 100;
            if (i%10 == 0)
            {
                x = 0;
                y += 30;
            }
            PAINTAREAITEMNode pi = new PAINTAREAITEMNode()
                .setBounds(x+";"+y+";90;25")
                .setSelected(true)
                .setFlush(true)
                .setSelectionenabled(true);
            nodes.add(pi);
            FIELDNode f = new FIELDNode()
                .setText(pbx("data['D"+i+"']"));
            m_data.put("D"+i,"Text "+i);
            pi.addSubNode(f);
        }
        m_dynContent.setContentNodes(nodes);
    }

}
