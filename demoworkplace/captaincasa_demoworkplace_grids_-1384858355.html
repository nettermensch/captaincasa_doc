The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • Grids & Trees
  • Grids and Trees
  • Grid Row Selector
  • Title Cell Icon

When using the attribute FIXGRID-SELECTORCOLUMN, so that a selector column is drawn as first column, then you can in addition place an icon into the title cell.

You do so by defining the attribute FIXGRID-SELCTORCOLUMNTITLEIMAGE.

On server side the method "onSelectorTitleIconInvoked(...)" is called within the grid binding. You need to override e.g. the FIXGRIDListBinding class in order to add your own implementation.

In the example, the icon is rendered like a checkbox - so pay attention: the chekbox that you see on the top left is not a real checkbox, it is an icon - the image behind the icon is updated correspondingly.
When the user selects the icon, then in the example either all items are selected, or all items are de-selected.

This is the Java code:

package workplace;

import java.io.Serializable;

import org.eclnt.editor.annotations.CCGenClass;
import org.eclnt.jsfserver.elements.events.BaseActionEventGridSelectorTitleIcon;
import org.eclnt.jsfserver.elements.impl.FIXGRIDItem;
import org.eclnt.jsfserver.elements.impl.FIXGRIDListBinding;
import org.eclnt.workplace.IWorkpageDispatcher;

@CCGenClass (expressionBase="#{d.DemoGridSelectorTitleIcon}")

public class DemoGridSelectorTitleIcon
    extends PageBean 
    implements Serializable
{
    // ------------------------------------------------------------------------
    // inner classes
    // ------------------------------------------------------------------------
    
    public class MyGridItem extends FIXGRIDItem implements java.io.Serializable
    {
        protected String m_lastName;
        public String getLastName() { return m_lastName; }
        public void setLastName(String value) { m_lastName = value; }

        protected String m_firstName;
        public String getFirstName() { return m_firstName; }
        public void setFirstName(String value) { m_firstName = value; }
    }
    
    public class MyGrid extends FIXGRIDListBinding<MyGridItem>
    {
        @Override
        public void onSelectorTitleIconInvoked(BaseActionEventGridSelectorTitleIcon event)
        {
            int numberOfSelectedItems = m_grid.getSelectedItems().size();
            if (numberOfSelectedItems <= 1)
            {
                // select all...
                m_grid.deselectCurrentSelection();
                for (MyGridItem gi: m_grid.getItems())
                    m_grid.selectItem(gi);
                m_titleImage = "/images/icons/checkbox_true.png";                
            }
            else
            {
                // select nothing
                m_grid.deselectCurrentSelection();
                m_titleImage = "/images/icons/checkbox_false.png";                
            }
        }
    }

    // ------------------------------------------------------------------------
    // members
    // ------------------------------------------------------------------------
    
    MyGrid m_grid = new MyGrid();
    String m_titleImage = "/images/icons/checkbox_false.png";

    // ------------------------------------------------------------------------
    // constructors
    // ------------------------------------------------------------------------
    
    public DemoGridSelectorTitleIcon(IWorkpageDispatcher workpageDispatcher)
    {
        super(workpageDispatcher);        
        for (int i=0; i<100; i++)
        {
            MyGridItem gi = new MyGridItem();
            gi.setFirstName("FN " + i);
            gi.setLastName("LN " + i);
            m_grid.getItems().add(gi);
        }
    }

    // ------------------------------------------------------------------------
    // public usage
    // ------------------------------------------------------------------------
    
    public MyGrid getGrid() { return m_grid; }
    public String getTitleImage() { return m_titleImage; }
    
    // ------------------------------------------------------------------------
    // private usage
    // ------------------------------------------------------------------------

    
}
