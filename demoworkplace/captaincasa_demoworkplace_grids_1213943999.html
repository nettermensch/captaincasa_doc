The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • Grids & Trees
  • Grids and Trees
  • Grouping column headers
  • Grouped columns

The component FIXGRDIDWITHGROUPING is a "just normal" grid, but provides the possibility to define a grouping on header level.

Take a look at the drag&drop that is provided on column level: you only can drag/drop columns within one group - so that the grouping never can be destroyed.

The definition is done by using component GRIDICOLGROUP. When defining, pay attention to correctly define the grouping: the arrangement of groups must be 100% balanced. This means: all GRIDCOL nodes need to exactly be positioned on the same hierarchy level. It is not possible to have a grouping level of "1 level" in the one part of the grid, and a grouping level of "2 levels" in other parts.
