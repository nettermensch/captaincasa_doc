The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • Components
  • Animation & Graphics & Sound
  • Paint Area
  • Simple Usage

This is a very basic and static demo of the PAINTAREA component.

The PAINTAREA is a container holding PAINTAREAITEM components. Each component receives an explicit x;y;width;height(;z)-position by using the BOUNDS attribute. The PAINTAREAITEM may contain any "normal" component within its content area.

In addition there is the component PAINTAREALINEITEM, which is used for drawing lines.
