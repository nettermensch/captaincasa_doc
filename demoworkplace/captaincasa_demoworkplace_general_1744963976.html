The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • General
  • Workplace Management
  • Restart workplace

There are cases in which you want to reload a whole application - e.g. one user logs off - and you want to be sure that all objects are initialized "by guarantee".

In this case you may use the feature that is offered by the piece of the server side framework that translates the first name of a JSF expression ("#{d.xyz.abc" => the "d") into an object instance. The class that is responsible for doing this (VariableResolverByDelegation) provides the possibility to pass a new object for a given name at runtime.

When having a used a dispatcher-like variable approach then it is to re-initialize central objects of your server side UI processing.

This is the XML layout definition:

<?xml version="1.0" encoding="utf-8"?>
<t:layoutdefinition xmlns:t="http://www.CaptainCasa.com/controllibrary/t" xmlns:tpbc="http://www.CaptainCasa.com/controllibrary/tpbc" >
<t:rowdemobodypane id="g_1" objectbinding="#{d.DemoRestart}" rowdistance="#{ccstylevalue['@defaultrowdistance@']}" >
<t:row id="g_2" >
<t:button id="g_3" actionListener="#{d.DemoRestart.onRestart}" text="&#34;Hard&#34; Restart" />
</t:row>
<t:row id="g_4" >
<t:button id="g_5" actionListener="#{d.DemoRestart.onRestartSoft}" text="&#34;Soft&#34; Restart" />
</t:row>
</t:rowdemobodypane>
<t:pageaddons id="g_pa"/>
</t:layoutdefinition>

This is the Java code:

package workplace;

import java.io.Serializable;

import javax.faces.event.ActionEvent;

import org.eclnt.editor.annotations.CCGenClass;
import org.eclnt.jsfserver.elements.phases.PhaseManager;
import org.eclnt.jsfserver.util.HttpSessionAccess;
import org.eclnt.jsfserver.util.VariableResolverByDelegation;
import org.eclnt.workplace.IWorkpageDispatcher;

@CCGenClass (expressionBase="#{d.DemoRestart}")

public class DemoRestart 
    extends WorkpageDispatchedPageBean
    implements Serializable
{
    public DemoRestart(IWorkpageDispatcher dispatcher)
    {
        super(dispatcher);
    }

    public void onRestart(ActionEvent event) 
    {
        HttpSessionAccess.getCurrentDialogSession().invalidate();
    }

    public void onRestartSoft(ActionEvent event) 
    {
        getWorkpageContainer().closeAllWorkpages(false,new Runnable() 
        {
            public void run()
            {
                HttpSessionAccess.getCurrentDialogSession().invalidate();
            }
        });
    }

}
