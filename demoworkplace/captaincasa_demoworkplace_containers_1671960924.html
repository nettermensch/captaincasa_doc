The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • Containers
  • Container sizing
  • Column distance in rows

The ROW components provides an attribute COLDISTANCE - which is the default horizontal distance between the contained components.

This is the horizontal counter part of the ROWDISTANCE attribute, that is provided in container components.
