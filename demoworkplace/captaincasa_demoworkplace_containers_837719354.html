The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • Containers
  • Adaptive Containers - Controls
  • Adaptive tile cotainer

The ADAPTIVETILECONTAINER component allows to place content components ("tiles") which are arranged within a matrix. If the screen space is not sufficient then several pages are created internally. The user can navigate between the pages by dragging and dropping, or by using the corresponding navigation buttons.

Contained components are measured by their defined (minimum) height. You may not use percentage sizing within the contained components.
