The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • General
  • RISC Client Integration
  • System icon

The component SYSTEMICON is an icon that directly executes dedicated client functions.

The function is passed by setting attribute SYSTEMFUNCTION.
