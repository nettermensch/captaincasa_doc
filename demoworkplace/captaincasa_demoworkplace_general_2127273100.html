The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • General
  • RISC Client Integration
  • Iframe inclusion with JS interface

The component IFRAMEINCLUDE is a generic component to integrate an HTML-page that resides on the same server.

You can talk to this HTML-page by sending commands via a JavaScript API that the HTML page has to implement.

Please check the file <webcontentcc>/eclnt/risc/ext_iframeinclude_ccexample/example.html - which explains by example how to do the JavaScript integration.

This is the Java code:

package workplace;

import java.io.Serializable;

import javax.faces.event.ActionEvent;

import org.eclnt.editor.annotations.CCGenClass;
import org.eclnt.jsfserver.defaultscreens.Statusbar;
import org.eclnt.jsfserver.elements.events.BaseActionEventIframeIncludeCallback;
import org.eclnt.jsfserver.elements.events.BaseActionEventPdfprinterNotification;
import org.eclnt.jsfserver.elements.util.Trigger;
import org.eclnt.workplace.IWorkpageDispatcher;
import org.eclnt.workplace.WorkpageDispatchedPageBean;

@CCGenClass (expressionBase="#{d.DemoIframeInclude}")

public class DemoIframeInclude
    extends PageBean 
    implements Serializable
{
    // ------------------------------------------------------------------------
    // members
    // ------------------------------------------------------------------------

    String m_configuration = "Some configuration.";

    // ------------------------------------------------------------------------
    // constructors & initialization
    // ------------------------------------------------------------------------

    public DemoIframeInclude(IWorkpageDispatcher workpageDispatcher)
    {
        super(workpageDispatcher);        
    }

    public String getPageName() { return "/workplace/demoiframeinclude.jsp"; }
    public String getRootExpressionUsedInPage() { return "#{d.DemoIframeInclude}"; }

    // ------------------------------------------------------------------------
    // public usage
    // ------------------------------------------------------------------------

    public String getConfiguration() { return m_configuration; }
    public void setConfiguration(String value) { this.m_configuration = value; }

    public void onSendConfigurationAction(javax.faces.event.ActionEvent event) 
    {
    }
    
    public void onCallbackAction(ActionEvent event)
    {
        if (event instanceof BaseActionEventIframeIncludeCallback)
        {
            BaseActionEventIframeIncludeCallback e = (BaseActionEventIframeIncludeCallback)event;
            String paramString = e.getCallbackParam();
            Statusbar.outputSuccessWithPopup(paramString);
        }
    }

    // ------------------------------------------------------------------------
    // private usage
    // ------------------------------------------------------------------------
}
