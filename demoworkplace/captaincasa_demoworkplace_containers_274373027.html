The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • Containers
  • Scroll Pane - Extended Functions
  • Scrollbar Mode

By using the attributes HORIZONTALSCROLLMODE and VERTICALSCROLLMODE you can control the scrolling behavior of the SCROLLPANE component.

Please note: the attribute value "autowithresize" is reserved for FIXGRID components only - it cannot be applied with SCROLLPANE components.
