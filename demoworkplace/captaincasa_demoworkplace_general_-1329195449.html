The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • General
  • Client Integration
  • Window focus info (Java Client only)

The component WINDOWFOCUSINFO is a quite special one. It tells you with every roundtrip if the window, that is holding the component, has the focus (true) or not (false).

There are special use cases, in which this component makes sense: when e.g. retrieving an event from a serial scanner device (via CLIENTSERIALLRECEIVER or CLIENTDATAGRAMRECEIVER component) then you need to know, in which window the user currently works - so that your logic on server side can relate the event to the proper processing.

Please pay attention - the checkbox "My Window is active" seems to be "one step behin" always. But: the information about the window focus is passed always "with the next roundtrip" - so everything is fine!

This is the Java code:

package workplace;

import java.io.Serializable;

import javax.faces.event.ActionEvent;

import org.eclnt.editor.annotations.CCGenClass;
import org.eclnt.jsfserver.defaultscreens.ModelessPopup;
import org.eclnt.jsfserver.defaultscreens.Statusbar;
import org.eclnt.jsfserver.elements.util.DefaultModelessPopupListener;
import org.eclnt.util.log.CLog;
import org.eclnt.workplace.IWorkpageDispatcher;

@CCGenClass (expressionBase="#{d.DemoWindowFocusInfo}")

public class DemoWindowFocusInfo
    extends PageBean 
    implements Serializable
{
    ModelessPopup m_modelessPopup;
    
    protected String m_windowSelected;
    protected boolean m_windowPopupSelected;
    
    public DemoWindowFocusInfo(IWorkpageDispatcher workpageDispatcher)
    {
        super(workpageDispatcher);        
    }
    
    public String getWindowSelected() { return m_windowSelected; }
    public void setWindowSelected(String value) 
    {
        CLog.L.log(CLog.LL_INF,"##### setWindowSelected: " + value);
        m_windowSelected = value; 
    }

    public boolean getWindowPopupSelected() { return m_windowPopupSelected; }
    public void setWindowPopupSelected(boolean value) 
    { 
        CLog.L.log(CLog.LL_INF,"##### setWindowPopupSelected: " + value);
        m_windowPopupSelected = value; 
    }

    public void onRefresh(ActionEvent event) 
    {
        Statusbar.outputMessage("Refresh was executed. " + System.currentTimeMillis());
    }
    
    public void onOpenPopup(ActionEvent event)
    {
        if (m_modelessPopup != null)
        {
            m_modelessPopup.requestFocus();
            return;
        }
        m_modelessPopup = getOwningDispatcher().createModelessPopup();
        m_modelessPopup.open
        (
                "/workplace/demowindowfocusinfopopup.jsp",
                "Hello!",
                800,
                600,
                new DefaultModelessPopupListener(m_modelessPopup)
                {
                    @Override
                    public void reactOnPopupClosedByUser()
                    {
                        super.reactOnPopupClosedByUser();
                        m_modelessPopup = null;
                    }
                }
        );
    }

}
