The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • General
  • Arranging Content
  • Page Beans
  • Page Bean Component

This demo shows the usage of a page bean component.

A page bean component is a special form of a page bean. All information that is required by the page bean at runtime is contained in the JAR file of the bean - there are no dependencies to other resource (e.g. JSP-files).

A page bean can be configured by defining its class name and by defining the initialization parameters.

This is the XML layout definition:

<?xml version="1.0" encoding="utf-8"?>
<t:layoutdefinition xmlns:t="http://www.CaptainCasa.com/controllibrary/t" xmlns:tpbc="http://www.CaptainCasa.com/controllibrary/tpbc" >
<t:rowdemobodypane id="g_1" objectbinding="#{d.DemoPageBeanComponent}" rowdistance="#{ccstylevalue['@defaultrowdistance@']}" >
<t:row id="g_2" >
<t:pagebeancomponent id="g_3" pagebeanbinding="#{d.DemoPageBeanComponent.multiText}" pagebeanclass="democontrols.DemoPBCMultiText" pagebeaninitdata="height:300;width:300" />
</t:row>
</t:rowdemobodypane>
<t:pageaddons id="g_pa"/>
</t:layoutdefinition>

This is the Java code:

package workplace;

import java.io.Serializable;
import java.util.ArrayList;
import java.util.List;

import org.eclnt.editor.annotations.CCGenClass;
import org.eclnt.jsfserver.defaultscreens.Statusbar;
import org.eclnt.jsfserver.util.pagebeancomponents.ComplexValueEditor;
import org.eclnt.workplace.IWorkpageDispatcher;
import org.eclnt.workplace.WorkpageDispatchedPageBean;

import democontrols.DemoPBCMultiText;

import javax.faces.event.ActionEvent;

@CCGenClass (expressionBase="#{d.DemoPageBeanComponent}")

public class DemoPageBeanComponent
    extends PageBean 
    implements Serializable
{
    // ------------------------------------------------------------------------
    // members
    // ------------------------------------------------------------------------

    DemoPBCMultiText m_multiText = new DemoPBCMultiText();
    
    // ------------------------------------------------------------------------
    // constructors & initialization
    // ------------------------------------------------------------------------

    public DemoPageBeanComponent(IWorkpageDispatcher workpageDispatcher)
    {
        super(workpageDispatcher);
        List<String> texts = new ArrayList<String>();
        texts.add("This is some existing text which is already avaiable.");
        m_multiText.prepare(texts,new DemoPBCMultiText.IListener()
        {
            public void reactOnUpdate()
            {
            }
        });
    }

    public String getPageName() { return "/workplace/demopagebeancomponent.jsp"; }
    public String getRootExpressionUsedInPage() { return "#{d.DemoPageBeanComponent}"; }

    // ------------------------------------------------------------------------
    // public usage
    // ------------------------------------------------------------------------

    public DemoPBCMultiText getMultiText() { return m_multiText; }
    
    // ------------------------------------------------------------------------
    // private usage
    // ------------------------------------------------------------------------
}
