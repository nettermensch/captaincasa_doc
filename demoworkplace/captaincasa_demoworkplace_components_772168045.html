The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • Components
  • Straight Controls
  • Value Representation
  • Slider - Flush timer

The SLIDER component provides a FLUSHTIMER attribute by which you can define when flush events (value change events) are sent to the server side.

Without defining a FLUSHTIMER, the flush event is sent when releasing the mouse button. With defining a FLUSHTIMER the flush event already is sent if the user stops moving the mouse for a certain duration.
