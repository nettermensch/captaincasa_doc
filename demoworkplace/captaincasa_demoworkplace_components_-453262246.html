The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • Components
  • Straight Controls
  • Field Control
  • Image in front of text
  • Text image with action

All field components (FIELD, COMBOFIELD, COMBOBOX, SPINNER, ...) now provide a new attribute WITHTEXTIMAGEINVOKE. If set to true then the user may click onto the TEXTIMAGE and a corresponding action listener will be called on server side.

The event on server side is of typ "BaseActionEventTextImageInvoke".

This is the XML layout definition:

<?xml version="1.0" encoding="utf-8"?>
<t:layoutdefinition xmlns:t="http://www.captaincasa.com/controllibrary/t" >
<t:beanprocessing id="g_1" beanbinding="#{d.DemoFieldTextImageAction}" />
<t:rowdemobodypane id="g_4" objectbinding="#{d.DemoFieldTextImageAction}" rowdistance="#{ccstylevalue[&#39;@defaultrowdistance@&#39;]}" >
<t:row id="g_8" >
<t:textpane id="g_2" text="Click onto the plus icon - a corresponding action will be invoked on server side:" width="100%" />
</t:row>
<t:row id="g_6" coldistance="#{ccstylevalue[&#39;@defaultcoldistance@&#39;]}" >
<t:field id="g_10" actionListener="#{d.DemoFieldTextImageAction.onField1Action}" labeltext="Field 1" text="#{d.DemoFieldTextImageAction.field1}" textimage="/images.iconssvg.add_16x16.@iconDark@.16x16.ccsvg" width="200" withtextimageinvoke="true" />
</t:row>
</t:rowdemobodypane>
</t:layoutdefinition>

This is the Java code:

package workplace;

import java.io.Serializable;

import org.eclnt.editor.annotations.CCGenClass;
import org.eclnt.jsfserver.defaultscreens.OKPopup;
import org.eclnt.jsfserver.elements.events.BaseActionEventTextImageInvoke;
import org.eclnt.workplace.IWorkpageDispatcher;

@CCGenClass (expressionBase="#{d.DemoFieldTextImageAction}")

public class DemoFieldTextImageAction
    extends WorkpageDispatchedPageBean 
    implements Serializable
{
    String m_field1;

    public DemoFieldTextImageAction(IWorkpageDispatcher workpageDispatcher)
    {
        super(workpageDispatcher);        
    }

    public String getPageName() { return "/workplace/demofieldtextimageaction.xml"; }
    public String getRootExpressionUsedInPage() { return "#{d.DemoFieldTextImageAction}"; }

    public String getField1() { return m_field1; }
    public void setField1(String value) { this.m_field1 = value; }


    public void onField1Action(javax.faces.event.ActionEvent event) 
    {
        if (event instanceof BaseActionEventTextImageInvoke)
        {
            OKPopup.createInstance("","Text image invoked on field1!");
        }
    }
}
