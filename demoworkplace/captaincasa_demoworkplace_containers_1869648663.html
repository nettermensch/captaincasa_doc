The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • Containers
  • Scale Pane
  • Scale automatically set

Resize the green part - the content will be automatically scaled to fit inside.

By setting SCALEPANE-AUTOSCALE to "true" the content will be scaled automatically to fit into the available screen space.

package workplace;

import org.eclnt.editor.annotations.CCGenClass;
import org.eclnt.workplace.IWorkpageDispatcher;

@CCGenClass (expressionBase="#{d.DemoScaleAuto}")

public class DemoScaleAuto
    extends PageBean 
    implements Serializable
{
    public DemoScaleAuto(IWorkpageDispatcher workpageDispatcher)
    {
        super(workpageDispatcher);        
    }

    public String getPageName() { return "/workplace/demoscaleauto.jsp"; }
    public String getRootExpressionUsedInPage() { return "#{d.DemoScaleAuto}"; }
}
