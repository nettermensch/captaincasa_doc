The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • Components
  • Straight Controls
  • Label Control
  • Data Type Formatting

You can pass data format information into the LABEL - using attributes FORMAT, FORMATMASK and TIMEZONE. The attributes are the same ones that you also use within the FORMATTEDFIELD definition.

Please note: the TIMEZONE is inevitably important when working with FORMAT values "time", "date", "datetime"!
