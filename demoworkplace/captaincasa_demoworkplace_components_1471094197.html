The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • Components
  • Animation & Graphics & Sound
  • 2D transformations (rotating, skewing)

There is a new attribute TRANSFORM which is available for many controls and which allows to apply 2D transformations.

The transformations are directly passed into the CSS-transform-style-attribute. Typical values are "rotate(...deg)", "skewX(...deg)", "skewY(...deg)", skew(...deg,...deg)".

The control itself is positioned at its "normal" position and then is transformed accordingly. Sometimes you have to make sure that the physical screen space around the transformed control is sufficient. In this case you may position a PANE with some PADDING around.

This is the Java code:

package workplace;

import java.io.Serializable;

import org.eclnt.editor.annotations.CCGenClass;
import org.eclnt.workplace.IWorkpageDispatcher;

@CCGenClass (expressionBase="#{d.DemoTransform}")

public class DemoTransform
    extends PageBean 
    implements Serializable
{
    public DemoTransform(IWorkpageDispatcher workpageDispatcher)
    {
        super(workpageDispatcher);        
    }

    public String getPageName() { return "/workplace/demotransform.xml"; }
    public String getRootExpressionUsedInPage() { return "#{d.DemoTransform}"; }
}
