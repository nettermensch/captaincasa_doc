The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • Components
  • Straight Controls
  • Value Representation
  • Progress Bar

The progress bar renders a percentage value - passed via the VALUE attribute. The percentage value on the one hand defines the size of the progress area, on the other hand it is by default as string on top of the bar.

If defining in addition a text via the TEXT attribute, then the text is rendered on top of the bar.

Parameters like foreground, background color, font, border, etc. are available. - You (unfortunately) cannot change the color of the text...

This is the Java code:

package workplace;

import java.io.Serializable;
import org.eclnt.editor.annotations.CCGenClass;
import org.eclnt.workplace.IWorkpageDispatcher;

@CCGenClass (expressionBase="#{d.DemoProgressbar}")

public class DemoProgressbar
    extends PageBean
    implements Serializable
{
    public DemoProgressbar(IWorkpageDispatcher dispatcher)
    {
        super(dispatcher);
        initRefComponent("progressbar");
    }
    
    protected int m_progressValue = 25;
    public int getProgressValue() { return m_progressValue; }
    public void setProgressValue(int value) { m_progressValue = value; }
}
