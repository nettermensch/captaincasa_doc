The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • General
  • Client Integration
  • Hyperlink

The HYPERLINK components is a link, opening a certain URL within the browser.

Core parameters are:

(.) the TEXT to be shown
(.) the URL to be opened when the user clicks the text

This is the XML layout definition:

<?xml version="1.0" encoding="utf-8"?>
<t:layoutdefinition xmlns:t="http://www.CaptainCasa.com/controllibrary/t" xmlns:tpbc="http://www.CaptainCasa.com/controllibrary/tpbc" >
<t:rowdemobodypane id="g_1" rowdistance="#{ccstylevalue['@defaultrowdistance@']}" objectbinding="#{d.DemoHyperLink}">
<t:row id="g_2" >
<t:hyperlink id="g_3" font="size:16" text="CaptainCasa Web Site" url="http://www.CaptainCasa.com" />
</t:row>
<t:row id="g_4" >
<t:hyperlink id="g_5" text="http://www.google.com" url="http://www.google.com" />
</t:row>
</t:rowdemobodypane>
<t:pageaddons id="g_pa"/>
</t:layoutdefinition>
