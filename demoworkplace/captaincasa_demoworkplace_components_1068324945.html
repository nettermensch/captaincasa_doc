The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • Components
  • Straight Controls
  • Text Input/Output
  • Code / XML Editor
  • Code editor with flushing

The FLUSH of the code editor is automatically associated with a FLUSHTIMER function: if the user updates the editor content then the editor is observed for subsequent user changes within the duration defined by FLUSHTIMER.

If there are no changes then the value is automatically transferred to the server side - together with some flush event.

The default flush duration is 2000ms.

This is the XML layout definition:

<?xml version="1.0" encoding="utf-8"?>
<t:layoutdefinition xmlns:t="http://www.CaptainCasa.com/controllibrary/t" xmlns:tpbc="http://www.CaptainCasa.com/controllibrary/tpbc" >
<t:beanprocessing id="g_1" beanbinding="#{d.DemoCodeEditorFlush}" />
<t:rowdemobodypane id="g_2" objectbinding="#{d.DemoCodeEditorFlush}" rowdistance="20" >
<t:row id="g_8" >
<t:button id="g_9" text="Button" />
</t:row>
<t:row id="g_3" >
<t:codeeditor id="g_4" actionListener="#{d.DemoCodeEditorFlush.onCodeEditorAction}" codetype="xml" flush="true" flushtimer="10000" height="100%" text="#{d.DemoCodeEditorFlush.text}" width="100%" />
</t:row>
<t:row id="g_5" >
<t:textarea id="g_6" enabled="false" height="200" text="#{d.DemoCodeEditorFlush.text}" width="100%" />
</t:row>
</t:rowdemobodypane>
<t:pageaddons id="g_pa"/>
</t:layoutdefinition>

This is the Java code:

package workplace;

import java.io.Serializable;

import org.eclnt.editor.annotations.CCGenClass;
import org.eclnt.jsfserver.defaultscreens.Statusbar;
import org.eclnt.workplace.IWorkpageDispatcher;

@CCGenClass (expressionBase="#{d.DemoCodeEditorFlush}")

public class DemoCodeEditorFlush
    extends PageBean 
    implements Serializable
{
    // ------------------------------------------------------------------------
    // members
    // ------------------------------------------------------------------------

    String m_text = 
            "<test>\n"
            + "    <companyName>CaptainCasa</companyName>\n" 
            + "</test>";

    // ------------------------------------------------------------------------
    // constructors & initialization
    // ------------------------------------------------------------------------

    public DemoCodeEditorFlush(IWorkpageDispatcher workpageDispatcher)
    {
        super(workpageDispatcher);        
    }

    public String getPageName() { return "/workplace/democodeeditorflush.jsp"; }
    public String getRootExpressionUsedInPage() { return "#{d.DemoCodeEditorFlush}"; }

    // ------------------------------------------------------------------------
    // public usage
    // ------------------------------------------------------------------------

    public String getText() { return m_text; }
    public void setText(String value) { this.m_text = value; }

    public void onCodeEditorAction(javax.faces.event.ActionEvent event) 
    {
        Statusbar.outputSuccess("Editor action: " + m_text.length() + ", " + System.currentTimeMillis());
    }

}
