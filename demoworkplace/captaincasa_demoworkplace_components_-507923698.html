The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • Components
  • Straight Controls
  • Check Box & Radio Button
  • Radio Button Group

The RADIOBUTTONGROUP binds to a ValidValuesBinding instance on server side - in the same way as a COMBOBOX.

Per value an individual RADIOBUTTON is rendered. You can select between a horizontal and a vertical arrangement.

This is the XML layout definition:

This is the Java code:

package workplace;

import java.io.Serializable;

import org.eclnt.editor.annotations.CCGenClass;
import org.eclnt.jsfserver.elements.util.ValidValuesBinding;
import org.eclnt.workplace.IWorkpageDispatcher;

@CCGenClass (expressionBase="#{d.DemoRadioButtonGroup}")

public class DemoRadioButtonGroup
    extends PageBean 
    implements Serializable
{
    // ------------------------------------------------------------------------
    // members
    // ------------------------------------------------------------------------

    ValidValuesBinding m_departmentVVS = new ValidValuesBinding();
    String m_department = "0002";

    // ------------------------------------------------------------------------
    // constructors & initialization
    // ------------------------------------------------------------------------

    public DemoRadioButtonGroup(IWorkpageDispatcher workpageDispatcher)
    {
        super(workpageDispatcher);        
        m_departmentVVS.addValidValue("0001","Sales");
        m_departmentVVS.addValidValue("0002","Marketing");
        m_departmentVVS.addValidValue("0003","Production");
    }

    public String getPageName() { return "/workplace/demoradiobuttongroup.jsp"; }
    public String getRootExpressionUsedInPage() { return "#{d.DemoRadioButtonGroup}"; }

    // ------------------------------------------------------------------------
    // public usage
    // ------------------------------------------------------------------------

    public ValidValuesBinding getDepartmentVVS() { return m_departmentVVS; }

    public String getDepartment() { return m_department; }
    public void setDepartment(String value) { this.m_department = value; }
}
