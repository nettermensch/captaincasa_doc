The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • General
  • Arranging Content
  • Includes
  • Simple page navigation

This example demonstrates a simple page navigation: when double-clicking into the list then the corresponding item is opened in a subsequent screen.

Internally there is...
(.) an outer page, including other pages by using ROWPAGEBANINCLUDE
(.) two inner pages, one for the list, the other for the detail screen

The navigation is done through the outer page.

The full sources of all 3 pages and beans classes are available within the /tomcat/webapps/demos folder:
(.) /workplace/demonaviouter.jsp and /workplacesrc/DemoNaviOuter.java
(.) /workplace/demonavilist.jsp and /workplacesrc/DemoNaviList.java
(.) /workplace/demonavidetail.jsp and /workplacesrc/DemoNaviDetail.java

This is the XML layout definition:

<?xml version="1.0" encoding="utf-8"?>
<t:layoutdefinition xmlns:t="http://www.CaptainCasa.com/controllibrary/t" xmlns:tpbc="http://www.CaptainCasa.com/controllibrary/tpbc" >
<t:rowdemobodypane id="g_1" objectbinding="#{d.DemoNaviOuter}" >
<t:rowpagebeaninclude id="g_2" pagebeanbinding="#{d.DemoNaviOuter.contentBean}" />
</t:rowdemobodypane>
<t:pageaddons id="g_pa"/>
</t:layoutdefinition>

This is the Java code:

package workplace;

import java.io.Serializable;

import org.eclnt.editor.annotations.CCGenClass;
import org.eclnt.jsfserver.pagebean.IPageBean;
import org.eclnt.workplace.IWorkpageDispatcher;

import workplace.DemoNaviList.Person;

@CCGenClass (expressionBase="#{d.DemoNaviOuter}")

public class DemoNaviOuter
    extends WorkpageDispatchedPageBean 
    implements Serializable
{
    IPageBean m_contentBean;
    
    DemoNaviList m_naviList;

    public DemoNaviOuter(IWorkpageDispatcher workpageDispatcher)
    {
        super(workpageDispatcher);        
        m_naviList = new DemoNaviList(getOwningDispatcher());
        m_naviList.prepare(new DemoNaviList.IListener()
        {
            public void reactOnSelection(Person p)
            {
                switchToPerson(p);
            }
        });
        m_contentBean = m_naviList;
    }

    public IPageBean getContentBean() { return m_contentBean; }
    
    public void switchToList() { m_contentBean = m_naviList; }
    
    public void switchToPerson(Person p)
    {
        DemoNaviDetail detail = new DemoNaviDetail(getOwningDispatcher());
        detail.prepare(new DemoNaviDetail.IListener()
                       {
                           public void reactOnBackToList()
                           {
                               switchToList();
                           }
                       },
                       p);
        m_contentBean = detail;
    }

}
