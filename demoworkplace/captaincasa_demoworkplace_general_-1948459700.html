The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • General
  • General Issues
  • Online Help

Online help is available for all input components: if the user pressed the F1 key within a focussed component then corresponding help shwows up.

Each component can be associated with the online help by defining the HELPID attribute within the component. Behind the HELPID value there is a resolution of a text, which can either be a plain text, an HTML text or an RTF text. Because of using the Swing brwoser component by default, only limited formatting elements are applicable when using HTML or RTF as content type.
