The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • Grids & Trees
  • Grids and Trees
  • Coloring grid lines

You now can directly define the background of one row by using the FIXGRID attirbutes ROWBACKGROUND and ROWBGPAINT.

You can define:
1. a fix value ("#F0F0F0")
2. a "normal" expression addressing the page bean ("#{...}")
3. an item expression addressing the item (".{...}")

Of course you can still define some own background color for the individual cell components.

Internally the corresponding value is applied to each individual cell of the row and is added to the background color settings of the cell component.
This means: if e.g. the field brings in some own background (by default: white), then the additional background is drawn on top.

Consequence: you typically define some background color with transparency, so that the cell component's background can still be visible.

This is the XML layout definition:

<?xml version="1.0" encoding="utf-8"?>
<t:layoutdefinition xmlns:t="http://www.captaincasa.com/controllibrary/t" >
<t:beanprocessing id="g_1" beanbinding="#{d.DemoGridRowbgpaint}" />
<t:rowdemobodypane id="g_4" objectbinding="#{d.DemoGridRowbgpaint}" >
<t:row id="g_6" >
<t:fixgrid id="g_8" height="100%" objectbinding="#{d.DemoGridRowbgpaint.grid}" rowbackground=".{rowBackground}" width="100%" >
<t:gridcol id="g_9" text="Some name" width="100%;100" >
<t:label id="g_5" text=".{name}" />
</t:gridcol>
<t:gridcol id="g_11" text="Some street" width="100%;100" >
<t:field id="g_12" text=".{street}" width="100" />
</t:gridcol>
<t:gridcol id="g_14" text="Some town" width="100%;100" >
<t:label id="g_15" background="#00008010" text=".{town}" />
</t:gridcol>
</t:fixgrid>
</t:row>
</t:rowdemobodypane>
</t:layoutdefinition>

This is the Java code:

package workplace;

import java.io.Serializable;

import org.eclnt.editor.annotations.CCGenClass;
import org.eclnt.jsfserver.elements.impl.FIXGRIDItem;
import org.eclnt.jsfserver.elements.impl.FIXGRIDListBinding;
import org.eclnt.workplace.IWorkpageDispatcher;

@CCGenClass (expressionBase="#{d.DemoGridRowbgpaint}")

public class DemoGridRowbgpaint
    extends PageBean 
    implements Serializable
{
    // ------------------------------------------------------------------------
    // inner classes
    // ------------------------------------------------------------------------

    public class GridItem extends FIXGRIDItem implements java.io.Serializable
    {
        String i_name;
        String i_street;
        String i_town;
        String i_rowBackground;
        public String getStreet() { return i_street; }
        public void setStreet(String value) { this.i_street = value; }
        public String getTown() { return i_town; }
        public void setTown(String value) { this.i_town = value; }
        public String getName() { return i_name; }
        public void setName(String value) { this.i_name = value; }
        public String getRowBackground() { return i_rowBackground; }
        public void setRowBackground(String rowBackground) { i_rowBackground = rowBackground; }
        public void onRowSelect()
        {
        }
        public void onRowExecute()
        {
        }
    }

    // ------------------------------------------------------------------------
    // members
    // ------------------------------------------------------------------------

    FIXGRIDListBinding<GridItem> m_grid = new FIXGRIDListBinding<GridItem>();

    // ------------------------------------------------------------------------
    // constructors & initialization
    // ------------------------------------------------------------------------

    public DemoGridRowbgpaint(IWorkpageDispatcher workpageDispatcher)
    {
        super(workpageDispatcher);        
        for (int i=0; i<100; i++)
        {
            GridItem gi = new GridItem();
            gi.setName("Name " + i);
            gi.setStreet("Street " + i);
            gi.setTown("Town " + i);
            // set bgpaint for some of the columns
            if (i%5 == 0)
                gi.setRowBackground("#a8d08d40");
            if (i%7 == 0)
                gi.setRowBackground("#8eaadb40");
            m_grid.getItems().add(gi);
        }
    }

    public String getPageName() { return "/workplace/demogridrowbgpaint.xml"; }
    public String getRootExpressionUsedInPage() { return "#{d.DemoGridRowbgpaint}"; }

    // ------------------------------------------------------------------------
    // public usage
    // ------------------------------------------------------------------------

    public FIXGRIDListBinding<GridItem> getGrid() { return m_grid; }

    // ------------------------------------------------------------------------
    // private usage
    // ------------------------------------------------------------------------
}
