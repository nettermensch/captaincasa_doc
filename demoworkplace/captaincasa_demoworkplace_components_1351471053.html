The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • Components
  • Straight Controls
  • Field Control
  • Color Chooser

The COLORFIELD allows the user to input a color value. The format of a color value is "#rrggbb" or "#rrggbbtt", with...:

(rr) representing the red value
(gg) representing the green value
(bb) representing the blue value
(tt) representing the transparency value

All values are to be defined as hexadecimal values.

Example: the color code for "red" is "#FF0000". The color code for "black" is "#000000". The color code for a black that lets the background shine through is e.g. "#00000030".

This is the Java code:

package workplace;

import java.io.Serializable;

import javax.faces.event.ActionEvent;

import org.eclnt.editor.annotations.CCGenClass;
import org.eclnt.jsfserver.util.ClientTestLog;
import org.eclnt.jsfserver.util.IClientLogAware;
import org.eclnt.workplace.IWorkpageDispatcher;

@CCGenClass (expressionBase="#{d.DemoColorField}")

public class DemoColorField
    extends PageBean
    implements Serializable, IClientLogAware
{
    public DemoColorField(IWorkpageDispatcher dispatcher)
    {
        super(dispatcher);
        initRefComponent("colorfield"); 
    }
    
    protected String m_color = null;
    public String getColor() { return m_color; }
    public void setColor(String value) { m_color = value; }
    
    public void onColorAction(ActionEvent event)
    {
    }
    
    public boolean onCreateClientTestLogMessage()
    {
        ClientTestLog.addClientTestLogMessage("Color is: " + m_color);
        return true;
    }

    
}
