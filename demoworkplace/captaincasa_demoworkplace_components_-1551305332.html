The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • Components
  • Animation & Graphics & Sound
  • Video
  • Explicit control

There are certain attributes which enable you to directly take over control over the VIDEO component:

(.) WITHCONTROLS - if set to "false" then the component will not show any inner controls, which are normally shown to play/pause a video
(.) PLAYTRIGGER/PAUSETRIGGER - if triggered then the component will react correspondingly. As result you can completely control the component by your application logic.

You may get event-notifications about the component's inner activities by defining attribute MEDIAEVENTFILTER. Here you define these event names you are interested in as semicolon separated list. The events are "ended", "play" and "pause".

If an event occurs then the ACTIONLISTENER of the component is called - passing the event via method "getMessage()".

This is the Java code:

package workplace;

import java.io.Serializable;

import org.eclnt.editor.annotations.CCGenClass;
import org.eclnt.jsfserver.elements.events.BaseActionEventMediaPlayer;
import org.eclnt.jsfserver.elements.util.Trigger;
import org.eclnt.workplace.IWorkpageDispatcher;

@CCGenClass (expressionBase="#{d.DemoVideoDetails}")

public class DemoVideoDetails
    extends PageBean 
    implements Serializable
{
    // ------------------------------------------------------------------------
    // members
    // ------------------------------------------------------------------------

    String m_eventProtocol = "";
    boolean m_withPlayerControls = true;
    Trigger m_pauseTrigger = new Trigger();
    Trigger m_playTrigger = new Trigger();

    // ------------------------------------------------------------------------
    // constructors & initialization
    // ------------------------------------------------------------------------

    public DemoVideoDetails(IWorkpageDispatcher workpageDispatcher)
    {
        super(workpageDispatcher);        
    }

    public String getPageName() { return "/workplace/demovideodetails.jsp"; }
    public String getRootExpressionUsedInPage() { return "#{d.DemoVideoDetails}"; }

    // ------------------------------------------------------------------------
    // public usage
    // ------------------------------------------------------------------------

    public void onVideoAction(javax.faces.event.ActionEvent event) 
    {
        if (event instanceof BaseActionEventMediaPlayer)
        {
            BaseActionEventMediaPlayer e = (BaseActionEventMediaPlayer)event;
            m_eventProtocol = e.getMessage() + "\n" + m_eventProtocol;
        }
    }

    public String getEventProtocol() { return m_eventProtocol; }

    public boolean getWithPlayerControls() { return m_withPlayerControls; }
    public void setWithPlayerControls(boolean value) { this.m_withPlayerControls = value; }

    public void onPauseAction(javax.faces.event.ActionEvent event) { m_pauseTrigger.trigger(); }
    public void onPlayAction(javax.faces.event.ActionEvent event) { m_playTrigger.trigger(); }

    public Trigger getPauseTrigger() { return m_pauseTrigger; }

    public Trigger getPlayTrigger() { return m_playTrigger; }

    // ------------------------------------------------------------------------
    // private usage
    // ------------------------------------------------------------------------
}
