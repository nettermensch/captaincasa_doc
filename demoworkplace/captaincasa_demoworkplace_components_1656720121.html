The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • Components
  • Straight Controls
  • Drag & Drop
  • Drop Zones

When dragging the label over the various drop zones you will recognize that there is a certain "zone painting" within the drop zones.

The information about the type of drop zones is passed as aprt of the DROPRECEIVE attribute: per drop-receive-type you can add the drop zone information ("dropReceiveType:dropZoneInfo"). In case of multiple drop-receive-types you just concatenate the information ("dropReceiveType:dropZoneInfo;dropReceiveType:dropZoneInfo").

The "drop zones" are just a certain painting that is done on client side. It is your task to interpret the drop position on server side when a drop is done by the user. To do so you receive the exact drop position as part of the event information on server side.

On server side there are helper functions within the BaseActionEventDrop event (methods "checkIfDroppedInZone...") thay you may use for transferringt a concrete drop position into a drop zone.

The drop zones are defined in the following way (left,top,width,height):

(.) "horizontalsplit": left (0,0,50%,100%)/ right (50%,0,50%,100%)
(.) "verticalsplit": top (0,0,100%,50%)/ bottom (0,50%,100%,50%)
(.) "edges": left-top (0,0,50%,50%)/ left-bottom (0,50%,50%,50%)/ right-top (50%,0,50%,50%)/ right-bottom (50%,50%,50%,50%)
(.) "sides": top (0,0,100%,25%)/ bottom (0,75%,100%,25%) /left (0,25%,25%,50%) /right (50%,25%,50%,50%)
(.) "sidesandcenter": top (0,0,100%,25%)/ bottom (0,75%,100%,25%) /left (0,25%,25%,50%) /right (75%,25%,25%,50%) /center (25%,25%,50%,50%)

This is the Java code:

package workplace;

import javax.faces.event.ActionEvent;

import org.eclnt.jsfserver.defaultscreens.Statusbar;
import org.eclnt.jsfserver.elements.events.BaseActionEventDrop;
import org.eclnt.workplace.IWorkpageDispatcher;

public class DemoDropZones extends PageBean
{
    public DemoDropZones(IWorkpageDispatcher dispatcher)
    {
        super(dispatcher);
    }

    public void onProcessDrop(ActionEvent event)
    {
        if (event instanceof BaseActionEventDrop)
        {
            BaseActionEventDrop baed = (BaseActionEventDrop)event;
            Statusbar.outputSuccess("Drop processed: " + 
                                    baed.getDragInfo() + " // " + 
                                    baed.getPercentageHorizontaAsFloat() + " // " +
                                    baed.getPercentageVerticalAsFloat());
        }
    }
}
