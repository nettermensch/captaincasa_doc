The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • Components
  • Straight Controls
  • Client Clock
  • Count down / up

You may set attribute STARTAT to a certain value (in milliseconds). The clock will then count relative to this value.

By setting STARTATCOUNTDOWN to "true" the clock will count downwards from the value of STARTAT on.

By triggering a reset via attribute RESETTRIGGER the clock will switch back to the value of STARTAT.

This is the Java code:

package workplace;

import java.io.Serializable;

import org.eclnt.editor.annotations.CCGenClass;
import org.eclnt.jsfserver.elements.util.Trigger;
import org.eclnt.workplace.IWorkpageDispatcher;
import org.eclnt.workplace.WorkpageDispatchedPageBean;

@CCGenClass (expressionBase="#{d.DemoClockStartAt}")

public class DemoClockStartAt
    extends PageBean 
    implements Serializable
{
    public DemoClockStartAt(IWorkpageDispatcher workpageDispatcher)
    {
        super(workpageDispatcher);        
    }

    public String getPageName() { return "/workplace/democlockstartat.jsp"; }
    public String getRootExpressionUsedInPage() { return "#{d.DemoClockStartAt}"; }

    Trigger m_resetTrigger = new Trigger();
    public Trigger getResetTrigger() { return m_resetTrigger; }

    public void onResetAction(javax.faces.event.ActionEvent event) 
    {
        m_resetTrigger.trigger();
    }

}
