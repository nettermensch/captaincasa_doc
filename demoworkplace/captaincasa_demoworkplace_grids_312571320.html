The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • Grids & Trees
  • Grids and Trees
  • Grid Data Binding

The focus of this demo is the data binding between the grid columns and the server side objects.

Normally you use an expression ".{xxx}" for property binding within a component that serves as cell component. As consequence the expression "xxx" is executed from the grid item on.

You can still use the "#{...}" expression style for dynamic binding. In this case of course there is no item / row dependency anymore. In the example the "editable" control is implemented using "#{...}" expressions.

An interesting aspect in this example is the radio button implementation: the radio button maintains a central property, but the reference values are taken from the row items.

This is the XML layout definition:

This is the Java code:

package workplace;

import org.eclnt.jsfserver.elements.impl.FIXGRIDItem;
import org.eclnt.jsfserver.elements.impl.FIXGRIDListBinding;
import org.eclnt.workplace.IWorkpageDispatcher;

public class DemoGridbinding extends PageBean
{
    public class MyRow extends FIXGRIDItem
    {
        String m_firstName;
        String m_lastName;
        public void setFirstName(String value) { m_firstName = value; }
        public String getFirstName() { return m_firstName; }
        public void setLastName(String value) { m_lastName = value; }
        public String getLastName() { return m_lastName; }
        public int getIndex() { return m_rows.getItems().indexOf(this); }
    }
    
    FIXGRIDListBinding<MyRow> m_rows = new FIXGRIDListBinding<MyRow>();    
    boolean m_enabledFirstName = true;
    boolean m_enabledLastName = true;
    int m_radioButtonIndex = -1;
    
    public DemoGridbinding(IWorkpageDispatcher dispatcher)
    {
        super(dispatcher);
        for (int i=0; i<5; i++)
        {
            MyRow row = new MyRow();
            row.setFirstName("FN" + i);
            row.setLastName("LN"+i);
            m_rows.getItems().add(row);
        }
    }
    
    public FIXGRIDListBinding<MyRow> getRows() { return m_rows; }
    public void setEnabledFirstName(boolean value) { m_enabledFirstName = value; }
    public boolean getEnabledFirstName() { return m_enabledFirstName; }
    
    public void setEnabledLastName(boolean value) { m_enabledLastName = value; }
    public boolean getEnabledLastName() { return m_enabledLastName; }
    
    public void setRadioButtonIndex(int value) { m_radioButtonIndex = value; }
    public int getRadioButtonIndex() { return m_radioButtonIndex; }
    
    public String getColorFirstName()
    {
        if (getEnabledFirstName() == true)
            return "#FFFFFF";
        else
            return "#E0E0FF";
    }
    
    public String getColorLastName()
    {
        if (getEnabledLastName() == true)
            return "#FFFFFF";
        else
            return "#E0E0FF";
    }
    
}
