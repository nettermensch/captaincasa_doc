The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • Containers
  • Structuring page content
  • ...in "exposed" style

For the components BOX, FOLDABLEPANE, TABBEDPANE there is a style variant "ccexposed" which explicitly renders some border around the components - so that they are more exposed in front of the background.

This is the Java code:

package workplace;

import java.io.Serializable;

import org.eclnt.editor.annotations.CCGenClass;
import org.eclnt.workplace.IWorkpageDispatcher;
import org.eclnt.workplace.WorkpageDispatchedPageBean;

@CCGenClass (expressionBase="#{d.DemoBoxing2}")

public class DemoBoxing2
    extends WorkpageDispatchedPageBean 
    implements Serializable
{
    public DemoBoxing2(IWorkpageDispatcher workpageDispatcher)
    {
        super(workpageDispatcher);        
    }

    public String getPageName() { return "/workplace/demoboxing2.xml"; }
    public String getRootExpressionUsedInPage() { return "#{d.DemoBoxing2}"; }

}
