The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • General
  • Workplace Management
  • Start content as Popup

Enter your name into the field "Your Name" and press "Hello!" afterwards. The result field will be updated accordingly.

Compare the graphical layout with the layout definition which is shown inside the "Layout XML" area. The layout is assembled out of a set of components, each component defining its graphical layout and its relation to the backend processing.

Have a look onto the Java code of the managed bean which reflects the graphical processing on backend side. For each field you find one property definition, the button is linked with a method.

This is the XML layout definition:

This is the Java code:

package workplace;

import javax.faces.event.ActionEvent;

import org.eclnt.jsfserver.defaultscreens.Statusbar;
import org.eclnt.workplace.IWorkpageDispatcher;

public class DemoHelloWorld extends PageBean
{
    String m_name;
    String m_output;
    
    public DemoHelloWorld(IWorkpageDispatcher dispatcher)
    {
        super(dispatcher);
    }
    
    public void setName(String value) { m_name = value; }
    public String getName() { return m_name; }
    
    public String getOutput() 
    { 
        return m_output; 
    }
    
    public void onHello(ActionEvent ae)
    {
        if (m_name == null)
        {
            m_output = "No name defined.";
            Statusbar.outputError("Please define some name!");
        }
        else
        {
            m_output = "Hello World, "+m_name+"!";
        }
    }
    
}
