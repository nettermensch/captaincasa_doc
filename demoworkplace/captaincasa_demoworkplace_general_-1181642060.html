The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • General
  • Arranging Content
  • Popup Dialogs
  • Screen background color

By using method ModalPopup.setScreeBackgroundColor() you may define the background color of the shadow that is drawn behind the popup. If not defined then the color is taken from the CSS style.

This is the XML layout definition:

<?xml version="1.0" encoding="utf-8"?>
<t:layoutdefinition xmlns:t="http://www.CaptainCasa.com/controllibrary/t" xmlns:tpbc="http://www.CaptainCasa.com/controllibrary/tpbc" >
<t:rowdemobodypane id="g_1" objectbinding="#{d.DemoPopupScreenBackground}" rowdistance="#{ccstylevalue['@defaultrowdistance@']}" >
<t:row id="g_2" >
<t:button id="g_3" actionListener="#{d.DemoPopupScreenBackground.onOpenDefault}" text="Open popup - default" width="1%;200" />
</t:row>
<t:row id="g_4" >
<t:button id="g_5" actionListener="#{d.DemoPopupScreenBackground.onOpenAttention}" text="Open popup - attention!" width="1%;200" />
</t:row>
<t:row id="g_6" >
<t:button id="g_7" actionListener="#{d.DemoPopupScreenBackground.onOpenOK}" text="Open popup - OK!" width="1%;200" />
</t:row>
</t:rowdemobodypane>
<t:pageaddons id="g_pa"/>
</t:layoutdefinition>

This is the Java code:

package workplace;

import java.io.Serializable;
import org.eclnt.editor.annotations.CCGenClass;
import org.eclnt.jsfserver.defaultscreens.ModalPopup;
import org.eclnt.workplace.IWorkpageDispatcher;
import org.eclnt.workplace.WorkpageDispatchedPageBean;

import javax.faces.event.ActionEvent;

@CCGenClass (expressionBase="#{d.DemoPopupScreenBackground}")

public class DemoPopupScreenBackground
    extends PageBean
    implements Serializable
{
    // ------------------------------------------------------------------------
    // constructors & initialization
    // ------------------------------------------------------------------------

    public DemoPopupScreenBackground(IWorkpageDispatcher workpageDispatcher)
    {
        super(workpageDispatcher);        
    }

    public String getPageName() { return "/workplace/demopopupscreenbackground.jsp"; }
    public String getRootExpressionUsedInPage() { return "#{d.DemoPopupScreenBackground}"; }

    // ------------------------------------------------------------------------
    // public usage
    // ------------------------------------------------------------------------

    public void onOpenOK(javax.faces.event.ActionEvent event) 
    {
        openPopup("#00800040");
    }

    public void onOpenAttention(javax.faces.event.ActionEvent event) 
    {
        openPopup("#80000040");
    }

    public void onOpenDefault(javax.faces.event.ActionEvent event) 
    {
        openPopup(null);
    }

    // ------------------------------------------------------------------------
    // private usage
    // ------------------------------------------------------------------------
    
    private void openPopup(String backgroundColor)
    {
        final DemoHelloWorld ui = new DemoHelloWorld(getOwningDispatcher());
        ModalPopup mp = openModalPopup(ui,"Hello world",700,400,new ModalPopup.IModalPopupListener()
        {
            public void reactOnPopupClosedByUser()
            {
                closePopup(ui);
            }
        });
        mp.setScreenBackgroundColor(backgroundColor);
    }
}
