The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • Components
  • Straight Controls
  • Text Input/Output
  • HTML Editor
  • HTML Editor - with flushing

Key in some text into the HTML-editor - after some time of inactivity the text will automatically be flushed to the server side processing.

The SIMPLEHTMLEDITOR component also provides some - limited - functions to flush its content on change.

By defining FLUSH to "true" and by assigning a number value to FLUSHTIMER the component will flush back changes within the text content after some duration of user inactivity. The duration is defined by the FLUSHTIMER value.

This is the Java code:

package workplace;

import java.io.Serializable;
import org.eclnt.editor.annotations.CCGenClass;
import org.eclnt.jsfserver.defaultscreens.Statusbar;
import org.eclnt.workplace.IWorkpageDispatcher;
import org.eclnt.workplace.WorkpageDispatchedPageBean;

import javax.faces.event.ActionEvent;

@CCGenClass (expressionBase="#{d.DemoSimpleHtmlEditorFlush}")

public class DemoSimpleHtmlEditorFlush
    extends PageBean 
    implements Serializable
{
    // ------------------------------------------------------------------------
    // members
    // ------------------------------------------------------------------------

    String m_htmlText = "Some <b>HTML Text</b>.";

    // ------------------------------------------------------------------------
    // constructors & initialization
    // ------------------------------------------------------------------------

    public DemoSimpleHtmlEditorFlush(IWorkpageDispatcher workpageDispatcher)
    {
        super(workpageDispatcher);        
    }

    public String getPageName() { return "/workplace/demosimplehtmleditorflush.jsp"; }
    public String getRootExpressionUsedInPage() { return "#{d.DemoSimpleHtmlEditorFlush}"; }

    public void onEditorAction(javax.faces.event.ActionEvent event) 
    {
        Statusbar.outputMessage("Editor content was updated. ("+System.currentTimeMillis()+")");
    }

    // ------------------------------------------------------------------------
    // public usage
    // ------------------------------------------------------------------------

    public String getHtmlText() { return m_htmlText; }
    public void setHtmlText(String value) { this.m_htmlText = value; }

}
