The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • Containers
  • Adaptive Containers - Controls
  • Matrix area

The ROWMATRIXAREA is an adaptive container, arranging its content components within one or several rows.

Each contained component is measured - the one with the highest minimum width defines the sizing of the matrix, so that each column of the matrix has the same width. All the matrix components that are part of the same row receive the same height.
