The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • General
  • RISC Client Integration
  • Insert HTML

This client is only usable within the RISC Client.

The HTML that is passed by the TEXT attribute is directly inserted into the rectangle area that is defined by the component's WIDTH and HEIGHT definition.

The sizing of the component is independent from the content that you pass!

PLEASE NOT - AND PLEASE PAY ATTENTION:
This component by intention directly moves the HTML into the corresponding area. You are responsible for checking the HTML for sanity (e.g. that it does not contain embedded scripting). Only use this component if you are 100% sure that the HTML that is passed is coming from some trusted source.

This component must be used with extreme care from security point of view!
