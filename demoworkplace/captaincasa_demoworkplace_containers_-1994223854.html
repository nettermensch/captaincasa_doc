The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • Containers
  • Background painting
  • BGPAINT command with conditions

All BGPAINT commands can be combined with conditions:

The syntax is:

<command><condition>

The command is only executed if the condition is fulfilled.

There are the following conditions:

1. _focus
2. _nofocus
3. _empty
4. _enabled
5. _disabled

Please note: the BGPAINT commands should only be used for special background coloring within the component. They should not be used for general styling issues!

This is the Java code:

package workplace;

import java.io.Serializable;
import org.eclnt.editor.annotations.CCGenClass;
import org.eclnt.workplace.IWorkpageDispatcher;
import org.eclnt.workplace.WorkpageDispatchedPageBean;

import javax.faces.event.ActionEvent;

@CCGenClass (expressionBase="#{d.DemoBgpaintConditions}")

public class DemoBgpaintConditions
    extends PageBean 
    implements Serializable
{
    // ------------------------------------------------------------------------
    // members
    // ------------------------------------------------------------------------

    boolean m_fieldEnabled = true;
    
    // ------------------------------------------------------------------------
    // constructors & initialization
    // ------------------------------------------------------------------------

    public DemoBgpaintConditions(IWorkpageDispatcher workpageDispatcher)
    {
        super(workpageDispatcher);        
    }

    public String getPageName() { return "/workplace/demobgpaintconditions.jsp"; }
    public String getRootExpressionUsedInPage() { return "#{d.DemoBgpaintConditions}"; }

    // ------------------------------------------------------------------------
    // public usage
    // ------------------------------------------------------------------------

    public boolean getFieldEnabled() { return m_fieldEnabled; }
    public void setFieldEnabled(boolean value) { this.m_fieldEnabled = value; }

    public void onToggleEnabledAction(javax.faces.event.ActionEvent event) 
    {
        m_fieldEnabled = !m_fieldEnabled;
    }

    // ------------------------------------------------------------------------
    // private usage
    // ------------------------------------------------------------------------
}
