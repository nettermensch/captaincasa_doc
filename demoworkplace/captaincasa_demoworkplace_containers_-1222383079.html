The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • Containers
  • Grid layout pane
  • Simple examples

The GRIDLAYOUTPANE is a layout component that internally opens up a matrix of areas.

The matrix is defined by the content of its GRIDLAYOUTROWS. Each element of a row is places into this matrix according to its position in the row and according to its definition of COLSPAN AND ROWSPAN.

The sizing of the matrix is automatically calculated so that the minimum size and the preferred size of each component is respected.

package workplace;

import java.io.Serializable;

import org.eclnt.editor.annotations.CCGenClass;
import org.eclnt.workplace.IWorkpageDispatcher;

@CCGenClass (expressionBase="#{d.DemoGLPSimple}")

public class DemoGLPSimple
    extends WorkpageDispatchedPageBean 
    implements Serializable
{
    public DemoGLPSimple(IWorkpageDispatcher workpageDispatcher)
    {
        super(workpageDispatcher);        
    }

    public String getPageName() { return "/workplace/demoglpsimple.jsp"; }
    public String getRootExpressionUsedInPage() { return "#{d.DemoGLPSimple}"; }
}
