The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • Containers
  • Adaptive Containers - Controls
  • Adaptive matrix area
  • Occupying more space than matrix width

In this demo some of the components that are contained in the ROWSPANMATRIXAREA exceed the definition of the MATRIXWIDTH value.

Consequence: they occupy this number of matrix columns that they require. In the example 2 matrix columns are occupied by the "wide" columns.

This is the Java code:

package workplace;

import java.io.Serializable;

import org.eclnt.editor.annotations.CCGenClass;
import org.eclnt.workplace.IWorkpageDispatcher;

@CCGenClass (expressionBase="#{d.DemoAdaptiveSpans01}")

public class DemoAdaptiveSpans02
    extends WorkpageDispatchedPageBean 
    implements Serializable
{
    // ------------------------------------------------------------------------
    // inner classes
    // ------------------------------------------------------------------------

    // ------------------------------------------------------------------------
    // members
    // ------------------------------------------------------------------------

    int m_scaleInt = 60;

    // ------------------------------------------------------------------------
    // constructors & initialization
    // ------------------------------------------------------------------------

    public DemoAdaptiveSpans02(IWorkpageDispatcher workpageDispatcher)
    {
        super(workpageDispatcher);        
    }

    public String getPageName() { return "/workplace/demoadaptivespans01.xml"; }
    public String getRootExpressionUsedInPage() { return "#{d.DemoAdaptiveSpans01}"; }

    // ------------------------------------------------------------------------
    // public usage
    // ------------------------------------------------------------------------

    public int getScaleInt() { return m_scaleInt; }
    public void setScaleInt(int value) { m_scaleInt = value; }
    
    public double getScaleDouble() 
    {
        if (m_scaleInt == 100) 
            return 1d;
        else
            return (double)m_scaleInt/100d;
    }

    public void onScaleIntAction(javax.faces.event.ActionEvent event) 
    {
    }

    // ------------------------------------------------------------------------
    // private usage
    // ------------------------------------------------------------------------
}
