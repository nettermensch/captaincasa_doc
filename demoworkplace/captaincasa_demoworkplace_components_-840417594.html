The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • Components
  • Straight Controls
  • Invokers
  • Links

The LINK component is a nice, lightweight component for invoking server side functions.

This is the XML layout definition:

This is the Java code:

package workplace;

import java.util.Date;

import javax.faces.event.ActionEvent;

import org.eclnt.jsfserver.defaultscreens.Statusbar;
import org.eclnt.workplace.IWorkpageDispatcher;

public class DemoLink extends PageBean
{
    public DemoLink(IWorkpageDispatcher dispatcher)
    {
        super(dispatcher);
        initRefComponent("link"); 
    }

    public void onLink(ActionEvent ae)
    {
        Statusbar.outputSuccess("Link was pressed. " + (new Date()).toString());
    }
}
