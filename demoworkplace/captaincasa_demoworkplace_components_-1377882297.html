The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • Components
  • Animation & Graphics & Sound
  • Paint Area
  • Line drawing
  • Line arrow size

By using the attribute ARROWSIZE you can define the size of both the from- and to-arrow.

This is the XML layout definition:

<?xml version="1.0" encoding="utf-8"?>
<t:layoutdefinition xmlns:t="http://www.captaincasa.com/controllibrary/t" >
<t:beanprocessing id="g_1" beanbinding="#{d.DemoPaintAreaLineArrowSize}" />
<t:rowdemobodypane id="g_4" objectbinding="#{d.DemoPaintAreaLineArrowSize}" >
<t:row id="g_6" >
<t:scrollpane id="g_3" height="100%" width="100%" >
<t:row id="g_8" >
<t:paintarea id="g_2" height="100%" width="100%" >
<t:paintarealineitem id="g_5" arrowfrom="1" arrowto="1" bounds="20;20;200;0" text="Normal arrow size" />
<t:paintarealineitem id="g_10" arrowfrom="1" arrowsize="1.25" arrowto="1" bounds="20;50;200;0" text="Arrow size 1.25" />
<t:paintarealineitem id="g_12" arrowfrom="1" arrowsize="1.5" arrowto="1" bounds="20;80;200;0" text="Arrow size 1.5" />
<t:paintarealineitem id="g_14" arrowfrom="1" arrowsize="1.75" arrowto="1" bounds="20;110;200;0" text="Arrow size 1.75" />
<t:paintarealineitem id="g_16" arrowfrom="1" arrowsize="2" arrowto="1" bounds="20;140;200;0" text="Arrow size 2" />
</t:paintarea>
</t:row>
</t:scrollpane>
</t:row>
</t:rowdemobodypane>
</t:layoutdefinition>

This is the Java code:

package workplace;

import java.io.Serializable;

import org.eclnt.editor.annotations.CCGenClass;
import org.eclnt.workplace.IWorkpageDispatcher;

@CCGenClass (expressionBase="#{d.DemoPaintAreaLineArrowSize}")

public class DemoPaintAreaLineArrowSize
    extends PageBean 
    implements Serializable
{
    public DemoPaintAreaLineArrowSize(IWorkpageDispatcher dispatcher)
    {
        super(dispatcher);
    }

    public String getPageName() { return "/workplace/demopaintarealinearrowsize.xml"; }
    public String getRootExpressionUsedInPage() { return "#{d.DemoPaintAreaLineArrowSize}"; }
}
