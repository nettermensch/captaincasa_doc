The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • Components
  • Straight Controls
  • Field Control
  • Field Value Help - Details
  • Item background, item select-ability

This demo shows some extended functions of the IdTextSelection class, that may be used to implement the dialog that is shown by a COMBOFIELD:

1. on item level you may pass an individual BGPAINT
2. on item level you may pass the information if the item is select-able (default) or not.

This is the XML layout definition:

<?xml version="1.0" encoding="utf-8"?>
<t:layoutdefinition xmlns:t="http://www.CaptainCasa.com/controllibrary/t" xmlns:tpbc="http://www.CaptainCasa.com/controllibrary/tpbc" >
<t:rowdemobodypane id="g_1" objectbinding="#{d.DemoComboFieldExt}" >
<t:row id="g_2" >
<t:combofield id="g_3" actionListener="#{d.DemoComboFieldExt.onCostCenterAction}" labeltext="Cost center" text="#{d.DemoComboFieldExt.costCenter}" width="200" />
</t:row>
</t:rowdemobodypane>
<t:pageaddons id="g_pa"/>
</t:layoutdefinition>

This is the Java code:

package workplace;

import java.io.Serializable;
import org.eclnt.editor.annotations.CCGenClass;
import org.eclnt.jsfserver.defaultscreens.ISetId;
import org.eclnt.jsfserver.defaultscreens.IdTextSelection;
import org.eclnt.jsfserver.defaultscreens.IdTextSelection.IdTextLine;
import org.eclnt.jsfserver.elements.events.BaseActionEventValueHelp;
import org.eclnt.jsfserver.pagebean.PageBean;
import org.eclnt.workplace.IWorkpageDispatcher;

import javax.faces.event.ActionEvent;

@CCGenClass (expressionBase="#{d.DemoComboFieldExt}")

public class DemoComboFieldExt
    extends PageBean 
    implements Serializable
{
    // ------------------------------------------------------------------------
    // constructors & initialization
    // ------------------------------------------------------------------------

    public DemoComboFieldExt(IWorkpageDispatcher dispatcher)
    {
        super(dispatcher);
    }
    
    // ------------------------------------------------------------------------
    // public usage
    // ------------------------------------------------------------------------

    String m_costCenter;
    public String getCostCenter() { return m_costCenter; }
    public void setCostCenter(String value) { this.m_costCenter = value; }

    public void onCostCenterAction(ActionEvent event) 
    {
        if (event instanceof BaseActionEventValueHelp)
        {
            IdTextSelection idts = IdTextSelection.createInstance();
            IdTextLine idtl;
            idtl = idts.addLine(null,"Production"); idtl.setBgpaint("background(#CCFFCC)"); idtl.setSelectable(false);
            idts.addLine("0001","Gear Systems");
            idts.addLine("0002","Engines");
            idts.addLine("0003","Chassis");
            idtl = idts.addLine(null,"Administration"); idtl.setBgpaint("background(#CCFFCC)"); idtl.setSelectable(false);
            idts.addLine("1001","Board");
            idts.addLine("1002","HR");
            idtl = idts.addLine(null,"Sales"); idtl.setBgpaint("background(#CCFFCC)"); idtl.setSelectable(false);
            idts.addLine("2001","Sales North");
            idts.addLine("2002","Sales South");
            idtl = idts.addLine(null,"Others"); idtl.setBgpaint("background(#CCFFCC)"); idtl.setSelectable(false);
            for (int i=3001; i<3050; i++)
            {
                idts.addLine(""+i,"Cost Center " + i);
            }
            idts.setCallBack(new ISetId()
            {
                public void setId(String id)
                {
                    m_costCenter = id;
                }
            });
            idts.preselect(m_costCenter);
        }
    }

    // ------------------------------------------------------------------------
    // private usage
    // ------------------------------------------------------------------------
}
