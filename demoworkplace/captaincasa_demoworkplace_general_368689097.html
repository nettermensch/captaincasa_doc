The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • General
  • Focus Management
  • Focus hot key

By using attribute REQUESTFOCUSHOTKEY you can explicitly define some keyboard combination which the user may use to directly focus a certain component.

The hot key definition is valid throughout the whole dialog.

This is the XML layout definition:

<?xml version="1.0" encoding="utf-8"?>

<t:row id="g_2" >
<t:box id="g_3" rowdistance="#{ccstylevalue['@defaultrowdistance@']}" width="100%" >
<t:row id="g_4" coldistance="5" >

This is the Java code:

package workplace;

import java.io.Serializable;

import org.eclnt.editor.annotations.CCGenClass;
import org.eclnt.workplace.IWorkpageDispatcher;

@CCGenClass (expressionBase="#{d.DemoFocusHotkey}")

public class DemoFocusHotkey
    extends PageBean
    implements Serializable
{
    public DemoFocusHotkey(IWorkpageDispatcher workpageDispatcher)
    {
        super(workpageDispatcher);
    }

    @Override
    public String getPageName() { return "/workplace/demofocushotkey.jsp"; }
    @Override
    public String getRootExpressionUsedInPage() { return "#{d.DemoFocusHotkey}"; }
}
