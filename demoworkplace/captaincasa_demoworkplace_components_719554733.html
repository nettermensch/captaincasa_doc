The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • Components
  • Straight Controls
  • Invokers
  • Various Buttons

The button can be formatted in various way - as you can see.

Please pay attention: the normal button comes with a coloring that depends on the Java look and feel that you use. There is an attribute CONTENTAREAFILLED that you need to explicitly define to be "false" prior to using BACKGROUND, BGPAINT and FOREGROUND. - The same is true for borders: a button without borders requires CONTENTAREAFILLED to be set to "false".
