The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • Grids & Trees
  • Grids and Trees
  • Large Output Lists
  • Setting of Control Type

You can fine-control the array grid by assigning an explicit control type per column.
Example: you might set the control type "text-area" for columns containing long texts, that the user might scroll.

This is the XML layout definition:

<?xml version="1.0" encoding="utf-8"?>
<t:layoutdefinition xmlns:t="http://www.CaptainCasa.com/controllibrary/t" xmlns:tpbc="http://www.CaptainCasa.com/controllibrary/tpbc" >
<t:rowdemobodypane id="g_1" objectbinding="#{d.DemoArrayGridControlTypes}" >
<t:row id="g_2" >
<t:arraygrid id="g_3" bordercolor="#{ccstylevalue.ccLightBorderColor}" borderheight="1" borderwidth="1" height="100%" objectbinding="#{d.DemoArrayGridControlTypes.grid}" rowheight="50" sbvisibleamount="15" width="100%" />
</t:row>
</t:rowdemobodypane>
<t:pageaddons id="g_pa"/>
</t:layoutdefinition>

This is the Java code:

package workplace;

import java.io.Serializable;

import org.eclnt.editor.annotations.CCGenClass;
import org.eclnt.jsfserver.elements.impl.ARRAYGRIDItem;
import org.eclnt.jsfserver.elements.impl.ARRAYGRIDListBinding;
import org.eclnt.workplace.IWorkpageDispatcher;

@CCGenClass (expressionBase="#{d.DemoArrayGridControlTypes}")

public class DemoArrayGridControlTypes
    extends PageBean
    implements Serializable
{
    ARRAYGRIDListBinding<GridItem> m_grid = new ARRAYGRIDListBinding<GridItem>();
    public ARRAYGRIDListBinding<GridItem> getGrid() { return m_grid; }

    public class GridItem extends ARRAYGRIDItem implements java.io.Serializable
    {
    }

    // ------------------------------------------------------------------------
    // constructors & initialization
    // ------------------------------------------------------------------------

    public DemoArrayGridControlTypes(IWorkpageDispatcher workpageDispatcher)
    {
        super(workpageDispatcher);
        m_grid.setTitles(new String[] {"First Name","Last Name","Comment","Image"});
        m_grid.setWidths(new String[] {"100","100","100%;100","100"});
        m_grid.setControltypes(new int[] 
        {
            ARRAYGRIDListBinding.CONTROLTYPE_DEFAULT,
            ARRAYGRIDListBinding.CONTROLTYPE_DEFAULT,
            ARRAYGRIDListBinding.CONTROLTYPE_TEXTAREA,
            ARRAYGRIDListBinding.CONTROLTYPE_IMAGE
        });
        for (int i=0; i<50; i++)
        {
            GridItem gi = new GridItem();
            String firstName = "First" + i;
            String lastName = "Last" + i;
            StringBuffer comment = new StringBuffer();
            for (int j=0; j<i%5; j++)
                comment.append("This is some comment about the person. ");
            String image = "/images/iconssvg/running_blue_16x16.svg";
            if (i%3 == 0) image = "/images/iconssvg/running_green_16x16.svg";
            if (i%4 == 0) image = null;
            gi.setValues(new String[] {firstName,lastName,comment.toString(),image});
            m_grid.getItems().add(gi);
        }
    }

    // ------------------------------------------------------------------------
}
