The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • Components
  • Straight Controls
  • Images
  • Image fill mode

When images are explicitly sized then you can define how images should fit into the available space.

The "cover" mode is especially useful for images that serve as background of a screen because it always guarantees the full area to be filled without breaking the image's ratio.

This is the Java code:

package workplace;

import java.io.Serializable;

import org.eclnt.editor.annotations.CCGenClass;
import org.eclnt.workplace.IWorkpageDispatcher;

@CCGenClass (expressionBase="#{d.DemoImageFillMode}")

public class DemoImageFillMode
    extends PageBean 
    implements Serializable
{
    public DemoImageFillMode(IWorkpageDispatcher workpageDispatcher)
    {
        super(workpageDispatcher);        
    }

    public String getPageName() { return "/workplace/demoimagefillmode.jsp"; }
    public String getRootExpressionUsedInPage() { return "#{d.DemoImageFillMode}"; }
}
