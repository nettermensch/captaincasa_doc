The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • Containers
  • Hideable Container
  • Type of hiding

If hiding/showing the top area, then the contained BROWSER component always restarts with the URL that is defined behind. Some inner navigation that you might have done is lost.

If hiding/showing the bottom area, then the contained BROWSER component is opened exactly at the state that it had before hiding. Some inner navigation that you might have done is kept.

By default the client side components are removed from the client side component tree when hiding the area.

But you can fine-control this behavior:

By setting attribute TYPEOFHIDING to "resize" (instead of default "remove") the content of the HIDEABLE component is kept as it is, and is removed to widht 0 and height 0. By hiding the content this way, the client state of the content is kept.

This is the Java code:

package workplace;

import java.io.Serializable;
import org.eclnt.editor.annotations.CCGenClass;
import org.eclnt.workplace.IWorkpageDispatcher;
import org.eclnt.workplace.WorkpageDispatchedPageBean;

import javax.faces.event.ActionEvent;

@CCGenClass (expressionBase="#{d.DemoHideableType}")

public class DemoHideableType
    extends PageBean 
    implements Serializable
{
    // ------------------------------------------------------------------------
    // members
    // ------------------------------------------------------------------------

    boolean m_hideBottom = false;
    boolean m_hideTop = false;

    // ------------------------------------------------------------------------
    // constructors & initialization
    // ------------------------------------------------------------------------

    public DemoHideableType(IWorkpageDispatcher workpageDispatcher)
    {
        super(workpageDispatcher);        
    }

    public String getPageName() { return "/workplace/demohideabletype.jsp"; }
    public String getRootExpressionUsedInPage() { return "#{d.DemoHideableType}"; }

    // ------------------------------------------------------------------------
    // public usage
    // ------------------------------------------------------------------------

    public boolean getHideBottom() { return m_hideBottom; }
    public void setHideBottom(boolean value) { this.m_hideBottom = value; }

    public boolean getHideTop() { return m_hideTop; }
    public void setHideTop(boolean value) { this.m_hideTop = value; }

    // ------------------------------------------------------------------------
    // private usage
    // ------------------------------------------------------------------------
}
