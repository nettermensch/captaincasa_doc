The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • General
  • Client Integration
  • System Tray (Java Client only)

By using component SYSTEMTRAY - typically inside your top page - a system tray icon is created.

Windows users: have a look onto the bottom right - this is the default location of your task bar's system tray icon region. There is one new icon when having opened this page - a person with blue shirt. When closing this page the icon will disappear.

The tray icon supports two commands: "hide" the whole client, and "show" the whole client. When hiding the client then the user will not even see the client in the operating system's task bar anymore. Hiding the application is useful when openeing the client and pushing it into some kind of "background" mode.

In addition the tray icon can display messages (also when the whole client is not visible), that can show certain information to the user which may trigger him/her to re-open the client. It's similar to the small info popups that you get from a mail client, once a mail has arrived inside your inbox.

This is the Java code:

package workplace;

import java.io.Serializable;

import javax.faces.event.ActionEvent;

import org.eclnt.editor.annotations.CCGenClass;
import org.eclnt.jsfserver.defaultscreens.ModelessPopup;
import org.eclnt.jsfserver.defaultscreens.OKPopup;
import org.eclnt.jsfserver.defaultscreens.Statusbar;
import org.eclnt.jsfserver.elements.util.DefaultModelessPopupListener;
import org.eclnt.jsfserver.elements.util.Trigger;
import org.eclnt.workplace.IWorkpageDispatcher;

@CCGenClass (expressionBase="#{d.DemoSystemTray}")

public class DemoSystemTray
    extends PageBean
    implements Serializable
{
    public DemoSystemTray(IWorkpageDispatcher dispatcher)
    {
        super(dispatcher);
    }

    protected Trigger m_messageTrigger = new Trigger();
    Trigger m_triggerShow = new Trigger();
    Trigger m_triggerHide = new Trigger();
    protected String m_messageText;
    boolean m_timerActive = false;
    
    public String getMessageText() { return m_messageText; }
    public void setMessageText(String value) { m_messageText = value; }
    
    public boolean getTimerActive() { return m_timerActive; }
    public void setTimerActive(boolean timerActive) { m_timerActive = timerActive; }
    
    public Trigger getMessageTrigger() { return m_messageTrigger; }
    public Trigger getTriggerShow() { return m_triggerShow; }
    public Trigger getTriggerHide() { return m_triggerHide; }
    
    public int getTimerDuration()
    {
        if (m_timerActive == true)
            return 5000;
        else
            return 0;
    }

    public void onCreateTrayMessage(ActionEvent event) 
    {
        m_messageTrigger.trigger();
    }
    
    public void onTriggerHide(ActionEvent event)
    {
        m_triggerHide.trigger();
    }
    
    public void onFirstMenuItem(ActionEvent event)
    {
        Statusbar.outputSuccess("First Menu Item was clicked!");
    }

    public void onSecondMenuItem(ActionEvent event)
    {
        Statusbar.outputSuccess("Second Menu Item was clicked!");
    }

}
