The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • Components
  • Straight Controls
  • Text Input/Output
  • Text with Hyperlinks

The TEXTWITHLINKS component feeds back the link information when it contains hyperlink definitions. In this example the server side logic opens up a corresponding browser, showing the link's content.

This is the XML layout definition:

<?xml version="1.0" encoding="utf-8"?>
<t:layoutdefinition xmlns:t="http://www.CaptainCasa.com/controllibrary/t" xmlns:tpbc="http://www.CaptainCasa.com/controllibrary/tpbc" >
<t:beanprocessing id="g_1" >
<t:jshowurl id="g_2" trigger="#{d.DemoTextWithHyperLinks.urlTrigger}" url="#{d.DemoTextWithHyperLinks.url}" usedesktop="true" />
</t:beanprocessing>
<t:rowdemobodypane id="g_3" objectbinding="#{d.DemoTextWithHyperLinks}" >
<t:row id="g_4" >
<t:textwithlinks id="g_5" actionListener="#{d.DemoTextWithHyperLinks.onLinkAction}" height="100%" text="#{d.DemoTextWithHyperLinks.text}" width="100%" />
</t:row>
</t:rowdemobodypane>
<t:pageaddons id="g_pa"/>
</t:layoutdefinition>

This is the Java code:

package workplace;

import java.io.Serializable;

import javax.faces.event.ActionEvent;

import org.eclnt.editor.annotations.CCGenClass;
import org.eclnt.jsfserver.elements.events.BaseActionEventLink;
import org.eclnt.jsfserver.elements.util.Trigger;
import org.eclnt.workplace.IWorkpageDispatcher;

@CCGenClass (expressionBase="#{d.DemoTextWithHyperLinks}")

public class DemoTextWithHyperLinks
    extends WorkpageDispatchedPageBean 
    implements Serializable
{
    protected String m_url;
    public String getUrl() { return m_url; }
    public void setUrl(String value) { m_url = value; }

    protected Trigger m_urlTrigger = new Trigger();
    public Trigger getUrlTrigger() { return m_urlTrigger; }
    public void setUrlTrigger(Trigger value) { m_urlTrigger = value; }

    protected String m_text = 
        "<html>" +
        "<body>" +
        "This is a link to CaptainCasa: <a href='http://www.captaincasa.com'>CaptainCasa</a><br>" +
        "This is a link to Google: <a href='http://www.google.com'>Google</a>" +
        "</body>" +
        "</html>";

    public DemoTextWithHyperLinks(IWorkpageDispatcher workpageDispatcher)
    {
        super(workpageDispatcher);        
        initRefComponent("textwithlinks");
    }

    public String getText() { return m_text; }

    public void onLinkAction(ActionEvent event) 
    {
        if (event instanceof BaseActionEventLink)
        {
            BaseActionEventLink le = (BaseActionEventLink)event;
            m_url = le.getLinkId();
            m_urlTrigger.trigger();
        }
    }

}
