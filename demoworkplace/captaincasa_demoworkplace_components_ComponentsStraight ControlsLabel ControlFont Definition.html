The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • Components
  • Straight Controls
  • Label Control
  • Font Definition

All components rendering text provide a FONT attribute. This demo shows examples how to use the FONT attribute.

This is the XML layout definition:

<?xml version="1.0" encoding="utf-8"?>
<t:layoutdefinition xmlns:t="http://www.CaptainCasa.com/controllibrary/t" xmlns:tpbc="http://www.CaptainCasa.com/controllibrary/tpbc" >
<t:rowdemobodypane id="g_1" objectbinding="#{d.DemoFont}" rowdistance="#{ccstylevalue['@defaultrowdistance@']}" >
<t:row id="g_2" >
<t:label id="g_3" text="default (no font settings)" />
</t:row>
<t:row id="g_4" >
<t:label id="g_5" font="weight:bold" text="weight:bold" />
</t:row>
<t:row id="g_6" >
<t:label id="g_7" font="weight:bold;size:16" text="weight:bold;size:16" />
</t:row>
<t:row id="g_8" >
<t:label id="g_9" font="strikethrough:true" text="strikethrough:true" />
</t:row>
<t:row id="g_10" >
<t:label id="g_11" font="strikethrough:true;size:18" text="strikethrough:true;size:18" />
</t:row>
<t:row id="g_12" >
<t:label id="g_13" font="posture:italic" text="posture:italic" />
</t:row>
<t:row id="g_14" >

</t:row>

<t:pageaddons id="g_pa"/>
</t:layoutdefinition>

This is the Java code:

package workplace;

import java.io.Serializable;
import org.eclnt.editor.annotations.CCGenClass;
import org.eclnt.workplace.IWorkpageDispatcher;
import org.eclnt.workplace.WorkpageDispatchedBean;

@CCGenClass (expressionBase="#{d.DemoFont}")

public class DemoFont
    extends PageBean 
    implements Serializable
{
    public DemoFont(IWorkpageDispatcher workpageDispatcher)
    {
        super(workpageDispatcher);  
        initRefComponent("label"); 
    }
}
