The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • General
  • Client Integration
  • Client Datagram Receiver (Java Client only)

The component CLIENTDATAGRAMRECEIVER allows to send IP-datagrams (UDP messages) to the user interface. The component receives the message and passes it to the server side processing by calling its action listener.

The component is typically used in the context of sub-system integration: e.g. there are barcode scanners sending the event about having read a barcode via http datagram.

The byte stream that is received through the datagram is part of the event inforation on server side. You can either read it as hexadeicmal string representation or as byte array.

This is the XML layout definition:

<?xml version="1.0" encoding="utf-8"?>
<t:layoutdefinition xmlns:t="http://www.CaptainCasa.com/controllibrary/t" xmlns:tpbc="http://www.CaptainCasa.com/controllibrary/tpbc" >
<t:beanprocessing id="g_1" >
<t:clientdatagramreceiver id="g_2" actionListener="#{d.DemoClientDatagramReceiver.onReceive}" buffersize="1000" port="54001" timeout="50000" />
</t:beanprocessing>
<t:rowdemobodypane id="g_3" objectbinding="#{d.DemoClientDatagramReceiver}">
<t:row id="g_4" >
<t:label id="g_5" text="Datagram" width="120" />
<t:field id="g_6" actionListener="#{d.DemoClientDatagramReceiver.onReceive}" text="#{d.DemoClientDatagramReceiver.datagram}" width="150" />
</t:row>
</t:rowdemobodypane>
<t:pageaddons id="g_pa"/>
</t:layoutdefinition>

This is the Java code:

package workplace;

import java.io.Serializable;

import javax.faces.event.ActionEvent;

import org.eclnt.editor.annotations.CCGenClass;
import org.eclnt.jsfserver.elements.events.BaseActionEventClientDatagramReceive;
import org.eclnt.workplace.IWorkpageDispatcher;

@CCGenClass (expressionBase="#{d.DemoClientDatagramReceiver}")

public class DemoClientDatagramReceiver
    extends PageBean
    implements Serializable
{
    public DemoClientDatagramReceiver(IWorkpageDispatcher dispatcher)
    {
        super(dispatcher);
    }
    
    public void onReceive(ActionEvent event) 
    {
        if (event instanceof BaseActionEventClientDatagramReceive)
        {
            BaseActionEventClientDatagramReceive baecdr = (BaseActionEventClientDatagramReceive)event;
            m_datagram = baecdr.getDatagramHexString();
        }
    }

    protected String m_datagram;
    public String getDatagram() { return m_datagram; }
    public void setDatagram(String value) { m_datagram = value; }

}
