The context of this demo is the creation of a dialog using the CaptainCasa framework for the following purpose:

  • Components
  • Mobile Periphery
  • QR Code Scanner

The component QRCODESCANNERJSQR uses the JSQR library for scanning barcodes.

Please find details and terms of usage on page https://github.com/cozmo/jsQR

The component is integrated as independent IFRAME so that the JSQR framework is running in some own, independent environment.

Please pay attention: the component requires access to your client's camera. The browser can only access the camera when starting the page with "https://"! (Exception: working in localhost-environment.)

This is the XML layout definition:

<?xml version="1.0" encoding="utf-8"?>
<t:layoutdefinition xmlns:t="http://www.CaptainCasa.com/controllibrary/t" xmlns:tpbc="http://www.CaptainCasa.com/controllibrary/tpbc" >
<t:rowdemobodypane id="g_1" objectbinding="#{d.DemoQRCodeScannerJQRS}" rowdistance="#{ccstylevalue['@defaultrowdistance@']}" >
<t:row id="g_2" >
<t:label id="g_3" stylevariant="ccbig" text="Place a QR Code in front of your camera..." />
</t:row>
<t:row id="g_4" >
<t:qrcodescannerjsqr id="g_5" actionListener="#{d.DemoQRCodeScannerJQRS.onQRCodeAction}" height="300" scandelay="1000" width="400" />
</t:row>
<t:row id="g_6" >
<t:textarea id="g_7" enabled="false" height="100%" text="#{d.DemoQRCodeScannerJQRS.protocol}" width="100%" />
</t:row>
</t:rowdemobodypane>
<t:pageaddons id="g_pa"/>
</t:layoutdefinition>

This is the Java code:

package workplace;

import java.io.Serializable;
import java.util.Date;

import org.eclnt.editor.annotations.CCGenClass;
import org.eclnt.jsfserver.defaultscreens.Statusbar;
import org.eclnt.jsfserver.elements.events.BaseActionEventQRCodeScan;
import org.eclnt.workplace.IWorkpageDispatcher;
import org.eclnt.workplace.WorkpageDispatchedPageBean;

import javax.faces.event.ActionEvent;

@CCGenClass (expressionBase="#{d.DemoQRCodeScannerJQRS}")

public class DemoQRCodeScannerJQRS
    extends WorkpageDispatchedPageBean 
    implements Serializable
{
    // ------------------------------------------------------------------------
    // inner classes
    // ------------------------------------------------------------------------

    /* Listener to the user of the page bean. */
    public interface IListener
    {
    }

    // ------------------------------------------------------------------------
    // members
    // ------------------------------------------------------------------------

    private IListener m_listener;
    String m_protocol = "Protocol:\n";
    
    // ------------------------------------------------------------------------
    // constructors & initialization
    // ------------------------------------------------------------------------

    public DemoQRCodeScannerJQRS(IWorkpageDispatcher workpageDispatcher)
    {
        super(workpageDispatcher);        
    }

    public String getPageName() { return "/workplace/demoqrcodescannerjqrs.jsp"; }
    public String getRootExpressionUsedInPage() { return "#{d.DemoQRCodeScannerJQRS}"; }

    // ------------------------------------------------------------------------
    // public usage
    // ------------------------------------------------------------------------

    public String getProtocol() { return m_protocol; }
    public void setProtocol(String value) { this.m_protocol = value; }

    public void onQRCodeAction(javax.faces.event.ActionEvent event) 
    {
        if (event instanceof BaseActionEventQRCodeScan)
        {
            BaseActionEventQRCodeScan e = (BaseActionEventQRCodeScan)event;
            m_protocol += "QRCode was scanned: " + e.getQRCodeAsUTF8String() + ", " + new Date().toString() + "\n";
        }
    }

}
